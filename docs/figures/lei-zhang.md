# 张磊——做时间的朋友

> 如果用一个词概括张磊的投资生涯，那就是**长期主义**。他用 20 年时间证明：在中国这片充满不确定性的市场上，做时间的朋友是最确定的策略。

## 人物速写

张磊，高瓴资本（Hillhouse Capital）创始人兼 CEO，亚洲管理规模最大的私募股权基金之一。他将耶鲁捐赠基金的投资理念引入中国，用"长期主义"和"重仓中国"重新定义了中国价值投资。

| 核心贡献 | 将长期主义引入中国私募、重仓中国核心资产                         |
| 代表公司 | 高瓴资本（Hillhouse Capital）                                    |

## 传奇人生

### 早年经历：从驻马店到人大

1972 年，张磊出生于河南驻马店一个普通家庭。在那个年代，驻马店是中国最贫穷的地区之一。张磊的父母都是普通职工，家庭条件并不富裕。

但张磊从小就展现出了对知识的渴望。他回忆说：

> "小时候家里没有太多书，我就去图书馆借。能借到什么就读什么，从历史到经济，从文学到科学。"

1990 年，张磊以优异的成绩考入中国人民大学国际金融专业。在人大，他如饥似渴地学习经济学和金融学，阅读了大量西方投资理论的著作。

### 投资启蒙：耶鲁与史文森

1998 年，张磊获得了耶鲁大学的奖学金，前往攻读 MBA。这是他人生最重要的转折点。

在耶鲁，他遇到了大卫·史文森（David Swensen）——耶鲁大学捐赠基金的首席投资官。史文森是全球机构投资的传奇人物，他管理的耶鲁捐赠基金从 1985 年的 13 亿美元增长到 2020 年的超过 300 亿美元，年化收益率超过 13%，远超同行（来源：耶鲁大学捐赠基金年报，2020）。

史文森教给张磊最重要的三件事：

1. **长期视角**——不要关注短期波动，关注资产的长期价值
2. **多元化配置**——不要把所有鸡蛋放在一个篮子里，但也不要过度分散
3. **寻找优秀管理者**——投资本质上是投资"人"

张磊后来回忆：

> "史文森教会了我什么是真正的长期主义。他不是在看下一季度的回报，而是在看未来 10 年、20 年的趋势。"

### 关键转折点：重仓中国

2002 年，张磊从耶鲁毕业后，在全球投资基金管理公司（Global Emerging Markets Investment Management）工作。期间，他负责研究新兴市场投资，特别是中国市场。

2003 年前后，张磊做了一个让所有人意外的决定：**回国创业。**

当时中国资本市场刚刚起步，A 股市场还在股权分置改革的阵痛中，私募股权行业几乎是一片空白。大多数耶鲁毕业生选择留在华尔街，但张磊看到了一个巨大的机会：

> "中国经济正在经历人类历史上最大规模的转型。互联网、消费、医疗——这些行业正在重塑中国。我想参与其中。"

2005 年，张磊用耶鲁大学捐赠基金提供的 3000 万美元种子资金，在北京创立了高瓴资本（Hillhouse Capital）。公司名字取自耶鲁大学附近的一条街——Hillhouse Avenue，那里是耶鲁校园的核心地带，也是张磊求学时经常散步的地方。

**关于首投腾讯的说明：** 高瓴 2005 年成立时，腾讯已于 2004 年在港交所上市。张磊对腾讯的投资并非"首投"，而是高瓴成立后在二级市场的早期重要布局。高瓴在腾讯上市后持续加仓，成为其长期重要股东（来源：港交所披露易、高瓴 13F 文件）。

### 巅峰时刻：高瓴的崛起

高瓴资本成立后的 20 年，是中国资本市场发展最快的 20 年。张磊带领高瓴完成了从一家小型私募到亚洲最大投资机构之一的蜕变。

**关键里程碑：**

- **2005 年**：高瓴资本成立，获耶鲁捐赠基金 3000 万美元种子资金
- **2010 年**：投资京东，成为其最大机构股东之一
- **2014 年**：高瓴管理规模突破 100 亿美元（来源：公开报道）
- **2017 年**：牵头 531 亿港元收购百丽国际，创下亚洲最大私募收购纪录
- **2019 年**：投资格力电器，成为其重要股东
- **2021 年**：高瓴管理规模超过 6000 亿人民币（来源：中国证券投资基金业协会）
- **2022 年**：高瓴拆分为高瓴资本（私募股权）、HHLR Advisors（二级市场）、高瓴创投（早期投资）等多个独立运营实体

**高瓴拆分后各实体管理规模（2024 年估计）：**

| 实体                  | 主要方向           | 估计管理规模                      |
| --------------------- | ------------------ | --------------------------------- |
| 高瓴资本（Hillhouse） | 私募股权、并购     | 约 300-400 亿美元                 |
| HHLR Advisors         | 二级市场股票投资   | 约 100-150 亿美元（SEC 13F 披露） |
| 高瓴创投              | 早期风险投资       | 约 50-80 亿美元                   |
| 高济医疗              | 医药零售、医疗服务 | 独立运营，门店超 2 万家           |

**数据来源：** SEC 13F 文件（HHLR Advisors）、中国证券投资基金业协会、晚点 LatePost 报道（2022）

张磊的投资版图覆盖了互联网、消费、医疗、硬科技、企业服务等多个领域。他投资的企业包括：腾讯、京东、美团、拼多多、百济神州、格力电器、百丽国际、蓝月亮等。

2020 年，张磊出版了《价值：我对投资的思考》一书，系统阐述了他的投资哲学。这本书成为中国投资界的畅销书，也让更多人了解了"长期主义"的力量。

## 经典战役

### 战役一：持续投资腾讯（2005-2014）

高瓴资本成立后，张磊在二级市场持续买入腾讯股票，这是高瓴最早的重大投资之一。

**背景：**

- 2004 年 5 月，腾讯在港交所上市，发行价 3.70 港元
- 2005 年的腾讯，QQ 用户数已经超过 1 亿，但商业模式还在探索中
- 公司 2005 年收入约 14.3 亿元，净利润约 7.0 亿元（来源：腾讯 2005 年年报）
- 当时市场对腾讯的争议很大：QQ 是"玩具"还是"平台"？

**张磊的逻辑：**

- 中国互联网用户正在快速增长，QQ 是当时最大的即时通讯平台
- 网络效应（Network Effect）一旦形成，护城河会非常深
- 腾讯团队年轻、有执行力，马化腾是罕见的产品型 CEO
- 3000 万美元的种子资金中，张磊决定将相当比例配置给腾讯

**结果：**

- 腾讯后来发展成为中国最大的互联网公司之一
- 2024 年，腾讯市值超过 3 万亿港元
- 高瓴在腾讯上的投资回报超过 **100 倍**
- 高瓴在腾讯上市后多年持续持有，后逐步减持（来源：港交所披露易）

| 指标         | 数值                               |
| ------------ | ---------------------------------- |
| 开始投资时间 | 2005 年（腾讯 2004 年上市后）      |
| 投资金额     | 种子资金的重要部分（约数千万美元） |
| 当时市值     | 约 100 亿港元                      |
| 当前市值     | 超过 3 万亿港元（2024 年）         |
| 回报率       | 超过 100 倍                        |
| 持有时间     | 多年（逐步减持）                   |

**数据来源：** 腾讯年报（2005-2024）、港交所披露易、《价值》（张磊著，2020）

### 战役二：投资京东（2010）

2010 年，张磊做出了另一个让市场惊讶的决定：**大举投资京东。**

**背景：**

- 2010 年的京东，年收入约 100 亿元，但还在亏损
- 自建物流的模式被很多人质疑——"太重了"
- 阿里巴巴已经占据电商市场主导地位
- 京东需要大量资金来建设物流基础设施

**张磊的逻辑：**

- 京东的自建物流是长期竞争力的核心
- 中国电商市场足够大，容得下多个玩家
- 刘强东的执行力极强，对用户体验的执着令人印象深刻
- 物流基础设施一旦建成，将成为极深的护城河

张磊后来回忆：

> "很多人问我为什么投京东。我说，如果你只看到京东在亏钱，你就错过了重点。重点在于，京东在建造一个别人建不了的东西。"

高瓴资本向京东投资了约 **3 亿美元**，成为京东最大的机构股东之一（来源：京东招股书，2014）。

**结果：**

- 2014 年 5 月，京东在纳斯达克上市，发行价 19 美元，市值超过 250 亿美元
- 京东物流后来独立上市，成为中国最大的自营物流网络
- 高瓴在京东上的投资回报超过 **10 倍**
- 京东成为中国第二大电商平台

| -------- | ------------------------ |
| 投资时间 | 2010 年                  |
| 投资金额 | 约 3 亿美元              |
| 当时收入 | 约 100 亿元              |
| 上市时间 | 2014 年 5 月（纳斯达克） |
| 上市市值 | 超过 250 亿美元          |

**数据来源：** 京东招股书（2014）、Crunchbase、公开报道

### 战役三：收购百丽国际（2017）

2017 年，高瓴资本牵头财团以 **531 亿港元**（约 68 亿美元）收购了百丽国际，创下亚洲最大私募收购纪录。

**背景：**

- 百丽国际是中国最大的鞋类零售商，旗下拥有百丽、天美意、他她等品牌
- 2017 年前后，受电商冲击，百丽的业绩持续下滑
- 股价从高点下跌超过 70%，市值大幅缩水
- 创始人邓耀和于武决定退出

**张磊的逻辑：**

- 百丽的问题不是品牌不行，而是数字化转型滞后
- 百丽拥有中国最大的鞋类零售网络（超过 2 万家门店）
- 如果能将数字化能力注入百丽，就能实现"老树发新芽"
- 私有化后可以从容转型，不受短期业绩压力影响

高瓴资本联合鼎晖投资，以每股 6.3 港元的价格私有化百丽国际。

**结果：**

- 高瓴入主后，百丽开始了大规模的数字化转型
- 门店数字化、供应链优化、线上渠道拓展
- 2020 年 9 月，百丽旗下运动业务（滔搏 International）在港交所上市，市值超过 600 亿港元
- 百丽的数字化转型被认为是中国传统零售转型的成功案例

| -------- | --------------------------- |
| 收购时间 | 2017 年 7 月                |
| 收购金额 | 531 亿港元（约 68 亿美元）  |
| 收购价格 | 每股 6.3 港元               |
| 门店数量 | 超过 2 万家                 |
| 后续动作 | 滔搏 2020 年 9 月港交所上市 |
| 战略方向 | 数字化转型、供应链优化      |

**数据来源：** 百丽国际公告（2017）、滔搏招股书（2020）、财新报道

### 战役四：投资格力电器（2019）

2019 年，高瓴资本以约 **417 亿元** 受让格力电器 15% 的股份，成为其重要股东。

**背景：**

- 格力电器是中国最大的空调制造商
- 珠海国资委决定转让格力集团持有的 15% 股份
- 参与竞标的包括高瓴资本、厚朴投资等
- 市场关注：谁来接盘？

**张磊的逻辑：**

- 格力是中国制造业的标杆企业，董明珠的管理能力毋庸置疑
- 空调行业虽然增速放缓，但存量市场依然巨大
- 格力在智能制造、新能源等领域的布局值得关注
- 高瓴可以为格力带来数字化转型和国际化的资源

**结果：**

- 高瓴成为格力电器的重要股东后，积极参与公司治理
- 格力在数字化转型、渠道改革等方面持续推进
- 这笔投资也引发了市场关于"产业资本 + 金融资本"模式的讨论

| 持股比例 | 15%                |

| 战略方向 | 数字化转型、国际化 |

**数据来源：** 格力电器公告（2019）、公开报道

### 战役五：重仓中国医疗（2015-至今）

张磊在医疗领域的布局是高瓴最重要的战略方向之一。

**布局逻辑：**

- 中国人口老龄化趋势不可逆转
- 医疗健康是长期确定性最高的赛道之一
- 中国创新药、医疗器械、医疗服务等领域正在快速崛起

**代表性投资：**

- **百济神州**：中国创新药龙头企业，高瓴是早期投资者（来源：百济神州招股书）
- **药明康德**：全球领先的 CRO 企业，高瓴是重要股东
- **高济医疗**：高瓴旗下医药零售平台，门店超过 2 万家
- **惠泰医疗**：心血管医疗器械企业

高瓴在医疗领域的投资超过 **1000 亿元**，覆盖创新药、医疗器械、医疗服务、医药零售等全产业链。

## 投资哲学

### 原则一：长期主义

这是张磊投资哲学的核心。

> "长期主义不仅仅是一种投资理念，更是一种价值观。它意味着你要相信时间的力量，相信价值的回归，相信优秀的企业和企业家。"

张磊的长期主义体现在：

- **持有期限长**——很多投资持有 5 年、10 年甚至更久
- **关注长期价值**——不关注短期业绩波动，关注企业 5-10 年后的竞争力
- **陪伴企业成长**——不仅仅是财务投资，更要为企业创造价值

### 原则二：重仓中国

张磊是最早提出"重仓中国"理念的投资者之一。

> "中国正在经历人类历史上最大规模的创新转型。作为投资者，最大的错误就是错过这个时代。"

他的逻辑是：

- 中国拥有全球最大的消费市场
- 中国正在从"世界工厂"转型为"创新中心"
- 中国企业在互联网、新能源、生物科技等领域已经具备全球竞争力
- 中国资本市场的深度和广度在持续提升

### 原则三：寻找伟大企业家

张磊非常看重企业家精神。

> "投资本质上是投资'人'。找到那些有远见、有执行力、有格局的企业家，和他们一起创造价值。"

他评价企业家的标准：

- **远见**——能否看到 5-10 年后的趋势
- **执行力**——能否把想法变成现实
- **格局**——是否以长期价值为导向，而非短期利益
- **学习能力**——能否持续进化

### 原则四：赋能式投资

张磊提出了"赋能式投资"的理念——不仅仅是给钱，更要为企业提供资源和支持。

> "高瓴的角色不仅仅是投资者，更是'合伙人'。我们要帮助企业做数字化转型、国际化、人才引进。"

高瓴的赋能能力包括：

- 数字化团队——帮助企业进行数字化转型
- 全球化网络——连接中国企业和全球资源
- 人才资源——帮助企业引进高端人才
- 产业资源——连接产业链上下游

### 原则五：守正用奇

张磊在《价值》一书中提出了"守正用奇"的理念。

> "守正，就是坚持价值投资的基本原则；用奇，就是在坚守原则的基础上，勇于创新。"

具体来说：

- **守正**——坚持长期主义、价值投资、重仓中国
- **用奇**——在投资方法、行业选择、交易结构上不断创新

## 策略拆解

### 选股标准

张磊的选股标准可以总结为"四个看"：

| 标准   | 具体要求                             |
| ------ | ------------------------------------ |
| 看赛道 | 行业空间大、增长确定性强、有长期逻辑 |
| 看公司 | 商业模式好、护城河深、有定价权       |
| 看管理 | 企业家有远见、执行力强、格局大       |
| 看价格 | 估值合理或偏低，有安全边际           |

具体关注点：

- **行业空间**：万亿级以上市场优先
- **竞争格局**：龙头或潜在龙头
- **商业模式**：可复制、可扩展、有网络效应
- **财务质量**：ROE 高、现金流好、负债合理

### 买入时机

张磊的买入时机判断：

1. **行业拐点**——行业从导入期进入成长期
2. **企业转型**——优秀企业遇到暂时困难，估值被错杀
3. **政策驱动**——政策变化带来行业机会
4. **私有化机会**——优质企业被低估，可以私有化后重塑

他特别强调：**不要在市场狂热时追高，要在市场恐慌时买入。**

### 持仓管理

张磊的持仓管理特点：

- **集中持仓**——前十大投资占组合的大部分
- **主动赋能**——不只是被动持有，而是积极参与企业治理
- **长期陪伴**——很多投资持有 5-10 年以上
- **动态调整**——根据企业基本面变化调整仓位

### 卖出纪律

张磊的卖出条件：

1. **基本面恶化**——企业核心竞争力丧失
2. **逻辑证伪**——投资逻辑被证明是错误的
3. **估值极度泡沫**——价格远超内在价值
4. **发现更好的机会**——换仓到性价比更高的标的

他特别强调：**不要因为短期业绩压力而卖出。** 如果长期逻辑还在，就应该继续持有。

### 普通人能学到什么

张磊的方法对普通投资者的启示：

1. **做时间的朋友**——不要频繁交易，让时间成为你的盟友
2. **投资你相信的东西**——只投资你真正理解的行业和企业
3. **关注长期趋势**——不要被短期波动干扰，关注 5-10 年的大趋势
4. **持续学习**——张磊每天阅读大量资料，持续学习新行业
5. **保持耐心**——好的投资机会不常有，出现时要敢于重仓

**具体建议：**

- 选择 3-5 家你真正理解的好公司
- 以 5 年以上的视角持有
- 每年检查一次基本面是否发生变化
- 保持阅读和学习的习惯
