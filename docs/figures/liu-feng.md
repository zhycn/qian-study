# 冯柳——弱者体系

> 他没有金融科班背景，没有华尔街经历，只是一个普通散户。但他在雪球上以"茅台 03"为 ID 分享投资心得，吸引了数十万粉丝。2015 年，他被高毅资产创始人邱国鹭邀请加入，管理规模迅速突破百亿。他叫冯柳，中国资本市场最传奇的"民间高手"。

## 人物速写

冯柳，高毅资产基金经理，中国最知名的"民间高手"转型机构的典范。他以"弱者体系"和"逆向投资"闻名，是私募行业最耀眼的明星基金经理之一。

## 传奇人生

### 早年经历：普通散户的起点

冯柳的早年经历并不显赫。他没有金融科班背景，没有华尔街经历，只是一个普通的 A 股散户。

2000 年代初期，冯柳开始接触 A 股市场。和大多数散户一样，他经历了：

- 追涨杀跌的亏损
- 听消息的失败
- 技术研究的迷茫

但冯柳有一个与众不同的特点：**他善于总结和反思。**

> "我亏了很多钱，但每一次亏损都让我学到了一些东西。慢慢我发现，散户最大的优势不是信息，而是灵活。"

### 投资启蒙：雪球时代的崛起

2010 年前后，冯柳开始在雪球上以"茅台 03"为 ID 分享投资心得。

**他的特点：**

- 文章深入浅出，逻辑清晰
- 不回避自己的错误
- 有独特的投资框架
- 语言生动，接地气

**他的影响力：**

- 吸引了数十万粉丝
- 成为雪球最受欢迎的投资者之一
- 投资观点被广泛传播和讨论
- 建立了良好的投资声誉

### 关键转折点：加入高毅资产

2015 年，高毅资产创始人邱国鹭注意到了冯柳。

邱国鹭后来回忆：

> "冯柳的文章我看了很多。他的投资框架非常独特，不是科班出身，但逻辑非常清晰。我决定邀请他加入高毅。"

2015 年，冯柳正式加入高毅资产，成为一名基金经理。

这个消息在投资圈引起了轰动：

- 一个"民间高手"被顶级私募邀请
- 非科班出身能否适应机构投资？
- 他的投资框架能否在更大规模下有效？

### 巅峰时刻：百亿基金经理

冯柳加入高毅资产后，用业绩回答了所有质疑。

**业绩表现：**

- 管理规模迅速增长
- 投资业绩优秀
- 投资风格稳定
- 受到投资者认可

到 2024 年，冯柳的管理规模已经突破 **100 亿元**，成为高毅资产最重要的基金经理之一。

冯柳的成功证明了：

- 投资不需要科班背景
- 民间高手同样可以成功
- 独特的投资框架可以经受住市场考验
- 散户逆袭是可能的

## 经典战役

### 战役一：重仓茅台（2016-2021）

冯柳加入高毅资产后，贵州茅台成为其标志性持仓之一。

**背景：**

- 2016 年，茅台股价约 300 元左右
- 白酒行业刚从 2012-2014 年的寒冬中恢复
- 市场对白酒行业依然有疑虑

**冯柳的逻辑：**

- 茅台的护城河极深，品牌壁垒无法复制
- 白酒行业集中度提升，龙头受益
- 估值合理，有安全边际

**结果：**

- 茅台股价从 300 元涨到 2021 年的超过 2600 元
- 回报率超过 **8 倍**（来源：上市公司前十大流通股东数据，2021）
- 冯柳在茅台上获利丰厚

### 战役二：投资恒瑞医药（2018-2020）

冯柳在医药板块的代表性投资。

**背景：**

- 2018 年，恒瑞医药受集采政策影响，股价回调
- 市场担心创新药企的盈利能力
- 恒瑞医药是中国创新药龙头

**冯柳的逻辑：**

- 恒瑞医药的研发实力国内领先
- 集采政策短期利空，长期利好行业集中度提升
- 估值回到合理区间

**结果：**

- 恒瑞医药股价从 2018 年的约 50 元涨到 2021 年的超过 120 元
- 回报率超过 **2 倍**（来源：上市公司前十大流通股东数据，2021）
- 冯柳在恒瑞医药上获利丰厚

| 买入价格 | 约 50 元 |

### 战役三：逆向投资白酒板块（2013-2014）

在白酒行业最低谷时，冯柳作为散户已经开始布局。

**背景：**

- 2013-2014 年，受"八项规定"影响，白酒行业遭遇寒冬
- 茅台股价从 200 多元跌到 100 多元
- 市场普遍认为白酒行业的"黄金时代"已经结束

**冯柳的逻辑：**

- 白酒是中国文化的一部分，需求不会消失
- 行业低谷是买入良机
- 龙头企业将受益于行业集中度提升

**结果：**

- 白酒行业在 2015 年后恢复增长
- 茅台、五粮液等龙头股价大幅上涨
- 冯柳的逆向投资获得丰厚回报

### 战役四："弱者体系"的实践

冯柳的"弱者体系"是其投资框架的核心。

**体系核心：**

- 承认自己是弱者——没有信息优势、没有研究优势
- 利用弱者的优势——灵活、耐心、不受约束
- 在弱者能赢的地方下注——赔率高、风险低的机会
- 不追求完美——接受不完美，追求大概率正确

### 战役五：雪球时代的影响力

冯柳在雪球时代的影响力是巨大的。

**影响：**

- 影响了大量散户的投资理念
- 推广了逆向投资和弱者体系
- 建立了良好的投资声誉
- 为后来加入高毅资产奠定了基础

## 投资哲学

### 原则一：弱者体系

这是冯柳投资哲学的核心。

> "承认自己是弱者，这是投资的第一步。"

弱者体系的核心：

1. **承认无知**——没有信息优势、没有研究优势
2. **利用优势**——灵活、耐心、不受约束
3. **选择战场**——在弱者能赢的地方下注
4. **接受不完美**——不追求完美，追求大概率正确

### 原则二：逆向投资

冯柳是逆向投资的坚定实践者。

> "在别人恐惧时贪婪，在别人贪婪时恐惧。"

逆向投资的逻辑：

- 市场情绪会导致价格偏离价值
- 恐慌时好公司的价格会被错杀
- 逆向投资是获取超额收益的重要方法
- 散户的优势在于灵活，可以快速决策

### 原则三：赔率优先

冯柳强调赔率优先于胜率。

> "投资不是追求高胜率，而是追求高赔率。"

赔率优先的逻辑：

- 高赔率的机会往往出现在市场恐慌时
- 即使胜率不高，只要赔率足够高，期望值（Expected Value）就是正的
- 散户应该寻找高赔率的机会
- 不要追求每次都正确

### 原则四：集中持仓

冯柳强调集中持仓。

> "集中持仓是获取超额收益的关键。"

集中持仓的逻辑：

- 分散投资只能获得平均收益
- 集中持仓可以放大收益
- 但前提是深入研究、确信逻辑
- 散户应该集中持仓，但要做好风险管理

### 原则五：长期视角

冯柳强调长期视角。

> "投资是一场马拉松，不是短跑。"

长期视角的体现：

- 以 1-3 年为视角持有
- 不受短期市场波动干扰
- 关注公司基本面变化
- 让时间成为盟友

## 策略拆解

### 选股标准

冯柳的选股标准：

| 标准 | 具体要求 |
| --- | --- |
| 赔率高 | 价格远低于内在价值 |
| 逆向 | 市场恐慌、被抛弃的标的 |
| 逻辑清晰 | 投资逻辑简单明了 |
| 能力圈内 | 只投资真正理解的东西 |

### 买入时机

冯柳的买入时机判断：

1. **市场恐慌**——好公司被错杀
2. **行业低谷**——优质企业在周期底部
3. **利空出尽**——坏消息已经反映在价格中
4. **赔率足够高**——价格远低于内在价值

### 持仓管理

冯柳的持仓管理：

- **集中持仓**——少数几只股票占大部分仓位
- **长期持有**——以 1-3 年为视角
- **不频繁交易**——除非基本面发生变化
- **持续跟踪**——关注公司基本面变化

### 卖出纪律

冯柳的卖出条件：

1. **逻辑实现**——投资逻辑已经实现
2. **赔率下降**——价格接近内在价值
3. **发现更好的机会**——换仓到赔率更高的标的
4. **基本面恶化**——公司核心竞争力丧失

### 普通人能学到什么

冯柳的方法对普通投资者特别友好：

1. **承认自己是弱者**——这是投资的第一步
2. **逆向投资**——在市场恐慌时买入
3. **赔率优先**——关注赔率而非胜率
4. **集中持仓**——少数几只股票
5. **长期视角**——以 1-3 年为视角

**具体建议：**

- 选择 3-5 家你理解的公司
- 在市场恐慌时敢于买入
- 以 1-3 年为视角持有
- 定期跟踪公司基本面
