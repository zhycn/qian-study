# 王亚伟——中国私募一哥

> 他是中国公募基金历史上最耀眼的明星。管理的华夏大盘精选在 2005-2012 年间年化收益约 20%，创造了公募基金的历史记录。2012 年，他选择离开公募，转型私募，继续以重组股投资和逆向选股闻名。他叫王亚伟，中国资本市场最具传奇色彩的基金经理之一。

## 人物速写

王亚伟，中国最知名的基金经理之一。从"公募一哥"到"私募一哥"，他用业绩证明了自己的投资能力。他以重组股投资和逆向选股闻名，是中国资本市场最具传奇色彩的投资者之一。

| 核心贡献 | 华夏大盘精选传奇、重组股投资典范                          |
| 代表公司 | 华夏基金（公募）→ 千合资本（私募）                        |

## 传奇人生

### 早年经历：从清华到基金

1971 年，王亚伟出生于安徽。1989 年，他考入清华大学，学习电子工程专业。

毕业后，王亚伟进入了金融行业。他先后在中国信达信托投资公司、华夏证券等机构工作，积累了丰富的投资经验。

1998 年，王亚伟加入了华夏基金，成为一名基金经理。

### 投资启蒙：华夏大盘精选的崛起

2005 年，王亚伟开始管理华夏大盘精选基金。

**背景：**

- 2005 年，A 股市场处于历史底部
- 股权分置改革刚刚开始
- 市场情绪极度悲观

**王亚伟的操作：**

- 逆向买入——在市场恐慌时买入
- 重组股投资——关注有重组预期的公司
- 独立研究——不依赖卖方报告
- 集中持仓——少数几只股票占大部分仓位

**业绩表现：**

- 2005-2007 年牛市——华夏大盘精选涨幅超过 1000%（来源：华夏基金年报，2007）
- 2008 年熊市——回撤控制优于同行
- 2009-2012 年——持续跑赢市场

2005-2012 年，华夏大盘精选的年化收益率约 **20%**，创造了公募基金的历史记录（来源：华夏基金年报，2012）。

### 关键转折点：转型私募

2012 年，王亚伟做出了一个让市场震惊的决定：**离开华夏基金，转型私募。**

**原因分析：**

- 公募基金的限制太多（仓位、投资范围、短期业绩压力）
- 私募可以更灵活地执行投资策略
- 重组股投资在私募中更容易操作
- 个人发展空间更大

2012 年，王亚伟创立了千合资本，正式转型私募。

这个消息在投资圈引起了轰动：

- "公募一哥"转型私募
- 他的投资策略能否在私募中复制？
- 管理规模能否继续增长？

### 巅峰时刻：私募时代的延续

王亚伟转型私募后，继续以重组股投资和逆向选股闻名。

**投资特点：**

- 重组股投资——关注有重组预期的公司
- 逆向选股——在市场恐慌时买入
- 独立研究——不依赖卖方报告
- 低调神秘——很少接受采访

王亚伟的私募业绩虽然不再公开披露，但据市场估计，他的管理规模已经突破 **100 亿元**，继续保持着"私募一哥"的地位（来源：《中国基金报》报道，2024）。

**关于私募业绩数据的说明：** 王亚伟转型私募后，千合资本的业绩数据不再公开披露。市场对其管理规模和业绩的估计来源于行业报道和投资者反馈，非官方数据（来源：《中国基金报》《证券时报》报道）。

### 战役二：投资招商银行（2005-2007）

王亚伟在华夏大盘精选期间，招商银行是其重要持仓之一。

**背景：**

- 2005 年，招商银行股价处于低位
- 市场对中国银行业前景存在疑虑
- 股权分置改革带来不确定性

**王亚伟的逻辑：**

- 招商银行是中国最具竞争力的股份制银行
- 零售业务领先，资产质量优良
- 股权分置改革后估值将重估

**结果：**

- 招商银行股价从 2005 年的约 5 元涨到 2007 年的超过 40 元
- 回报率超过 **8 倍**（来源：华夏基金年报、招商银行年报，2007）
- 王亚伟在招商银行上获利丰厚

| -------- | --------------------- |
| 买入时间 | 2005 年               |
| 买入价格 | 约 5 元               |
| 峰值价格 | 超过 40 元（2007 年） |
| 回报率   | 超过 8 倍             |

**数据来源：** 华夏基金年报、招商银行年报

### 战役三：投资广汇能源（2009-2012）

广汇能源是王亚伟在华夏大盘精选期间的另一代表性投资。

**背景：**

- 广汇能源是一家新能源和传统能源结合的公司
- 市场对其业务模式存在争议
- 股价处于低位

**王亚伟的逻辑：**

- 广汇能源在 LNG（液化天然气）领域有布局
- 能源行业长期看好
- 估值偏低，有安全边际

**结果：**

- 广汇能源股价上涨
- 王亚伟获得丰厚回报

### 战役四：重组股投资

王亚伟以重组股投资闻名。

**投资逻辑：**

- 重组股有巨大的价值重估空间
- 通过独立研究可以发现重组机会
- 重组成功后股价往往大幅上涨
- 但风险也很高，需要深入研究

**典型案例：**

- 投资有重组预期的公司
- 在重组前买入
- 重组成功后获得丰厚回报

### 战役五：转型私募

2012 年，王亚伟转型私募。

**转型逻辑：**

- 公募基金的限制太多
- 私募可以更灵活地执行投资策略
- 重组股投资在私募中更容易操作
- 个人发展空间更大

**效果：**

- 管理规模持续增长
- 投资风格更加灵活
- 继续获得优秀业绩

## 投资哲学

### 原则一：重组股投资

这是王亚伟最著名的投资方法。

> "重组股有巨大的价值重估空间。"

重组股投资的逻辑：

- 重组后公司基本面可能发生根本性变化
- 价值重估空间巨大
- 但需要深入研究，判断重组概率
- 风险较高，需要控制仓位

### 原则二：逆向选股

王亚伟是逆向选股的坚定实践者。

> "在别人恐惧时贪婪，在别人贪婪时恐惧。"

逆向选股的逻辑：

- 市场情绪会导致价格偏离价值
- 恐慌时好公司的价格会被错杀
- 逆向投资是获取超额收益的重要方法
- 但需要独立研究和判断

### 原则三：独立研究

王亚伟强调独立研究。

> "不依赖卖方报告，自己调研，自己判断。"

独立研究的体现：

- 自己调研公司
- 自己判断价值
- 形成独立的投资观点
- 不盲目跟随市场共识

### 原则四：集中持仓

王亚伟强调集中持仓。

> "集中持仓是获取超额收益的关键。"

集中持仓的逻辑：

- 分散投资只能获得平均收益
- 集中持仓可以放大收益
- 但前提是深入研究、确信逻辑
- 做好风险管理

### 原则五：低调做事

王亚伟以低调著称。

> "投资不需要宣传，需要的是业绩。"

低调的体现：

- 很少接受采访
- 不公开讨论投资观点
- 专注于投资本身
- 用业绩说话

## 策略拆解

### 选股标准

王亚伟的选股标准：

| 标准     | 具体要求               |
| -------- | ---------------------- |
| 重组预期 | 有重组可能的公司       |
| 逆向     | 市场恐慌、被抛弃的标的 |
| 独立研究 | 自己调研、自己判断     |
| 价值低估 | 价格低于内在价值       |

### 买入时机

王亚伟的买入时机判断：

1. **市场恐慌**——好公司被错杀
2. **重组预期**——有重组可能的公司
3. **价值低估**——价格低于内在价值
4. **独立研究确信**——自己研究后确信逻辑

### 持仓管理

王亚伟的持仓管理：

- **集中持仓**——少数几只股票占大部分仓位
- **长期持有**——以 1-3 年为视角
- **不频繁交易**——除非基本面发生变化
- **持续跟踪**——关注公司基本面变化

### 卖出纪律

王亚伟的卖出条件：

1. **逻辑实现**——投资逻辑已经实现
2. **价值回归**——价格接近内在价值
3. **发现更好的机会**——换仓到更好的标的
4. **基本面恶化**——公司竞争力丧失

### 普通人能学到什么

王亚伟的方法对普通投资者的启示：

1. **独立研究**——不依赖卖方报告，自己调研
2. **逆向投资**——在市场恐慌时买入
3. **集中持仓**——少数几只股票
4. **低调做事**——专注于投资本身
5. **持续学习**——不断提高研究能力

**具体建议：**

- 选择 3-5 家你深入研究的公司
- 在市场恐慌时敢于买入
- 以 1-3 年为视角持有
- 定期跟踪公司基本面
