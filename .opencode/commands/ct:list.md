查看金融词条目录的状态。**仅做展示和查看**，修复不一致请用 `/ct:structure`。

## 执行步骤

### 步骤 1：读取索引

用 Read 工具读取 `docs/glossary/index.md`，解析分类和词条列表。

### 步骤 2：扫描实际文件

用 Glob 工具扫描 `docs/` 下各分类目录的 `*.md` 文件。

### 步骤 3：对比分析

对比 index.md 与实际文件，分为三类：

| 状态 | 判断条件 |
|------|----------|
| 已完成 | 文件存在且字数 > 1000 |
| 空壳 | 文件存在但仅 frontmatter（字数 < 100） |
| 仅索引 | index.md 列出但文件不存在 |

### 步骤 4：输出清单

```
## 词条状态总览

### 基础概念（10/10 完成）
- [x] 什么是金融（1,200 字）
- [x] 货币（1,500 字）
- [ ] 信用（仅标题）

### 货币与银行（8/10 完成）
...
```

## 容错

- index.md 解析失败 → 改用 Glob 扫描所有 `.md` 文件，按目录推断分类
- 无词条 → 输出"词条目录为空"
- 用户要求修复 → 明确提示使用 `/ct:structure` 命令
