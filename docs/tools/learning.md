## 选择建议

- **零基础入门**：《小狗钱钱》+ 知行小酒馆
- **系统学习**：Coursera 金融市场课程 + 《漫步华尔街》
- **价值投资**：《聪明的投资者》+ 雪球社区
- **量化方向**：《打开量化投资的黑箱》+ 聚宽平台
