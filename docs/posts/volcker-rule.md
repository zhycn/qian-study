# 沃尔克规则

2008 年金融危机时，老百姓的存款被银行拿去在华尔街"赌桌上"豪赌，结果赌输了，政府还得用纳税人的钱去救。沃尔克规则就是给银行立的一条规矩："拿存款的银行，不该去赌场。"

## 概述

沃尔克规则（Volcker Rule）是 2010 年美国《多德-弗兰克法案》（Dodd-Frank Act）第 619 条的核心内容，由前美联储主席保罗·沃尔克（Paul Volcker）提出，旨在限制商业银行从事高风险的自营交易（Proprietary Trading）和对对冲基金/私募基金的投资。

核心内容：
- **禁止自营交易**：银行不得用自有资金进行与客户服务无关的交易
- **限制基金投资**：银行对对冲基金和私募基金的投资不得超过基金资本的 3% 和银行一级资本的 3%
- **合规要求**：银行需建立内部合规程序，证明交易符合豁免条件

**历史背景**：2008 年金融危机暴露了商业银行"既做存贷款，又做高风险交易"的利益冲突问题。沃尔克认为，享受联邦存款保险的银行不应从事高风险投机活动（来源：沃尔克在参议院听证会证词，2009）。

## 为什么需要沃尔克规则

### 危机前的乱象

2008 年金融危机前，大型商业银行：
- 用存款资金进行高风险自营交易
- 大量投资对冲基金和私募基金
- 通过复杂衍生品放大风险
- "大而不能倒"——赌赢了高管拿奖金，赌输了政府兜底

### 道德风险

```
赌赢了 → 高管拿巨额奖金
赌输了 → 政府用纳税人钱救助
```

这种"收益私有化、风险社会化"的模式， incentivize 银行承担过度风险。

## 核心规定

### 禁止自营交易

**自营交易定义**：银行作为 principal（ principal）为自己的账户买卖证券、衍生品等金融工具，目的是从短期价格波动中获利。

**允许的例外**：
| 豁免类型 | 说明 | 例子 |
| -------- | ---- | ---- |
| 做市交易 | 为客户提供流动性 | 银行作为做市商买卖债券 |
| 对冲交易 | 管理银行自身风险 | 用衍生品对冲利率风险 |
| 承销交易 | 证券承销相关业务 | IPO 承销中的库存管理 |
| 政府债券 | 买卖政府债券 | 买卖美国国债 |
| 保险业务 | 保险公司投资 | 保险公司为匹配负债的投资 |

### 限制基金投资

| 限制 | 具体要求 |
| ---- | -------- |
| 投资上限 | 对单只基金投资不超过基金资本的 3% |
| 总上限 | 对所有基金投资总额不超过银行一级资本的 3% |
| 名称限制 | 银行不能使用"基金"名称误导客户 |

## 为什么重要

沃尔克规则的意义：

- **降低系统性风险**：限制银行高风险交易，减少"大而不能倒"问题
- **保护存款人利益**：防止银行用存款进行高风险投机
- **减少道德风险**：打破"赌赢拿奖金、赌输政府救"的恶性循环
- **回归银行本源**：让银行专注存贷款和客户服务，而不是投机

## 生活化类比

沃尔克规则就像给"赌场"和"银行"之间建一堵墙。

银行就像你家楼下的杂货店——你存钱进去（存款），它借给别人（贷款），赚点利息差。这是它的本分。

但有些银行不满足，拿着你的钱去隔壁赌场玩（自营交易）。赢了，老板开豪车；输了，找政府要钱（救助）。

沃尔克规则说："不行！杂货店就该好好开杂货店，别去赌场。"

当然，杂货店也需要一些"风险管理工具"——比如买保险防火灾（对冲）、备点货满足客户需求（做市）。这些是允许的。但纯粹为了赚钱去赌博（自营交易），不行。

## 实施与修订

### 实施时间线

| 时间 | 事件 |
| ---- | ---- |
| 2010 年 | 《多德-弗兰克法案》通过，沃尔克规则写入法律 |
| 2013 年 | 五大监管机构（美联储、OCC、FDIC、SEC、CFTC）发布最终规则 |
| 2014 年 | 规则正式生效，银行开始合规整改 |
| 2018 年 | 特朗普政府签署放松监管法案，放宽对中小银行的限制 |
| 2020 年 | 美联储等修订规则，简化合规要求 |

### 争议与评价

**支持方观点**：
- 降低了银行系统性风险
- 减少了自营交易规模
- 保护了存款人利益

**反对方观点**：
- 降低了市场流动性（做市活动减少）
- 合规成本高昂
- 限制了银行盈利能力

**实际效果**：
- 大型银行自营交易规模显著下降（来源：美联储评估报告，2020）
- 银行合规成本增加，但系统性风险降低
- 市场流动性在某些领域有所下降

## 实战应用

### 对全球银行业的影响

沃尔克规则虽是美国法规，但对全球银行业有深远影响：

- **跨国银行**：在美经营的外国银行也需遵守
- **监管趋势**：多国考虑类似限制（如英国"围栏"改革）
- **业务调整**：全球大型银行缩减自营交易业务

### 对投资者的启示

**银行股投资**：
- 沃尔克规则限制了银行盈利来源，可能影响估值
- 但降低了风险，长期看有利于银行股稳定
- 关注银行合规成本和业务结构调整

**市场流动性**：
- 做市活动减少可能影响某些债券的流动性
- 交易成本可能上升，投资者需关注买卖价差

## 关键要点

- 沃尔克规则禁止银行用自有资金进行自营交易
- 限制银行对对冲基金和私募基金的投资
- 核心目标是降低系统性风险、保护存款人
- 做市、对冲、承销等客户服务活动被豁免
- 2018 年后有所放宽，但核心原则未变
- 对全球银行业有深远影响

## 常见误区

- **"沃尔克规则禁止所有交易"**——做市、对冲、承销等客户服务活动是允许的
- **"沃尔克规则让银行不赚钱"**——限制的是高风险自营交易，传统存贷款业务不受影响
- **"沃尔克规则只影响美国"**——跨国银行在美业务也需遵守，影响全球
- **"沃尔克规则完全解决问题"**——规则有漏洞，执行有难度，不能完全消除风险
