# CDS 信用违约互换

CDS 的本质很简单：你买了一只债券，担心发行人还不上钱，于是找另一家机构"买保险"——如果债券违约了，保险公司赔你钱；如果没违约，你白交保费。

这个"保险"就是信用违约互换（Credit Default Swap, CDS）。它是金融世界最知名的衍生品之一，也是 2008 年次贷危机中最具争议的角色。《大空头》电影里，主角们就是通过买入 CDS 做空次贷市场，一夜暴富。

## 概述

信用违约互换（Credit Default Swap, CDS）是一种信用衍生品合约，用于转移债券或贷款的信用风险。

**合约结构**：

- **买方**（Protection Buyer）：支付定期保费（Premium），获得信用保护
- **卖方**（Protection Seller）：收取保费，承诺在"信用事件"发生时赔付买方
- **参考实体**（Reference Entity）：被保险的债券或贷款的发行人
- **名义本金**（Notional Amount）：合约的面值，用于计算保费和赔付金额
- **期限**（Tenor）：通常为 1-10 年，标准合约为 5 年

**信用事件**（Credit Events）触发赔付：

- 破产（Bankruptcy）
- 无法支付（Failure to Pay）
- 债务重组（Debt Restructuring）
- 拒付/延期偿付（Repudiation/Moratorium）

**赔付方式**：

- **实物交割**（Physical Settlement）：买方将违约债券交给卖方，卖方按面值（100%）支付现金
- **现金交割**（Cash Settlement）：卖方直接支付面值与违约后市场价值的差额

**CDS 定价**：

CDS 的价格用"基点"（bps）报价，1bp = 0.01%。例如 CDS 报价 100bps，意味着买方每年支付名义本金的 1% 作为保费。

$$\text{年保费} = \text{名义本金} \times \text{CDS 利差}$$

CDS 利差反映了市场对参考实体违约风险的定价。利差越高，违约风险越大。

**关键特征**：

| 维度 | 说明 |
|------|------|
| 交易场所 | 场外交易（OTC），非标准化 |
| 监管 | 2008 年后逐步纳入中央清算 |
| 用途 | 对冲信用风险 / 投机 / 套利 |
| 最大风险 | 买方：已付保费；卖方：名义本金 |
| 流动性 | 大型发行人（如美国国债、大型蓝筹）流动性好，小型发行人差 |

## 演进历史

**CDS 的发明（1994 - 1997）**：CDS 由摩根大通（J.P. Morgan）的团队在 1994 年发明，最初名为"BISTRO"（Broad Index Secured Trust Offering）（来源：摩根大通内部档案，1997）。目的是将银行贷款的信用风险转移出去，释放资本金。1997 年 ISDA（国际掉期与衍生工具协会）发布标准 CDS 合约，市场开始标准化（来源：ISDA，1997）。

**快速扩张期（1998 - 2007）**：全球 CDS 名义本金从 1998 年的约 3000 亿美元飙升到 2007 年底的 62 万亿美元（来源：ISDA 市场数据，2008）。CDS 从单纯的对冲工具演变为投机工具——你不需要持有债券也能买 CDS（"裸 CDS"），相当于对赌别人违约。

**次贷危机与 AIG 危机（2008）**：AIG 金融产品部门（AIGFP）卖出了价值约 4400 亿美元的 CDS 保护，主要参考次贷 MBS 和 CDO（来源：纽约联储，2008）。当次贷违约率上升，AIG 无力赔付，被迫接受政府 1820 亿美元救助（来源：美国财政部，2008）。CDS 从"保险"变成了"系统性风险的放大器"。

**改革与规范化（2010 至今）**：《多德-弗兰克法案》要求标准化 CDS 纳入中央清算（来源：美国国会，2010）。ISDA 2014 年修订 CDS 定义，引入"债务重组"条款的细化。欧洲通过 EMIR 法规规范衍生品交易（来源：欧盟，2012）。CDS 市场规模从峰值回落，2023 年全球未平仓名义本金约 4 万亿美元（来源：ISDT，2023）。

## 核心分类

- **单一名称 CDS**（Single-Name CDS）：参考单一发行人（如"苹果公司 5 年期 CDS"）。最基础的 CDS 类型
- **CDS 指数**（CDS Index）：参考一篮子发行人的 CDS 组合。最知名的是 CDX（北美）和 iTraxx（欧洲）。流动性好，是市场情绪的"晴雨表"
- **资产支持 CDS**（ABX/CMBX）：参考资产证券化产品（如次贷 MBS、商业地产 MBS）的 CDS。次贷危机中的核心角色
- **主权 CDS**（Sovereign CDS）：参考国家主权债务的 CDS。如"希腊 5 年期 CDS"在欧债危机期间飙升至 1000bps 以上（来源：Markit，2012）
- **贷款 CDS**（Loan CDS）：参考银行贷款的 CDS，也称"总收益互换"（Total Return Swap）

## 为什么重要

CDS 是信用风险管理的"核武器"：

- **风险转移**：银行通过 CDS 将贷款信用风险转移给愿意承担的投资者，释放资本金
- **价格发现**：CDS 利差是市场对企业信用风险的实时定价，比评级更灵敏
- **投机工具**：投资者可以通过 CDS 做空信用，而不需要实际持有债券
- **系统性风险指标**：CDS 利差和 CDS 指数是衡量市场恐慌程度的重要指标
- **危机放大器**：2008 年危机中，CDS 将局部风险扩散为系统性风险，AIG 几乎因此破产

《大空头》电影中，迈克尔·伯里（Michael Burry）等人通过买入次贷 MBS 的 CDS，在房地产市场崩溃时获得巨额回报。他们的逻辑很简单：当次贷违约率上升，MBS 价值暴跌，CDS 卖方必须按面值赔付——而他们当初支付的保费只是面值的很小一部分。

## 生活化类比

CDS 就像给邻居的房子买火灾保险。

正常情况下，你给自己的房子买保险——这是对冲风险（你持有债券，买 CDS 保护）。

但 CDS 允许你给邻居的房子买保险——你并不拥有那栋房子，只是赌它会着火（裸 CDS，投机）。如果邻居房子着火了，你获得赔付，大赚一笔。如果没着火，你白交保费。

问题来了：如果很多人都给同一栋房子买保险，而保险公司（CDS 卖方）收的保费不够赔付所有保单，一旦房子真的着火，保险公司就破产了——这就是 AIG 在 2008 年遇到的情况。

更糟糕的是，如果有人故意纵火（恶意做空），那问题就更复杂了。虽然 CDS 合约有"信用事件"的严格定义，但在极端情况下，市场恐慌本身就可能加速违约的到来。

## 实战应用

### CDS 定价与估值

**CDS 利差的含义**：

| CDS 利差 | 含义 | 参考案例 |
|----------|------|----------|
| < 50bps | 极低风险 | 美国国债、德国国债 |
| 50-100bps | 低风险 | 大型蓝筹企业（苹果、微软） |
| 100-300bps | 中等风险 | 投资级企业 |
| 300-500bps | 较高风险 | 垃圾级企业 |
| 500-1000bps | 高风险 | 困境企业 |
| > 1000bps | 极高风险 | 接近违约 |

（来源：Markit，2024 年参考数据）

**CDS 与债券利差的关系**：

理论上，CDS 利差应该等于同期限债券的信用利差（CDS-Bond Basis）。如果存在差异，就产生了套利机会：

- CDS 利差 > 债券利差：买入债券 + 买入 CDS 保护（锁定正收益）
- CDS 利差 < 债券利差：做空债券 + 卖出 CDS 保护

但实际上，由于流动性、资金成本、Counterparty Risk 等因素，CDS-Bond Basis 经常偏离零。

### 2008 年危机中的 CDS

**AIG 的 CDS 敞口**：

AIGFP 部门卖出的 CDS 保护约 4400 亿美元，主要参考超优先档（Super Senior）MBS 和 CDO（来源：纽约联储，2008）。这些产品被评级机构评为 AAA，AIG 认为违约概率极低，因此没有预留足够准备金。

当次贷违约率上升：

1. MBS/CDO 价值下跌，触发 CDS 合约的抵押品追加要求（Margin Call）
2. AIG 需要支付数十亿美元的抵押品，流动性迅速枯竭
3. 评级机构下调 AIG 评级，触发更多抵押品要求
4. AIG 无力支付，被迫接受政府救助

**"大空头"的 CDS 策略**：

以迈克尔·伯里（Michael Burry）的 Scion Capital 为例：

- 2005-2006 年：发现次贷市场风险，开始买入次贷 MBS 的 CDS
- 支付的 CDS 保费：约 1-2%/年
- 2007-2008 年：次贷违约率飙升，MBS 价值暴跌
- CDS 赔付：面值（100%）减去残值（可能只有 10-20%）
- 回报：投入约 1 亿保费，获得约 7 亿赔付，回报约 7 倍（来源：《大空头》原著，迈克尔·刘易斯）

### CDS 在投资中的应用

**信用观点表达**：

- 看空某企业信用：买入 CDS 保护（无需持有债券）
- 看多某企业信用：卖出 CDS 保护（收取保费，承担违约风险）

**相对价值交易**：

- 同一行业不同企业的 CDS 利差比较
- 同一企业不同期限的 CDS 利差比较（CDS 曲线）
- CDS 与债券的基差交易

**组合对冲**：

- 基金持有大量债券，用 CDS 指数对冲系统性信用风险
- 银行用 CDS 管理贷款组合的信用风险敞口

## 关键要点

- CDS 是信用风险的"保险合约"，买方付保费，卖方承诺违约时赔付
- CDS 可以用于对冲（持有债券 + 买 CDS）或投机（裸 CDS）
- CDS 利差是市场对企业信用风险的实时定价，比评级更灵敏
- 2008 年危机中，CDS 将局部风险扩散为系统性风险，AIG 几乎因此破产
- 危机后 CDS 市场逐步规范化，标准化合约纳入中央清算
- CDS 市场仍然是全球信用风险定价的重要参考

## 常见误区

- **CDS 就是保险**：CDS 与保险有本质区别。保险要求"可保利益"（你必须拥有被保险资产），但 CDS 允许"裸买"（不持有债券也能买），更像对赌
- **CDS 利差 = 违约概率**：CDS 利差反映的是"风险中性违约概率"，包含风险溢价，不等于真实违约概率
- **CDS 只用于投机**：CDS 最初是作为风险管理工具发明的，对冲仍然是重要用途
- **CDS 市场已经消失**：CDS 市场依然存在，只是规模从峰值回落，监管更加严格
- **买入 CDS 就能稳赚**：如果参考实体不违约，CDS 买方损失全部保费。2007-2008 年很多早期买入 CDS 的投资者在危机前就被"震出局"了
