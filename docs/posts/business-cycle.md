# 经济周期

经济周期就像一年四季——春天复苏万物生长，夏天繁荣人人赚钱，秋天衰退生意难做，冬天萧条熬过去就是春天，聪明人知道什么季节穿什么衣服。

## 概述

经济周期（Business Cycle）是指经济活动沿着长期增长趋势出现的有规律的扩张与收缩交替往复的现象。一个完整的经济周期通常包括四个阶段：繁荣（Expansion）、衰退（Recession）、低谷（Trough）、复苏（Recovery）。

通俗理解：经济不是一直往上走的，而是像海浪一样有起有伏，有高峰也有低谷，循环往复。

经济周期不是指 GDP 的绝对下降，而是指增长率的变化。即使 GDP 仍在增长，只要增速放缓，就可能处于周期的下行阶段。

## 演进历史

经济周期的研究可以追溯到 19 世纪。

**1860 年代**：法国医生克莱门特·朱格拉（Clément Juglar）首次提出经济存在 7-11 年的周期性波动，被称为"朱格拉周期"。他发现设备投资的周期性更新是驱动因素。

**1920 年代**：苏联经济学家康德拉季耶夫（Nikolai Kondratiev）提出 50-60 年的"长波周期"（康波周期），认为技术革命是长周期的驱动力。

**1930 年代**：熊彼特（Joseph Schumpeter）综合了多种周期理论，提出经济周期是"创新"的产物——新技术的出现引发投资热潮，随后是产能过剩和衰退。

**1946 年**：伯恩斯（Arthur Burns）和米切尔（Wesley Mitchell）出版《衡量经济周期》，奠定了现代经济周期研究的实证基础。美国国家经济研究局（NBER）成为官方认定经济周期拐点的权威机构。

**1980 年代至今**：实际经济周期理论（RBC）和新凯恩斯主义模型成为主流，强调技术冲击、价格粘性和货币政策在周期中的作用。

## 核心分类

经济周期按长度和驱动因素可以分为不同类型：

**基钦周期（Kitchin Cycle）**：

- 长度：3-5 年
- 驱动因素：库存周期
- 逻辑：企业根据市场需求调整库存，库存积压 → 减产去库存 → 库存不足 → 增产补库存

**朱格拉周期（Juglar Cycle）**：

- 长度：7-11 年
- 驱动因素：设备投资周期
- 逻辑：企业更新设备 → 投资增加 → 产能扩张 → 产能过剩 → 投资减少

**库兹涅茨周期（Kuznets Cycle）**：

- 长度：15-25 年
- 驱动因素：房地产和建筑周期
- 逻辑：房地产建设热潮 → 供给过剩 → 市场调整 → 新一轮建设

**康德拉季耶夫周期（Kondratiev Wave）**：

- 长度：50-60 年
- 驱动因素：技术革命
- 逻辑：新技术出现 → 大规模投资 → 技术成熟 → 增长放缓 → 下一次技术革命

**经济周期的四个阶段**：

1. **复苏（Recovery）**：经济触底回升，就业改善，企业开始投资，政策保持宽松
2. **繁荣（Expansion）**：经济高速增长，就业充分，通胀上升，政策逐步收紧
3. **衰退（Recession）**：经济增速放缓甚至负增长，企业裁员，股市下跌，政策转向宽松
4. **低谷（Trough）**：经济跌至谷底，失业率高企，通缩风险，政策全力刺激

## 为什么重要

理解经济周期能帮助我们在不同阶段做出更好的决策：繁荣期警惕泡沫，衰退期寻找机会，萧条期保住本金，复苏期大胆布局。政府也根据经济周期调整政策，试图"熨平"周期波动。

具体来说：

- **投资决策**：不同资产在不同周期阶段表现不同。复苏期股票好，繁荣期商品好，衰退期债券好
- **职业选择**：繁荣期跳槽涨薪容易，衰退期保住工作更重要
- **购房时机**：萧条期和复苏初期往往是购房的好时机
- **政策预判**：理解周期位置可以预判央行和政府的政策方向

## 生活化类比

经济就像一年四季：春天（复苏）万物复苏，生意好做；夏天（繁荣）热火朝天，人人赚钱；秋天（衰退）开始转凉，生意难做；冬天（萧条）寒风刺骨，熬过去就是春天。聪明人知道什么季节穿什么衣服。

但经济的"四季"没有固定时间表——有时夏天很长，有时冬天很冷，有时还会出现"倒春寒"（二次衰退）。

## 关键要点

- 经济周期长度不一，短则几年，长则十几年
- 没有两个完全相同的经济周期
- 政策可以缓解但无法消除周期波动
- 不同资产类别在不同周期阶段表现不同
- 经济周期的拐点只能在事后确认，事前难以精确判断
- 库存周期是最短、最容易观察的周期

## 常见误区

- **误区一：经济周期是固定时间的循环**——周期长度不固定，无法精确预测拐点。NBER 通常在衰退开始后数月才正式确认
- **误区二：这次不一样**——每次周期驱动因素不同，但人性不变，繁荣-衰退的规律始终存在。霍华德·马克斯在《周期》中写道："树不会长到天上去"
- **误区三：政策能消除周期**——政策只能"熨平"波动，无法消灭周期。过度干预反而可能扭曲市场信号，积累更大风险
- **误区四：萧条就是世界末日**——萧条是周期的一部分，也是出清和重新配置的必经阶段。每一次萧条之后都有新的繁荣

## 与其他概念的关系

- 与 [GDP](/posts/gdp)：GDP 增速变化是判断周期阶段的核心指标。连续两个季度 GDP 负增长通常被视为技术性衰退
- 与 [货币政策](/posts/monetary-policy)：央行逆周期调节（Counter-cyclical Policy）试图平滑经济波动——衰退时降息、繁荣时加息
- 与 [滞胀](/posts/stagflation)：滞胀是经济周期中的一种异常状态，打破了传统的周期规律
- 与 [财政政策](/posts/fiscal-policy)：政府通过逆周期财政政策（衰退时扩张、繁荣时紧缩）来稳定经济
- 与 [黑天鹅](/posts/black-swan)：黑天鹅事件可能触发或加速经济周期的转折
