# 周蔚文——公募常青树

> 在中国公募基金行业，能穿越三轮以上牛熊、从业 20 年依然保持优秀业绩的基金经理屈指可数。周蔚文是其中之一。他用"景气度投资"和"行业轮动"策略，在 A 股市场实现了年化超过 18% 的长期回报，被称为公募界的"常青树"。

## 人物速写

周蔚文，中欧基金副总经理、资深基金经理，中国公募基金行业最资深的投资老将之一。他从 2000 年代初开始管理基金，历经多轮牛熊更迭，凭借景气度投资和行业轮动策略，长期业绩稳居行业前列。

## 传奇人生

### 早年经历：从复旦到投研

1967 年，周蔚文出生于上海。他后来考入复旦大学，接受了系统的金融学教育。复旦大学是中国金融学教育的重镇，培养了大批优秀的金融从业者。

1990 年代，周蔚文进入证券行业，开始了他的投研生涯。当时的中国证券市场还处于早期阶段，市场波动剧烈，投资理念尚未成熟。

周蔚文从研究员做起，先后覆盖过多个行业。这段经历为他后来的行业轮动策略打下了坚实的基础。

> "做研究员的时候，我覆盖过很多行业。每个行业都有自己的周期和逻辑。我逐渐发现，理解行业景气度的变化，比单纯看个股更重要。"

### 投资启蒙：发现景气度的力量

在早期的投研工作中，周蔚文观察到一个现象：

- 同一只股票，在不同行业周期阶段，表现差异巨大
- 行业景气度向上时，即使是二线公司也能涨得很好
- 行业景气度向下时，龙头公司也难以独善其身

这个发现让他意识到：**行业景气度是驱动股票回报的最重要因素之一。**

他开始系统性地研究行业周期：

- **宏观经济周期**——经济上行还是下行？
- **行业生命周期**——导入期、成长期、成熟期还是衰退期？
- **政策周期**——政策鼓励还是限制？
- **供需周期**——供给紧张还是过剩？

这套方法论后来成为他投资框架的核心。

### 关键转折点：加入中欧基金

2000 年代中期，周蔚文加入中欧基金，开始了他在公募平台的长期投资生涯。

中欧基金成立于 2006 年，是中国公募基金行业的重要参与者。周蔚文在中欧基金的角色不断升级：

- 基金经理——管理多只公募产品
- 投资总监——负责整体投研方向
- 副总经理——参与公司治理和战略决策

在中欧基金期间，周蔚文管理的基金涵盖了多个主题和风格：

- **行业主题基金**——聚焦特定景气度向上的行业
- **全市场基金**——基于行业轮动进行配置
- **稳健增长基金**——追求长期稳健回报

他的管理风格有几个鲜明特点：

- **不押注单一行业**——通过行业轮动分散风险
- **不追热点**——在景气度拐点前布局，而不是等热点爆发后追入
- **注重回撤控制**——在熊市中控制损失，为下一轮牛市保存实力

### 巅峰时刻：穿越多轮牛熊

周蔚文的投资生涯跨越了 A 股市场最重要的几个阶段：

**关键里程碑：**

- **2000 年代中期**：开始管理基金，经历了股权分置改革带来的市场变革
- **2007 年**：A 股历史上最大牛市之一，上证指数达到 6124 点
- **2008 年**：全球金融危机，A 股暴跌超过 60%
- **2014-2015 年**：杠杆牛市与暴跌
- **2016-2017 年**：价值投资风格占优
- **2019-2021 年**：核心资产牛市
- **2022-2024 年**：市场调整与结构分化

在每一轮牛熊更迭中，周蔚文都展现出了出色的适应能力：

- **牛市中**——通过行业轮动捕捉结构性机会，不盲目追高
- **熊市中**——控制仓位和回撤，保存实力
- **震荡市中**——精选景气度向上的行业，获取超额收益

**关于长期业绩的数据：** 根据晨星中国数据，周蔚文管理的代表基金在 20 年+ 的投资生涯中，年化收益超过 18%，大幅跑赢同期沪深 300 指数和同类平均水平（来源：晨星中国，截至 2024 年）。

周蔚文在中欧基金最核心的贡献是：**将景气度投资从理念变成了可执行的体系。** 他建立了一套完整的行业轮动框架：

1. **宏观判断**——经济处于什么周期？
2. **行业比较**——哪些行业景气度在上升？
3. **个股选择**——在景气度向上的行业中选龙头
4. **动态调整**——根据景气度变化进行轮动

## 经典战役

### 战役一：2007 年牛市——提前布局周期股

2006-2007 年，A 股经历了历史上最大的牛市之一。周蔚文通过提前布局周期股，获得了丰厚的回报。

**背景：**

- 2006-2007 年，中国经济处于高速增长期
- 股权分置改革释放了市场活力
- 有色金属、煤炭、银行等周期股大幅上涨
- 上证指数从 1000 点左右涨到 6124 点

**周蔚文的判断：**

- 经济高速增长，周期股受益最大
- 有色金属、煤炭等资源品价格持续上涨
- 银行股估值低、盈利增长快
- 周期股的景气度还在上升通道中

**具体操作：**

- 提前布局有色金属、煤炭、银行等周期板块
- 在景气度上升阶段持有，不轻易卖出
- 当市场情绪极度狂热时，逐步减仓

**结果：**

- 2007 年，周蔚文管理的基金业绩大幅领先同类（来源：晨星中国，2007 年基金排名）
- 周期股板块成为当年表现最好的板块
- 上证指数全年涨幅超过 100%

### 战役二：2008 年金融危机——控制回撤

2008 年全球金融危机爆发，A 股暴跌超过 60%。周蔚文通过及时的仓位调整和行业切换，有效控制了回撤。

**背景：**

- 2008 年，全球金融危机爆发
- A 股从 6124 点暴跌至 1664 点，跌幅超过 70%
- 绝大多数基金回撤超过 50%
- 市场恐慌情绪蔓延

**周蔚文的应对：**

- 在 2007 年底市场狂热时，已经开始逐步减仓
- 降低周期股仓位，增加防御性板块配置
- 控制整体仓位，不盲目抄底

**结果：**

- 2008 年，周蔚文管理的基金回撤小于市场平均水平（来源：晨星中国，2008 年基金年报）
- 在熊市中保存了实力，为下一轮行情做好准备

| 指标         | 数值                           |
| ------------ | ------------------------------ |
| 上证指数跌幅 | 从 6124 点到 1664 点（-73%）   |
| 基金回撤     | 小于市场平均（来源：晨星中国） |
| 应对策略     | 提前减仓、降低周期股、控制仓位 |

**数据来源：** 晨星中国基金评级、基金定期报告

### 战役三：2013-2015 年——布局成长股

2013-2015 年，A 股市场风格从价值转向成长。周蔚文敏锐地捕捉到了这一变化，提前布局成长股。

**背景：**

- 2013 年前后，中国经济增速放缓，传统周期股景气度下降
- 互联网、传媒、医药等成长行业景气度上升
- "互联网+"概念兴起，创业板大幅跑赢主板
- 2015 年，杠杆资金推动 A 股再次冲高

**周蔚文的判断：**

- 经济转型期，成长行业景气度上升
- 互联网、传媒、医药是长期增长确定性高的赛道
- 传统周期股景气度下行，应该降低配置
- 成长股的估值虽然不低，但景气度可以消化估值

**具体操作：**

- 降低周期股仓位，增加成长股配置
- 重点布局互联网、传媒、医药等成长板块
- 在 2015 年市场狂热时，逐步减仓

**结果：**

- 2013-2015 年，周蔚文管理的基金业绩持续领先（来源：晨星中国，2013-2015 年排名）
- 成长股板块成为这一时期表现最好的板块
- 在 2015 年暴跌前及时减仓，控制了回撤

### 战役四：2019-2021 年——重仓核心资产

2019-2021 年，A 股市场迎来了"核心资产"牛市。周蔚文通过布局消费、医药、新能源等景气度向上的行业，再次获得了优秀业绩。

**背景：**

- 2019 年，中美贸易摩擦缓和，市场情绪回暖
- 消费升级趋势明显，白酒、家电等消费股表现优异
- 医药行业受益于人口老龄化和创新药发展
- 新能源汽车政策推动，产业链景气度大幅上升

**周蔚文的判断：**

- 消费、医药、新能源是长期景气度向上的行业
- 核心资产（龙头公司）的估值虽然不低，但成长性可以支撑
- 行业集中度提升，龙头公司受益
- 新能源是未来 5-10 年确定性最高的赛道之一

**具体操作：**

- 增加消费、医药、新能源板块配置
- 选择各行业的龙头公司
- 在景气度上升阶段持有，不轻易卖出

**结果：**

- 2019-2021 年，周蔚文管理的基金业绩持续领先（来源：晨星中国，2019-2021 年排名）
- 消费、医药、新能源板块成为这一时期表现最好的板块
- 新能源汽车产业链相关公司涨幅巨大

### 战役五：2022-2024 年——防守反击

2022-2024 年，A 股市场经历了深度调整。周蔚文通过防守反击策略，在熊市中控制回撤，在反弹中捕捉机会。

**背景：**

- 2022 年，全球加息周期开启，A 股市场承压
- 房地产下行、消费疲软、出口放缓
- 市场情绪低迷，多数板块下跌
- 2023-2024 年，市场结构分化，红利资产和科技股表现分化

**周蔚文的应对：**

- 降低整体仓位，控制风险
- 增加红利资产（高股息、低估值）配置
- 关注景气度触底反弹的行业
- 在反弹中捕捉结构性机会

**结果：**

- 2022-2024 年，周蔚文管理的基金回撤小于市场平均（来源：晨星中国，2022-2024 年基金年报）
- 在熊市中保存了实力
- 在反弹中捕捉到了结构性机会

## 投资哲学

### 原则一：景气度是核心驱动

周蔚文投资哲学的核心是景气度投资。

> "股票的价格最终取决于盈利。而盈利取决于行业景气度。找到景气度向上的行业，就找到了赚钱的方向。"

他的景气度分析框架：

- **宏观经济景气度**——经济处于扩张还是收缩？
- **行业景气度**——行业供需格局如何？价格趋势如何？
- **公司景气度**——公司订单、收入、利润趋势如何？
- **政策景气度**——政策是鼓励还是限制？

### 原则二：行业轮动是超额收益来源

周蔚文认为，行业轮动是获取超额收益的重要手段。

> "没有永远上涨的行业，也没有永远下跌的行业。关键是找到景气度正在上升的行业。"

他的行业轮动逻辑：

- **周期轮动**——经济周期不同阶段，不同行业表现不同
- **成长轮动**——新兴产业从导入期到成长期，景气度持续上升
- **政策轮动**——政策变化带来行业景气度变化
- **估值轮动**——估值极端的行业会回归合理

### 原则三：不押注单一行业

周蔚文强调分散配置的重要性。

> "即使你看对了某个行业，也不应该把所有仓位押在上面。分散配置可以降低风险，提高长期回报的稳定性。"

他的分散配置原则：

- 单一行业仓位不超过组合的 30%
- 覆盖 3-5 个景气度向上的行业
- 根据景气度变化动态调整

### 原则四：在拐点前布局

周蔚文强调前瞻性布局。

> "等所有人都看到景气度上升的时候，股价已经反映了。要在拐点前布局，而不是等热点爆发后追入。"

他的拐点判断方法：

- **数据跟踪**——关注行业高频数据（价格、库存、订单等）
- **政策跟踪**——关注政策变化和行业动向
- **产业链调研**——了解上下游景气度变化
- **估值参考**——估值处于历史低位时，拐点可能临近

### 原则五：控制回撤比追求收益更重要

周蔚文非常重视回撤控制。

> "在熊市中少亏 10%，在牛市中就多 10% 的本金可以增值。控制回撤是长期复利的基础。"

他的回撤控制方法：

- 市场狂热时减仓
- 景气度下行时降低仓位
- 不盲目抄底
- 保持一定的现金仓位

## 策略拆解

### 选股标准

周蔚文的选股标准可以总结为"三个景气度"：

| 标准       | 具体要求                     |
| ---------- | ---------------------------- |
| 行业景气度 | 行业处于上升期，供需格局向好 |
| 公司景气度 | 公司订单、收入、利润趋势向上 |
| 估值景气度 | 估值合理或偏低，有安全边际   |

**具体筛选条件：**

- 行业景气度：供需格局改善、价格趋势向上、政策鼓励
- 公司景气度：收入增长加速、利润增速提升、订单饱满
- 估值：PE/PB 处于历史合理区间，PEG < 1.5

### 买入时机

周蔚文的买入时机判断：

1. **景气度拐点**——行业景气度触底回升
2. **政策催化**——政策变化带来行业机会
3. **估值回归**——好公司遇到暂时困难，估值被错杀
4. **数据验证**——高频数据确认景气度上升

他特别强调：**不要在景气度已经充分反映在股价中时追入。** 要在拐点前布局。

### 持仓管理

周蔚文的持仓管理特点：

- **行业分散**——覆盖 3-5 个景气度向上的行业
- **个股集中**——在每个行业中选择 2-3 只龙头
- **动态调整**——根据景气度变化进行轮动
- **仓位灵活**——根据市场整体景气度调整仓位

### 卖出纪律

周蔚文的卖出条件：

1. **景气度拐点向下**——行业景气度开始下降
2. **估值极度泡沫**——价格远超内在价值
3. **逻辑证伪**——投资逻辑被证明是错误的
4. **发现更好的机会**——换仓到景气度更高的行业

他特别强调：**不要因为短期波动而卖出。** 如果景气度还在上升，就应该继续持有。

### 普通人能学到什么

周蔚文的方法对普通投资者有重要借鉴意义：

1. **关注行业景气度**——不要只看个股，要看行业趋势
2. **学会行业轮动**——没有永远上涨的行业，要动态调整
3. **在拐点前布局**——等所有人都看到机会时，已经晚了
4. **控制回撤**——熊市中少亏，牛市中才能多赚
5. **分散配置**——不押注单一行业，降低风险

**具体建议：**

- 跟踪 3-5 个你熟悉的行业
- 关注行业高频数据（价格、库存、订单等）
- 在景气度触底时布局，在景气度高峰时减仓
- 保持 20-30% 的现金仓位，应对不确定性
- 每年检查一次行业景气度变化
