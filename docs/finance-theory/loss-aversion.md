# 损失厌恶

> 丢了 100 块钱的难受程度，大概需要捡到 200 块钱才能弥补——这就是损失厌恶，人类大脑的"出厂设置"。

## 概述

损失厌恶（Loss Aversion）是指**人们对损失的敏感度远高于对同等收益的敏感度**。行为经济学研究表明，损失带来的心理痛苦大约是同等收益带来的快乐的 **2 到 2.5 倍**。

损失厌恶是前景理论（[Prospect Theory](/finance-theory/prospect-theory)）的核心组成部分，由丹尼尔·卡尼曼（Daniel Kahneman）和阿莫斯·特沃斯基（Amos Tversky）在 1979 年提出。

在投资中的典型表现：

- **处置效应**（Disposition Effect）：过早卖出盈利的股票（落袋为安），却长期持有亏损的股票（死扛不卖）
- **风险规避**：宁愿存银行拿低利息，也不愿承担波动去投资
- **框架效应**（Framing Effect）：同一个方案，说"存活率 90%"比说"死亡率 10%"更容易被接受
- **现状偏差**（Status Quo Bias）：倾向于维持当前状态，即使改变可能更好

## 为什么重要

- **解释投资行为**：为什么散户总是"拿不住利润，却死扛亏损"——这是损失厌恶最直接的体现
- **资产配置**：损失厌恶导致投资者过度偏好低风险资产，可能错失长期收益。股权溢价之谜（Equity Premium Puzzle）部分可用损失厌恶解释
- **产品设计**：金融机构通过"保本""稳健""最大回撤"等话术迎合损失厌恶心理
- **决策质量**：理解损失厌恶有助于做出更理性的投资决策，避免被情绪驱动
- **谈判与定价**：损失厌恶在商业谈判、薪资协商、消费决策中无处不在

## 核心原理

### 前景理论中的价值函数

损失厌恶在数学上体现为价值函数在损失区域比收益区域更陡峭：

$$v(x) = \begin{cases} x^\alpha & x \geq 0 \\ -\lambda(-x)^\beta & x < 0 \end{cases}$$

其中 $\lambda$ 就是损失厌恶系数（Loss Aversion Coefficient）。卡尼曼和特沃斯基的原始估计 $\lambda \approx 2.25$，后续研究的估计范围在 1.5 到 2.75 之间。

这意味着：**1 元的损失在心理上的影响 ≈ 2.25 元的收益**。

### 神经科学证据

功能性磁共振成像（fMRI）研究表明：

- 预期损失会激活大脑的杏仁核（Amygdala）——与恐惧和威胁相关的区域
- 预期收益会激活腹侧纹状体（Ventral Striatum）——与奖赏相关的区域
- 损失区域的神经激活强度约为收益区域的 2 倍

这证明损失厌恶不是"想太多"，而是大脑的生理反应。

### 进化论解释

从进化角度看，损失厌恶是适应性特征：

- 在原始环境中，错过一次收益（如没找到食物）的代价通常小于一次损失（如被野兽攻击）
- 对损失的过度敏感帮助人类祖先在危险环境中生存下来
- 但在现代金融场景中，这种"出厂设置"可能导致次优决策

## 模型/方法详解

### 损失厌恶的测量方法

**1. 混合博弈法（Mixed Gamble Method）**

向受试者呈现一系列混合博弈：

- 50% 概率赢 $G$，50% 概率输 $L$
- 逐步调整 $G$，找到受试者愿意接受的最小 $G$
- 损失厌恶系数 $\lambda = G/L$

例如，如果某人需要至少赢 200 元才愿意承担输 100 元的风险，则 $\lambda = 200/100 = 2$。

**2. 确定性等值法（Certainty Equivalent Method）**

比较确定性损失与风险损失的偏好，间接估计 $\lambda$。

### 处置效应的数学模型

Odean（1998）定义了**实现盈亏比例**（Proportion of Gains Realized, PGR）和**实现亏损比例**（Proportion of Losses Realized, PLR）：

$$PGR = \frac{RG}{RG + UG}$$

$$PLR = \frac{RL}{RL + UL}$$

其中 RG = 已实现盈利交易数，UG = 未实现盈利交易数，RL = 已实现亏损交易数，UL = 未实现亏损交易数。

损失厌恶预测 PGR > PLR，即投资者更愿意实现盈利而非亏损。Odean 的实证结果证实了这一预测。

### 损失厌恶与股权溢价之谜

Mehra 和 Prescott（1985）发现，历史股票收益率比债券高出约 6%，远超传统模型可以解释的范围。Benartzi 和 Thaler（1995）提出**短视性损失厌恶**（Myopic Loss Aversion）解释：

- 投资者频繁评估投资组合（短视）
- 每次评估时，损失厌恶放大了对股票波动的恐惧
- 因此要求极高的风险溢价才愿意持有股票

## 主流方案对比

| 概念                              | 定义                       | 与损失厌恶的关系                                   |
| --------------------------------- | -------------------------- | -------------------------------------------------- |
| 风险厌恶（Risk Aversion）         | 对不确定性的偏好低于确定性 | 不同概念。风险厌恶关注波动性，损失厌恶关注亏损本身 |
| 损失厌恶（Loss Aversion）         | 对损失的敏感度高于收益     | 核心概念                                           |
| 后悔厌恶（Regret Aversion）       | 害怕做出错误决策后的后悔感 | 与损失厌恶相关但不同，后悔关注"如果当初…"          |
| 现状偏差（Status Quo Bias）       | 倾向于维持当前状态         | 损失厌恶的一种表现——改变可能带来损失               |
| 沉没成本谬误（Sunk Cost Fallacy） | 因已投入成本而继续投入     | 与损失厌恶相关——放弃意味着承认损失                 |

## 实战应用

### 克服损失厌恶的策略

1. **设定明确的止损规则**：在买入前就确定止损位，避免情绪化决策
2. **关注组合而非个股**：从整体组合角度评估表现，减少对单一亏损的关注
3. **拉长评估周期**：减少查看账户频率，降低短视性损失厌恶的影响
4. **重新框架**：将亏损视为"学费"或"保险成本"，而非纯粹的损失
5. **自动化投资**：使用定投、再平衡等规则化策略，减少主观判断

### 利用损失厌恶的产品设计

- **保本型基金**：承诺本金安全，迎合损失厌恶
- **目标日期基金**：默认选项设计，利用现状偏差
- **损失提醒**：在用户可能亏损时发送提醒，触发损失厌恶促进决策

### 在谈判中的应用

- 强调对方"不合作会失去什么"比"合作会得到什么"更有效
- 设置截止日期，制造"错过即损失"的紧迫感
- 先给出一个较高的"锚"，再让步，让对方感觉"赚到了"

## 关键要点

- 损失厌恶系数（$\lambda$）通常在 2.0 到 2.5 之间，即损失的心理影响是收益的 2-2.5 倍
- 损失厌恶不同于风险厌恶：风险厌恶是对不确定性的规避，损失厌恶是对"亏"本身的恐惧
- 处置效应是损失厌恶在投资中的直接体现
- 通过设定明确的投资纪律（如止损线、定投计划）可以部分克服损失厌恶
- 损失厌恶是进化形成的适应性特征，无法完全消除，只能管理

## 常见误区

- **误区一**："损失厌恶就是风险厌恶"——两者不同。风险厌恶是对不确定性的规避（即使期望值为正也不愿意赌），损失厌恶是对亏损的恐惧（亏损的痛苦大于等量收益的快乐）
- **误区二**："只要止损就能克服损失厌恶"——止损是工具，但执行止损本身就需要克服损失厌恶。知道该做什么和实际做到是两回事
- **误区三**："损失厌恶是缺点"——损失厌恶在进化中帮助人类规避危险，只是在现代金融场景中可能不利。在某些情境下（如风险管理），损失厌恶是有益的
- **误区四**："专业人士没有损失厌恶"——研究表明基金经理同样受损失厌恶影响，只是表现方式不同（如不愿承认错误投资、抱团取暖）
- **误区五**："损失厌恶系数对所有人都一样"——损失厌恶系数因文化、年龄、经验、性别而异。经验丰富的投资者系数可能更低

## 与其他概念的关系

- [前景理论](/finance-theory/prospect-theory) - 损失厌恶是前景理论的核心组成部分
- disposition-effect - 处置效应是损失厌恶在投资中的表现
- [行为金融学](/finance-theory/behavioral-finance) - 损失厌恶是行为金融学的基石概念之一
- [过度自信](/finance-theory/overconfidence) - 过度自信有时会与损失厌恶产生相反的效应（过度自信可能让人不愿止损）
- [锚定效应](/finance-theory/anchoring-effect) - 以买入价为锚，亏损时不愿卖出，是两种偏差的叠加
