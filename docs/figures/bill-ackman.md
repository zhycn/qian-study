# 比尔·阿克曼——激进投资者之王

> 阿克曼不是传统的投资者。他是**华尔街的角斗士**——用公开信、媒体战和股东提案来推动公司变革。他赢过，也输过，但从不无聊。

## 人物速写

比尔·阿克曼，潘兴广场资本管理（Pershing Square Capital Management）创始人，华尔街最著名的激进投资者。他以集中持仓、公开施压管理层和善于利用媒体闻名。

| 出生年 | 1962 年 5 月 11 日（纽约市） |
| 国籍 | 美国（犹太裔） |
| 核心贡献 | 激进投资、集中持仓、公开施压 |
| 代表基金 | 潘兴广场资本管理（Pershing Square） |
| 投资年限 | 20+ 年（2004 年至今） |
| 年化收益 | 约 14%（2004 年至今，扣除费用后） |
| 管理规模 | 峰值约 250 亿美元 |
| 个人财富 | 约 30 亿美元（2024 年福布斯） |
| 教育背景 | 哈佛大学 MBA |
| 经典战役 | 做空 MBIA、投资 Chipotle、Valeant 失败 |

## 传奇人生

### 早年经历：华尔街世家

1962 年，比尔·阿克曼出生于纽约市一个犹太家庭。他的父亲是一名房地产开发商，家境优渥。

阿克曼从小就展现出了对商业的兴趣。他在哈佛大学读本科时主修艺术史——一个与金融毫不相关的专业。他后来回忆：

> "我学艺术史是因为我喜欢艺术。但我同时也知道，我需要一份能赚钱的工作。"

本科毕业后，阿克曼进入哈佛大学商学院读 MBA。在哈佛商学院，他学到了两件事：
1. **如何分析公司**——案例教学法让他学会了深入分析商业模式
2. **如何谈判**——哈佛商学院的谈判课程让他学会了如何与管理层打交道

### 投资启蒙：从投行到对冲基金

1988 年，阿克曼毕业后加入了高盛（Goldman Sachs），担任投资银行分析师。他的工作是帮助公司进行并购和融资。

在高盛，阿克曼学到了华尔街的运作方式：
- **如何估值**——DCF、可比公司、先例交易
- **如何谈判**——与管理层、董事会、其他投资者
- **如何利用杠杆**——债务融资、衍生品、结构化产品

1988 年，阿克曼离开了高盛，加入了一家小型对冲基金——Ackman-Zitlow Investment Management。他的工作是做多和做空股票。

在这家基金，阿克曼第一次尝试了**激进投资**——买入一家公司的股票，然后推动管理层进行变革。

### 关键转折点：创立潘兴广场

2004 年，阿克曼做了一件改变人生的事——他创立了潘兴广场资本管理（Pershing Square Capital Management）。

"潘兴广场"这个名字来自纽约曼哈顿的一个地名。阿克曼选择这个名字是因为：
- 它听起来像一家老牌金融机构
- 它暗示着"广场"——一个公开、透明的地方
- 它与众不同，容易记住

潘兴广场的投资策略很独特：
- **极度集中**——通常只持有 8-12 只股票
- **激进投资**——买入后推动管理层变革
- **长期持有**——平均持有期 3-5 年
- **公开施压**——利用媒体和公开信向管理层施压

### 巅峰时刻：做空 MBIA

2007-2008 年，阿克曼做了一件让他名垂青史的事——**做空 MBIA，一家大型债券保险公司。**

**背景：**
- MBIA 是一家债券保险公司，主要为市政债券和企业债券提供保险
- 2000 年代，MBIA 开始为次贷相关证券提供保险
- 阿克曼通过研究发现，MBIA 的风险敞口被严重低估

**阿克曼的操作：**
- 2007 年初，开始建立 MBIA 空头头寸
- 发布公开报告，揭露 MBIA 的风险
- 利用媒体和公开信向监管层施压
- 逐步加大空头仓位

**结果：**
- 2008 年金融危机爆发，MBIA 濒临破产
- 阿克曼的空头头寸获利约 **数亿美元**
- 这笔交易成为潘兴广场最赚钱的交易之一

## 经典战役

### 战役一：做空 MBIA（2007-2008）

这是阿克曼最著名的战役，值得详细拆解。

**发现过程：**
- 2006 年，阿克曼开始研究债券保险行业
- 他发现 MBIA 和 Ambac 等公司为大量次贷相关证券提供保险
- 通过深入分析，他发现这些公司的风险敞口被严重低估
- 他判断一旦次贷市场出现问题，这些公司将面临巨额索赔

**公开报告：**
- 2007 年 2 月，阿克曼发布了一份长达 50 页的公开报告
- 报告详细分析了 MBIA 的风险敞口
- 报告指出 MBIA 的资本不足以覆盖潜在损失
- 报告呼吁监管层介入调查

**市场反应：**
- 报告发布后，MBIA 股价暴跌
- 监管层开始调查 MBIA
- MBIA 最终被迫承认风险敞口被低估
- 2008 年，MBIA 濒临破产，被政府救助

**关键数据：**

| 指标 | 数值 |
| --- | --- |
| 空头建仓时间 | 2007 年初 |
| MBIA 股价跌幅 | 从约 60 美元跌到不足 1 美元 |
| 潘兴广场获利 | 约数亿美元 |
| 持有时间 | 约 1-2 年 |

### 战役二：投资 Chipotle（2015-2018）

Chipotle 是阿克曼最成功的多头投资之一。

**背景：**
- 2015 年，Chipotle 遭遇食品安全危机
- 多起大肠杆菌和诺如病毒事件导致销售额大幅下滑
- 股价从约 750 美元跌到约 300 美元

**阿克曼的投资逻辑：**
- Chipotle 的商业模式依然优秀——快餐休闲（fast casual）
- 食品安全危机是暂时性的
- 公司正在改善食品安全流程
- 品牌依然强大，客户忠诚度依然高
- 估值已经反映了最坏情况

**操作：**
- 2015 年底，潘兴广场开始大量买入 Chipotle 股票
- 最终持仓占 Chipotle 约 10% 的股份
- 阿克曼加入 Chipotle 董事会，推动管理层变革

**结果：**
- Chipotle 逐渐恢复了销售额
- 股价从约 300 美元涨到超过 2000 美元（2024 年）
- 潘兴广场获利约 **数亿美元**

### 战役三：投资 Valeant（2014-2016）——失败案例

Valeant 是阿克曼最失败的投资，也是投资史上最著名的失败案例之一。

**背景：**
- Valeant 是一家加拿大制药公司，通过收购其他公司来增长
- CEO 迈克尔·皮尔逊（Michael Pearson）是一位激进的收购者
- Valeant 的股价在 2008-2015 年期间上涨了超过 20 倍

**阿克曼的投资逻辑：**
- Valeant 的收购策略有效，增长迅速
- 制药行业有强大的定价权
- 皮尔逊是一位优秀的 CEO
- 估值合理，有上涨空间

**操作：**
- 2014 年，潘兴广场开始大量买入 Valeant 股票
- 最终持仓占 Valeant 约 10% 的股份
- 阿克曼加入 Valeant 董事会

**崩盘：**
- 2015 年，Valeant 被揭露通过大幅提价来增长
- 引发政治和公众强烈反弹
- 公司被调查，CEO 皮尔逊被迫辞职
- 股价从约 250 美元暴跌到不足 20 美元

**结果：**
- 潘兴广场在 Valeant 上亏损约 **30 亿美元**
- 这是潘兴广场历史上最大的单笔亏损
- 阿克曼公开承认错误，向投资者道歉

**反思：**
阿克曼后来反思：

> "我在 Valeant 上犯了一个错误——我太相信 CEO 了。我没有充分调查公司的商业模式是否可持续。这是一个痛苦的教训。"

| 指标 | 数值 |
| --- | --- |
| 买入时间 | 2014 年 |
| 买入价格 | 约 100-150 美元 |
| 最低价格 | 不足 20 美元（2016 年） |
| 亏损金额 | 约 30 亿美元 |
| 亏损比例 | 约 80-90% |

### 战役四：投资 Howard Hughes Corporation（2010 至今）

Howard Hughes Corporation 是阿克曼持有时间最长的投资之一。

**背景：**
- Howard Hughes Corporation 是一家房地产开发公司
- 公司拥有大量未开发的土地储备
- 市场低估了这些土地的价值

**阿克曼的投资逻辑：**
- 公司的土地储备价值远超市值
- 管理层正在逐步开发这些土地
- 随着开发推进，价值将逐步释放
- 估值便宜，有上涨空间

**操作：**
- 2010 年，潘兴广场开始大量买入 Howard Hughes 股票
- 阿克曼加入董事会，推动管理层变革
- 持续持有超过 10 年

**结果：**
- Howard Hughes 股价从约 30 美元涨到超过 200 美元
- 潘兴广场获利约 **数亿美元**
- 这是阿克曼最成功的长期投资之一

### 战役五：2020 年新冠疫情交易

2020 年初，新冠疫情爆发。阿克曼做了一笔经典的宏观交易。

**操作：**
- 2020 年 2 月，阿克曼开始购买信用违约互换（CDS）
- CDS 是一种保险，如果债券违约，CDS 持有者获得赔付
- 阿克曼花费约 2700 万美元购买 CDS

**结果：**
- 2020 年 3 月，市场暴跌，CDS 价值飙升
- 阿克曼的 CDS 头寸价值达到约 26 亿美元
- 回报率超过 **100 倍**
- 阿克曼随后平仓，将获利分配给投资者

他在给投资者的信中说：

> "这笔交易是我们历史上最赚钱的交易之一。但更重要的是，它保护了我们的投资组合免受市场暴跌的影响。"

| 购买成本 | 约 2700 万美元 |
| 峰值价值 | 约 26 亿美元 |
| 回报率 | 超过 100 倍 |
| 持有时间 | 约 1 个月 |

## 投资哲学

### 原则一：集中持仓

阿克曼是集中持仓的终极信徒。

> "如果你最好的想法只占组合的 5%，那你的想法就不够好。"

他的集中持仓原则：
- **通常只持有 8-12 只股票**
- **单只股票可以占到 10-20% 的仓位**
- **看对的时候下重注**

### 原则二：激进投资

阿克曼是激进投资的代表人物。

> "买入股票后，你不应该被动等待。你应该积极推动管理层进行变革。"

他的激进投资方法：
- **公开信**——发布公开信向管理层施压
- **媒体战**——利用媒体曝光公司问题
- **股东提案**——在股东大会上提出变革提案
- **董事会席位**——争取董事会席位，直接参与决策

### 原则三：深入研究

阿克曼是深入研究的信徒。

> "你必须比管理层更了解他们的公司。"

他的研究方法：
- **阅读财报**——详细分析收入、利润、现金流、负债
- **调研行业**——了解竞争格局、行业趋势
- **与管理层交流**——评估能力、诚信、激励机制
- **估值**——用多种方法评估内在价值

### 原则四：长期持有

阿克曼是长期持有的信徒。

> "我们不是交易员。我们是投资者。我们的持有期是 3-5 年，甚至更长。"

他的长期持有逻辑：
- 价值释放需要时间
- 管理层变革需要时间
- 频繁交易产生摩擦成本

### 原则五：透明沟通

阿克曼是透明沟通的信徒。

> "投资者有权知道我们在做什么，为什么这么做。"

他的透明沟通体现在：
- **季度信**——每季度向投资者发送详细信件
- **公开报告**——发布公开报告分析公司
- **媒体采访**——接受媒体采访解释投资逻辑

## 策略拆解

### 选股标准

阿克曼的选股标准：

| 标准 | 具体要求 |
| --- | --- |
| 好业务 | 商业模式清晰，有竞争优势 |
| 好管理 | 管理层有能力（但可能需要推动变革） |
| 好财务 | 高 ROE、低负债、稳定现金流 |
| 好估值 | 价格低于内在价值 |
| 好催化剂 | 有推动价值回归的催化剂 |

### 买入时机

阿克曼的买入时机：

1. **公司被市场错误定价**——价格远低于内在价值
2. **有催化剂**——新管理层、行业变化、监管变化
3. **可以推动变革**——有机会通过激进投资创造价值

### 持仓管理

阿克曼的持仓管理：

- **极度集中**——8-12 只股票
- **积极推动**——推动管理层变革
- **长期持有**——3-5 年
- **定期沟通**——向投资者透明沟通

### 卖出纪律

阿克曼的卖出条件：

1. **目标达成**——价格达到内在价值
2. **逻辑被证伪**——基本面恶化
3. **发现更好的机会**——换仓

### 普通人能学到什么

阿克曼的方法对普通投资者的启示：

1. **集中持仓**——如果你最好的想法只占 5%，那想法就不够好
2. **积极研究**——你必须比市场更了解你投资的公司
3. **长期持有**——价值释放需要时间
4. **透明沟通**——向你的投资伙伴透明沟通
5. **承认错误**——错了就认错，不要死扛

**但要注意：** 阿克曼的激进投资方法需要大量资源、专业知识和影响力。普通投资者**不应该模仿他的具体操作**，而应该学习他的研究方法和思维方式。
