# 本杰明·格雷厄姆——价值投资之父

> 如果只能用一句话概括价值投资，那就是格雷厄姆说过的：**用 50 美分买价值 1 美元的东西。**

## 人物速写

本杰明·格雷厄姆，价值投资理论奠基人，巴菲特口中的"一生中最重要的人"。他用一套可量化、可复制的投资框架，把投资从赌博变成了一门科学。

| 维度 | 数据 |
| --- | --- |
| 生卒年 | 1894-1976（享年 82 岁） |
| 国籍 | 美国（生于伦敦，犹太裔） |
| 核心贡献 | 价值投资理论、安全边际、市场先生概念 |
| 代表著作 | 《证券分析》（1934）、《聪明的投资者》（1949） |
| 投资年限 | 约 30 年（1915-1956，管理格雷厄姆-纽曼基金） |
| 年化收益 | 格雷厄姆-纽曼基金约 20%（1936-1956） |
| 管理规模 | 峰值约 250 万美元（1950 年代） |
| 得意门生 | 沃伦·巴菲特、沃尔特·施洛斯、欧文·卡恩 |

## 传奇人生

### 早年经历：从天堂跌入地狱

1894 年 5 月 9 日，本杰明·格雷厄姆出生于伦敦一个犹太家庭。他原名本杰明·格罗斯鲍姆（Benjamin Grossbaum），后来为了避开一战期间对德国姓氏的敌意，才改名为格雷厄姆。

1 岁时，全家移民纽约。童年时代的格雷厄姆家境优渥——父亲是进口商，母亲来自富裕家庭。他在曼哈顿上东区长大，就读于最好的公立学校，看起来前途无量。

然而，命运在他 8 岁那年急转直下。1903 年，父亲因病去世，家道中落。更糟糕的是，母亲把全部家当投入股市，结果在 1907 年恐慌中亏得所剩无几。一家人从宽敞的公寓搬到了布鲁克林的廉租房，靠亲戚接济度日。

这段经历给年幼的格雷厄姆留下了两个深刻印记：**一是对股市的敬畏，二是对财务安全的极度渴望。** 这两点后来成为他投资哲学的核心。

### 求学与启蒙：天才少年的选择

格雷厄姆是个不折不扣的天才。高中毕业后，他获得了哥伦比亚大学的奖学金。校长希望他攻读数学或哲学——毕竟他在这两门学科上展现出了非凡的天赋。

但格雷厄姆做了一个让所有人意外的决定：**去商学院。**

他的理由很现实——家里穷，需要尽快赚钱。1912 年，18 岁的格雷厄姆进入哥伦比亚大学，主修数学，同时选修了商学院的课程。他成绩优异，毕业时年仅 20 岁。

毕业后，他面临三个工作机会：
- 华尔街投行，年薪 12 美元/周
- 证券公司，年薪 14 美元/周
- 一家出版公司，年薪 18 美元/周

他选择了第二家——纽伯格·亨德森·洛伯公司（Newburger, Henderson & Loeb），因为这份工作能让他接触证券市场。入职后，他从最底层的统计员做起，每天的工作就是整理和分析公司的财务数据。

**正是这份枯燥的工作，让他练就了阅读财务报表的硬功夫。**

### 第一次重大打击：1929 年大崩盘

1920 年代，格雷厄姆的事业蒸蒸日上。1923 年，他成立了自己的基金公司——格雷厄姆公司（Graham & Company），起步资金 50 万美元。到 1929 年，基金规模已经增长到 250 万美元，格雷厄姆个人净资产接近 50 万美元——在当年，这是不折不扣的巨富。

然而，1929 年 10 月的股灾改变了一切。

格雷厄姆的投资策略在当时看来很"先进"——他会用保证金（杠杆）买入被低估的股票。理论上，这些股票确实便宜，但杠杆放大了风险。当市场崩盘时，他遭遇了致命的打击：

- 1929-1932 年，他的基金亏损了约 **70%**
- 个人净资产从 50 万美元暴跌到几乎清零
- 一度濒临破产，靠朋友借款维持生活

多年后，格雷厄姆在回忆这段经历时说：

> "我经历了从富有到贫穷的完整过程。这让我明白了一个道理：**即使你的分析是对的，如果市场继续非理性，你照样会破产。**"

这次经历直接催生了他后来最著名的概念——**安全边际**。他意识到，仅仅找到便宜货是不够的，你必须留出足够的安全空间，以应对市场的非理性和不可预见的风险。

### 巅峰时刻：格雷厄姆-纽曼基金

1936 年，格雷厄姆与杰瑞·纽曼（Jerry Newman）合伙成立了格雷厄姆-纽曼公司（Graham-Newman Corp.）。这一次，他的投资策略更加成熟和保守：

- **不再使用杠杆**
- **坚持安全边际原则**
- **专注于套利和清算类投资**
- **分散投资，不押注单一标的**

结果如何？从 1936 年到 1956 年关闭基金，格雷厄姆-纽曼基金的年化收益率约为 **20%**，而同期标普 500 的年化收益率约为 12%。

更重要的是，这 20 年的收益曲线远比市场平稳——最大回撤远小于市场平均水平。这正是格雷厄姆追求的：**不追求最高的收益，而追求最可靠的收益。**

1956 年，62 岁的格雷厄姆决定关闭基金，退休去加州大学洛杉矶分校任教。他给出的理由是："我不想把晚年建立在运气之上。"

### 与巴菲特的师徒情缘

1950 年，20 岁的沃伦·巴菲特在哥伦比亚大学商学院读研究生，他的论文题目是"我最崇拜的投资大师"——答案显而易见，是本杰明·格雷厄姆。

巴菲特读了格雷厄姆的《聪明的投资者》后，深受震撼。他后来回忆说：

> "读到这本书的那一刻，我就像找到了光明。之前我所做的一切投资，突然有了意义。"

巴菲特主动给格雷厄姆写信，希望能到他的公司工作。格雷厄姆起初拒绝了——因为格雷厄姆-纽曼公司从不雇佣犹太人以外的员工（当时的行业潜规则），而巴菲特不是犹太人。

但巴菲特没有放弃。1951 年毕业后，他免费在格雷厄姆的公司工作了几个月。1954 年，格雷厄姆终于正式雇佣了巴菲特，年薪 1.2 万美元。

在格雷厄姆-纽曼公司工作的两年，是巴菲特投资生涯最重要的启蒙期。他后来评价：

> "格雷厄姆教给我的，不是如何赚钱，而是如何不亏钱。**安全边际**这个概念，是我一生投资的核心。"

## 经典战役

### 战役一：GEICO——改变巴菲特一生的投资

1948 年，格雷厄姆-纽曼基金做了一笔后来被证明极其伟大的投资——买入政府雇员保险公司（GEICO）的股票。

GEICO 当时是一家小型汽车保险公司，采用直销模式（跳过代理人），成本结构远优于同行。格雷厄姆通过细致的财务分析发现：

- GEICO 的每股净资产约为 70 美元
- 但股价仅为 3-4 美元
- 公司成立以来从未亏损，且增长迅速

格雷厄姆-纽曼基金以约 72 万美元的价格买入了 GEICO 约 33% 的股份。这笔投资后来涨到了数千万美元，成为基金历史上最赚钱的单笔投资。

更重要的是，**正是这笔投资让巴菲特注意到了 GEICO。** 几十年后，伯克希尔·哈撒韦在 1996 年全资收购了 GEICO，这笔投资最终为巴菲特带来了超过 200 亿美元的回报。

可以说，没有格雷厄姆当年对 GEICO 的发现，就没有后来巴菲特的这笔经典投资。

**关键数据：**

| 指标 | 数值 |
| --- | --- |
| 买入时间 | 1948 年 |
| 买入价格 | 约 3-4 美元/股 |
| 买入金额 | 约 72 万美元 |
| 持仓比例 | 约 33% |
| 最终价值 | 数千万美元（基金存续期间） |
| 回报率 | 超过 100 倍 |

### 战役二：北方管道公司——套利投资的经典

北方管道公司（Northern Pipe Line）是洛克菲勒标准石油公司拆分后的一家子公司。1926 年，格雷厄姆注意到一个有趣的现象：

北方管道公司的股票交易价格约为 65 美元/股，但公司账上持有的铁路债券（流动性极高、风险极低）价值就高达 95 美元/股。换句话说，**市场不仅白送了整个管道业务，还倒贴了 30 美元。**

格雷厄姆大量买入北方管道公司股票，然后联合其他股东向管理层施压，要求公司返还现金。最终，公司同意向股东返还每股 70 美元的现金。

这笔投资的逻辑非常格雷厄姆式：
- **下行风险极低**——即使管理层不妥协，债券价值也提供了安全垫
- **上行空间可观**——如果管理层妥协，可以获得丰厚回报
- **时间可控**——施压过程大约持续了一年

**关键数据：**

| 指标 | 数值 |
| --- | --- |
| 买入价格 | 约 65 美元/股 |
| 内在价值（仅债券） | 约 95 美元/股 |
| 最终返还现金 | 70 美元/股 + 保留股权 |
| 持有时间 | 约 1 年 |
| 回报率 | 约 100%+（含保留股权价值） |

### 战役三：洛克伍德公司——可可豆套利

1954 年，格雷厄姆-纽曼基金参与了洛克伍德公司（Rockwood & Co.）的可可豆套利交易。

背景是这样的：当时美国政府对可可豆实行配给制，但有一个例外——如果公司用可可豆换取可可脂，然后把可可脂卖给政府，就可以不受配给限制。

洛克伍德公司发现了一个套利机会：
- 市场上可可豆价格约 5 美分/磅
- 用可可豆换可可脂后，可可脂价值约 24 美分/磅
- 公司决定用可可脂换取股东手中的股票（相当于回购）

格雷厄姆-纽曼基金大量买入洛克伍德公司股票，然后用股票换取可可脂，再在市场上卖出可可脂获利。

这笔交易的本质是**利用政府管制政策造成的价格扭曲进行套利**——风险低、回报确定、时间可控，是格雷厄姆最喜欢的投资类型。

**关键数据：**

| 指标 | 数值 |
| --- | --- |
| 套利逻辑 | 可可豆与可可脂的价差 |
| 风险水平 | 极低（政府政策保障） |
| 持有时间 | 数月 |
| 回报率 | 约 50%+ |

### 战役四：1930 年代的"烟蒂投资"

1930 年代大萧条期间，格雷厄姆发现了一批价格低得离谱的股票——有些公司的市值甚至低于其持有的现金和流动资产价值。他把这类投资戏称为"烟蒂投资"（Cigar Butt Investing）：就像在地上捡到还能抽一口的烟蒂，虽然不雅观，但确实能赚到钱。

典型案例包括：

- **联合电车公司**：股价低于公司持有的现金
- **密苏里太平洋铁路**：破产重组过程中，债券价格远低于清算价值
- **多家封闭式基金**：交易价格低于净资产 50% 以上

这类投资的共同特征是：
1. 价格远低于清算价值
2. 公司资产以现金或高流动性资产为主
3. 市场极度悲观，但基本面并未完全崩溃

格雷厄姆后来在《证券分析》中详细记录了这类投资的逻辑和方法，成为价值投资的经典案例。

## 投资哲学

### 原则一：安全边际（Margin of Safety）

这是格雷厄姆投资哲学的核心，也是他一生最重要的贡献。

**安全边际的含义很简单：用 50 美分买价值 1 美元的东西。**

但背后的逻辑极其深刻：

1. **你的估值可能是错的**——即使你做了最仔细的分析
2. **未来是不可预测的**——黑天鹅事件随时可能发生
3. **市场是非理性的**——价格可能长期偏离价值

安全边际就是为这些不确定性留出缓冲空间。格雷厄姆认为，**安全边际越大，投资风险越小，预期回报越高**——这看似矛盾，却是价值投资的精髓。

> "安全边际的功能，就是让精确的估算变得不再必要。"——《聪明的投资者》

### 原则二：市场先生（Mr. Market）

格雷厄姆在《聪明的投资者》中提出了一个著名的寓言：

想象你和一个叫"市场先生"的人合伙做生意。每天，市场先生都会给你一个报价，要么买你的股份，要么卖给你他的股份。

市场先生有个特点：**他的情绪极不稳定。** 有时候他兴高采烈，报价很高；有时候他极度悲观，报价很低。

格雷厄姆的建议是：**利用市场先生，而不是被他利用。**

- 当他报价过高时，卖给他
- 当他报价过低时，从他手里买
- 其他时候，忽略他

这个寓言的核心启示是：**市场是为你服务的，不是来指导你的。** 价格波动不是风险，而是机会。

### 原则三：投资与投机的区别

格雷厄姆对投资和投机做了明确区分：

> "投资操作是基于透彻的分析，确保本金的安全并获得满意的回报。不符合这些条件的操作就是投机。"

他提出了三个标准来判断一项操作是投资还是投机：
1. **是否有透彻的分析**——不是凭感觉或跟风
2. **是否确保本金安全**——有安全边际
3. **是否追求满意的回报**——不是不切实际的暴利

这个区分在今天依然适用。格雷厄姆认为，大多数人所谓的"投资"，其实都是投机。

### 原则四：分散投资

格雷厄姆是分散投资的坚定支持者。他的建议是：

- **持有 10-30 只股票**，不要把所有鸡蛋放在一个篮子里
- **单只股票不超过总资产的 5%**
- **跨行业、跨市场分散**

他承认，分散投资会降低最高回报，但更重要的是**降低最坏情况下的损失**。对于格雷厄姆来说，投资的首要目标是不亏钱，而不是赚大钱。

### 原则五：能力圈

虽然"能力圈"这个概念后来被巴菲特发扬光大，但它的源头可以追溯到格雷厄姆。

格雷厄姆的投资方法有一个重要前提：**只投资你能理解的东西。** 他专注于财务分析，只买那些能通过数字验证其价值的公司。对于看不懂的商业模式、讲不出清晰财务故事的公司，他一律回避。

## 策略拆解

### 选股标准

格雷厄姆的选股标准非常量化，他提出了著名的"格雷厄姆筛选法"：

| 标准 | 具体要求 |
| --- | --- |
| 市盈率（P/E） | 低于 15 倍 |
| 市净率（P/B） | 低于 1.5 倍 |
| P/E × P/B | 低于 22.5 |
| 流动比率 | 大于 2（流动资产/流动负债） |
| 长期债务 | 不超过净流动资产 |
| 每股收益增长 | 过去 10 年至少增长 1/3 |
| 分红记录 | 连续 20 年分红 |
| 盈利记录 | 过去 10 年无亏损 |

这些标准的核心逻辑是：**寻找财务稳健、估值便宜的公司。** 格雷厄姆不关心公司有多"性感"，他只关心数字是否足够安全。

### 买入时机

格雷厄姆的买入时机判断基于两个维度：

1. **个股层面**：当股价低于内在价值（通常用净资产或清算价值衡量）足够多时买入
2. **市场层面**：当市场整体估值偏低时（如盈利收益率远高于债券收益率时）加大投入

他特别强调：**不要试图预测市场短期走势。** 你只需要知道当前价格是否足够便宜，至于市场什么时候"发现"这个便宜，不重要。

### 持仓管理

格雷厄姆的持仓管理原则：

- **买入后不频繁交易**——给市场时间发现价值
- **定期复查持仓**——确认基本面没有恶化
- **不追加投资**——除非价格进一步下跌且基本面不变
- **保持耐心**——价值回归可能需要 2-3 年甚至更久

### 卖出纪律

格雷厄姆的卖出条件非常明确：

1. **价格达到内在价值**——获利了结，不贪心
2. **基本面恶化**——安全边际消失，果断卖出
3. **发现更好的机会**——换仓到性价比更高的标的
4. **持有时间过长（3-5 年）仍未回归**——重新评估逻辑是否正确

他特别强调：**不要因为股价上涨就卖出，也不要因为股价下跌就卖出。** 卖出的唯一理由是价值与价格的关系发生了变化。

### 普通人能学到什么

格雷厄姆的方法对普通投资者特别友好，因为：

1. **不需要内幕消息**——所有数据都是公开的财务信息
2. **不需要预测未来**——只需要评估当前价格是否便宜
3. **不需要频繁交易**——买入后持有，等待价值回归
4. **可以系统化执行**——格雷厄姆筛选法可以用 Excel 或股票筛选器实现

**具体建议：**
