# 核心-卫星策略

核心-卫星策略（Core-Satellite Strategy）是"稳中求进"的投资方法：把 70%-80% 的钱买宽基指数基金长期持有（核心），20%-30% 的钱投资行业基金或个股追求超额收益（卫星）。核心部分保证你不跑输市场，卫星部分让你有机会跑赢市场。就像太阳系——太阳稳稳地发光发热，卫星们各显神通。

## 概述

核心-卫星策略将投资组合分为两部分：

**核心（Core）**：70%-80%

- 宽基指数基金（如沪深 300、标普 500）
- 长期持有，低换手率
- 目标：获取市场平均收益（Beta）
- 特点：低成本、分散化、稳健

**卫星（Satellite）**：20%-30%

- 行业基金、主题基金、个股
- 灵活调整，较高换手率
- 目标：获取超额收益（Alpha）
- 特点：高弹性、集中化、进取

$$组合收益 = 核心收益 \times 核心权重 + 卫星收益 \times 卫星权重$$

## 为什么这样设计

### 传统投资的困境

| 策略     | 问题                                       |
| -------- | ------------------------------------------ |
| 全指数   | 稳健但缺乏超额收益，投资者觉得"不过瘾"     |
| 全主动   | 追求超额收益，但大多数主动基金跑输指数     |
| 频繁交易 | 试图择时选股，但交易成本和错误决策侵蚀收益 |

### 核心-卫星的解决方案

- **核心部分**：承认"我可能选不准股票"，用指数基金获取市场收益
- **卫星部分**：满足"我想试试手气"，用少量资金追求超额收益
- **心理优势**：核心部分保底，卫星部分亏了不影响大局，赢了锦上添花

## 核心部分构建

### 宽基指数选择

| 市场 | 指数      | 特点               | 代表基金     |
| ---- | --------- | ------------------ | ------------ |
| A 股 | 沪深 300  | 大盘蓝筹，代表性强 | 沪深 300ETF  |
| A 股 | 中证 500  | 中盘成长，弹性好   | 中证 500ETF  |
| A 股 | 中证 800  | 大中盘综合         | 中证 800ETF  |
| 美股 | 标普 500  | 全球标杆           | 标普 500ETF  |
| 港股 | 恒生指数  | 港股代表           | 恒生 ETF     |
| 全球 | MSCI 全球 | 全球分散           | 全球股票 ETF |

### 核心配置建议

| 投资者类型 | 核心比例 | 推荐配置                                                   |
| ---------- | -------- | ---------------------------------------------------------- |
| 保守型     | 90%      | 沪深 300（50%）+ 债券指数（40%）                           |
| 稳健型     | 80%      | 沪深 300（40%）+ 中证 500（20%）+ 债券（20%）              |
| 平衡型     | 70%      | 沪深 300（30%）+ 中证 500（20%）+ 港股（10%）+ 债券（10%） |
| 积极型     | 60%      | 沪深 300（25%）+ 中证 500（20%）+ 美股（10%）+ 债券（5%）  |

## 卫星部分构建

### 卫星投资方向

| 类型     | 例子                  | 风险 | 预期超额收益 |
| -------- | --------------------- | ---- | ------------ |
| 行业基金 | 新能源、医药、半导体  | 中高 | 5%-15%       |
| 主题基金 | ESG、人工智能、碳中和 | 中高 | 5%-20%       |
| 个股     | 精选成长股            | 高   | 10%-50%+     |
| 另类资产 | REITs、商品、加密货币 | 中高 | 5%-20%       |
| 海外市场 | 美股、港股、新兴市场  | 中   | 3%-10%       |

### 卫星选择原则

- **与核心低相关**：卫星投资应与核心指数低相关，才能真正分散风险
- **有逻辑支撑**：不是"听消息"，而是有基本面或趋势逻辑
- **控制集中度**：单只卫星不超过总组合的 5%
- **定期评估**：每季度评估卫星表现，表现不佳的及时替换

## 生活化类比

核心-卫星策略就像"做饭"。

核心部分是"主食"——米饭或面条。它保证你吃饱（获得市场收益），虽然不惊艳，但不可或缺。

卫星部分是"菜"——红烧肉、清蒸鱼、炒蔬菜。它让饭好吃（追求超额收益），有滋有味。

只有主食没有菜 = 能活但没意思（全指数，缺乏超额收益）。
只有菜没有主食 = 好吃但不顶饱（全主动，风险太大）。
主食 + 菜 = 既吃饱又吃好（核心-卫星策略）。

## 实施步骤

### 步骤 1：确定核心-卫星比例

根据风险偏好和投资经验：

| 经验水平   | 核心比例 | 卫星比例 | 说明                     |
| ---------- | -------- | -------- | ------------------------ |
| 新手       | 90%      | 10%      | 先熟悉市场，卫星部分试水 |
| 有一定经验 | 80%      | 20%      | 核心为主，卫星适度进取   |
| 经验丰富   | 70%      | 30%      | 卫星部分可更灵活         |
| 专业投资者 | 60%      | 40%      | 卫星部分可加大配置       |

### 步骤 2：选择核心资产

- 选择 1-3 只宽基指数基金
- 关注费率（越低越好）
- 关注规模（越大流动性越好）
- 关注跟踪误差（越小越好）

### 步骤 3：选择卫星资产

- 根据市场热点和个人判断选择 3-5 只卫星
- 分散到不同行业/主题，避免过度集中
- 设定止损和止盈点

### 步骤 4：再平衡

| 再平衡方式 | 操作方法            | 频率        |
| ---------- | ------------------- | ----------- |
| 定期再平衡 | 恢复目标比例        | 每半年/每年 |
| 阈值再平衡 | 偏离超过 ±5% 时调整 | 触发时      |
| 卫星轮换   | 替换表现不佳的卫星  | 每季度      |

## 实战应用

### 示例组合

以 100 万资金为例：

**核心部分（80 万）**：

- 沪深 300ETF：40 万（40%）
- 中证 500ETF：20 万（20%）
- 纯债基金：20 万（20%）

**卫星部分（20 万）**：

- 新能源 ETF：5 万（5%）
- 半导体 ETF：5 万（5%）
- 医药 ETF：5 万（5%）
- 美股 QDII：5 万（5%）

### 中国投资者特别提示

- **ETF 选择**：A 股 ETF 费率低、流动性好，是核心的理想选择
- **行业轮动**：A 股行业轮动快，卫星部分需灵活调整
- **打新**：A 股打新是低风险超额收益来源，可作为卫星补充
- **港股通**：通过港股通配置港股，丰富卫星选择
- **税收优惠**：个人投资者买卖 ETF 免征印花税

## 关键要点

- 核心-卫星策略 = 宽基指数（70%-80%）+ 行业/个股（20%-30%）
- 核心部分获取市场收益（Beta），卫星部分追求超额收益（Alpha）
- 核心部分低成本、分散化、长期持有
- 卫星部分灵活调整，但需控制风险
- 定期再平衡维持目标比例
- 适合既想稳健又想进取的投资者

## 常见误区

- **"核心部分收益太低"**——核心部分是基础，卫星部分才是超额收益来源
- **"卫星部分可以随便买"**——卫星部分也需要逻辑和纪律，不是赌博
- **"核心-卫星就是分散投资"**——关键是核心和卫星的不同定位，不是简单分散
- **"再平衡就是卖涨买跌"**——再平衡是维持目标配置，不是择时
