# 现金流量表

现金流量表就像公司的"银行流水账单"——利润可以靠会计技巧调节，但银行账户里的数字很难造假，它告诉你公司到底收了多少钱、花了多少钱、还剩多少钱。

## 概述

现金流量表（Cash Flow Statement）追踪的是公司在**一段时间内**现金的实际变动。它与 [利润表](/posts/income-statement) 形成鲜明对比——利润表基于权责发生制，包含大量非现金项目；而现金流量表基于收付实现制，只记录真金白银的进出。

三大板块：

- **经营活动现金流**（Cash Flow from Operations, CFO）：日常业务产生的现金
- **投资活动现金流**（Cash Flow from Investing, CFI）：买卖长期资产和投资产生的现金
- **筹资活动现金流**（Cash Flow from Financing, CFF）：融资和股东回报相关的现金

核心公式：

$$
\text{期末现金} = \text{期初现金} + \text{CFO} + \text{CFI} + \text{CFF}
$$

这个公式看似简单，但它揭示了一个深刻的事实：公司现金的变化，只能来自经营、投资和筹资这三件事。

## 为什么重要

现金流量表被很多专业投资者视为"最不容易造假"的报表。原因很简单：

- 利润可以通过会计估计和判断来调节
- 但银行账户里的数字很难伪造

现金流量表能回答几个生死攸关的问题：

- 公司赚的利润变成真金白银了吗？还是只是一堆应收账款？
- 公司靠什么活着？是业务本身赚钱，还是靠借钱续命？
- 公司有没有足够的现金支付工资、还债、分红？

历史上无数公司账面盈利但最终破产——安然（Enron，2001）（来源：SEC 调查报告，2002）、雷曼兄弟（Lehman Brothers，2008）（来源：美国破产法院文件，2008）——都是因为现金流断裂。利润是观点，现金是事实。

## 核心原理

### 经营活动现金流：间接法 vs 直接法

**间接法**（Indirect Method）是最常用的编制方式，从净利润出发，逐项调整非现金项目：

$$
\begin{aligned}
\text{经营现金流} = &\text{净利润} \\
&+ \text{折旧和摊销（非现金费用，加回）} \\
&+ \text{资产减值损失（非现金，加回）} \\
&- \text{应收账款增加（收入已确认但现金未收到，扣减）} \\
&- \text{存货增加（现金已花但尚未形成费用，扣减）} \\
&+ \text{应付账款增加（费用已确认但现金未支付，加回）} \\
&\pm \text{其他营运资本变动}
\end{aligned}
$$

**直接法**（Direct Method）则直接列示现金收支的类别：

- 从客户收到的现金
- 支付给供应商的现金
- 支付给员工的现金
- 支付的各项税费

直接法更直观，但编制成本高，实务中使用较少。

### 自由现金流（Free Cash Flow, FCF）

自由现金流是现金流量表最重要的衍生指标：

$$
\text{自由现金流} = \text{经营现金流} - \text{资本支出（CAPEX）}
$$

自由现金流代表公司在维持正常运营和必要投资后，真正可以"自由支配"的现金。它是 [DCF 估值](/posts/dcf) 的核心输入，也是判断公司财务健康度的黄金指标。

更细致的分类：

- **FCFF**（Free Cash Flow to Firm）：企业自由现金流，归属于所有资本提供者（股东 + 债权人）

$$
\text{FCFF} = \text{EBIT} \times (1 - T_c) + \text{折旧摊销} - \text{CAPEX} - \Delta \text{营运资本}
$$

- **FCFE**（Free Cash Flow to Equity）：股权自由现金流，仅归属于股东

$$
\text{FCFE} = \text{FCFF} - \text{利息} \times (1 - T_c) + \text{净借款}
$$

### 现金转换周期（Cash Conversion Cycle, CCC）

衡量公司从花出现金到收回现金需要多长时间：

$$
\text{CCC} = \text{DSO} + \text{DIO} - \text{DPO}
$$

其中：

- **DSO**（Days Sales Outstanding）= 应收账款 / 日均收入，收款天数
- **DIO**（Days Inventory Outstanding）= 存货 / 日均营业成本，存货周转天数
- **DPO**（Days Payable Outstanding）= 应付账款 / 日均营业成本，付款天数

CCC 越短，说明公司现金周转越快，经营效率越高。亚马逊的 CCC 甚至是负数（约 -30 天）（来源：Amazon 10-K 年报，SEC 文件，2024）——它先收到客户的钱，很久之后才付给供应商，相当于用供应商的钱在做生意。

## 方法/类型详解

### 现金流量表的八种典型组合

经营、投资、筹资三项现金流的正负组合，可以判断公司处于什么阶段：

| CFO | CFI | CFF | 典型阶段 | 解读                                   |
| --- | --- | --- | -------- | -------------------------------------- |
| +   | -   | +   | 成长期   | 业务赚钱，大量投资扩张，同时外部融资   |
| +   | -   | -   | 成熟期   | 业务赚钱，继续投资，同时还款或分红     |
| +   | +   | -   | 收获期   | 业务赚钱，变卖资产，还款或分红         |
| +   | +   | +   | 特殊     | 业务赚钱，变卖资产，还在融资——需警惕   |
| -   | -   | +   | 创业期   | 业务烧钱，投资扩张，靠融资续命         |
| -   | +   | +   | 困境     | 业务烧钱，变卖资产，靠融资维持         |
| -   | -   | -   | 危机     | 三项全负，现金快速消耗，极度危险       |
| -   | +   | -   | 衰退期   | 业务烧钱，变卖资产，同时还款——不可持续 |

### 现金流质量分析框架

判断现金流质量的核心维度：

1. **CFO / 净利润比率**：持续大于 1 说明利润质量高，现金转化能力强
2. **CFO 的持续性**：是否依赖一次性因素（如退税、政府补助）
3. **CAPEX 的合理性**：资本支出是否与收入增长匹配
4. **FCF 的稳定性**：自由现金流是否持续为正且稳定增长

## 主流方案对比

### 经营现金流 vs 净利润

| 维度     | 净利润                 | 经营现金流             |
| -------- | ---------------------- | ---------------------- |
| 会计基础 | 权责发生制             | 收付实现制             |
| 包含折旧 | 扣减折旧               | 加回折旧（非现金）     |
| 应收账款 | 确认收入即计入         | 实际收到才计入         |
| 存货     | 销售时才计入成本       | 购买时就扣减现金       |
| 操纵难度 | 较高（会计估计空间大） | 较低（银行流水可验证） |

### 不同行业的现金流特征

- **科技/SaaS**：前期 CFO 可能为负（获客成本高），但边际成本极低，规模化后 FCF 爆发
- **制造业**：CAPEX 巨大，CFO 稳定但 FCF 受投资周期影响大
- **零售业**：CCC 短甚至为负，现金流周转快，CFO 通常强劲
- **房地产**：现金流高度周期性，项目期大量流出，销售期集中流入

## 实战应用

### 案例：现金流诊断

假设某公司年度现金流数据（单位：亿元）：

| 项目           | 金额    |
| -------------- | ------- |
| 净利润         | 8       |
| 折旧和摊销     | 3       |
| 应收账款增加   | -5      |
| 存货增加       | -3      |
| 应付账款增加   | 2       |
| **经营现金流** | **5**   |
| 购买固定资产   | -6      |
| 收购子公司     | -4      |
| **投资现金流** | **-10** |
| 银行借款       | 8       |
| 支付股利       | -1      |
| **筹资现金流** | **7**   |
| **现金净增加** | **2**   |

诊断：

- 净利润 8 亿但 CFO 只有 5 亿，利润现金转化率 62.5%，偏低
- 应收账款增加 5 亿，说明大量收入没有收回现金
- 投资现金流 -10 亿，公司正在大规模扩张
- 筹资现金流 +7 亿，靠借款支撑投资和分红
- 结论：公司处于高速扩张期，但盈利质量有待提高，需要关注应收账款回收情况

### 现金流预警信号

以下信号值得警惕：

- CFO 持续低于净利润
- 经营现金流靠应付账款拖延来维持
- 连续多年 FCF 为负
- 靠持续融资来维持运营（CFO 和 CFI 都为负，靠 CFF 续命）
- 经营现金流突然大幅波动且无合理业务解释

## 关键要点

- 经营现金流为正且持续增长，是最健康的信号
- 自由现金流（FCF）是估值的核心指标
- 净利润高但 CFO 低，通常意味着大量应收账款或存货积压
- 投资现金流为负不一定是坏事，可能是公司在扩张产能
- 现金转换周期（CCC）越短，经营效率越高
- 现金流比利润更难操纵，是检验利润质量的试金石

## 常见误区

- **经营现金流为正就一定健康** —— 如果靠拖延应付账款或变卖资产维持，不可持续
- **投资现金流为负就是坏事** —— 可能是公司在扩张产能，长期看是好事
- **现金越多越好** —— 闲置现金过多说明资金利用效率低，应该投入业务或返还股东
- **现金流量表不会造假** —— 虽然比利润表难操纵，但仍有空间（如分类操纵、timing 操作）
- **自由现金流等于经营现金流** —— FCF 还需扣除资本支出

## 与其他概念的关系

- 与 [利润表](/posts/income-statement) 对比，检验利润质量（CFO vs 净利润）
- 与 [资产负债表](/posts/balance-sheet) 中的货币资金科目衔接
- 自由现金流（FCF）是 [DCF 估值](/posts/dcf) 的核心输入
- 经营现金流是计算 [WACC](/posts/wacc) 中债务覆盖倍数的基础
- 现金流断裂是 [破产](/posts/bankruptcy) 的直接原因
