# 投资界风云人物

把高冷的投资大师还原成有血有肉的人——看他们如何从普通人成长为传奇，拆解他们的投资框架，让普通人也能学到一二。

## 关于本栏目

本栏目采用**混合式**写法：每个人物既有故事性又有干货。前半部分讲人物生平和经典战役，后半部分拆解投资框架和普通人能学到什么。

每篇文章约 3000-5000 字，包含：

- **人物速写**：一句话总结 + 关键数据
- **传奇人生**：从普通人到传奇的成长路径
- **经典战役**：3-5 个最具代表性的投资案例
- **投资哲学**：核心原则和方法论
- **策略拆解**：选股标准、买入时机、持仓管理、卖出纪律
- **经典名言**：3-5 条最具代表性的名言
- **延伸阅读**：推荐书籍、纪录片、相关概念
