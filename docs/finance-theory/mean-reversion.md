# 均值回归

> 没有只涨不跌的股票，也没有只跌不涨的股票。价格就像一根被拉伸的橡皮筋，拉得越远，弹回来的力量越大。

## 概述

均值回归（Mean Reversion）是指**资产价格或收益率在偏离长期平均水平后，倾向于回归均值的统计现象**。

均值回归的核心逻辑：

- **估值修复**：价格远高于内在价值时，卖压增加；远低于内在价值时，买盘增加
- **周期性**：经济和行业有周期性，高增长不可持续，低增长也不会永远持续
- **统计规律**：许多金融时间序列（如 PE、PB、波动率、利差）表现出均值回归特性

均值回归的数学表达（Ornstein-Uhlenbeck 过程）：

$$dX_t = \theta(\mu - X_t)dt + \sigma dW_t$$

其中：

- $\mu$：长期均值（Long-run Mean）
- $\theta$：回归速度（Speed of Reversion），越大回归越快
- $\sigma$：波动率（Volatility）
- $W_t$：标准布朗运动（Standard Brownian Motion）

该过程的解为：

$$X_t = \mu + (X_0 - \mu)e^{-\theta t} + \sigma \int_0^t e^{-\theta(t-s)} dW_s$$

均值项 $\mu + (X_0 - \mu)e^{-\theta t}$ 随时间指数衰减到 $\mu$，衰减速率由 $\theta$ 控制。

## 为什么重要

- **投资策略**：均值回归是配对交易（Pairs Trading）、统计套利（Statistical Arbitrage）等策略的理论基础
- **估值参考**：PE、PB 等估值指标的均值回归特性为估值提供锚
- **风险管理**：波动率的均值回归特性用于 VaR 模型和期权定价。GARCH 模型本质上就是波动率均值回归
- **资产配置**：利用均值回归特性进行再平衡（Rebalancing），"高卖低买"
- **宏观预测**：GDP 增长率、通胀率、利率等宏观经济变量具有均值回归特性

## 核心原理

### 均值回归的经济学直觉

均值回归的经济学基础可以归结为**竞争均衡**（Competitive Equilibrium）的力量：

1. **超额利润吸引竞争**：当某个行业利润率高时，新进入者增加，供给上升，利润率下降
2. **亏损导致退出**：当某个行业亏损时，企业退出，供给下降，利润率回升
3. **资本流动**：资金从低回报领域流向高回报领域，推动回报趋同
4. **政策干预**：央行在通胀过高时加息、过低时降息，推动经济回归目标

### 均值回归 vs 随机漫步

均值回归和[随机漫步](/finance-theory/random-walk)是两种对立的时间序列模型：

| 特征       | 随机漫步                        | 均值回归                                   |
| ---------- | ------------------------------- | ------------------------------------------ |
| 数学模型   | $P_t = P_{t-1} + \varepsilon_t$ | $dX_t = \theta(\mu - X_t)dt + \sigma dW_t$ |
| 预测性     | 不可预测                        | 可预测（向均值回归）                       |
| 方差       | 随时间线性增长                  | 有上限 $Var(X_t) \to \sigma^2/(2\theta)$   |
| 冲击持久性 | 永久性                          | 暂时性，指数衰减                           |
| 有效市场   | 支持                            | 挑战                                       |

### 半衰期（Half-Life）

均值回归的半衰期衡量偏离均值后回归到一半所需的时间：

$$\text{Half-Life} = \frac{\ln(2)}{\theta}$$

半衰期越短，回归越快。实务中，不同资产的半衰期差异很大：

- 波动率：几天到几周
- 行业利润率：1-3 年
- 估值指标（PE/PB）：3-5 年
- 汇率：2-4 年

## 模型/方法详解

### 均值回归的统计检验

**1. 方差比检验**（Variance Ratio Test）

Lo 和 MacKinlay（1988）提出检验价格序列是否服从随机漫步：

$$VR(k) = \frac{\text{Var}(R_t^{(k)})}{k \times \text{Var}(R_t)}$$

其中 $R_t^{(k)}$ 是 k 期收益率。如果 $VR(k) < 1$，表明存在均值回归；如果 $VR(k) > 1$，表明存在动量。

**2. ADF 检验**（Augmented Dickey-Fuller Test）

检验时间序列是否平稳（Stationary）：

$$\Delta X_t = \alpha + \beta t + \gamma X_{t-1} + \sum_{i=1}^p \delta_i \Delta X_{t-i} + \varepsilon_t$$

如果 $\gamma < 0$ 且统计显著，则序列是均值回归的。

**3. Hurst 指数**

Hurst 指数 $H$ 度量时间序列的持久性：

- $H = 0.5$：随机漫步
- $H < 0.5$：均值回归（$H$ 越小回归越强）
- $H > 0.5$：趋势延续（动量）

### 配对交易模型

配对交易（Pairs Trading）是均值回归最经典的应用：

1. **选择配对**：找到两只历史价格高度相关的股票（如可口可乐和百事可乐）
2. **计算价差**：$S_t = P_{A,t} - \beta P_{B,t}$
3. **检验均值回归**：对价差做 ADF 检验
4. **交易信号**：当价差偏离均值超过 2 倍标准差时，做空价差（卖高买低）
5. **平仓**：当价差回归均值时平仓

价差的收益：

$$\text{Profit} = (S_t - \mu) - (S_{t+\tau} - \mu) = S_t - S_{t+\tau}$$

其中 $\tau$ 是回归时间。

### GARCH 模型中的波动率均值回归

GARCH(1,1) 模型：

$$\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2$$

长期方差为：

$$\bar{\sigma}^2 = \frac{\omega}{1 - \alpha - \beta}$$

当 $\alpha + \beta < 1$ 时，波动率均值回归到 $\bar{\sigma}^2$。实证中 $\alpha + \beta$ 通常接近 1（约 0.9-0.98），表明波动率回归较慢。

## 主流方案对比

| 策略       | 原理                       | 适用场景   | 风险                         |
| ---------- | -------------------------- | ---------- | ---------------------------- |
| 配对交易   | 两只相关资产的价差均值回归 | 同行业股票 | 结构性变化导致配对失效       |
| 统计套利   | 多资产组合的统计均值回归   | 全市场     | 模型风险、流动性风险         |
| 价值投资   | 估值指标均值回归           | 个股选择   | 价值陷阱（基本面永久恶化）   |
| 波动率交易 | 波动率均值回归             | 期权市场   | 波动率持续高位（如金融危机） |
| 再平衡策略 | 资产配置比例的均值回归     | 组合管理   | 趋势市场中表现差             |

## 实战应用

### 估值均值回归

以 PE 为例，假设某股票历史平均 PE 为 15 倍，当前 PE 为 25 倍：

- 如果盈利不变，股价需要下跌 40% 才能回归均值
- 如果股价不变，盈利需要增长 67% 才能"消化"高估值
- 更可能的情况是两者共同作用

Jeremy Siegel 的研究表明，长期来看（20 年以上），股票的实际收益率会趋近于历史均值（约 6.5-7%）。

### 波动率交易

VIX 指数（恐慌指数）是典型的均值回归品种：

- 长期均值约 20
- 恐慌时可飙升至 80+（如 2008 年、2020 年）
- 但几乎总是会回归到 20 附近

交易策略：在 VIX 极高时卖出波动率（如卖出 VIX 期货或看涨期权），在 VIX 极低时买入波动率。

### 汇率均值回归

购买力平价（PPP）理论预测汇率长期均值回归到购买力平价水平。虽然短期可能严重偏离，但长期（5-10 年）有回归趋势。

## 关键要点

- 均值回归不等于"明天就回归"，回归可能需要很长时间。凯恩斯说得好："市场保持非理性的时间可能比你保持偿付能力的时间更长"
- 均值本身可能发生变化（结构性变化），不能简单用历史均值预测未来。技术革新、监管变化、商业模式转变都可能改变长期均值
- 均值回归在长期更明显，短期可能表现为动量（趋势延续）。短期动量、长期回归是金融市场的普遍模式
- 均值回归策略需要严格的纪律和风险管理，因为"接飞刀"的风险很大
- 均值回归不是"定律"，而是统计倾向。每次回归都是概率事件，不是必然事件

## 常见误区

- **误区一**："跌多了就一定会涨回来"——均值回归不是必然的。公司基本面可能永久性恶化（如诺基亚、柯达），价格不会回到历史均值
- **误区二**："历史均值就是未来均值"——结构性变化（如行业变革、技术革新、监管变化）会改变长期均值。用过去 10 年的均值预测未来可能严重失误
- **误区三**："均值回归策略稳赚不赔"——回归时间不确定，可能在你回归前就爆仓。LTCM 的破产就是经典案例
- **误区四**："均值回归和动量效应矛盾"——两者在不同时间尺度上主导。短期（3-12 个月）动量效应主导，长期（3-5 年）均值回归主导
- **误区五**："均值回归只适用于股票"——均值回广泛存在于利率、汇率、商品、波动率、宏观经济变量等各类金融和经济时间序列中

## 与其他概念的关系

- [随机漫步](/finance-theory/random-walk) - 随机漫步理论与均值回归形成对比（前者认为价格变动不可预测）
- [锚定效应](/finance-theory/anchoring-effect) - 均值回归的心理基础之一是锚定效应（投资者以历史均值为锚）
- momentum - 动量效应与均值回归在短期和长期呈现不同主导
- [价值投资](/investment/value-investing) - 价值投资的核心逻辑之一是均值回归
- [有效市场假说](/finance-theory/efficient-market-hypothesis) - 均值回归现象对有效市场假说提出了挑战
