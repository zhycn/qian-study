# 陈光明——A 股价值投资的标杆

> 当大多数人还在 A 股追涨杀跌的时候，有一个人用 20 年时间证明：**价值投资在中国不仅可行，而且可以做得很好。** 他叫陈光明，东方红资产管理的创始人，中国公募基金界最纯粹的价值投资者。

## 人物速写

陈光明，东方红资产管理创始人，中国公募基金行业价值投资的标杆人物。他从券商研究所的普通研究员起步，一路成长为管理千亿规模的资管大佬。他管理的公募产品长期年化收益超过 20%，以 ROE 选股和深度研究为核心方法论。

## 传奇人生

### 早年经历：从厦大到券商

1974 年，陈光明出生于福建。他后来考入厦门大学金融学相关专业，在厦大接受了系统的金融学教育。厦门大学是中国金融学研究的传统重镇，培养了一大批优秀的金融从业者。

大学期间，陈光明就对价值投资产生了浓厚的兴趣。他阅读了大量格雷厄姆、巴菲特、费雪的经典著作，逐渐形成了自己的投资框架。

2000 年前后，陈光明加入东方证券研究所，开始了他的投研生涯。当时的中国证券市场还处于早期阶段，大多数投资者关注的是题材炒作和短线交易，价值投资几乎无人问津。

### 投资启蒙：在 A 股寻找价值

在东方证券研究所期间，陈光明负责行业和公司的深度研究。他很快发现了一个现象：

> "A 股市场有很多好公司，但很少有人愿意花时间去研究它们。大多数人关注的是消息面、资金面、技术面，而不是企业的基本面。"

这个发现让陈光明意识到：**如果坚持做深度研究，在 A 股市场反而能获得巨大的超额收益。** 因为竞争者少，好公司的定价错误更常见。

他开始系统性地研究 A 股上市公司的财务报表，重点关注几个指标：

- **ROE（净资产收益率）**——衡量企业盈利能力的核心指标
- **自由现金流**——企业真正能赚到的钱
- **竞争优势**——护城河有多深
- **管理层质量**——是否以股东利益为导向

这套方法论后来成为东方红资管的核心投研框架。

### 关键转折点：创立东方红资管

2000 年代中期，陈光明从研究所转向投资岗位，开始直接管理资金。他管理的早期产品就展现出了与众不同的风格：

- **低换手率**——不频繁交易，买入后长期持有
- **集中持仓**——前十大重仓股占比很高
- **深度研究驱动**——每笔投资都基于扎实的基本面研究

2010 年前后，陈光明开始负责东方证券资产管理业务。他带领团队打造了"东方红"品牌，将价值投资理念系统性地引入公募资管领域。

**关于"东方红"品牌的命名：** "东方红"取自东方证券的品牌体系，但在陈光明的打造下，"东方红"逐渐成为中国价值投资的代名词之一。（来源：东方证券公告）

2014 年，东方红系列公募产品开始发行，其中**东方红中国优势**、**东方红睿丰**等产品成为市场关注的焦点。这些产品的共同特点是：

- 以 ROE 为核心选股指标
- 长期持有优质公司
- 不参与题材炒作
- 换手率远低于行业平均水平

### 巅峰时刻：东方红的崛起

2015-2021 年，是东方红资管最辉煌的时期。

**关键里程碑：**

- **2014 年**：东方红系列公募产品开始发行，首批产品即获得市场认可
- **2015 年**：A 股经历剧烈波动，东方红产品凭借低仓位和优质持仓，回撤远小于市场（来源：晨星中国，2015 年基金年报）
- **2016-2017 年**：价值投资风格占优，东方红产品业绩大幅领先同类（来源：晨星中国，2017 年排名）
- **2019 年**：东方红资管管理规模突破 2000 亿元（来源：中国证券投资基金业协会）
- **2021 年**：陈光明卸任东方证券资管董事长，创立睿远基金，开启新的征程

**关于管理规模的数据：** 根据中国证券投资基金业协会数据，东方证券资管在 2019-2021 年期间管理规模峰值超过 3000 亿元，其中公募产品规模超过 1500 亿元。

陈光明在东方红期间最核心的贡献是：**将价值投资从理念变成了可复制的体系。** 他建立了一套完整的投研框架：

1. **自上而下选赛道**——关注长期增长确定性高的行业
2. **自下而上选公司**——以 ROE 为核心，深度研究每一家公司
3. **长期持有**——买入后不频繁交易，让时间创造价值
4. **风险控制**——不碰看不懂的公司，不追高估值泡沫

这套体系不仅在东方红得到了验证，也在陈光明后来创立的睿远基金中继续发挥作用。

## 经典战役

### 战役一：东方红中国优势（2015-2017）

这是东方红公募系列的代表作之一，也是陈光明价值投资理念的经典实践。

**背景：**

- 2015 年 6 月，A 股经历了历史上最剧烈的暴跌之一，上证指数从 5178 点暴跌至 2850 点，跌幅超过 40%
- 市场恐慌情绪蔓延，大量股票被错杀
- 东方红中国优势于 2015 年 3 月成立，成立不久就遭遇了股灾

**陈光明的应对：**

- 不恐慌性抛售，坚持持有优质公司
- 利用市场恐慌，在低位加仓被错杀的好公司
- 严格控制仓位，不盲目抄底

**重仓方向：**

- 消费——白酒、家电等具有品牌护城河的公司
- 医药——长期增长确定性高的行业
- 金融——低估值、高分红的银行和保险

**结果：**

- 2015 年下半年，东方红中国优势的回撤远小于市场（来源：晨星中国，2015 年基金年报）
- 2016-2017 年，随着市场回归理性，价值股重新受到追捧，东方红中国优势业绩大幅领先
- 成立以来年化收益超过 20%（来源：东方红资管年报，截至 2021 年）

| ------------ | --------------------------------- |
| 成立时间     | 2015 年 3 月                      |
| 成立后遭遇   | 2015 年 A 股暴跌（上证指数 -40%） |
| 2015 年回撤  | 远小于市场（来源：晨星中国）      |
| 成立以来年化 | 超过 20%（截至 2021 年）          |
| 核心策略     | ROE 选股、长期持有、低换手率      |

**数据来源：** 东方红资管年报、晨星中国基金评级、公开报道

### 战役二：重仓白酒（2016-2021）

陈光明是 A 股市场最早系统性重仓白酒的价值投资者之一。

**背景：**

- 2012-2014 年，受"八项规定"影响，白酒行业经历了深度调整
- 茅台、五粮液等龙头公司股价从高点下跌超过 50%
- 市场普遍认为白酒行业的"黄金时代"已经结束
- 大量资金撤离白酒板块

**陈光明的逻辑：**

- 白酒是中国文化的一部分，需求不会因短期政策而消失
- 高端白酒的品牌护城河极深，无法复制
- 行业集中度提升，龙头公司受益
- 估值已经回到合理甚至偏低区间

**具体操作：**

- 2016 年前后，东方红产品开始系统性配置白酒龙头
- 以贵州茅台、五粮液、泸州老窖为核心持仓
- 持有时间长达 5 年以上

**结果：**

- 贵州茅台股价从 2016 年的约 250 元涨到 2021 年的超过 2600 元，涨幅超过 10 倍
- 五粮液股价从 2016 年的约 20 元涨到 2021 年的超过 300 元，涨幅超过 15 倍
- 白酒板块成为 2016-2021 年 A 股表现最好的板块之一

| -------- | ----------------------- | ---------------------- |
| 建仓时间 | 2016 年前后             | 2016 年前后            |
| 建仓价格 | 约 250 元               | 约 20 元               |

### 战役三：布局医药（2017-2021）

医药是陈光明长期看好的赛道之一。

**布局逻辑：**

- 中国人口老龄化趋势不可逆转
- 医疗健康是长期确定性最高的赛道之一
- 中国创新药、医疗器械、医疗服务等领域正在快速崛起
- 医药行业的 ROE 普遍较高，符合选股标准

**代表性持仓：**

- **恒瑞医药**——中国创新药龙头企业，ROE 长期保持在 20% 以上
- **迈瑞医疗**——中国医疗器械龙头，全球化布局
- **爱尔眼科**——医疗服务连锁模式，可复制性强

**结果：**

- 恒瑞医药股价从 2017 年的约 30 元涨到 2021 年的超过 90 元（前复权），涨幅超过 3 倍
- 迈瑞医疗上市后表现优异，成为 A 股医疗器械市值最高的公司
- 医药板块在 2017-2021 年期间持续跑赢大盘

| 指标     | 恒瑞医药              | 迈瑞医疗               |
| -------- | --------------------- | ---------------------- |
| 布局时间 | 2017 年前后           | 2018 年上市后          |
| 布局价格 | 约 30 元              | 上市初期               |
| 峰值价格 | 超过 90 元（2021 年） | 超过 500 元（2021 年） |
| 核心逻辑 | 创新药龙头、ROE 20%+  | 医疗器械龙头、全球化   |

**数据来源：** Wind 数据、恒瑞医药年报、迈瑞医疗年报、东方红基金定期报告

### 战役四：坚守金融（2015-2020）

陈光明在金融板块的配置体现了他对低估值、高分红公司的偏好。

**背景：**

- 2015 年股灾后，银行、保险等金融板块估值处于历史低位
- 大量银行股市净率（P/B）跌破 1 倍
- 市场普遍认为银行资产质量堪忧

**陈光明的逻辑：**

- 中国银行业虽然面临资产质量压力，但龙头银行的资产质量可控
- 低估值 + 高分红 = 安全边际
- 保险行业受益于居民财富管理需求增长
- 金融板块是组合的"压舱石"

**具体操作：**

- 配置招商银行、中国平安等龙头金融公司
- 看重其稳定的 ROE 和分红能力
- 作为组合的底仓配置

**结果：**

- 招商银行股价从 2016 年的约 15 元涨到 2021 年的超过 50 元，涨幅超过 3 倍
- 中国平安股价从 2016 年的约 25 元涨到 2021 年的超过 90 元，涨幅超过 3 倍
- 金融板块为组合提供了稳定的收益来源

| 指标     | 招商银行               | 中国平安              |
| -------- | ---------------------- | --------------------- |
| 配置时间 | 2016 年前后            | 2016 年前后           |
| 配置价格 | 约 15 元               | 约 25 元              |
| 峰值价格 | 超过 50 元（2021 年）  | 超过 90 元（2021 年） |
| 核心逻辑 | 零售银行龙头、ROE 15%+ | 保险龙头、综合金融    |

**数据来源：** Wind 数据、招商银行年报、中国平安年报、东方红基金定期报告

### 战役五：创立睿远基金（2021）

2021 年，陈光明做出了一个让市场关注的决定：**离开东方证券资管，创立睿远基金。**

**背景：**

- 陈光明在东方证券资管工作了 20 余年，从研究员做到董事长
- 2021 年，他选择卸任东方证券资管董事长
- 随后创立睿远基金，继续践行价值投资理念

**市场解读：**

- 这不是"离开"，而是"再出发"
- 睿远基金将延续东方红的价值投资理念
- 陈光明希望通过新的平台，将价值投资做得更纯粹

**结果：**

- 睿远基金成立后，发行的产品依然延续了 ROE 选股、深度研究、长期持有的风格
- 首批产品获得市场热烈认购
- 陈明明的价值投资理念在新的平台上继续传承

## 投资哲学

### 原则一：ROE 是核心选股指标

陈光明最核心的选股标准是 ROE（净资产收益率）。

> "ROE 是衡量企业盈利能力最重要的指标。一家能长期保持高 ROE 的公司，一定有其独特的竞争优势。"

他的 ROE 选股逻辑：

- **长期 ROE > 15%**——这是筛选优质公司的第一道门槛
- **ROE 的稳定性**——不仅看高低，更看是否稳定
- **ROE 的来源**——是靠高利润率、高周转率还是高杠杆？优先选择高利润率驱动的公司
- **ROE 的可持续性**——护城河是否足够深，能否长期维持高 ROE

### 原则二：深度研究驱动

陈光明强调深度研究的重要性。

> "投资不是猜谜游戏，而是基于深度研究的决策。你对一家公司了解得越深，你的投资就越有信心。"

他的深度研究方法：

- **读财报**——至少读过去 5-10 年的年报
- **调研**——实地走访企业，与管理层交流
- **产业链研究**——了解上下游，理解行业格局
- **持续跟踪**——买入后持续关注基本面变化

### 原则三：长期持有优质公司

陈光明是长期持有的坚定践行者。

> "好公司不需要频繁买卖。买入后，让时间成为你的朋友。"

他的长期持有逻辑：

- 好公司的价值会随时间增长
- 频繁交易产生摩擦成本（手续费、印花税、冲击成本）
- 找到好公司很难，不应该轻易卖掉
- A 股市场的波动性大，长期持有可以平滑波动

### 原则四：不碰看不懂的公司

陈光明严格遵守能力圈原则。

> "市场上永远有机会，但你只需要赚你能力圈内的钱。"

他明确回避的领域：

- 商业模式看不懂的公司
- 财务数据不透明的公司
- 管理层诚信有问题的公司
- 估值极度泡沫的公司

### 原则五：安全边际

陈光明注重买入价格的安全边际。

> "再好的公司，如果买贵了，也不是一笔好投资。"

他的安全边际逻辑：

- 好公司也要在合理或偏低的价格买入
- 估值过高时，即使公司质量再好，也要控制仓位
- 市场恐慌时，好公司往往被错杀，这是买入的好时机

## 策略拆解

### 选股标准

陈光明的选股标准可以总结为"五个好"：

| 标准   | 具体要求                             |
| ------ | ------------------------------------ |
| 好行业 | 长期增长确定性强，空间大             |
| 好公司 | ROE 高且稳定，竞争优势深             |
| 好管理 | 管理层诚信、有能力、以股东利益为导向 |
| 好财务 | 现金流好、负债合理、盈利质量高       |
| 好价格 | 估值合理或偏低，有安全边际           |

**具体 ROE 筛选条件：**

- 连续 5 年 ROE > 15%
- ROE 波动率小（稳定性强）
- ROE 主要由净利润率驱动（而非高杠杆）
- 自由现金流/净利润 > 80%（盈利质量高）

### 买入时机

陈光明的买入时机判断：

1. **市场恐慌**——好公司被错杀，估值回到合理区间
2. **行业低谷**——优质企业遇到暂时困难
3. **新股上市**——好公司 IPO 后，经过充分研究后买入
4. **定期调仓**——发现更具性价比的标的时换仓

他特别强调：**不要因为市场热点而买入。** 只买你真正理解的好公司。

### 持仓管理

陈光明的持仓管理特点：

- **集中持仓**——前十大重仓股占比通常在 60% 以上
- **低换手率**——年化换手率远低于行业平均水平
- **行业分散**——虽然个股集中，但行业相对分散
- **持续跟踪**——定期复查基本面，但不频繁操作

### 卖出纪律

陈光明的卖出条件：

1. **基本面恶化**——ROE 持续下降，竞争优势丧失
2. **估值极度泡沫**——价格远超内在价值
3. **发现更好的机会**——换仓到性价比更高的标的
4. **逻辑证伪**——投资逻辑被证明是错误的

他特别强调：**不要因为短期波动而卖出。** 如果长期逻辑还在，就应该继续持有。

### 普通人能学到什么

陈光明的方法对普通投资者特别有借鉴意义：

1. **用 ROE 选股**——这是一个简单但有效的筛选工具
2. **做深度研究**——不要只看 K 线，去读财报、了解公司
3. **长期持有**——减少交易频率，让时间创造价值
4. **坚守能力圈**——不懂不投，不碰看不懂的公司
5. **注重安全边际**——好公司也要在合理价格买入

**具体建议：**

- 用 ROE > 15% 且连续 5 年稳定作为初筛条件
- 选择 5-10 家你真正理解的好公司
- 以 3-5 年以上的视角持有
- 每年检查一次基本面是否发生变化
- 保持阅读财报的习惯
