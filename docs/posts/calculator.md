# 在线计算器

金融计算器就像你的"数字军师"——不管是算复利、房贷还是定投,它都能用数据帮你做出更理性的决策,不再拍脑袋做决定。

## 复利计算器

计算复利收益，感受时间的力量。

**公式：** `FV = PV × (1 + r)^n`

| 参数 | 说明                  |
| ---- | --------------------- |
| PV   | 本金（Present Value） |
| r    | 年化收益率            |
| n    | 投资年数              |
| FV   | 终值（Future Value）  |

**示例：** 10 万元，年化 8%，30 年后 = 10 万 × (1 + 0.08)^30 ≈ **100.6 万元**

> 💡 推荐使用：[天天基金复利计算器](https://fund.eastmoney.com/trade/dczh.html)

## 房贷计算器

计算月供，选择还款方式。

### 等额本息

每月还款金额固定，前期利息占比高。

$$M = P \times \frac{r(1+r)^n}{(1+r)^n - 1}$$

### 等额本金

每月还款本金固定，利息逐月递减，总利息更少。

| 对比项   | 等额本息   | 等额本金         |
| -------- | ---------- | ---------------- |
| 月供     | 固定       | 逐月递减         |
| 总利息   | 较多       | 较少             |
| 前期压力 | 较小       | 较大             |
| 适合人群 | 收入稳定者 | 前期还款能力强者 |

> 💡 推荐使用：[贝壳房贷计算器](https://www.ke.com/posts/mortgage-calculator)

## 定投计算器

模拟定投收益，验证投资策略。

**核心参数：**

- 每月投入金额
- 预期年化收益率
- 投资年限
- 是否考虑波动（定投微笑曲线）

**示例：** 每月定投 2000 元，年化 10%，20 年后 ≈ **153 万元**
（总投入 48 万，收益 105 万）

> 💡 推荐使用：[蛋卷基金定投计算器](https://danjuanapp.com/calculator)

## 常用金融公式速查

### 收益率计算

| 公式       | 说明                                   |
| ---------- | -------------------------------------- |
| 简单收益率 | `(期末价值 - 期初价值) / 期初价值`     |
| 年化收益率 | `(1 + 总收益率)^(1/年数) - 1`          |
| 夏普比率   | `(年化收益 - 无风险利率) / 年化波动率` |

### 估值指标

| 指标      | 公式              | 说明             |
| --------- | ----------------- | ---------------- |
| 市盈率 PE | 股价 / 每股收益   | 回本年限         |
| 市净率 PB | 股价 / 每股净资产 | 相对净资产的溢价 |
| 股息率    | 每股分红 / 股价   | 现金分红回报率   |
| ROE       | 净利润 / 净资产   | 股东权益回报率   |

### 定投相关

| 公式     | 说明                         |
| -------- | ---------------------------- |
| 定投终值 | `PMT × [((1+r)^n - 1) / r]`  |
| 72 法则  | `72 / 年化收益率 ≈ 翻倍年数` |
