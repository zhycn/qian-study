# 学习路径

> 不知道怎么开始？按你的需求选择一条学习路线吧。

## 📖 新手入门

适合零基础读者，从最核心的概念开始建立金融认知框架。

- [什么是金融](/basics/what-is-finance) — 金融是什么？为什么重要？
- [货币](/basics/money) → [利息](/basics/interest) → [通货膨胀](/basics/inflation) — 理解钱的底层逻辑
- [信用](/basics/credit) — 现代金融的基石
- [风险与收益](/basics/risk-return) — 金融决策的核心权衡

**学完你能**：理解日常听到的金融术语，看懂经济新闻的基本逻辑。

## 🎯 实用场景

结合生活实际，学完就能用上的金融知识。

### 个人理财

- [储蓄](/personal-finance/savings) → [信用卡](/personal-finance/credit-card) → [房贷](/personal-finance/mortgage) → [保险](/personal-finance/insurance)
- [资产配置](/personal-finance/asset-allocation) — 不把鸡蛋放在一个篮子里
- [紧急备用金](/personal-finance/emergency-fund) — 应对突发状况的"救命钱"

### 投资入门

- [股票](/investing/stock) → [基金](/investing/fund) → [ETF](/investing/etf) → [指数](/investing/index)
- [市盈率](/investing/pe-ratio) — 判断股票贵不贵
- [分红](/investing/dividend) — 公司给股东的"红包"

### 公司金融基础

- [资产负债表](/corporate-finance/balance-sheet) → [利润表](/corporate-finance/income-statement) → [现金流量表](/corporate-finance/cash-flow-statement)
- [估值](/corporate-finance/valuation) — 一个公司到底值多少钱

**学完你能**：管理个人财务，看懂上市公司财报，开始自己的投资实践。
