# 查理·芒格——巴菲特的黄金搭档

> 如果巴菲特是伯克希尔的发动机，芒格就是方向盘。没有芒格，巴菲特可能依然伟大，但不会如此伟大。

## 人物速写

查理·芒格（Charlie Munger），伯克希尔·哈撒韦副主席，巴菲特 60 年的商业伙伴和人生挚友。他以"多元思维模型"（Mental Models）闻名，是投资界最具智慧的思考者之一。

| 维度     | 数据                                                  |
| -------- | ----------------------------------------------------- |
| 生卒年   | 1924 年 1 月 1 日 - 2023 年 11 月 28 日（享年 99 岁） |
| 国籍     | 美国                                                  |
| 核心贡献 | 多元思维模型、好公司比好价格重要                      |
| 代表公司 | 伯克希尔·哈撒韦（副主席）、Daily Journal 董事长       |
| 投资年限 | 约 60 年（1960 年代至今）                             |
| 个人财富 | 约 26 亿美元（2023 年福布斯）                         |
| 教育背景 | 哈佛大学法学博士（J.D.）                              |
| 职业背景 | 律师 → 房地产 → 投资                                  |
| 代表著作 | 《穷查理宝典》（Poor Charlie's Almanack）             |

**数据来源：** 《穷查理宝典》（2005）、伯克希尔·哈撒韦年报、Daily Journal 年报、福布斯富豪榜（2023）

## 传奇人生

### 早年经历：大萧条中的成长

1924 年 1 月 1 日，查理·芒格出生于内布拉斯加州奥马哈市。他和巴菲特在同一条街上长大，但两人直到 1959 年才第一次见面。

芒格的童年正值大萧条。他的祖父是联邦法官，家境尚可，但大萧条依然给这个家庭带来了巨大冲击。芒格后来回忆：

> "大萧条教会了我一件事：**永远不要认为好时光会永远持续下去。**"

中学毕业后，芒格进入密歇根大学学习数学。二战期间，他加入美国陆军航空队，被派往新墨西哥州和加利福尼亚州的气象站。战争结束后，他利用《退伍军人权利法案》进入了哈佛法学院。

### 求学与启蒙：从数学到法律

在哈佛法学院，芒格展现出了非凡的才华。他不需要太努力就能取得优异成绩——因为他有一个独特的学习方法：**跨学科思考。**

他后来回忆：

> "我从不局限于法律思维。我会用数学的逻辑、心理学的洞察、经济学的框架来理解法律问题。这让我比同学们看得更深。"

1948 年，芒格以优异成绩从哈佛法学院毕业。他没有去华尔街，而是回到了洛杉矶，进入了一家律师事务所。

### 关键转折点：从律师到投资者

1960 年代，芒格开始涉足房地产投资。他和一位建筑师朋友合作，在南加州购买土地、建造住宅、然后出售或出租。这笔生意非常成功——到 1960 年代末，芒格通过房地产投资积累了约 140 万美元的财富。

与此同时，芒格也开始投资股票市场。他的早期投资风格是典型的"格雷厄姆式"——寻找便宜的股票，等待价值回归。

**1959 年，芒格遇到了巴菲特。**

这次相遇改变了两个人的一生。

### 遇见巴菲特：从猩猩到人类

1959 年，35 岁的芒格和 29 岁的巴菲特在奥马哈的一次聚会上第一次见面。两人一见如故——他们都热爱阅读、热爱思考、热爱投资。

巴菲特后来回忆：

> "我们第一次见面就聊了几个小时。我发现这个人和我想得一模一样——甚至想得更好。"

芒格对巴菲特的影响是深远的。巴菲特曾经说：

> "芒格让我从猩猩变成了人类。他拓宽了我的视野，让我不再只是一个纯粹的格雷厄姆式投资者。"

具体来说，芒格教会了巴菲特三件事：

1. **好公司比好价格重要**——用合理价格买伟大公司，胜过用伟大价格买合理公司
2. **护城河的价值**——品牌、转换成本、网络效应等持久竞争优势
3. **跨学科思维**——用心理学、物理学、生物学等多元模型来理解商业

### 巅峰时刻：伯克希尔的黄金时代

1978 年，芒格正式加入伯克希尔·哈撒韦，担任副主席。从此，"巴菲特 + 芒格"的黄金组合开始了长达 40 多年的合作。

芒格在伯克希尔的角色很独特：

- 他不是日常运营者——运营由各子公司 CEO 负责
- 他不是交易执行者——交易由巴菲特决策
- 他是**思想的碰撞者**——巴菲特做决定前，一定会和芒格讨论

芒格的贡献主要体现在：

- **推动巴菲特买入喜诗糖果**（1972）——这是巴菲特投资哲学转变的标志性事件
- **推动巴菲特买入可口可乐**（1988）——品牌护城河的终极案例
- **推动巴菲特投资 Costco**——芒格是 Costco 的长期董事
- **推动伯克希尔收购 See's、BNSF 铁路等优质资产**

2023 年 11 月 28 日，芒格在加州一家医院安详离世，享年 99 岁。巴菲特在悼词中说：

> "伯克希尔是芒格和我共同构建的。没有他，伯克希尔不会有今天的成就。"

## 经典战役

### 战役一：推动买入喜诗糖果（1972）

这是芒格对巴菲特影响最深远的案例。

1972 年，巴菲特和芒格考虑收购喜诗糖果（See's Candies）。喜诗糖果是加州一家高端巧克力品牌，成立于 1921 年，以优质产品和卓越客户服务著称。

**巴菲特的犹豫：**

- 喜诗糖果的净资产约 800 万美元
- 卖方要价 2500 万美元
- 市盈率超过 20 倍，市净率超过 3 倍
- 按照格雷厄姆的标准，这太贵了

**芒格的坚持：**

芒格告诉巴菲特：

- 喜诗糖果在加州有强大的品牌忠诚度
- 每年圣诞节前可以提价而不影响销量——这是**定价权**（Pricing Power）
- 资本支出极低，几乎不需要额外的资本投入
- 可以持续产生自由现金流

巴菲特被说服了。这笔投资后来成为了他投资哲学转变的标志性事件。

**结果：**

- 喜诗糖果从 1972 年到 2007 年，累计为伯克希尔创造了超过 **20 亿美元** 的税前利润
- 投入 2500 万美元，回报超过 20 亿美元—— **超过 80 倍的回报**

巴菲特后来多次提到：

> "如果没有芒格，我不会买喜诗糖果。如果没有喜诗糖果，我不会理解好生意的价值。"

| 指标         | 数值                        |
| ------------ | --------------------------- |
| 收购时间     | 1972 年                     |
| 收购价格     | 2500 万美元                 |
| 净资产       | 约 800 万美元               |
| 累计税前利润 | 超过 20 亿美元（1972-2007） |
| 回报率       | 超过 80 倍                  |

**数据来源：** 巴菲特致股东信（1972、2007、2014）、《穷查理宝典》（2005）

### 战役二：投资 Costco（1990 年代至今）

芒格是 Costco 的长期董事，也是 Costco 最坚定的支持者。

**芒格的投资逻辑：**

- Costco 的商业模式极其简单——会员制仓储超市
- 会员费是主要利润来源，商品几乎零利润销售
- 这种模式创造了极强的客户忠诚度
- 规模效应让 Costco 的成本优势无法复制

芒格在 1990 年代开始大量买入 Costco 股票，并通过 Daily Journal 公司和伯克希尔持续持有。

**结果：**

- Costco 股价从 1990 年代初的约 10 美元（拆股调整后）涨到 2023 年的超过 550 美元
- 芒格通过 Costco 获得了 **超过 50 倍** 的回报
- Costco 成为芒格最成功的投资之一

**50 倍回报的计算：** 假设芒格在 1993 年以约 10 美元/股（拆股调整后）买入 Costco，到 2023 年股价约 550 美元，回报率 = 550 ÷ 10 = 55 倍。加上 30 年间的分红，总回报更高。（来源：Yahoo Finance 历史数据、Daily Journal 年报）

芒格评价 Costco：

> "Costco 是我见过的最好的零售商业模式。它把客户利益放在第一位，利润自然会来。"

| 指标     | 数值                         |
| -------- | ---------------------------- |
| 买入时间 | 1990 年代初                  |
| 买入价格 | 约 10 美元（拆股调整后）     |
| 当前价格 | 超过 550 美元（2023 年）     |
| 回报率   | 超过 50 倍                   |
| 芒格角色 | Costco 长期董事（1997-2023） |

**数据来源：** Yahoo Finance 历史数据、Daily Journal 年报、Costco 年度报告

### 战役三：Daily Journal 公司的转型

芒格长期担任 Daily Journal 公司的董事长。这原本是一家出版法律公告的小公司，在芒格的带领下，逐渐转型为一家投资公司。

**芒格的操作：**

- 用公司现金流投资股票
- 集中持仓——主要持有富国银行、美国银行、Costco、比亚迪等
- 长期持有——不频繁交易

**结果：**

- Daily Journal 的股票组合从 2000 年代初的约 1000 万美元增长到 2021 年的超过 4 亿美元
- 年化收益率超过 **20%**（2000-2021，来源：Daily Journal 年报）
- 芒格通过这家公司展示了他自己的投资能力——不依赖巴菲特

**20%+年化的来源：** Daily Journal 2021 年年报显示，其股票组合从 2000 年的约 1000 万美元增长到 2021 年的超过 4 亿美元，21 年间的年化收益率约 20.5%。同期标普 500 年化约 8%。（来源：Daily Journal 2021 年年报）

### 战役四：投资比亚迪（2008）

2008 年，芒格推动伯克希尔投资了中国电动汽车制造商比亚迪（BYD）。

**背景：**

- 芒格认识了比亚迪创始人王传福
- 他被王传福的才华和比亚迪的技术实力所打动
- 芒格告诉巴菲特："这个人比爱迪生还厉害，他既能做工程，又能做管理"

**交易：**

- 伯克希尔通过中美能源（MidAmerican Energy）投资 2.32 亿美元
- 买入比亚迪约 10% 的股份（约 2.25 亿股 H 股）
- 买入价格约 8 港元/股

**结果：**

- 比亚迪股价在 2020-2022 年期间一度超过 330 港元
- 这笔投资的峰值价值超过 **100 亿美元**
- 回报率超过 **40 倍**

**减持情况：** 伯克希尔从 2022 年 8 月开始减持比亚迪股份。截至 2024 年 6 月，持股比例已从约 10% 降至约 5% 以下，累计减持套现超过 80 亿港元。减持原因包括：比亚迪估值已充分反映其价值、伯克希尔需要现金进行其他投资、以及税务规划考虑。（来源：港交所披露易、伯克希尔 13F 文件）

芒格后来评价王传福：

> "他是一个天才。我从未见过像他这样的人。"

| 买入价格 | 约 8 港元/股                   |

| 峰值价格 | 超过 330 港元（2022 年）       |

**数据来源：** 伯克希尔·哈撒韦年报（2008-2024）、港交所披露易、《穷查理宝典》（2005）

### 战役五：韦斯科金融公司（Wesco Financial）

1959 年，芒格开始控制韦斯科金融公司（Wesco Financial），这是一家储蓄贷款协会。在芒格的管理下，韦斯科逐渐转型为一家投资控股公司。

**芒格的操作：**

- 用韦斯科的保险浮存金投资股票
- 集中持仓——主要持有可口可乐、吉列、富国银行等
- 长期持有——很多股票持有超过 20 年

**结果：**

- 韦斯科的每股账面价值从 1960 年代的约 1 美元增长到 2011 年的超过 100 美元
- 年化收益率超过 **20%**
- 2011 年，韦斯科被伯克希尔全资收购

这笔投资展示了芒格独立的投资能力——不依赖巴菲特，他同样能取得卓越的回报。

## 投资哲学

### 原则一：多元思维模型（Mental Models）

这是芒格最著名的思想贡献。

**核心观点：** 你必须掌握多个学科的核心模型，才能做出好的投资决策。

> "在手拿锤子的人眼里，世界就像一根钉子。"

芒格认为，大多数投资者只用一种思维模式（通常是财务分析）来看问题，这是远远不够的。他建议掌握以下学科的核心模型：

| 学科   | 核心模型                     |
| ------ | ---------------------------- |
| 数学   | 复利、概率、排列组合         |
| 物理学 | 临界质量、断裂点             |
| 生物学 | 进化、适应、生态位           |
| 心理学 | 认知偏差、激励机制、社会认同 |
| 经济学 | 供需、比较优势、规模效应     |
| 工程学 | 冗余设计、安全边际、反馈循环 |

芒格建议：

> "你不需要成为每个领域的专家。你只需要掌握每个领域最重要的 2-3 个模型，然后把它们组合起来使用。"

### 原则二：好公司比好价格重要

这是芒格对巴菲特最重要的影响。

传统格雷厄姆式投资关注的是"便宜"——用 50 美分买价值 1 美元的东西。但芒格认为：

> "用合理的价格买一家伟大的公司，远胜过用伟大的价格买一家合理的公司。"

他的逻辑是：

- 伟大的公司会随时间创造更多价值
- 便宜的公司可能永远便宜——因为它本身就不够好
- 时间是好公司的朋友，是烂公司的敌人

### 原则三：能力圈

芒格和巴菲特都强调能力圈，但芒格的表述更加深刻：

> "如果你确有能力圈，那它的大小不重要。重要的是你知道它的边界在哪里。"

芒格的能力圈主要集中在：

- 零售（Costco、喜诗糖果）
- 金融（银行、保险）
- 科技（比亚迪）
- 房地产

他明确回避的领域：

- 他不理解的复杂金融产品
- 他无法评估长期竞争力的行业

### 原则四：逆向思考

芒格是逆向思考的大师。他最著名的话：

> "我只想知道我会死在哪里，这样我就永远不去那个地方。"

这种"逆向思考"的方法体现在：

- **不做什么比做什么更重要**——避免愚蠢比追求聪明更有效
- **先考虑风险，再考虑回报**——先想清楚什么会导致失败
- **检查清单**——芒格在做决策前会使用检查清单，确保不遗漏关键因素

### 原则五：耐心与纪律

芒格是耐心和纪律的终极信徒。

> "赚大钱不在于买卖，而在于等待。"

他的耐心体现在：

- 长期持有——很多股票持有超过 20 年
- 不频繁交易——"好的投资机会很少，出现时应该下重注"
- 等待好机会——"我们大部分时间什么都不做，等待好机会出现"

## 策略拆解

### 选股标准

芒格的选股标准可以总结为"三个好"：

| 标准   | 具体要求                                     |
| ------ | -------------------------------------------- |
| 好生意 | 商业模式简单、有护城河、能持续产生自由现金流 |
| 好管理 | 管理层诚信、有能力、以股东利益为导向         |
| 好价格 | 估值合理（不一定便宜）                       |

具体财务指标：

- **ROE**：持续高于 15%
- **毛利率**：稳定或上升
- **自由现金流**：持续为正且增长
- **资本支出**：相对较低
- **客户忠诚度**：高复购率、低流失率

### 买入时机

芒格的买入时机判断：

1. **好公司遇到暂时性困难**——股价被错杀
2. **市场整体恐慌**——好公司被连带抛售
3. **新公司展现出强大竞争力**——如比亚迪

他特别强调：**不要试图择时。** 如果你找到了一家伟大的公司，价格又合理，那就买入。

### 持仓管理

芒格的持仓管理极其简单：

- **集中持仓**——少数几只股票占绝大部分仓位
- **买入后不操作**——除非基本面发生变化
- **不关注短期波动**——"如果你因为股价下跌 50% 就恐慌，你不适合投资股票"

### 卖出纪律

芒格的卖出条件：

1. **基本面恶化**——护城河消失、管理层变质
2. **估值极度泡沫**——价格远超内在价值
3. **发现更好的机会**——换仓到性价比更高的标的

他特别强调：**不要因为股价上涨就卖出。** 如果一家公司还在创造价值，持有它比卖掉它更好。

### 普通人能学到什么

芒格的方法对普通投资者的启示：

1. **多元思维**——不要只用一种方式看问题，学习多个学科的基础知识
2. **避免愚蠢**——先想清楚什么会导致失败，然后避免它
3. **长期持有**——找到好公司，然后耐心持有
4. **持续学习**——芒格被称为"长着两条腿的书架"，他每天阅读数小时
5. **检查清单**——在做重要决策前，使用检查清单确保不遗漏关键因素

**具体建议：**

- 每年读 10-20 本好书（不限于投资）
- 建立自己的检查清单
- 只投资你能理解的生意
- 保持耐心，不要频繁交易
