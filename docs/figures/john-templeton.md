# 约翰·邓普顿——逆向投资之父

> 当全世界都在恐慌性抛售时，他逆着人流买入。他说："牛市在悲观中诞生，在怀疑中成长，在乐观中成熟，在兴奋中死亡。"他不仅是一位投资大师，更是一位虔诚的基督徒——他把投资哲学和宗教信仰完美地融合在了一起。

## 人物速写

约翰·邓普顿，逆向投资理论的奠基人。他用一生证明了：**最赚钱的投资机会，往往出现在最恐慌的时刻。**

| 维度       | 数据                                                                          |
| ---------- | ----------------------------------------------------------------------------- |
| 生卒年     | 1912-2008（享年 95 岁）                                                       |
| 国籍       | 美国（后移居巴哈马）                                                          |
| 核心贡献   | 逆向投资（Contrarian Investing）、全球投资策略                                |
| 代表著作   | 《邓普顿教你逆向投资》（Templeton's Way with Money，1983）                    |
| 投资年限   | 超过 65 年（1937-2002）                                                       |
| 年化收益   | 邓普顿成长基金约 14.5%（1954-1992）（来源：《邓普顿教你逆向投资》及基金年报） |
| 管理规模   | 峰值约 220 亿美元                                                             |
| 最著名交易 | 1939 年二战爆发时买入 100 多只低价股                                          |

**一句话总结：** 他用一生证明，别人恐惧时贪婪，别人贪婪时恐惧，不是口号，而是可以量化执行的策略。

## 传奇人生

### 早年经历：田纳西州的节俭男孩

1912 年 9 月 29 日，约翰·邓普顿出生于美国田纳西州的一个小镇。他的家庭并不富裕——父亲是一名律师，收入勉强维持家用。

邓普顿从小就展现出了两个特点：**极度节俭和极度好奇。**

- 他会把每一分钱都记在账本上，从不乱花
- 他会借遍镇上所有的书，如饥似渴地阅读
- 他会在暑假打工赚钱，而不是像其他孩子一样玩耍

16 岁时，邓普顿进入范德堡大学（Vanderbilt University）。他主修经济学，同时选修了哲学和宗教课程。这段经历对他后来的投资哲学产生了深远影响——**他开始思考财富的意义，以及金钱与幸福的关系。**

1934 年，邓普顿以全班第一的成绩从范德堡大学毕业，获得了耶鲁大学法学院的奖学金。

### 求学与启蒙：从法律到投资

在耶鲁法学院，邓普顿依然保持着极度节俭的生活方式。他会把每一笔开支都记录下来，甚至精确到美分。

1937 年，邓普顿从耶鲁法学院毕业。但他没有选择做律师，而是做了一个让所有人意外的决定：**去华尔街做投资。**

他的理由很简单：**"法律是零和游戏——有人赢就有人输。但投资可以创造价值——所有人都可以赢。"**

邓普顿的第一份工作是在华尔街的一家证券公司做分析师。但他的工作方式与同行截然不同：

- 别人关注热门股，他关注被市场抛弃的低价股
- 别人在美国市场找机会，他开始研究海外市场
- 别人听消息炒股，他坚持自己做独立分析

这种"逆向"的思维方式，让他在早期就展现出了与众不同的投资眼光。

### 投资启蒙：1939 年的"疯狂"决定

1939 年 9 月 1 日，德国入侵波兰，第二次世界大战爆发。

全球市场恐慌性暴跌。投资者疯狂抛售股票，不计代价。华尔街弥漫着绝望的情绪——很多人认为，战争将摧毁全球经济，股票市场可能会归零。

就在这一天，27 岁的邓普顿做了一件让所有人觉得"疯狂"的事：

**他向老板借了 1 万美元，买入了 100 多只价格低于 1 美元的股票。**（来源：《邓普顿教你逆向投资》，1983 年）

这些股票包括多家濒临破产的铁路公司、保险公司和工业企业——它们被市场抛弃，价格跌到了谷底。

他的逻辑是：

- 战争确实会带来破坏，但也会带来重建
- 市场过度恐慌，股价已经远低于内在价值
- 历史上，每次危机之后，市场都会恢复并创新高
- 低价股虽然风险高，但上行空间巨大

当时，几乎所有同事都认为他疯了。有人劝他赶紧止损，有人预言他会破产。

但邓普顿不为所动。他坚信自己的判断：**"别人恐慌时，我要贪婪。"**

结果如何？

- 1939-1943 年，市场逐渐恢复，邓普顿买入的低价股大幅上涨
- 他的 1 万美元投资，最终变成了约 4 万美元——**4 年赚了 4 倍**
- 这笔交易奠定了他"逆向投资之父"的地位

更重要的是，**这笔交易验证了邓普顿一生坚守的投资信念：最赚钱的机会，往往出现在最恐慌的时刻。**

> 生活化类比：就像疫情期间大家都在抛售机票酒店股票，你却偷偷买入。一年后疫情结束，旅游反弹，你赚了好几倍。邓普顿就是那个在"所有人都在逃命"时，逆着人流走进去的人。

### 关键转折点：创立邓普顿基金

1954 年，邓普顿与合伙人创立了邓普顿成长基金（Templeton Growth Fund）。

这只基金的投资策略与当时所有基金都不同：

- **逆向投资**——在市场恐慌时买入，在市场乐观时卖出
- **全球投资**——不局限于美国市场，而是全球寻找机会
- **低价股策略**——关注被市场抛弃的便宜股票
- **长期持有**——不频繁交易，给市场时间发现价值

邓普顿基金的业绩证明了这个策略的有效性。从 1954 年到 1992 年，邓普顿成长基金的年化收益率约为 **14.5%**，远超同期市场平均水平（来源：《邓普顿教你逆向投资》及基金年报）。

### 巅峰时刻：日本股市投资

1960-1970 年代，邓普顿做了一笔让他名声大噪的投资——**重仓日本股市。**

当时，日本经济正在快速复苏，但全球投资者对日本市场仍然持怀疑态度。邓普顿通过深入研究，发现：

- 日本公司的市盈率普遍低于美国公司
- 日本经济增长速度远超全球平均水平
- 日本企业的竞争力在不断增强
- 日本股市被严重低估

基于这些判断，邓普顿开始大量买入日本股票。当时，大多数同行认为他"疯了"——日本市场流动性差、信息不透明、文化差异大，为什么要去那里投资？

但邓普顿坚持自己的判断。结果：

- 1960-1980 年代，日本股市大幅上涨，日经指数从约 400 点涨到 1989 年的近 39000 点
- 邓普顿的日本投资获得了数十倍的回报（来源：《邓普顿教你逆向投资》）
- 邓普顿基金因此成为全球表现最好的基金之一

这笔投资不仅让邓普顿赚到了钱，更重要的是，**它开创了全球投资的先河。** 在邓普顿之前，几乎没有美国投资者会认真考虑投资日本市场。邓普顿用行动证明了：**机会不分国界，关键是你愿不愿意去寻找。**

### 宗教信仰与投资的融合

邓普顿是一位虔诚的基督徒。他一生都在思考投资与信仰的关系。

他认为：

- **投资是一种服务**——帮助人们实现财务目标，是一种善举
- **贪婪是投资的大敌**——过度追求回报会导致错误决策
- **感恩是投资的基石**——对市场保持敬畏，对收益保持感恩
- **慈善是财富的意义**——财富的最终目的是帮助他人

1972 年，邓普顿移居巴哈马。他给出的理由是：**"那里更适合我思考和祈祷。"**

在巴哈马，邓普顿继续管理他的基金，同时投入大量时间从事慈善事业。他创立了邓普顿基金会（Templeton Foundation），资助宗教、科学和慈善领域的研究。

2008 年 4 月 8 日，邓普顿在巴哈马去世，享年 95 岁。他一生积累的巨额财富，大部分都捐给了慈善事业。

## 经典战役

### 战役一：1939 年二战爆发——逆向投资的经典

1939 年 9 月，二战爆发，全球市场恐慌性暴跌。

邓普顿的行动：

- 向老板借了 1 万美元
- 买入了 100 多只价格低于 1 美元的股票，包括多家濒临破产的铁路公司、保险公司和工业企业
- 这些股票大多是被市场抛弃的"垃圾股"

他的逻辑：

- 市场过度恐慌，股价远低于内在价值
- 战争会带来破坏，但也会带来重建
- 历史上，每次危机之后市场都会恢复
- 低价股虽然风险高，但上行空间巨大

**结果：**

| 指标     | 数值                 |
| -------- | -------------------- |
| 投资金额 | 1 万美元             |
| 投资时间 | 1939 年              |
| 持有时间 | 约 4 年              |
| 最终价值 | 约 4 万美元          |
| 回报率   | 约 300%（4 年 4 倍） |

这笔交易是逆向投资最经典的案例之一，也是邓普顿投资生涯的起点（来源：《邓普顿教你逆向投资》，1983 年）。

### 战役二：日本股市——全球投资的先驱

1960 年代，邓普顿开始重仓日本股市。

当时日本市场的情况：

- 日本经济正在快速复苏
- 但全球投资者对日本市场持怀疑态度
- 日本公司市盈率普遍低于美国公司
- 日本股市流动性差、信息不透明

邓普顿的判断：

- 日本经济增长速度远超全球平均水平
- 日本企业竞争力在不断增强
- 日本股市被严重低估
- 风险可控，回报可观

**结果：**

| 指标     | 数值                                   |
| -------- | -------------------------------------- |
| 投资时间 | 1960 年代开始                          |
| 持有时间 | 20+ 年                                 |
| 回报率   | 数十倍（来源：《邓普顿教你逆向投资》） |
| 影响     | 开创了全球投资的先河                   |

### 战役三：1970 年代石油危机——逆向买入

1970 年代，石油危机爆发，全球市场再次恐慌。

邓普顿的行动：

- 在市场恐慌时买入被抛售的股票
- 特别关注能源和原材料行业的低价股
- 坚持长期持有，不因为短期波动而改变策略

他的逻辑：

- 石油危机是周期性事件，不是永久性破坏
- 市场过度反应，创造了买入机会
- 能源和原材料行业长期前景依然看好

**结果：** 邓普顿基金在 1970 年代表现优异，年化收益远超市场平均水平。

### 战役四：1987 年黑色星期一——恐慌中买入

1987 年 10 月 19 日，"黑色星期一"，道琼斯指数单日暴跌 22.6%，创下历史最大单日跌幅。

全球投资者恐慌性抛售，市场弥漫着绝望情绪。

邓普顿的反应是什么？**买入。**

他在回忆中说：

> "那天，我打电话给交易员，告诉他：'买，买，买。'所有人都觉得我疯了。但我知道，这是几十年一遇的机会。"

**结果：** 市场在之后几个月内迅速恢复，邓普顿的买入决策为他带来了丰厚的回报。

## 投资哲学

### 原则一：逆向投资——在别人恐慌时买入

这是邓普顿最著名的投资原则。

**逆向投资（Contrarian Investing）的核心逻辑是：** 市场情绪往往是反向指标。当所有人都乐观时，市场可能已经见顶；当所有人都悲观时，市场可能已经见底。

邓普顿的逆向投资方法：

1. **识别市场情绪**——通过新闻、成交量、估值等指标判断市场情绪
2. **逆向思考**——当市场极度悲观时，考虑买入；当市场极度乐观时，考虑卖出
3. **独立判断**——不盲从市场共识，坚持自己的分析
4. **耐心等待**——逆向投资需要时间，市场可能需要很久才能"纠正"

邓普顿的名言：

> "牛市在悲观中诞生，在怀疑中成长，在乐观中成熟，在兴奋中死亡。"

这句话完美概括了市场情绪与投资回报的关系。

> 生活化类比：就像去菜市场买菜，傍晚时分摊主急着收摊，菜价便宜得离谱。这时候买入，第二天早上转手就能赚钱。邓普顿就是那个专门在"傍晚"去买菜的人。

### 原则二：全球投资——机会不分国界

邓普顿是全球投资的先驱。他认为：

> "世界上有那么多国家，那么多公司，为什么要把自己局限在一个市场里？"

邓普顿的全球投资方法：

- **不局限于美国市场**——全球寻找被低估的机会
- **关注新兴市场**——这些市场往往被忽视，但增长潜力巨大
- **分散国别风险**——不把所有资金投入一个国家
- **深入了解当地市场**——不是盲目投资，而是深入研究

在邓普顿之前，几乎没有美国投资者会认真考虑投资日本、欧洲以外的市场。邓普顿用行动证明了：**机会不分国界，关键是你愿不愿意去寻找。**

### 原则三：低价股策略——在垃圾中找黄金

邓普顿是低价股投资的先驱。他认为：

> "最便宜的东西，往往有最大的上行空间。"

邓普顿的低价股策略：

- **关注价格低于 1 美元的股票**——这些股票往往被市场抛弃
- **寻找基本面没有恶化的公司**——价格下跌是因为市场情绪，不是因为公司出了问题
- **分散投资**——低价股风险高，需要分散投资来降低风险
- **长期持有**——给市场时间发现价值

### 原则四：长期持有——时间是朋友

邓普顿是长期持有的坚定信仰者。他认为：

> "投资不是赌博，而是用时间和耐心换取回报。"

邓普顿的长期持有逻辑：

- **市场需要时间发现价值**——便宜的东西不会立刻变贵
- **频繁交易增加成本**——交易费用和税收会侵蚀回报
- **长期持有降低风险**——短期波动会在长期中被平滑
- **复利的力量**——时间越长，复利效应越明显

### 原则五：投资与信仰的融合

邓普顿是一位虔诚的基督徒。他把投资哲学和宗教信仰完美地融合在一起。

他认为：

- **投资是一种服务**——帮助人们实现财务目标
- **贪婪是投资的大敌**——过度追求回报会导致错误决策
- **感恩是投资的基石**——对市场保持敬畏
- **慈善是财富的意义**——财富的最终目的是帮助他人

## 策略拆解

### 选股标准

邓普顿的选股标准可以概括为"三低一高"：

| 标准     | 具体要求                               |
| -------- | -------------------------------------- |
| 低价格   | 股价低于 1 美元或远低于历史高点        |
| 低估值   | 市盈率、市净率低于行业平均水平         |
| 低关注度 | 不被市场关注，没有被分析师覆盖         |
| 高潜力   | 公司基本面没有恶化，有恢复或增长的潜力 |

邓普顿特别强调：**低价不等于低质。** 他寻找的是"被市场错误定价的好公司"，而不是"本身就烂的公司"。

### 买入时机

邓普顿的买入时机判断：

- **市场极度悲观时**——恐慌性抛售创造买入机会
- **个股被错误定价时**——价格远低于内在价值
- **全球市场出现危机时**——危机往往创造最好的买入机会
- **不追求最低点**——分批买入，不试图买在最低点

### 持仓管理

邓普顿的持仓管理原则：

- **买入后不频繁交易**——给市场时间发现价值
- **定期复查基本面**——确认投资逻辑仍然成立
- **不因短期波动而改变策略**——保持耐心
- **全球分散**——不把所有资金投入一个国家或行业

### 卖出纪律

邓普顿的卖出条件：

1. **价格达到或超过内在价值**——获利了结
2. **基本面恶化**——投资逻辑不再成立
3. **发现更好的机会**——换仓到性价比更高的标的
4. **市场极度乐观时**——逆向卖出

他特别强调：**不要因为市场恐慌就卖出，也不要因为市场乐观就买入。** 逆向投资的核心是"反着来"。

### 普通人能学到什么

邓普顿的方法对普通投资者的启示：

1. **逆向思考**——当所有人都恐慌时，考虑买入；当所有人都乐观时，考虑卖出
2. **全球视野**——不要局限于本国市场，全球寻找机会
3. **长期持有**——减少交易频率，让时间成为你的朋友
4. **独立判断**——不盲从市场共识，坚持自己的分析
5. **感恩与慈善**——财富的最终目的是帮助他人

**具体建议：**

- 在市场恐慌时（如金融危机、疫情等），考虑买入被错杀的优质资产
- 关注全球市场，特别是新兴市场
- 用指数基金实现全球分散投资
- 保持耐心，不要因短期波动而改变长期策略
- 定期做慈善，回馈社会
