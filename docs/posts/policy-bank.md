# 政策性银行

如果说商业银行是"赚钱的银行"，那政策性银行就是"干活的银行"。它们不追求利润最大化，而是执行国家战略——修高铁、建水利、支持农业、促进进出口。这些项目周期长、风险高、利润薄，商业银行不愿意干，但国家必须有人干。

## 概述

政策性银行（Policy Bank）是由政府创立、出资或担保，以贯彻国家产业政策和区域发展政策为目标的特殊金融机构。与商业银行不同，政策性银行不以盈利为主要目的，而是服务于国家宏观经济政策。

核心特征：
- **政策导向**：业务方向由国家政策决定，不是市场利润驱动
- **中长期融资**：主要提供中长期贷款，支持基础设施和战略性产业
- **政府信用背书**：债券享有准主权信用评级，融资成本低
- **不吸收公众存款**：资金主要通过发行金融债券筹集

全球主要的政策性银行：
- 中国：国家开发银行、中国进出口银行、中国农业发展银行
- 日本：日本政策投资银行（DBJ）
- 德国：德国复兴信贷银行（KfW）
- 韩国：韩国产业银行（KDB）

## 中国的三大政策性银行

### 国家开发银行（CDB）

**定位**：支持国家基础设施、基础产业、支柱产业（"两基一支"）的中长期融资。

**业务范围**：
- 铁路、公路、机场、港口等交通基础设施
- 电力、石油、天然气等能源项目
- 城镇化建设、棚户区改造
- "一带一路"项目融资

**关键数据**：截至 2023 年末，资产总额超过 18 万亿元，是全球最大的开发性金融机构（来源：国家开发银行年报，2023）。

### 中国进出口银行（Exim Bank）

**定位**：支持对外经贸发展和跨境投资。

**业务范围**：
- 出口信贷：支持中国设备、技术、服务出口
- 进口信贷：支持资源性产品、先进技术进口
- 对外承包工程贷款
- 境外投资贷款

**关键数据**：2023 年发放贷款超过 3 万亿元，支持了数千个进出口项目（来源：中国进出口银行年报，2023）。

### 中国农业发展银行（ADBC）

**定位**：支持"三农"（农业、农村、农民）发展。

**业务范围**：
- 粮食收购贷款：保障国家粮食安全
- 农业农村基础设施建设
- 扶贫贷款：支持脱贫攻坚和乡村振兴
- 农业综合开发贷款

**关键数据**：2023 年末贷款余额超过 7 万亿元，是中国唯一的农业政策性银行（来源：中国农业发展银行年报，2023）。

## 为什么重要

政策性银行在经济中扮演"补位者"角色：

- **市场失灵时的补充**：商业银行追求利润，对周期长、风险高的项目回避。政策性银行填补这一空白
- **逆周期调节工具**：经济下行期，政策性银行加大投放，刺激经济；经济过热期，收缩投放
- **国家战略的执行者**："一带一路"、乡村振兴、新型城镇化等国家战略，政策性银行是主要资金提供方
- **降低融资成本**：政府信用背书使政策性银行能以较低成本发债融资，进而提供低息贷款

## 生活化类比

政策性银行就像小区里的"物业维修基金"。

小区里，路灯坏了、管道漏了、外墙脱落了——这些事每家每户都受益，但没人愿意单独掏钱修。物业维修基金就是大家提前交的钱，专门用于这些"公共事务"。

政策性银行也一样。修一条高铁，沿线城市都受益，但商业银行觉得投资回收期太长（20-30 年）、利润太薄，不愿意贷。这时候政策性银行站出来——它不以赚钱为目的，而是为了让整个国家受益。

区别在于，物业维修基金用的是业主的钱，政策性银行用的是国家信用借来的钱（发行政策性金融债），最终由全社会受益。

## 运作机制

### 资金来源

政策性银行不吸收公众存款，资金主要来自：

| 来源 | 说明 | 占比 |
| ---- | ---- | ---- |
| 政策性金融债 | 在银行间市场发行债券 | 约 80% |
| 财政拨款 | 政府注资和利润留存 | 约 10% |
| 国际借款 | 向世界银行、亚行等借款 | 约 5% |
| 其他 | 存款、同业拆借等 | 约 5% |

### 资金运用

| 领域 | 典型项目 | 贷款期限 |
| ---- | -------- | -------- |
| 基础设施 | 高铁、高速公路、机场 | 10-30 年 |
| 能源 | 电站、油气管道 | 10-20 年 |
| 农业 | 粮食收购、水利建设 | 1-10 年 |
| 进出口 | 出口买方信贷、境外投资 | 3-15 年 |
| 城镇化 | 棚改、城市更新 | 5-20 年 |

### 与商业银行的区别

| 维度 | 政策性银行 | 商业银行 |
| ---- | ---------- | -------- |
| 目标 | 执行政策，服务国家战略 | 盈利最大化 |
| 资金来源 | 发行债券为主 | 吸收存款为主 |
| 贷款对象 | 大型项目、国家战略 | 企业和个人 |
| 利率 | 通常低于市场利率 | 市场化定价 |
| 风险承担 | 政府隐性担保 | 自担风险 |

## 实战应用

### 对企业和投资者的意义

**企业融资**：
- 符合条件的大型项目可申请政策性银行贷款，利率更低、期限更长
- "一带一路"项目、绿色能源、乡村振兴等领域优先支持

**债券投资**：
- 政策性金融债（"政金债"）享有准主权信用，风险极低
- 收益率略高于国债，是稳健投资的选择
- 流动性好，银行间市场交易活跃

**宏观经济判断**：
- 政策性银行信贷投放是观察国家政策方向的窗口
- 投放加速 = 政策宽松、基建发力
- 投放放缓 = 政策收紧、去杠杆

### 中国投资者特别提示

- **政金债**：可通过银行柜台或债券基金间接投资，门槛较低
- **政策性银行 ETF**：部分债券 ETF 主要配置政金债
- **关注年报**：三大政策性银行每年发布年报，披露投放方向和规模

## 关键要点

- 政策性银行不以盈利为目的，而是执行国家战略
- 中国有三大政策性银行：国开行、进出口银行、农发行
- 资金主要通过发行金融债券筹集，不吸收公众存款
- 政金债享有准主权信用，是稳健投资品种
- 政策性银行信贷投放是观察国家政策方向的重要指标
- 2015 年后国开行定位为"开发性金融机构"，但功能与政策性银行类似

## 常见误区

- **"政策性银行就是商业银行"**——目标不同，政策性银行服务国家战略，不追求利润最大化
- **"政策性银行亏钱"**——虽然不以盈利为目的，但大多数政策性银行仍能保持微利或盈亏平衡
- **"政策性银行只给国企贷款"**——也支持民企和外资企业，只要项目符合国家政策方向
- **"政策性银行风险很高"**——有政府信用背书，违约风险极低，但需关注项目质量
