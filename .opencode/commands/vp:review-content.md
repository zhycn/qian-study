description: 审查文档内容质量
agent: build
---

对 VitePress 文档内容进行深度质量审查。

如果用户未指定范围，询问：

1. 审查全部文档还是指定文件？
2. 重点关注什么？（准确性、完整性、语言、格式）

审查维度：

1. 内容准确性
   - 技术描述是否正确
   - 代码示例是否可运行
   - API 文档是否与代码一致
2. 内容完整性
   - 是否有未完成的章节
   - 是否有 TODO 标记
   - 关键信息是否缺失
3. 语言一致性
   - 是否统一使用中文
   - 术语是否一致
   - 语气是否统一
4. 格式规范性
   - 标题层级是否正确
   - 代码块是否标注语言
   - 链接格式是否统一
5. 内容时效性
   - 是否有过时的信息
   - 版本号是否需要更新

生成审查报告，按严重程度列出问题。

示例：`/vp:review-content` 或 `/vp:review-content docs/guide/`
