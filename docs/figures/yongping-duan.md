# 段永平——中国巴菲特

> 他是中国最被低估的投资大师。作为企业家，他创立了小霸王、步步高，幕后孵化了 OPPO 和 vivo；作为投资者，他在 2002 年网易濒临退市时大举买入，获利超过 100 倍。他叫段永平，一个把"本分"做到极致的人。

## 人物速写

段永平，企业家转型投资者的典范。他创立的步步高系（OPPO、vivo）是中国智能手机市场的两大巨头，而他个人的投资业绩更是令人瞩目——2002 年投资网易获利超 100 倍，长期持有苹果获得丰厚回报。

## 传奇人生

### 早年经历：从江西到广东

1961 年，段永平出生于江西南昌一个普通知识分子家庭。父亲是江西工学院（现南昌大学）的教师，母亲是中学教师。

段永平从小就展现出了对电子技术的浓厚兴趣。1978 年，他考入浙江大学无线电工程专业。在大学期间，他不仅学习成绩优异，还积极参与各种电子制作活动。

1982 年大学毕业后，段永平被分配到北京电子管厂工作。但他很快发现，国营体制下的工作节奏和创新空间都无法满足他的野心。

1988 年，段永平做出了一个改变人生的决定：**南下广东。**

### 投资启蒙：小霸王的崛起

1989 年，段永平加入了一家名为"日华电子"的小厂，这是一家生产游戏机配件的企业。段永平凭借自己的技术背景和管理才能，很快成为了公司的核心人物。

1991 年，段永平主导推出了"小霸王学习机"——一款基于任天堂 FC 游戏机兼容技术的产品。这款产品在中国市场大获成功：

- 1993 年，小霸王学习机销量超过 100 万台
- 1995 年，小霸王产值超过 10 亿元
- 段永平个人也因此获得了丰厚的回报

但好景不长。由于产权问题，段永平与日华电子的矛盾日益激化。1995 年，段永平选择离开小霸王，在东莞创立了**步步高电子**。

### 关键转折点：步步高的辉煌

1995 年创立步步高后，段永平带领公司进入了无绳电话、VCD、学习机等多个领域。

**步步高的成功产品：**
- 无绳电话——中国市场第一
- VCD/DVD——中国市场前三
- 学习机——延续小霸王时代的优势

2001 年前后，段永平做出了一个让所有人意外的决定：**退休。**

当时的步步高正处于巅峰期，但段永平认为：
- 公司已经建立了完善的管理体系
- 年轻的管理团队可以独立运营
- 他自己需要寻找新的人生方向

段永平将步步高拆分为三家独立运营的公司：
1. **步步高教育电子**——继续做学习机等产品
2. **OPPO**——由陈明永负责，主攻智能手机
3. **vivo**——由沈炜负责，主攻智能手机

段永平作为幕后股东，不再参与日常运营。他移居美国，开始了自己的投资生涯。

### 巅峰时刻：从企业家到投资者

2001 年移居美国后，段永平开始系统学习投资。他阅读了大量巴菲特和芒格的著作，逐渐形成了自己的投资框架。

2002 年，他做出了投资生涯最重要的一笔交易——**买入网易。**

此后，段永平的投资版图不断扩大：
- 2000 年代：投资网易、搜狐等中国互联网公司
- 2010 年代：大举投资苹果，成为其坚定持有者
- 2020 年代：投资拼多多（黄峥是段永平的"门徒"）、腾讯等

段永平在雪球上以"大道无形我有型"为 ID 活跃多年，分享自己的投资心得。他的发言简洁、直接、充满智慧，被投资者视为"中国版巴菲特"的言论。

## 经典战役

### 战役一：投资网易（2002）

这是段永平投资生涯最经典的案例，也是中国投资史上最成功的逆向投资之一。

**背景：**
- 2001-2002 年，互联网泡沫破裂，纳斯达克指数暴跌超过 70%
- 网易因财务问题被纳斯达克停牌，股价从高峰期的超过 30 美元暴跌到 **0.8 美元** 以下
- 市场普遍认为网易可能退市，甚至破产
- 丁磊一度想卖掉网易

**段永平的逻辑：**
- 他仔细研究了网易的财务状况，发现公司账上现金超过 2 亿美元
- 每股现金超过 2 美元，而股价只有不到 1 美元——**股价低于现金**
- 网易的核心业务（游戏和门户）依然在运营
- 丁磊是一个有能力的创业者

段永平后来回忆：

> "我当时算了一笔账：网易账上有 2 亿多美元现金，总股本约 1 亿股，每股现金超过 2 美元。而股价只有 0.8 美元。这意味着你花 0.8 美元就能买到 2 美元的现金，还白送一个公司。"

段永平投入了约 **200 万美元**，以每股不到 1 美元的价格买入了网易股票。

**结果：**
- 2002 年底，网易恢复交易，股价开始回升
- 2003 年，网易凭借游戏业务（《大话西游》）实现盈利
- 2004 年，网易股价超过 30 美元
- 段永平在 20-30 美元区间逐步卖出，获利超过 **100 倍**
- 这笔投资让段永平获利超过 **1 亿美元**

| --- | --- |
| 买入时间 | 2002 年 |
| 买入价格 | 不到 1 美元/股（最低约 0.8 美元） |

| 卖出价格 | 20-30 美元/股 |

### 战役二：投资苹果（2011-至今）

段永平是苹果最早的坚定持有者之一，他的投资逻辑与巴菲特不谋而合。

**背景：**
- 2011 年前后，苹果 iPhone 已经改变了手机行业
- 但市场对苹果的争议很大：硬件公司能否持续创新？
- 苹果市盈率只有 10-15 倍，被很多投资者认为"太贵"或"不可持续"

**段永平的逻辑：**
- 苹果拥有强大的品牌忠诚度和生态系统
- iPhone 的换机周期长，用户粘性极高
- 苹果的现金流极其强劲，每年产生数百亿美元自由现金流
- 库克的管理能力被市场低估

段永平在 2011 年前后开始大量买入苹果股票，并在之后的多年里持续持有。

**结果：**
- 苹果股价从 2011 年的约 50 美元（拆股调整后）涨到 2024 年的超过 200 美元
- 加上分红，回报率超过 **10 倍**
- 苹果成为段永平持仓中最重要的标的之一

| 指标 | 数值 |
| --- | --- |
| 买入时间 | 2011 年前后 |
| 买入价格 | 约 50 美元（拆股调整后） |
| 当前价格 | 超过 200 美元（2024 年） |
| 回报率 | 超过 10 倍（含分红） |
| 持有时间 | 10+ 年 |

### 战役三：投资拼多多（2018）

段永平与拼多多创始人黄峥的关系，是中国投资界"师徒传承"的佳话。

**背景：**
- 黄峥是段永平的"门徒"，曾跟随段永平学习投资和商业
- 2018 年，拼多多在争议中上市，被质疑"低端""假货"
- 市场对拼多多的商业模式存在巨大分歧

**段永平的逻辑：**
- 他了解黄峥的能力和人品
- 拼多多的社交电商模式有创新性
- 中国下沉市场的消费潜力被低估
- 黄峥的执行力极强

段永平在拼多多上市后持续买入，成为其重要股东。

**结果：**
- 拼多多股价从上市初期的约 20 美元涨到 2024 年的超过 150 美元
- 拼多多成为中国第三大电商平台
- 段永平的投资回报超过 **5 倍**

| 指标 | 数值 |
| --- | --- |
| 买入时间 | 2018 年（上市后） |
| 买入价格 | 约 20 美元 |
| 当前价格 | 超过 150 美元（2024 年） |
| 回报率 | 超过 5 倍 |
| 与创始人关系 | 黄峥是段永平的"门徒" |

### 战役四：投资茅台（2013-2014）

段永平在 2013-2014 年白酒行业低谷时买入贵州茅台，是又一个经典的逆向投资案例。

**背景：**
- 2012-2013 年，受"八项规定"影响，白酒行业遭遇寒冬
- 茅台股价从 2012 年的高点下跌超过 50%
- 市场普遍认为白酒行业的"黄金时代"已经结束

**段永平的逻辑：**
- 茅台的品牌价值不会因为短期政策变化而消失
- 白酒是中国文化的一部分，需求是长期的
- 茅台的护城河（品牌 + 产地 + 工艺）无法复制
- 估值已经回到合理区间

**结果：**
- 茅台股价从 2014 年的约 100 元涨到 2021 年的超过 2600 元
- 回报率超过 **20 倍**
- 茅台成为 A 股市值最高的公司之一

| 指标 | 数值 |
| --- | --- |
| 买入时间 | 2013-2014 年 |
| 买入价格 | 约 100-150 元 |
| 峰值价格 | 超过 2600 元（2021 年） |
| 回报率 | 超过 20 倍 |
| 持有时间 | 多年 |

### 战役五：与巴菲特共进午餐（2006）

2006 年，段永平以 **62 万美元** 拍下了与巴菲特共进午餐的机会。

**背景：**
- 这是中国企业家第一次参加巴菲特午餐
- 段永平带了当时 26 岁的黄峥（拼多多创始人）一同前往
- 这次午餐对段永平和黄峥都产生了深远影响

**午餐收获：**
- 段永平后来回忆，午餐本身并没有获得具体的投资建议
- 但这次经历让他更加坚定了自己的投资理念
- 对黄峥来说，这次经历影响了他后来的创业和投资理念

段永平评价这次午餐：

> "和巴菲特吃饭最大的收获是，他让我更加确信自己做的事情是对的。投资不需要复杂的东西，只需要坚持做对的事。"

| 同行人员 | 黄峥（拼多多创始人） |

## 投资哲学

### 原则一：本分

"本分"是段永平投资和企业经营的核心哲学。

> "本分就是做对的事，把事做对。"

段永平的"本分"包含三层含义：
1. **做对的事**——选择正确的方向，不投机、不取巧
2. **把事做对**——在执行层面做到极致
3. **不对的事坚决不做**——即使短期有诱惑，也不做违背原则的事

在投资中，"本分"意味着：
- 不懂不投
- 不借钱投资
- 不做空
- 不频繁交易

### 原则二：平常心

段永平强调保持"平常心"。

> "平常心就是不被市场情绪左右，坚持自己的判断。"

平常心体现在：
- 不因股价上涨而兴奋
- 不因股价下跌而恐慌
- 不因市场噪音而改变自己的判断
- 坚持长期视角

### 原则三：能力圈

段永平严格遵守能力圈原则。

> "投资最重要的是知道自己不懂什么。"

他的能力圈主要集中在：
- 互联网（网易、腾讯等）
- 消费电子（苹果、OPPO、vivo）
- 消费品（茅台等）
- 他深度了解的行业

他明确回避的领域：
- 他不理解的复杂金融产品
- 他无法评估长期竞争力的行业

### 原则四：长期持有

段永平是长期持有的坚定践行者。

> "如果你不想持有一只股票 10 年，那就不要持有它 10 分钟。"

他的长期持有逻辑：
- 好公司的价值会随时间增长
- 频繁交易产生摩擦成本
- 找到好公司很难，不应该轻易卖掉

### 原则五：独立思考

段永平强调独立思考的重要性。

> "投资最重要的是独立思考，不要人云亦云。"

他建议投资者：
- 自己做研究，不要依赖别人的推荐
- 形成自己的判断框架
- 不盲目跟随市场热点
- 保持理性和客观

## 策略拆解

### 选股标准

段永平的选股标准可以总结为"三个好"：

| 标准 | 具体要求 |
| --- | --- |
| 好生意 | 商业模式简单易懂，有持久竞争优势 |
| 好管理 | 管理层诚信、有能力、以股东利益为导向 |
| 好价格 | 估值合理或偏低，有安全边际 |

具体关注点：
- **商业模式**：简单、可理解、可持续
- **护城河**：品牌、网络效应、转换成本
- **现金流**：自由现金流强劲
- **管理层**：诚信、有能力、长期导向

### 买入时机

段永平的买入时机判断：

1. **好公司遇到暂时困难**——股价被错杀
2. **市场恐慌**——好公司被连带抛售
3. **估值回到合理区间**——价格低于内在价值
4. **发现新的好公司**——在能力圈内发现新机会

他特别强调：**不要因为市场热点而买入。** 只买你真正理解的好公司。

### 持仓管理

段永平的持仓管理极其简单：

- **集中持仓**——少数几只股票占大部分仓位
- **买入后不操作**——除非基本面发生变化
- **不关注短期波动**——"股价涨跌不重要，重要的是公司好不好"
- **持续跟踪**——定期关注公司基本面变化

### 卖出纪律

段永平的卖出条件：

1. **基本面恶化**——公司核心竞争力丧失
2. **发现更好的机会**——换仓到性价比更高的标的
3. **估值极度泡沫**——价格远超内在价值
4. **投资逻辑证伪**——当初买入的理由不再成立

他特别强调：**不要因为股价上涨就卖出。** 如果公司还在创造价值，就应该继续持有。

### 普通人能学到什么

段永平的方法对普通投资者特别友好：

1. **不懂不投**——只投资你能理解的生意
2. **保持耐心**——找到好公司后耐心持有
3. **独立思考**——不要人云亦云
4. **本分做事**——做对的事，把事做对
5. **平常心**——不被市场情绪左右

**具体建议：**
- 选择 3-5 家你真正理解的好公司
- 以 5 年以上的视角持有
- 每年检查一次基本面
- 保持阅读和学习的习惯
- 不借钱投资，不做空
