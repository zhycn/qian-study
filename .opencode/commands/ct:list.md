查看金融词条目录的状态。**仅做展示和查看**，修复不一致请用 `/ct:structure`。

## 执行步骤

### 步骤 1：读取索引

用 Read 工具读取 `docs/glossary/index.md`，解析分类和词条列表。分类名从 `## 分类名` 二级标题获取。

### 步骤 2：扫描实际文件

用 Glob 工具扫描 `docs/` 下各分类目录的 `*.md` 文件。

### 步骤 3：对比分析

对比 index.md 与实际文件，分为四类（与 ct:stats 保持一致）：

| 状态   | 判断条件                                   | 标记  |
| ------ | ------------------------------------------ | ----- |
| 已完成 | 文件存在且正文字数 > 1000                  | `[x]` |
| 进行中 | 文件存在且正文字数 100-1000                | `[~]` |
| 未开始 | 文件存在但正文字数 < 100（仅 frontmatter） | `[ ]` |
| 仅索引 | index.md 列出但文件不存在                  | `[-]` |

字数统计方法：剔除 frontmatter 后的正文字数（使用 `awk '/^---$/{c++; next} c==2' 文件 | wc -m`）。

### 步骤 4：输出清单

```markdown
## 词条状态总览

### 基础概念（8/10 完成）

- [x] 什么是金融（1,200 字）
- [x] 货币（1,500 字）
- [~] 信用（500 字，进行中）
- [ ] 借贷（仅 frontmatter）
- [-] 利率（文件不存在）

### 货币与银行（6/10 完成）

...
```

## 工具绑定

| 步骤     | 工具                |
| -------- | ------------------- |
| 读取索引 | Read                |
| 扫描文件 | Glob                |
| 字数统计 | Bash（awk + wc -m） |
| 对比分析 | Read + Glob         |

## 容错

- index.md 解析失败 → 改用 Glob 扫描所有 `.md` 文件，按目录推断分类
- 无词条 → 输出"词条目录为空"
- 用户要求修复 → 明确提示使用 `/ct:structure` 命令
- index.md 与 sidebar 不一致 → 提示用户使用 `/ct:structure` 同步
