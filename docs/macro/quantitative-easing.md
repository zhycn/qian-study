# 量化宽松

> 央行通过大规模购买资产向市场"撒钱"的非常规货币政策

## 概述

量化宽松（Quantitative Easing，简称 QE）是中央银行在常规货币政策（如降息）失效时采取的一种非常规货币政策。央行通过创造新货币，大规模购买国债、企业债、MBS（Mortgage-backed Securities，抵押贷款支持证券）等金融资产，向金融体系注入大量流动性。

通俗理解：当降息已经降到零了还没用，央行就直接"印钱"去买债券，把钱硬塞进市场里。

QE 的核心机制是：央行购买资产 → 资产价格上涨 → 收益率下降 → 长期利率降低 → 刺激借贷和投资 → 提振经济。同时，央行资产负债表大幅扩张。

## 演进历史

QE 并非 2008 年才出现，它的"鼻祖"是日本。

**2001-2006 年**（来源：维基百科，2001-2006）：日本银行（BOJ）在全球首次实施 QE，以应对通缩和经济停滞。当时日本购买了大量长期国债，将银行在央行的准备金从 5 万亿日元提高到 35 万亿日元（来源：日本银行，2001-2006）。效果有限，但开创了先河。

**2008-2014 年**：金融危机后，美联储开启三轮 QE：

- QE1（2008.11-2010.3）（来源：美联储，2008）：购买 1.75 万亿美元（来源：美联储，2008）MBS 和国债
- QE2（2010.11-2011.6）（来源：美联储，2010）：购买 6000 亿美元（来源：美联储，2010）国债
- QE3（2012.9-2014.10）（来源：美联储，2012）：开放式购买，每月 850 亿美元（来源：美联储，2012），直到就业市场改善

同期，欧洲央行（ECB）和英国央行（BOE）也启动了各自的 QE 计划。

**2017-2019 年**（来源：美联储，2017-2019）：美联储开始"缩表"（Quantitative Tightening，QT），逐步缩减资产负债表。但 2019 年（来源：美联储，2019）回购市场动荡迫使美联储暂停缩表。

**2020 年**（来源：维基百科，2020）：新冠疫情爆发后，美联储在短短两个月内购买了约 3 万亿美元（来源：美联储，2020）资产，规模超过前三轮 QE 总和。欧洲央行推出 PEPP（大流行紧急购买计划）。

**2022 年至今**（来源：美联储，2022）：为应对高通胀，美联储和欧洲央行开启激进缩表，全球流动性进入收缩周期。

## 核心分类

QE 可以根据购买资产类型和操作方式分类：

**按购买资产类型**：

- **国债购买**：最常见的方式，压低长期利率
- **MBS 购买**：美联储在 QE1 和 QE3 中大量购买，支持房地产市场
- **企业债购买**：疫情中美联储首次购买企业债，直接支持实体经济
- **ETF 购买**：日本银行独特做法，直接购买股票 ETF，支撑股市

**按操作方式**：

- **封闭式 QE**：设定总规模和期限，如 QE1、QE2
- **开放式 QE**：不设上限，直到达到特定目标，如 QE3
- **收益率曲线控制（YCC）**：日本银行做法，承诺将 10 年期国债收益率控制在 0% 附近

**QE 的退出方式（缩表）**：

- **停止再投资**：到期债券不再购买新债，自然缩减
- **主动卖出**：央行主动出售持有资产，速度更快但冲击更大
- **渐进式缩表**：设定每月缩减上限，给市场适应时间

## 为什么重要

量化宽松是 2008 年金融危机后各国央行广泛使用的"大招"。它能快速降低长期利率、刺激信贷、提振资产价格，但也带来资产泡沫、贫富差距扩大等副作用。理解 QE 有助于理解为什么股市在实体经济不好时依然能涨。

具体来说：

- **降低长期利率**：通过购买长期债券，压低房贷利率和企业融资成本
- **稳定金融市场**：在危机中充当"最后买家"，防止资产价格崩盘
- **财富效应**：推高股市和房价，让持有资产的人感觉"更有钱"，从而增加消费
- **汇率影响**：大量放水通常导致本币贬值，有利于出口
- **通胀预期管理**：通过承诺持续宽松，防止通缩预期自我实现

## 生活化类比

经济像一个干涸的鱼塘，降息就像往鱼塘里滴水，但水太少了。量化宽松就是直接开消防车往鱼塘里灌水——水多了，鱼（经济）才能活过来，但水太多也可能把鱼塘撑破（通胀或泡沫）。

更形象地说：QE 就像央行给金融体系做"心肺复苏"。心脏（市场）不跳了，常规药物（降息）无效，只能用电击（印钱买资产）来重启。

## 关键要点

- QE 通常在利率接近零时使用，是"最后的手段"
- 央行购买资产会推高资产价格、压低长期利率
- QE 会导致央行资产负债表大幅扩张
- 退出 QE（缩表）过程可能引发市场波动
- QE 主要影响金融市场，对实体经济的传导效果存在争议
- QE 不是"直升机撒钱"，钱主要流入金融机构而非普通居民

## 常见误区

- **误区一：QE 就是直接印钞票发给老百姓**——QE 是央行买金融资产，钱主要流入金融机构和资本市场，不会直接发到个人手中。这与"直升机撒钱"（Helicopter Money）有本质区别
- **误区二：QE 一定会引发恶性通胀**——如果资金停留在金融体系内没有进入实体经济，可能只推高资产价格而不引发消费通胀。2008-2019 年（来源：各国统计局，2008-2019）美国 CPI 长期低于 2% 就是证明
- **误区三：QE 是万能药**——QE 能缓解流动性危机，但解决不了结构性问题（如人口老龄化、产业空心化），过度依赖会延缓必要的改革
- **误区四：QE 退出很容易**——缩表过程如果操作不当，可能引发市场恐慌和流动性危机。2013 年（来源：维基百科，2013）"缩减恐慌"（Taper Tantrum）就是前车之鉴

## 与其他概念的关系

- 与 [货币政策](/macro/monetary-policy)：QE 是非常规货币政策的极端形式，在常规工具用尽后启用
- 与 [加息与降息](/macro/rate-hike-cut)：QE 是降息无效后的"升级版"宽松。加息周期中，央行通常同时停止 QE 并缩表
- 与 [通胀](/macro/cpi)：大量放水可能引发通胀，但也可能只是推高资产价格。QE 与通胀的关系比直觉更复杂
- 与 [黑天鹅](/macro/black-swan)：2008 年金融危机（黑天鹅）后全球开启 QE 时代，2020 年疫情（另一只黑天鹅）后 QE 规模再创新高
- 与资产价格：QE 通过流动性效应和组合再平衡效应推高股票、债券和房地产价格
