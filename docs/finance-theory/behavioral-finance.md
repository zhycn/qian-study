# 行为金融学

> 传统金融学假设人是理性的，但行为金融学说：别逗了，人会恐惧、贪婪、跟风、过度自信——这些心理偏差会让市场变得"不那么有效"。

## 概述

行为金融学（Behavioral Finance）是心理学与金融学的交叉学科，研究**心理偏差如何系统性地影响投资者的决策和资产价格**。

传统金融学（如有效市场假说、[CAPM](/finance-theory/capm)）假设投资者是理性的"经济人"（Homo Economicus），会最大化自身效用。但大量实证研究表明，真实投资者会犯系统性错误：

- **损失厌恶**（Loss Aversion）：亏钱的痛苦远大于赚钱的快乐
- **过度自信**（Overconfidence）：高估自己的判断能力
- **锚定效应**（Anchoring Effect）：被第一个接触到的数字影响
- **羊群效应**（Herd Behavior）：跟着别人买，不做独立分析
- **代表性偏差**（Representativeness Bias）：根据近期表现推断未来趋势
- **确认偏差**（Confirmation Bias）：只关注支持自己观点的信息

行为金融学的代表人物包括丹尼尔·卡尼曼（Daniel Kahneman，2002 年诺奖得主（来源：诺贝尔奖官网））、理查德·塞勒（Richard Thaler，2017 年诺奖得主（来源：诺贝尔奖官网））、罗伯特·席勒（Robert Shiller，2013 年诺奖得主（来源：诺贝尔奖官网））。

## 为什么重要

- **解释市场异象**：传统金融学无法解释的现象（如泡沫、崩盘、动量效应、股权溢价之谜），行为金融学给出了合理解释
- **投资策略**：理解心理偏差可以帮助投资者避免常见错误，甚至利用他人的非理性行为获利
- **产品设计**：金融机构利用行为金融学设计更好的养老金计划、理财产品（如默认选项设计）
- **监管政策**：助推（Nudge）理论被广泛应用于金融消费者保护
- **风险管理**：行为偏差会放大市场波动，理解这一点有助于更好地管理尾部风险

## 核心原理

行为金融学建立在两个核心支柱之上：

### 支柱一：有限理性（Bounded Rationality）

赫伯特·西蒙（Herbert Simon）提出，人类的认知能力是有限的。在面对复杂决策时，人们不会进行完全的最优化计算，而是依赖**启发式**（Heuristics）——即经验法则或心理捷径。

常见的启发式偏差包括：

| 启发式     | 描述                       | 投资中的表现                 |
| ---------- | -------------------------- | ---------------------------- |
| 代表性     | 根据相似性判断概率         | 认为过去涨的股票未来还会涨   |
| 可得性     | 根据容易回忆的程度判断概率 | 刚经历股灾就认为股市风险极高 |
| 锚定与调整 | 从初始值出发做不充分调整   | 以买入价为锚，不愿止损       |
| 情感启发式 | 根据情感反应判断风险       | 喜欢的公司股票觉得风险低     |

### 支柱二：有限套利（Limits to Arbitrage）

即使市场中存在理性投资者，他们也不一定能纠正错误定价。原因包括：

1. **基本面风险**（Fundamental Risk）：套利头寸可能因基本面变化而亏损
2. **噪声交易者风险**（Noise Trader Risk）：非理性行为可能持续甚至加剧，导致套利者在回归前爆仓
3. **实施成本**（Implementation Costs）：交易成本、卖空限制、流动性约束
4. **模型风险**（Model Risk）：套利者不确定自己是否"正确"

Shleifer 和 Vishny（1997）（来源：Shleifer & Vishny, 1997）的经典论文证明，**套利是有风险的、有成本的、有约束的**，因此错误定价可以长期存在。

## 模型/方法详解

### 前景理论（Prospect Theory）

卡尼曼和特沃斯基（1979）（来源：Kahneman & Tversky, 1979）提出的前景理论是行为金融学的核心模型。价值函数为：

$$v(x) = \begin{cases} x^\alpha & x \geq 0 \\ -\lambda(-x)^\beta & x < 0 \end{cases}$$

其中 $\lambda \approx 2.25$（损失厌恶系数（来源：Tversky & Kahneman, 1992）），$\alpha, \beta < 1$（敏感度递减）。

概率权重函数：

$$w(p) = \frac{p^\gamma}{(p^\gamma + (1-p)^\gamma)^{1/\gamma}}$$

人们倾向于高估小概率事件（$w(p) > p$ 当 $p$ 很小时），低估中高概率事件。

### BSV 模型

Barberis、Shleifer 和 Vishny（1998）（来源：Barberis, Shleifer & Vishny, 1998）提出了一个基于心理偏差的资产定价模型：

- 投资者相信收益在"均值回归"和"趋势延续"两种机制之间切换
- 实际上收益是随机游走
- 投资者的**保守主义偏差**（Conservatism）导致对单一事件反应不足
- **代表性偏差**导致对连续事件反应过度
- 模型成功解释了动量效应和反转效应

### DHS 模型

Daniel、Hirshleifer 和 Subrahmanyam（1998）（来源：Daniel, Hirshleifer & Subrahmanyam, 1998）基于**过度自信**和**自我归因偏差**（Self-Attribution Bias）构建模型：

- 过度自信的投资者高估私人信息的精度
- 自我归因偏差使投资者将成功归因于能力、失败归因于噪声
- 模型解释了过度交易、动量和波动率聚集

## 主流方案对比

| 理论框架       | 核心假设              | 解释的现象             | 局限             |
| -------------- | --------------------- | ---------------------- | ---------------- |
| 传统金融学     | 完全理性、有效市场    | 资产定价基准           | 无法解释异象     |
| 前景理论       | 参考点依赖、损失厌恶  | 处置效应、股权溢价之谜 | 参考点选择主观   |
| BSV 模型       | 保守主义 + 代表性偏差 | 反应不足 + 反应过度    | 参数难以校准     |
| DHS 模型       | 过度自信 + 自我归因   | 动量、过度交易         | 难以区分其他解释 |
| 适应性市场假说 | 进化论视角            | 市场效率动态变化       | 预测能力有限     |

## 实战应用

### 避免认知偏差的实用技巧

1. **投资清单**（Investment Checklist）：在每次交易前检查是否受到特定偏差影响
2. **事前验尸**（Pre-mortem Analysis）：假设投资已经失败，反向分析原因
3. **决策日志**：记录每次决策的理由和预期，定期复盘
4. **多元化视角**：主动寻找与自己观点相反的信息
5. **自动化决策**：使用定投、再平衡等规则化策略减少主观判断

### 行为 alpha 策略

一些量化基金专门利用行为偏差获利：

- **反转策略**：利用投资者对坏消息的过度反应
- **盈余公告后漂移**（PEAD）：利用对财报信息的反应不足
- **52 周高点效应**：投资者在股票接近 52 周高点时反应不足

### 智能投顾中的行为设计

- 默认选项（Default Option）：将最优配置设为默认，利用现状偏差
- 框架效应（Framing）：将长期收益用可视化方式呈现，减少短期损失厌恶
- 社会证明（Social Proof）：展示"大多数用户选择"引导行为

## 关键要点

- 行为金融学不否定传统金融学，而是补充了"人非理性"这一现实
- 心理偏差是**系统性的**，不是随机错误，因此可以被研究和预测
- 行为金融学有两个支柱：**认知偏差**（信息处理错误）和**有限套利**（理性投资者无法完全纠正错误定价）
- 行为金融学已被主流学术界接受，多位代表人物获得诺贝尔奖
- 行为金融学不是"万能解释"，好的行为金融模型必须做出可检验的预测

## 常见误区

- **误区一**："行为金融学就是心理学"——行为金融学是心理学与金融学的交叉学科，有严格的数学和实证基础
- **误区二**："市场非理性就可以随便赚钱"——市场非理性不等于容易获利，错误定价可能持续很久，"市场保持非理性的时间可能比你保持偿付能力的时间更长"
- **误区三**："传统金融学已经过时"——传统金融学在多数场景下仍然有效，行为金融学是补充而非替代
- **误区四**："心理偏差可以完全克服"——认知偏差是大脑的"出厂设置"，只能减轻无法消除
- **误区五**："行为金融学可以解释一切"——好的理论必须有可证伪性，行为金融学有时被批评为"事后解释力过强"
