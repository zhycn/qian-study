# 风险与收益

> 想要赚得多，就得承受亏得多的可能——这是金融市场最铁的定律

## 概述

风险与收益（Risk and Return）是金融学中最核心的关系：**预期收益越高，承担的风险就越大**。天下没有免费的午餐，更没有"高收益零风险"的投资。

两个关键概念：

- **收益**（Return）：投资带来的回报，通常用年化收益率表示。包括资本利得（价格涨跌）和现金流收益（利息、分红）
- **风险**（Risk）：实际收益偏离预期的可能性，通常用波动率（Volatility，收益率的标准差）衡量。波动越大，风险越高

常见资产的风险收益排序（从低到高）：

| 资产类型 | 年化收益（历史平均） | 风险等级 |
| -------- | -------------------- | -------- |
| 国债     | 2%-4%                | 极低     |
| 银行存款 | 1%-3%                | 极低     |
| 债券基金 | 3%-6%                | 低       |
| 股票基金 | 8%-12%               | 中高     |
| 个股     | -50% ~ +100%         | 高       |
| 加密货币 | -80% ~ +500%         | 极高     |

> 注：以上数据为历史区间统计参考（来源：彭博社/万得，长期历史数据）

注意：历史收益不代表未来表现，以上数据仅供参考。

## 演进历史

**早期风险认知（古代 - 17 世纪）**：古代商人已通过分散运输（把货物分装多条船）来降低风险。意大利商人发明了"康曼达契约"（Commenda），投资者承担有限风险，这是现代有限责任公司的雏形。

**概率论与保险（17 - 18 世纪）**：1654 年帕斯卡和费马通过通信奠定了概率论基础（来源：维基百科，概率论历史）。1688 年伦敦劳埃德咖啡馆（Lloyd's Coffee House）成为海上保险交易中心（来源：维基百科，劳合社）。风险开始被量化和定价。

**现代投资组合理论（20 世纪）**：1952 年哈里·马科维茨（Harry Markowitz）发表《投资组合选择》，提出"分散投资可以降低风险而不降低预期收益"，被称为"金融学唯一的免费午餐"（来源：马科维茨，1952）。1964 年威廉·夏普（William Sharpe）提出资本资产定价模型（CAPM），用β系数衡量系统性风险（来源：夏普，1964）。1973 年费雪·布莱克（Fischer Black）和迈伦·斯科尔斯（Myron Scholes）提出期权定价公式（Black-Scholes Model），让风险定价更加精确（来源：Black-Scholes 论文，1973）。

**行为金融学（21 世纪）**：丹尼尔·卡尼曼（Daniel Kahneman）和阿莫斯·特沃斯基（Amos Tversky）发现人类在风险决策中存在系统性偏差。2002 年卡尼曼获诺贝尔经济学奖（来源：诺贝尔奖官网，2002）。行为金融学揭示了"风险"不仅是数学概念，更是心理感受。

## 核心分类

- **系统性风险**（Systematic Risk）：影响整个市场的风险，如经济衰退、战争、疫情。无法通过分散投资消除，也称"不可分散风险"或"市场风险"
- **非系统性风险**（Unsystematic Risk）：只影响特定公司或行业的风险，如管理层变动、产品召回。可以通过分散投资降低甚至消除
- **信用风险**（Credit Risk）：借款人违约的风险。债券投资者面临的主要风险
- **流动性风险**（Liquidity Risk）：资产无法快速变现或变现时大幅折价的风险
- **操作风险**（Operational Risk）：因内部流程、人员、系统缺陷或外部事件导致损失的风险

## 为什么重要

理解风险收益关系是做出理性投资决策的基础：

- **避免被骗**：识别"高收益零风险"的骗局。庞氏骗局（Ponzi Scheme）和各类非法集资都利用人们对"高收益低风险"的幻想
- **匹配风险承受能力**：年轻人可以承受更高风险（有更多时间恢复），临近退休则应降低风险
- **资产配置**：通过在不同风险等级的资产间分配资金，在风险和收益之间找到个人最优平衡

所有金融产品的定价，本质上都是在给风险定价。国债收益率是"无风险利率"，其他资产的收益率 = 无风险利率 + 风险溢价（Risk Premium）。风险越高，要求的风险溢价越大。

## 生活化类比

风险收益就像游乐园的游乐项目。

旋转木马（银行存款）很安全，但没啥刺激感（收益低）。海盗船（债券基金）有点晃，但基本安全（收益适中）。过山车（股票）可能让你尖叫（亏损），也可能让你爽翻天（高收益）。蹦极（加密货币）——刺激到极点，但不是每个人都受得了。

如果有人告诉你有个"既像过山车一样刺激，又像旋转木马一样安全"的项目——那一定是骗子。

聪明的游客不会只玩一个项目，而是每个都体验一下（分散投资）。这样即使过山车故障了（某只股票暴跌），其他项目还能玩（整体组合不至于全军覆没）。

## 关键要点

- 风险不等于亏损，而是"不确定性"——可能赚更多，也可能亏更多
- 分散投资可以降低非系统性风险而不降低预期收益，是"金融学唯一的免费午餐"
- 夏普比率（Sharpe Ratio）衡量"每承担一单位风险获得多少超额收益"，是评价投资绩效的重要指标
- β系数（Beta）衡量资产相对于市场的波动程度，β > 1 表示比市场波动更大
- 历史收益不代表未来表现，过去的数据只能参考不能预测
- 风险承受能力随年龄、收入、家庭状况变化，应定期重新评估

## 常见误区

- **高风险一定高收益**：高风险只是"可能"高收益，也可能高亏损。风险和收益是正相关关系，而非因果关系。高风险投资失败的案例比比皆是
- **分散投资会降低收益**：分散投资降低的是非系统性风险，长期来看不会降低预期收益，反而能提高风险调整后收益
- **历史收益能预测未来**：过去表现好的资产未来不一定好，市场环境、政策、技术都在变化。"过往业绩不代表未来表现"不是套话，是血泪教训
- **低风险等于没风险**：即使是国债也有通胀风险（实际收益可能为负）和利率风险（价格会波动）。"无风险"只是相对概念

## 与其他概念的关系

- [复利](/basics/compound-interest)：长期投资利用复利效应平滑短期风险波动
- [流动性](/basics/liquidity)：流动性差的资产需要更高的风险溢价补偿
- [利息](/basics/interest)：无风险利率（国债收益率）是所有风险资产定价的基准
- [机会成本](/basics/opportunity-cost)：选择低风险资产的机会成本是错过的高收益
