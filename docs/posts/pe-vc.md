# PE/VC 私募股权/风险投资

VC 投的是"梦想"——给刚起步的创业公司几百万，赌它成为下一个阿里巴巴。PE 投的是"现实"——花几十亿买下成熟公司，优化管理后再高价卖出。两者都是拿别人的钱，博取超高回报。

## 概述

**私募股权**（Private Equity, PE）和**风险投资**（Venture Capital, VC）都是**向投资者募集资金，投资于非上市公司股权，通过企业增值后退出获取回报的投资方式**。

PE 和 VC 的关系：

```
创业阶段：种子轮 → 天使轮 → A 轮 → B 轮 → C 轮 → ... → IPO → 成熟期
           ↑________ VC 投资区间 ________↑    ↑____ PE 投资区间 ____↑
```

PE 和 VC 的核心区别：

| 维度     | VC（风险投资）           | PE（私募股权）             |
| -------- | ------------------------ | -------------------------- |
| 投资阶段 | 早期、成长期             | 成熟期、Pre-IPO            |
| 投资金额 | 几十万到几千万美元       | 几千万到几十亿美元         |
| 持股比例 | 通常 10-30%              | 通常控股（50%+）           |
| 风险水平 | 极高（大部分项目失败）   | 中等（企业已有稳定现金流） |
| 回报倍数 | 目标 10-100 倍           | 目标 2-5 倍                |
| 投资方式 | 少数股权，不参与日常经营 | 控股，深度参与管理         |
| 典型策略 | 押注赛道、广撒网         | 价值创造、运营优化         |

私募股权行业的资金规模：

- 全球 PE/VC 管理资产（AUM）：约 13 万亿美元（2023 年，来源：Preqin）
- 中国 PE/VC 管理资产：约 20 万亿元人民币（2023 年，来源：中国证券投资基金业协会）
- 顶级基金年化回报：20-30%（但行业整体回报分化严重）

## 为什么重要

- **创新引擎**：VC 是科技创新的重要推手，苹果、谷歌、阿里巴巴、腾讯等都曾接受 VC 投资
- **资源配置**：PE/VC 将资本配置到最有潜力的企业和行业
- **就业创造**：PE/VC 投资的企业创造了大量就业机会
- **公司治理**：PE 通过控股和深度参与，改善企业治理和运营效率
- **资本市场**：PE/VC 是 IPO 市场的重要供给方

## 核心原理

### PE/VC 的运作模式

**基金结构**：

PE/VC 通常采用有限合伙制（Limited Partnership）：

- **普通合伙人**（GP, General Partner）：基金管理者，负责投资决策，承担无限责任
- **有限合伙人**（LP, Limited Partner）：基金投资者，不参与管理，承担有限责任

**收费模式**：

**2/20 模式**：

- **管理费**（Management Fee）：每年 2% 的 committed capital（承诺资本）
- **业绩提成**（Carried Interest / Carry）：利润的 20%

示例：

- 基金规模 10 亿元，管理费每年 2000 万元
- 基金最终回报 30 亿元，利润 20 亿元
- GP 获得业绩提成：20 亿 × 20% = 4 亿元
- LP 获得：20 亿 × 80% = 16 亿元

**回报瀑布**（Distribution Waterfall）：

1. **返还本金**：先返还 LP 的投入资本
2. **优先回报**（Hurdle Rate）：LP 获得 8% 的年化优先回报
3. **追补**（Catch-up）：GP 追补到 20% 的业绩提成
4. **分成**：剩余利润按 80/20 分配

### VC 的投资逻辑

**幂律分布**（Power Law）：

VC 投资回报遵循幂律分布：

- 大部分项目（60-70%）亏损或持平
- 少数项目（20-30%）获得中等回报
- 极少数项目（1-5%）获得超高回报（10-100 倍）

一个典型 VC 基金的回报分布：

| 项目     | 投资额      | 回报倍数   | 回报金额     |
| -------- | ----------- | ---------- | ------------ |
| 项目 A   | 500 万      | 0x（失败） | 0            |
| 项目 B   | 500 万      | 0.5x       | 250 万       |
| 项目 C   | 500 万      | 1x         | 500 万       |
| 项目 D   | 500 万      | 3x         | 1500 万      |
| 项目 E   | 500 万      | 50x        | 2.5 亿       |
| **合计** | **2500 万** |            | **2.725 亿** |

一个项目（项目 E）贡献了绝大部分回报。

**投资阶段**：

| 阶段   | 典型金额      | 估值范围      | 风险 | 用途               |
| ------ | ------------- | ------------- | ---- | ------------------ |
| 种子轮 | 50-200 万     | 1000-3000 万  | 极高 | 产品原型、团队组建 |
| 天使轮 | 200-1000 万   | 3000 万 -1 亿 | 极高 | 产品开发、市场验证 |
| A 轮   | 1000-5000 万  | 1-5 亿        | 高   | 市场扩张、团队建设 |
| B 轮   | 5000 万 -2 亿 | 5-20 亿       | 中高 | 规模化、商业化     |
| C 轮+  | 2 亿+         | 20 亿+        | 中   | 国际化、Pre-IPO    |

### PE 的价值创造

PE 通过以下方式创造价值：

**1. 运营优化**

- 降低成本、提高效率
- 优化供应链、改善现金流
- 引入专业管理团队

**2. 财务工程**

- 杠杆收购（LBO）：用债务融资放大回报
- 税盾效应：债务利息可以抵税
- 资本结构优化

**3. 战略重组**

- 剥离非核心业务
- 并购整合
- 市场扩张

**杠杆收购**（LBO）模型：

假设 PE 以 10 亿元收购一家公司：

- 自有资金：3 亿元
- 债务融资：7 亿元（利率 6%）

5 年后以 15 亿元出售：

- 偿还债务：约 7 亿元（假设只付利息）
- 净回报：15 - 7 = 8 亿元
- 回报率：(8 - 3) / 3 = 167%（年化约 22%）

如果没有杠杆：

- 回报率：(15 - 10) / 10 = 50%（年化约 8.5%）

杠杆显著放大了回报。

## 模型/方法详解

### VC 估值方法

早期创业公司估值是艺术而非科学：

**1. 可比公司法**（Comparable Company Analysis）

参考同行业、同阶段公司的估值：

$$\text{估值} = \text{可比公司估值倍数} \times \text{公司指标}$$

常用倍数：

- P/S（市销率）：适用于有收入但无盈利的公司
- P/GMV（市交易总额率）：适用于电商平台
- P/用户：适用于用户驱动的公司

**2. 风险调整现值法**（Risk-Adjusted DCF）

$$\text{估值} = \sum_{t=1}^n \frac{CF_t}{(1 + r)^t}$$

其中 $r$ 是高风险调整后的折现率（通常 40-60%）。

**3. 博克斯法**（Berkus Method）

Dave Berkus 提出的早期估值方法：

| 要素         | 价值           |
| ------------ | -------------- |
| 好主意       | 最高 50 万美元 |
| 原型产品     | 最高 50 万美元 |
| 管理团队     | 最高 50 万美元 |
| 战略关系     | 最高 50 万美元 |
| 产品销售     | 最高 50 万美元 |
| **最高估值** | **250 万美元** |

### PE 的 LBO 模型

LBO 模型的核心是计算 IRR（内部收益率）：

$$NPV = \sum_{t=0}^n \frac{CF_t}{(1 + IRR)^t} = 0$$

LBO 模型的关键输入：

- 收购价格
- 资本结构（债务/股权比例）
- 债务利率和还款计划
- 收入增长率
- 利润率变化
- 退出倍数

LBO 模型的输出：

- 股权 IRR
- 投资回报倍数（MOIC, Multiple on Invested Capital）
- 债务覆盖率

### PE/VC 的绩效度量

**IRR**（Internal Rate of Return）：

$$NPV = \sum_{t=0}^n \frac{CF_t}{(1 + IRR)^t} = 0$$

IRR 是 PE/VC 最常用的绩效指标，但有局限：

- 对早期回报敏感
- 不考虑资金规模
- 可能被操纵（如早期退出）

**MOIC**（Multiple on Invested Capital）：

$$MOIC = \frac{\text{总回报}}{\text{总投入}}$$

MOIC 简单直观，但不考虑时间价值。

**DPI**（Distributed to Paid-In Capital）：

$$DPI = \frac{\text{已分配给 LP 的金额}}{\text{LP 已投入的金额}}$$

DPI 反映基金的实际现金回报，是 LP 最关注的指标。

## 主流方案对比

| 策略        | 风险 | 回报    | 持有期  | 典型场景             |
| ----------- | ---- | ------- | ------- | -------------------- |
| 早期 VC     | 极高 | 10-100x | 7-10 年 | 科技创业公司         |
| 成长期 VC   | 高   | 3-10x   | 5-7 年  | 已验证商业模式的公司 |
| 并购 PE     | 中   | 2-5x    | 3-7 年  | 成熟企业             |
| 困境投资    | 中高 | 3-8x    | 2-5 年  | 财务困境企业         |
| 二级市场 PE | 中低 | 1.5-3x  | 1-3 年  | 上市公司私有化       |

## 实战应用

### 创业者如何获得 VC/PE 投资

**1. 准备阶段**

- **商业计划书**：清晰阐述商业模式、市场规模、竞争优势
- **财务模型**：展示收入预测、成本结构、资金需求
- **团队**：优秀的团队是 VC 最看重的因素

**2. 融资过程**

- **寻找合适的投资者**：不同 VC 有不同的投资阶段和行业偏好
- **路演**：向投资者展示项目
- **尽职调查**：投资者对团队、业务、财务进行全面调查
- **条款谈判**：估值、董事会席位、优先权等
- **交割**：签署协议，资金到账

**3. 关键条款**

| 条款       | 说明                             |
| ---------- | -------------------------------- |
| 估值       | 投前估值 vs 投后估值             |
| 清算优先权 | 退出时投资者优先获得回报         |
| 反稀释条款 | 保护投资者免受后续低价融资的稀释 |
| 董事会席位 | 投资者在董事会中的代表权         |
| 否决权     | 投资者对重大事项的否决权         |
| 对赌条款   | 业绩承诺和补偿机制               |

### PE/VC 投资的常见陷阱

**对创业者**：

- 估值过高：后续融资困难
- 条款过于苛刻：失去控制权
- 选择错误的投资者：缺乏行业资源
- 过度稀释：多轮融资后创始团队持股过低

**对投资者**：

- 过度自信：高估自己的判断能力
- 羊群效应：跟随热门赛道
- 忽视尽职调查：被表面数据迷惑
- 退出时机不当：过早或过晚退出

## 关键要点

- VC 投资回报遵循幂律分布，少数项目贡献绝大部分回报
- PE 通过杠杆和运营优化创造价值
- 2/20 收费模式是行业标准，但正在受到挑战
- 尽职调查是投资决策的关键环节
- PE/VC 是高风险高回报的投资，适合长期资金

## 常见误区

- **误区一**："VC 投资的是好点子"——VC 投资的是团队和执行能力，点子不值钱
- **误区二**："PE 就是炒公司"——PE 通过运营优化和价值创造获利，不是简单的低买高卖
- **误区三**："IRR 越高越好"——IRR 可能被操纵，需要结合 MOIC 和 DPI 综合评估
- **误区四**："估值越高越好"——对创业者来说，过高估值可能导致后续融资困难
- **误区五**："PE/VC 是稳赚的"——PE/VC 行业整体回报分化严重，大部分基金跑不赢指数
