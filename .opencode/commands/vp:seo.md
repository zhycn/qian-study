检查并优化 VitePress 的 SEO 设置。

检查项：

1. 站点元数据
   - title、description 是否设置
   - lang 是否正确
2. Open Graph 标签
   - og:type、og:locale、og:site_name
   - og:image 是否设置
3. Twitter Card
   - twitter:card 类型
4. Sitemap
   - hostname 是否配置
5. robots.txt 是否存在
6. 每个页面的 frontmatter description

提供优化建议并自动修复可自动化的问题。

示例：`/vp:seo`
