# 前景理论

前景理论就是人类决策的"双面镜"——面对收益时你变得胆小如鼠，面对损失时你却敢赌一把大的。亏 500 块的痛苦，要赚 1000 块才能弥补。

## 概述

前景理论（Prospect Theory）由丹尼尔·卡尼曼（Daniel Kahneman）和阿莫斯·特沃斯基（Amos Tversky）于 1979 年（来源：Kahneman & Tversky, 1979）提出，是**描述人们在不确定条件下如何做决策的行为模型**。它挑战了传统经济学中"理性人"（Homo Economicus）的假设。

前景理论的四大核心要素：

1. **参考点依赖**（Reference Dependence）：人们根据相对于某个参考点的"收益"或"损失"来评估结果，而不是根据最终财富总量
2. **损失厌恶**（Loss Aversion）：损失的心理影响约为同等收益的 2-2.5 倍
3. **敏感度递减**（Diminishing Sensitivity）：无论是收益还是损失，边际心理影响随金额增大而减小（价值函数呈 S 形）
4. **概率权重**（Probability Weighting）：人们倾向于高估小概率事件、低估中高概率事件

价值函数的形状：

$$v(x) = \begin{cases} x^\alpha & x \geq 0 \\ -\lambda(-x)^\beta & x < 0 \end{cases}$$

其中 $\lambda \approx 2.25$（损失厌恶系数（来源：Tversky & Kahneman, 1992）），$\alpha, \beta < 1$（敏感度递减）。

概率权重函数：

$$w(p) = \frac{p^\gamma}{(p^\gamma + (1-p)^\gamma)^{1/\gamma}}$$

其中 $\gamma < 1$，导致小概率被高估（$w(p) > p$），中高概率被低估（$w(p) < p$）。

卡尼曼因此获得 2002 年诺贝尔经济学奖（来源：诺贝尔奖官网）（特沃斯基已于 1996 年去世（来源：维基百科），诺奖不追授）。

## 为什么重要

- **行为经济学基石**：前景理论是行为经济学的核心理论，彻底改变了经济学对决策的理解
- **解释市场异象**：处置效应、股权溢价之谜、彩票型股票偏好等都可用前景理论解释
- **保险与赌博**：前景理论同时解释了为什么人们既买保险（高估小概率损失）又买彩票（高估小概率收益）——这在传统期望效用理论下是矛盾的
- **政策设计**：助推（Nudge）理论和框架效应广泛应用于公共政策，如养老金自动加入、器官捐献默认选项
- **金融实务**：前景理论被用于产品设计、客户沟通、风险管理等多个领域

## 核心原理

### 两阶段决策模型

前景理论将决策分为两个阶段：

**阶段一：编辑阶段**（Editing Phase）

人们对选项进行预处理：

- **编码**（Coding）：将结果编码为相对于参考点的收益或损失
- **合并**（Combining）：将相同结果的概率合并
- **分离**（Segregation）：将无风险部分与风险部分分离
- **取消**（Cancellation）：忽略选项间相同的部分

编辑阶段的关键是**参考点的选择**。参考点可以是：

- 现状（当前财富水平）
- 期望值（预期应该达到的水平）
- 社会比较（他人的表现）
- 心理账户（Mental Accounting）中的分类

**阶段二：评估阶段**（Evaluation Phase）

对编辑后的选项进行评估，选择价值最高的选项。每个选项的价值为：

$$V = \sum_{i} w(p_i) \times v(x_i)$$

其中 $w(p_i)$ 是决策权重（Decision Weight），$v(x_i)$ 是价值函数在结果 $x_i$ 处的值。

### 与传统期望效用理论的对比

| 维度          | 期望效用理论（EUT）  | 前景理论（PT）                   |
| ------------- | -------------------- | -------------------------------- |
| 评估对象      | 最终财富总量         | 相对于参考点的变化               |
| 风险态度      | 由效用函数曲率决定   | 由价值函数形状和概率权重共同决定 |
| 概率处理      | 客观概率             | 决策权重（非线性变换）           |
| 损失处理      | 对称                 | 损失厌恶（不对称）               |
| 描述性/规范性 | 规范性（应该怎么做） | 描述性（实际怎么做）             |

### 反射效应（Reflection Effect）

前景理论最反直觉的发现是**反射效应**：同一个人在收益框架下风险厌恶，在损失框架下风险偏好。

- 收益域：确定收益 < 风险收益的期望值 → 风险厌恶
- 损失域：确定损失 > 风险损失的期望值 → 风险偏好

这解释了为什么投资者在盈利时倾向于"落袋为安"，在亏损时倾向于"赌一把"。

## 模型/方法详解

### 累积前景理论（CPT）

1992 年（来源：Tversky & Kahneman, 1992），卡尼曼和特沃斯基提出了累积前景理论（Cumulative Prospect Theory, CPT），解决了原版前景理论的一些理论缺陷：

**原版 PT 的问题**：违反随机占优（Stochastic Dominance）。在某些情况下，一个明显更好的选项可能被评估为更差。

**CPT 的改进**：对累积概率分布而非单个概率应用权重函数。

CPT 的价值函数：

$$V(f) = \sum_{i=-m}^{0} w_i^- v(x_i) + \sum_{i=0}^{n} w_i^+ v(x_i)$$

其中决策权重由累积概率的变换得到：

$$w_i^+ = w^+(p_i + \cdots + p_n) - w^+(p_{i+1} + \cdots + p_n)$$

$$w_i^- = w^-(p_{-m} + \cdots + p_i) - w^-(p_{-m} + \cdots + p_{i-1})$$

### 参数估计

Tversky 和 Kahneman（1992）（来源：Tversky & Kahneman, 1992）估计的典型参数：

- $\alpha = \beta = 0.88$（敏感度递减）
- $\lambda = 2.25$（损失厌恶系数）
- $\gamma = 0.61$（概率权重曲率）
- $\delta = 0.69$（概率权重曲率，损失域）

后续研究在不同场景下估计的参数有所差异，但基本模式一致。

## 主流方案对比

| 理论         | 核心假设             | 优势                       | 局限                       |
| ------------ | -------------------- | -------------------------- | -------------------------- |
| 期望效用理论 | 理性人、最终财富     | 数学优雅、规范性强         | 描述性差，无法解释异象     |
| 前景理论     | 参考点依赖、损失厌恶 | 描述性强，解释力广         | 参考点选择主观，参数不稳定 |
| 累积前景理论 | 累积概率权重         | 解决随机占优问题           | 计算复杂，参数更多         |
| 后悔理论     | 关注后悔感最小化     | 解释某些 PT 无法解释的选择 | 与 PT 预测有时冲突         |
| 双曲贴现     | 时间偏好不一致       | 解释跨期选择异常           | 不适用于一次性风险决策     |

## 实战应用

### 解释处置效应

前景理论完美解释了处置效应（Disposition Effect）：

- **盈利股票**：处于价值函数的收益区域，函数是凹的（风险厌恶）→ 倾向于卖出锁定收益
- **亏损股票**：处于价值函数的损失区域，函数是凸的（风险偏好）→ 倾向于持有等待回本

### 股权溢价之谜

Benartzi 和 Thaler（1995）（来源：Benartzi & Thaler, 1995）用**短视性损失厌恶**（Myopic Loss Aversion）解释股权溢价之谜：

- 投资者频繁评估投资组合
- 每次评估时，损失厌恶使股票的高波动看起来更可怕
- 因此要求极高的风险溢价才愿意持有股票
- 如果投资者每年只看一次账户，股权溢价之谜就消失了

### 框架效应的应用

**投资产品设计**：

- "年化收益 8%"比"可能亏损 20%"更容易被接受
- 展示长期收益曲线比展示短期波动更友好
- "最大回撤 15%"比"可能亏 15%"更专业

**养老金计划**：

- 自动加入（Opt-out）比主动加入（Opt-in）参与率高得多
- 默认选项利用了现状偏差和损失厌恶（退出意味着"失去"已积累的收益）

### 交易策略启示

- **止损策略**：前景理论解释了为什么止损很难执行（处于损失区域时风险偏好）
- **定投策略**：定期定额投资减少了单次决策的心理负担
- **再平衡策略**：规则化的再平衡避免了情绪化决策

## 关键要点

- 前景理论是描述性理论（描述人实际怎么做），不是规范性理论（告诉人应该怎么做）
- 参考点可以是现状、期望值、他人表现，参考点的选择会极大影响决策
- 概率权重函数解释了为什么人们既买保险（高估小概率损失）又买彩票（高估小概率收益）
- 前景理论的扩展版本"累积前景理论"（CPT, 1992）解决了原版的一些理论缺陷
- 前景理论不是"非理性"的理论——它描述了人类在不确定条件下的真实决策模式，这种模式在进化环境中是适应性的

## 常见误区

- **误区一**："前景理论告诉我们怎么做决策"——前景理论是描述性的，不是指导性的。它告诉你人们实际怎么做，不是应该怎么做
- **误区二**："损失厌恶就是风险厌恶"——损失厌恶是对亏损的恐惧，风险厌恶是对不确定性的规避。两者相关但不同
- **误区三**："参考点总是现状"——参考点可以是期望值、他人表现、心理账户、或任何心理锚点。参考点的选择是前景理论中最灵活也最主观的部分
- **误区四**："前景理论只适用于金钱决策"——前景理论适用于健康、职业、人际关系等多种决策场景
- **误区五**："前景理论已经完全取代期望效用理论"——期望效用理论在规范性分析中仍然重要，前景理论是描述性补充
