# PMI 采购经理指数

如果 GDP 是经济的"年终总结"，那 PMI 就是经济的"月度体检报告"。

GDP 每季度才发布一次，而且数据出来时已经是"过去式"。PMI 每月 1 号就发布，覆盖制造业和服务业的最新状况，是判断经济走势最及时的先行指标。

PMI 的核心逻辑很简单：问企业的采购经理"这个月比上个月好吗？"如果回答"好"的人多，PMI 就高于 50（扩张）；如果回答"差"的人多，PMI 就低于 50（收缩）。50 就是那条"荣枯线"。

## 概述

PMI（Purchasing Managers' Index，采购经理指数）是通过对企业采购经理的月度调查数据计算得出的扩散指数（Diffusion Index），反映经济的景气程度。

**调查方法**：

- 每月向数千家企业的采购经理发放问卷
- 问题涵盖：新订单、生产、从业人员、供应商配送时间、原材料库存等
- 回答选项：好于上月、与上月持平、差于上月
- 计算方法：PMI = 正面回答比例 × 1 + 持平回答比例 × 0.5 + 负面回答比例 × 0

**PMI = 50 是荣枯线**：

- **PMI > 50**：经济扩张（回答"好"的人多于回答"差"的人）
- **PMI = 50**：经济持平
- **PMI < 50**：经济收缩

**PMI 的五大分项指数**（制造业 PMI）：

| 分项                                  | 权重 | 含义                           |
| ------------------------------------- | ---- | ------------------------------ |
| 新订单（New Orders）                  | 30%  | 需求端，最重要的分项           |
| 生产（Production）                    | 25%  | 供给端，反映企业开工情况       |
| 从业人员（Employment）                | 20%  | 就业情况                       |
| 供应商配送时间（Supplier Deliveries） | 15%  | 反向指标，配送越慢说明需求越旺 |
| 原材料库存（Inventories）             | 10%  | 企业备货意愿                   |

$$\text{制造业 PMI} = 0.3 \times \text{新订单} + 0.25 \times \text{生产} + 0.2 \times \text{从业人员} + 0.15 \times \text{配送时间} + 0.1 \times \text{库存}$$

**PMI 的类型**：

- **制造业 PMI**（Manufacturing PMI）：反映制造业景气度，最受关注
- **服务业 PMI**（Services PMI）：反映服务业景气度，在服务业占比高的国家更重要
- **综合 PMI**（Composite PMI）：制造业 + 服务业的加权平均
- **官方 PMI vs 民间 PMI**：官方由统计局发布，样本偏大企业；民间（如财新）样本偏中小企业，两者可能分化

## 演进历史

**PMI 的发明（1930 年代）**：PMI 最早由美国供应管理协会（ISM，原名 NAPM）在 1930 年代开发（来源：ISM 官网）。最初是为了监测美国制造业的月度景气变化。1948 年 ISM 开始定期发布制造业 PMI，成为全球最早的 PMI 指标（来源：ISM 历史档案）。

**全球化推广（1990 - 2000）**：1990 年代，标普全球（IHS Markit，现 S&P Global）与各国机构合作推出 PMI 调查，覆盖欧洲、亚洲等主要经济体（来源：S&P Global）。1998 年中国开始发布官方制造业 PMI（来源：国家统计局，1998）。

**金融危机与 PMI 的应用（2008 - 2009）**：2008 年金融危机期间，全球 PMI 暴跌至 30 以下（来源：Markit，2009），准确预警了经济衰退。PMI 成为央行和投资者判断经济拐点的首要指标。

**新冠疫情与 PMI 波动（2020 - 2022）**：2020 年 2 月中国制造业 PMI 暴跌至 35.7（来源：国家统计局，2020），创历史最低。随后快速反弹至 52，反映复工复产。全球 PMI 在 2020-2022 年经历剧烈波动，成为判断疫情冲击和复苏的核心指标。

**中国 PMI 体系**：

- 1998 年：国家统计局开始发布官方制造业 PMI
- 2005 年：引入非制造业 PMI（服务业 + 建筑业）
- 2012 年：财新 PMI（原汇丰 PMI）开始发布，聚焦中小企业
- 2018 年：统计局调整 PMI 样本结构，增加中小企业权重（来源：国家统计局，2018）

## 核心分类

- **官方 PMI**：由国家统计局发布，样本覆盖大型和中型企业，更具代表性。中国官方制造业 PMI 样本约 3000 家企业（来源：国家统计局）
- **财新 PMI**：由 S&P Global 与财新传媒合作发布，样本偏重中小企业和出口导向型企业。两者分化时，往往反映大中小企业景气度差异
- **制造业 PMI**：反映制造业景气度，是工业生产的先行指标
- **服务业 PMI**：反映服务业景气度，在消费驱动型经济体中更重要
- **综合 PMI**：制造业和服务业的加权平均，反映整体经济景气

## 为什么重要

PMI 是宏观经济最重要的先行指标之一：

- **及时性**：每月 1 号发布（中国通常在当月最后一天或次月 1 日发布），是所有宏观指标中最早的
- **先行性**：PMI 反映的是当月的变化趋势，领先 GDP 约 1-2 个季度
- **全面性**：覆盖新订单、生产、就业、库存等多个维度，是经济的"全景扫描"
- **国际可比**：全球 40+ 个国家发布 PMI，计算方法一致，便于横向比较
- **政策参考**：央行和政府在制定政策时高度关注 PMI。PMI 持续低于 50 可能触发刺激政策

**PMI 与 GDP 的关系**：

历史上，PMI 与 GDP 增长率高度相关。经验公式（仅供参考）：

$$\text{GDP 增长率} \approx (\text{PMI} - 50) \times 0.5 + 5$$

例如 PMI = 51，预测 GDP 增长率约 5.5%；PMI = 49，预测 GDP 增长率约 4.5%。

但这只是经验关系，精确预测需要更复杂的模型。

## 生活化类比

PMI 就像班级的"月度学习状态调查"。

老师每个月问全班同学："这个月学习状态比上个月好吗？"

- 如果超过一半的同学说"好"，PMI > 50，说明班级整体在进步
- 如果不到一半的同学说"好"，PMI < 50，说明班级整体在退步
- 如果刚好一半说"好"一半说"差"，PMI = 50，说明班级状态没变化

而且老师不只问一个笼统的问题，还细分了五个方面：

1. **新作业量**（新订单）：作业多了说明老师觉得你们在进步
2. **上课出勤率**（生产）：出勤率高说明学习状态好
3. **班干部数量**（从业人员）：班干部多了说明班级在扩充管理
4. **作业本发放速度**（供应商配送时间）：发放慢了说明作业量大、需求旺
5. **文具库存**（原材料库存）：囤文具说明准备大干一场

五个方面综合起来，就能判断班级是"蒸蒸日上"还是"每况愈下"。

## 实战应用

### PMI 数据解读

**趋势比绝对值更重要**：

- 单月 PMI 的绝对值意义有限，连续 3 个月的趋势才是关键
- PMI 从 48 升到 49，虽然仍在收缩区间，但说明收缩速度在放缓
- PMI 从 52 降到 51，虽然仍在扩张区间，但说明扩张速度在放缓

**分项指数分析**：

| 场景              | 可能含义                             |
| ----------------- | ------------------------------------ |
| 新订单↑ 生产↑     | 需求拉动型扩张，健康                 |
| 新订单↓ 库存↑     | 需求不足，库存积压，警惕             |
| 从业人员↓         | 企业裁员，就业压力增大               |
| 配送时间↑         | 供应链紧张（可能是需求旺或供给受限） |
| 新订单↑ 出口订单↑ | 内外需共振，经济向好                 |

**官方 PMI vs 财新 PMI 分化**：

- 官方 PMI > 财新 PMI：大型企业景气度好于中小企业
- 官方 PMI < 财新 PMI：中小企业景气度好于大型企业
- 分化持续扩大：结构性问题值得关注

### 中国 PMI 实践

**2024 年中国 PMI 走势**（示例数据）：

| 月份 | 制造业 PMI | 非制造业 PMI | 综合 PMI |
| ---- | ---------- | ------------ | -------- |
| 1 月 | 49.2       | 50.8         | 50.3     |
| 2 月 | 49.1       | 51.4         | 50.7     |
| 3 月 | 50.8       | 53.0         | 52.1     |
| 4 月 | 50.4       | 51.2         | 51.0     |
| 5 月 | 49.5       | 51.1         | 50.5     |

（来源：国家统计局，2024）

解读要点：

- 制造业 PMI 在 50 附近波动，反映制造业复苏基础不牢
- 非制造业 PMI 持续高于 50，服务业是经济的主要支撑
- 3 月 PMI 反弹与春节后复工复产有关

**PMI 与投资决策**：

| PMI 场景             | 股票             | 债券             | 商品             | 汇率     |
| -------------------- | ---------------- | ---------------- | ---------------- | -------- |
| PMI 持续 > 50 且上升 | 利好（盈利改善） | 利空（加息预期） | 利好（需求旺盛） | 本币升值 |
| PMI 持续 < 50 且下降 | 利空（盈利恶化） | 利好（降息预期） | 利空（需求萎缩） | 本币贬值 |
| PMI 从低位反弹       | 利好（预期改善） | 短期利空         | 利好（预期改善） | 本币升值 |
| PMI 从高位回落       | 短期利空         | 利好（政策预期） | 利空             | 本币贬值 |

### PMI 的局限性

- **扩散指数的特性**：PMI 只反映"方向"（扩张还是收缩），不反映"幅度"。PMI = 51 和 PMI = 60 都表示扩张，但幅度差异很大
- **样本偏差**：官方 PMI 样本偏大企业，可能无法完全反映中小企业状况
- **季节性因素**：春节、国庆等节假日会影响当月数据，需要关注季调后的数据
- **主观性**：基于采购经理的主观判断，可能存在偏差
- **不反映价格**：PMI 不包含价格信息，需要结合 PPI、CPI 等指标综合判断

## 关键要点

- PMI = 50 是荣枯线，高于 50 表示扩张，低于 50 表示收缩
- PMI 每月 1 号发布，是最及时的宏观经济指标
- 五大分项中，新订单（30% 权重）是最重要的分项
- 趋势比绝对值更重要，连续 3 个月的走势才有参考意义
- 官方 PMI 和财新 PMI 的分化反映大中小企业景气度差异
- PMI 是先行指标，领先 GDP 约 1-2 个季度

## 常见误区

- **PMI = 50 就是经济停滞**：PMI = 50 只表示与上月持平，不等于经济零增长。经济可能仍在以稳定速度增长
- **PMI 下降就是经济衰退**：PMI 从 55 降到 52 仍然是扩张，只是扩张速度放缓。只有 PMI < 50 才表示收缩
- **PMI 精确预测 GDP**：PMI 与 GDP 有经验关系，但不是精确公式。GDP 还受价格、净出口等因素影响
- **只看制造业 PMI**：在服务业占比超过 50% 的经济体中，服务业 PMI 同样重要
- **PMI 是滞后指标**：PMI 是先行指标，反映当月变化趋势，领先 GDP 等滞后指标

## 与其他概念的关系

- [GDP](/posts/gdp)：PMI 是 GDP 的先行指标，领先 1-2 个季度
- [CPI](/posts/cpi)：PMI 反映数量变化，CPI 反映价格变化，两者结合判断经济状况
- [PPI](/posts/ppi)：PMI 中的购进价格指数与 PPI 高度相关
- [经济周期](/posts/business-cycle)：PMI 是判断经济周期阶段（复苏、繁荣、衰退、萧条）的重要工具
