# 策略衰减

> 没有永远赚钱的策略，只有不断进化的投资者

## 概述

策略衰减（Strategy Decay），也叫 Alpha 衰减（Alpha Decay），是指一个原本盈利的交易策略随着时间的推移，赚钱能力逐渐下降甚至完全失效的现象。

这就像一条被越来越多人发现的小路——刚开始只有你一个人走，畅通无阻；后来走的人多了，路变得拥挤，甚至堵死了。量化策略也是如此：当一个赚钱的规律被太多人发现并使用，这个规律本身就会被"套利"掉，不再产生超额收益。

在量化行业，策略衰减不是"会不会发生"的问题，而是"什么时候发生"的问题。

## 为什么重要

策略衰减是量化投资无法回避的现实：

- **策略有生命周期**：没有永远赚钱的策略，只有不断进化的投资者
- **竞争加剧**：随着量化基金越来越多、算力越来越强，策略衰减的速度在加快
- **持续研发压力**：量化机构必须不断寻找新的 Alpha 来源，否则就会被市场淘汰
- **实盘监控必要**：即使策略上线，也需要持续跟踪其表现，及时发现衰减信号
- 据 AQR 研究，一个典型量化策略的 Alpha 半衰期（衰减到一半的时间）约为 3-5 年（来源：AQR 研究报告, "The Search for Alpha in a Crowded World"）

## 核心原理

### 策略衰减的机制

策略衰减的根本原因是**竞争**。当一个策略产生超额收益时，会吸引其他市场参与者模仿或开发类似策略，导致：

1. **价格效应**（Price Impact）：
   - 越来越多的人买入同一类资产，推高价格
   - 价格被推高后，未来预期收益下降
   - 最终价格反映所有已知信息，超额收益消失

2. **市场学习**（Market Learning）：
   - 市场参与者通过学习逐渐认识到某种规律
   - 学术论文发表、行业会议分享加速了知识传播
   - 一旦规律被广泛知晓，就会被套利掉

3. **市场结构变化**（Market Structure Change）：
   - 交易规则变化（如涨跌停制度调整）
   - 新的参与者进入（如 ETF 的普及）
   - 监管政策变化（如做空限制）

4. **宏观经济环境变迁**：
   - 利率环境变化影响价值/成长因子的表现
   - 通胀/通缩周期改变资产定价逻辑
   - 全球化/逆全球化影响跨境套利策略

### 衰减速度模型

策略衰减可以用指数衰减模型近似：

$$\alpha(t) = \alpha_0 \cdot e^{-\lambda t}$$

其中 $\alpha_0$ 是初始 Alpha，$\lambda$ 是衰减速率，$t$ 是时间。

不同策略类型的衰减速度差异很大：

- **高频策略**：几周或几个月，因为信息传播极快（来源：行业经验数据）
- **中频统计套利**：6 个月到 2 年
- **低频基本面策略**：3 到 10 年，因为基本面变化较慢

## 技术/方法详解

### 监控策略衰减的指标

**1. 滚动夏普比率**（Rolling Sharpe Ratio）：

计算策略在最近 $N$ 个月的滚动夏普比率，观察趋势：

$$\text{Rolling SR}_t = \frac{\text{Mean}(r_{t-N:t})}{\text{Std}(r_{t-N:t})} \cdot \sqrt{12}$$

持续下降趋势是衰减的早期信号。

**2. 信息系数衰减**（IC Decay）：

跟踪因子 IC（Information Coefficient）随时间的变化：

$$IC_t = \text{Correlation}(\text{Factor Score}_t, \text{Return}_{t+1})$$

IC 持续下降意味着因子预测能力减弱。

**3. 最大回撤扩大**：

策略的最大回撤（Max Drawdown）超出历史范围，可能意味着策略逻辑已失效。

**4. 与市场相关性上升**：

策略收益与市场收益的相关性上升，意味着 Alpha 在消失，策略退化为 Beta。

**5. 换手率变化**：

策略换手率异常升高，可能意味着市场结构变化导致策略需要更频繁调整。

### 应对策略衰减的方法

**1. 持续研发新策略**（Strategy Pipeline）：

头部量化机构采用"策略工厂"模式：

- 持续产生新策略想法
- 快速回测验证
- 小资金实盘测试
- 表现好的逐步加仓
- 表现差的及时淘汰

**2. 多策略组合**（Multi-Strategy Portfolio）：

将多个低相关性的策略组合在一起：

- 不同频率的策略（高频 + 中频 + 低频）
- 不同逻辑的策略（动量 + 均值回归 + 基本面）
- 不同资产类别的策略（股票 + 期货 + 外汇）

组合后的整体衰减速度远慢于单一策略。

**3. 动态权重调整**（Dynamic Weighting）：

根据策略近期表现动态调整权重：

- 表现好的策略增加权重
- 表现差的策略降低权重
- 使用**风险预算**（Risk Budgeting）框架分配权重

**4. 引入新数据源**：

- 使用另类数据获取新的信息优势
- 开发基于新数据源的策略
- 但需注意另类数据的 Alpha 衰减也在加速

**5. 策略参数自适应**：

- 使用在线学习（Online Learning）持续更新模型参数
- 根据市场状态（Market Regime）切换策略参数
- 但需警惕过度拟合近期数据

## 主流方案对比

| 应对方法       | 效果         | 实施难度 | 成本 | 适用场景             |
| -------------- | ------------ | -------- | ---- | -------------------- |
| 持续研发新策略 | 根本性解决   | 极高     | 极高 | 大型量化机构         |
| 多策略组合     | 有效降低衰减 | 高       | 高   | 中大型机构           |
| 动态权重调整   | 中等效果     | 中       | 中   | 各类机构             |
| 引入新数据源   | 短期有效     | 高       | 高   | 有数据工程能力的团队 |
| 参数自适应     | 有限效果     | 中       | 低   | 个人投资者           |

## 实战应用

### 策略生命周期管理

```
策略研发 → 回测验证 → 模拟交易 → 小资金实盘 → 逐步加仓
    ↓                                              ↓
策略衰减检测 ← 持续监控 ← 正常运营 ← 满仓运营
    ↓
衰减确认 → 降低权重 → 策略退役 → 经验归档
```

### 衰减检测的统计方法

使用**CUSUM**（Cumulative Sum）控制图检测策略表现的结构性变化：

$$S_t = \max(0, S_{t-1} + (r_t - \mu_0) - k)$$

其中 $\mu_0$ 是预期收益率，$k$ 是容忍度参数。当 $S_t$ 超过阈值时，发出衰减警报。

### Python 实现示例

```python
import numpy as np
import pandas as pd

def detect_strategy_decay(returns, window=60, threshold=0.5):
    """检测策略衰减"""
    rolling_sr = returns.rolling(window).mean() / returns.rolling(window).std() * np.sqrt(12)

    # 计算夏普比率的衰减趋势
    recent_sr = rolling_sr.iloc[-window:]
    trend = np.polyfit(range(len(recent_sr)), recent_sr.values, 1)[0]

    if trend < -threshold:
        return {
            'status': 'DECAYING',
            'trend': trend,
            'current_sr': rolling_sr.iloc[-1],
            'peak_sr': rolling_sr.max()
        }
    return {'status': 'NORMAL', 'trend': trend}
```

## 关键要点

- 策略衰减的原因：市场参与者学习、市场结构变化、监管政策调整、宏观经济环境变迁
- 衰减速度因策略类型而异：高频最快（几周或几个月），低频基本面最慢（几年）
- 应对方法：持续研发新策略、多策略组合、动态调整权重、引入新数据源
- 监控指标：滚动收益率、夏普比率变化、最大回撤扩大、与市场相关性上升
- "策略工厂"模式：头部量化机构像工厂一样持续生产新策略，以对冲老策略的衰减
- 策略衰减是正常现象，即使最初抓住的是真实规律，也会因竞争而消失

## 常见误区

- **"好策略永远有效"**：没有永远赚钱的策略，只有不断进化的投资者
- **"衰减 = 策略错了"**：策略衰减是正常现象，即使最初抓住的是真实规律也会衰减
- **"低频策略不会衰减"**：低频策略衰减较慢，但同样会因市场参与者学习而失效
- **"发现衰减再行动"**：应该持续监控，在衰减早期就及时调整或切换
- **"策略衰减无法应对"**：通过多策略组合和持续研发可以有效应对
- **"论文发表的策略可以直接用"**：论文发表时策略可能已经衰减，公开的策略往往已失效
