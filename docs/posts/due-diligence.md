# 尽职调查

买房子前你要验房，投资前你要"验公司"——查财务、查法律、查业务、查团队，把企业的底细摸清楚再决定掏不掏钱。这就是尽职调查。

## 概述

尽职调查（Due Diligence, DD）是指**在投资、并购或合作前，对目标企业进行全面、系统的调查和评估，以识别风险、验证信息、支持决策的过程**。

尽职调查的核心目的：

- **验证信息**：确认企业提供的信息是否真实、准确、完整
- **识别风险**：发现潜在的法律、财务、业务、技术风险
- **支持估值**：为估值和定价提供依据
- **交易设计**：根据调查结果设计交易结构和保护条款
- **整合规划**：为并购后的整合做准备

尽职调查的主要类型：

| 类型             | 调查内容                       | 负责方          | 关键问题                               |
| ---------------- | ------------------------------ | --------------- | -------------------------------------- |
| 财务尽职调查     | 财务报表、会计政策、税务       | 会计师事务所    | 财务数据是否真实？盈利质量如何？       |
| 法律尽职调查     | 公司治理、合同、诉讼、知识产权 | 律师事务所      | 有没有未披露的诉讼？知识产权是否清晰？ |
| 业务尽职调查     | 市场地位、竞争格局、客户集中度 | 投资方/咨询机构 | 商业模式是否可持续？竞争优势是什么？   |
| 技术尽职调查     | 技术架构、代码质量、技术债务   | 技术专家        | 技术是否领先？有没有技术债务？         |
| 人力资源尽职调查 | 团队背景、薪酬福利、劳动关系   | 人力资源顾问    | 核心团队是否稳定？有没有劳动纠纷？     |
| ESG 尽职调查     | 环境、社会、治理               | ESG 专家        | 有没有环境合规风险？治理结构是否健全？ |

## 为什么重要

- **风险控制**：尽职调查是投资前最重要的风险控制环节
- **信息对称**：减少信息不对称，避免"买柠檬"（Akerlof 的柠檬市场理论）
- **谈判筹码**：调查结果可以作为谈判筹码，影响价格和条款
- **交易成败**：重大风险发现可能导致交易终止
- **并购整合**：尽职调查为并购后的整合提供基础

## 核心原理

### 尽职调查的流程

**1. 准备阶段**

- 确定调查范围和重点
- 组建调查团队（财务、法律、业务、技术等）
- 制定调查计划和时间表
- 向目标企业发送信息清单（Information Request List）

**2. 信息收集**

- 收集目标企业提供的文件和数据
- 进行管理层访谈
- 进行客户、供应商、合作伙伴访谈
- 进行行业和市场调研

**3. 分析评估**

- 财务数据分析
- 法律风险评估
- 业务模式评估
- 技术能力评估
- 风险识别和量化

**4. 报告输出**

- 撰写尽职调查报告
- 列出发现的问题和风险
- 提出建议和解决方案
- 向投资决策委员会汇报

**5. 交易支持**

- 根据调查结果调整估值
- 设计交易保护条款（如陈述与保证、赔偿条款）
- 支持交易谈判

### 财务尽职调查的核心内容

**1. 财务报表分析**

- **收入确认**：收入确认政策是否合理？有没有提前确认收入？
- **成本费用**：成本结构是否合理？有没有隐藏费用？
- **资产负债**：资产质量如何？负债是否完整披露？
- **现金流**：经营性现金流是否与净利润匹配？

**2. 盈利质量分析**

$$\text{盈利质量} = \frac{\text{经营性现金流}}{\text{净利润}}$$

- 比值 > 1：盈利质量高（现金流充沛）
- 比值 < 1：盈利质量低（利润可能来自非现金项目）

**3. 财务预测验证**

- 历史数据与预测数据的对比
- 关键假设的合理性
- 敏感性分析

**4. 税务尽职调查**

- 税务合规性
- 税收优惠的可持续性
- 潜在的税务风险

### 法律尽职调查的核心内容

**1. 公司治理**

- 股权结构是否清晰？
- 公司章程和治理结构是否规范？
- 有没有关联交易和利益输送？

**2. 合同审查**

- 重大合同是否有效？
- 有没有不利条款（如排他性、高额违约金）？
- 合同到期和续约情况？

**3. 诉讼和仲裁**

- 有没有未决诉讼？
- 潜在的法律风险？
- 历史诉讼的处理情况？

**4. 知识产权**

- 专利、商标、著作权是否清晰？
- 有没有侵权风险？
- 知识产权的有效期和保护范围？

## 模型/方法详解

### 财务尽职调查的分析框架

**杜邦分析**（DuPont Analysis）：

$$ROE = \text{净利润率} \times \text{总资产周转率} \times \text{权益乘数}$$

$$ROE = \frac{\text{净利润}}{\text{收入}} \times \frac{\text{收入}}{\text{总资产}} \times \frac{\text{总资产}}{\text{股东权益}}$$

通过杜邦分析，可以识别 ROE 变化的驱动因素：

- 利润率变化：盈利能力变化
- 周转率变化：运营效率变化
- 权益乘数变化：杠杆变化

**财务预警模型**：

Altman Z-Score 模型（来源：Altman, 1968）：

$$Z = 1.2X_1 + 1.4X_2 + 3.3X_3 + 0.6X_4 + 1.0X_5$$

其中：

- $X_1$ = 营运资本 / 总资产
- $X_2$ = 留存收益 / 总资产
- $X_3$ = EBIT / 总资产
- $X_4$ = 股东权益市值 / 总负债
- $X_5$ = 收入 / 总资产

判断标准：

- $Z > 2.99$：安全区
- $1.81 < Z < 2.99$：灰色区
- $Z < 1.81$：困境区

### 业务尽职调查的分析框架

**波特五力模型**（Porter's Five Forces）：

| 力量           | 分析内容                         |
| -------------- | -------------------------------- |
| 行业内竞争     | 竞争对手数量、市场份额、竞争强度 |
| 新进入者威胁   | 进入壁垒、规模经济、品牌忠诚度   |
| 替代品威胁     | 替代品的性价比、转换成本         |
| 供应商议价能力 | 供应商集中度、转换成本、前向整合 |
| 客户议价能力   | 客户集中度、转换成本、后向整合   |

**SWOT 分析**：

|      | 有利                  | 不利               |
| ---- | --------------------- | ------------------ |
| 内部 | 优势（Strengths）     | 劣势（Weaknesses） |
| 外部 | 机会（Opportunities） | 威胁（Threats）    |

### 技术尽职调查的分析框架

**技术评估维度**：

| 维度       | 评估内容                       |
| ---------- | ------------------------------ |
| 技术先进性 | 与行业领先水平的对比           |
| 技术成熟度 | 技术是否经过市场验证           |
| 技术债务   | 代码质量、架构合理性、维护成本 |
| 技术团队   | 团队规模、能力、稳定性         |
| 技术风险   | 技术依赖、单点故障、安全风险   |
| 可扩展性   | 技术架构是否支持业务增长       |

**技术债务评估**：

$$\text{技术债务} = \text{重构成本} + \text{维护成本} + \text{机会成本}$$

高技术债务意味着：

- 未来需要投入大量资源进行重构
- 系统稳定性和安全性风险
- 新功能开发速度受限

## 主流方案对比

| 调查类型       | 深度 | 时间   | 成本 | 适用场景               |
| -------------- | ---- | ------ | ---- | ---------------------- |
| 全面尽职调查   | 深   | 4-8 周 | 高   | 大型并购、PE 投资      |
| 有限尽职调查   | 中   | 2-4 周 | 中   | 小额投资、跟投         |
| 确认性尽职调查 | 浅   | 1-2 周 | 低   | 已有意向，确认关键问题 |
| 卖方尽职调查   | 中   | 2-4 周 | 中   | 卖方主动披露，加速交易 |

## 实战应用

### 尽职调查的常见红旗信号

**财务红旗**：

- 收入增长但现金流为负
- 应收账款大幅增加
- 关联交易占比过高
- 频繁更换审计师
- 会计政策频繁变更

**法律红旗**：

- 未披露的诉讼或仲裁
- 核心资产权属不清
- 关键合同存在不利条款
- 合规问题（如环保、劳动）
- 股权结构复杂，存在代持

**业务红旗**：

- 客户集中度过高（单一客户占比 > 50%）
- 供应商集中度过高
- 市场份额持续下降
- 核心团队成员流失
- 商业模式不可持续

**技术红旗**：

- 核心技术依赖第三方
- 代码质量差，技术债务高
- 安全漏洞频发
- 技术团队不稳定
- 技术路线落后于行业

### 尽职调查的常见陷阱

**1. 信息不对称**

- 目标企业可能隐瞒不利信息
- 需要多渠道验证（客户访谈、供应商访谈、行业调研）

**2. 时间压力**

- 交易时间紧迫，尽职调查可能被压缩
- 需要优先关注关键风险

**3. 确认偏差**

- 投资方可能只关注支持投资的信息
- 需要保持客观，主动寻找反面证据

**4. 过度依赖第三方**

- 会计师事务所、律师事务所的报告可能不全面
- 投资方需要有自己的判断

### 数字化尽职调查

随着技术发展，尽职调查也在数字化：

- **数据分析工具**：自动化财务数据分析
- **AI 辅助**：合同审查、风险识别
- **区块链**：供应链溯源、交易验证
- **大数据**：客户评价、舆情分析
- **网络安全评估**：渗透测试、漏洞扫描

## 关键要点

- 尽职调查是投资前最重要的风险控制环节
- 财务、法律、业务、技术尽职调查缺一不可
- 尽职调查不仅是为了发现问题，更是为了支持交易设计和估值
- 红旗信号需要高度重视，但不一定意味着交易终止
- 数字化正在改变尽职调查的方式和效率

## 常见误区

- **误区一**："尽职调查就是查账"——尽职调查涵盖财务、法律、业务、技术等多个维度
- **误区二**："发现问题就终止交易"——很多问题可以通过交易条款设计来解决
- **误区三**："尽职调查越全面越好"——需要在深度和效率之间平衡，关注关键风险
- **误区四**："尽职调查是投资后的事"——尽职调查应该在投资前完成，为决策提供依据
- **误区五**："第三方报告就够了"——投资方需要有自己的判断和验证
