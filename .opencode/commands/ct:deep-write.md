对指定词条执行深度写作全流程：自动识别词条类型 → 匹配写作策略 → 生成大纲 → 撰写正文 → 质量审查 → 自动修复。

如果用户未指定词条，询问：

1. 为哪个词条执行深度写作？
2. 是否覆盖已有内容？（如有内容，先确认）

## 执行步骤

### 步骤 1：识别词条类型

读取 `docs/glossary/index.md`，根据词条所属分类确定写作策略：

| 分类                     | 写作策略     |
| ------------------------ | ------------ |
| 基础概念、经济学经典理论 | 知识科普     |
| 金融学核心理论           | 理论深度解析 |
| 货币与银行、投资市场     | 体系讲解     |
| 衍生品与风控             | 原理+实战    |
| 公司金融                 | 实务指南     |
| 宏观经济、国际金融       | 宏观视角     |
| 个人理财                 | 实操指南     |
| 金融科技、新金融概念     | 前沿解读     |
| 金融史与重大危机         | 案例分析     |
| 金融监管与机构           | 制度解读     |
| 投资流派与方法论         | 方法论对比   |
| 量化投资                 | 技术深度解析 |
| 工具资源                 | 选型指南     |

### 步骤 2：生成大纲

根据写作策略生成对应结构的大纲，要求：

- 使用 Markdown 标题层级（## 一级章节，### 二级章节）
- 每个章节包含简短说明和建议字数
- 标注需要图表或公式的位置
- 列出需要交叉引用的相关词条（3-5 个）

### 步骤 3：撰写正文

按大纲逐章节撰写，要求：

- **专业性**：概念描述准确，数据引用最新
- **通俗性**：使用生活化类比解释复杂概念
- **可读性**：段落简短（不超过 5 行），逻辑清晰
- **术语规范**：金融术语首次出现标注英文
- **对比分析**：对比类词条必须包含方案对比表格
- **交叉引用**：内部链接使用相对路径（如 `/investing/stock`）
- **延伸阅读**：外部链接指向权威资料

### 步骤 4：质量审查

审查维度：

1. **内容准确性**：金融概念描述正确，无过时信息
2. **内容完整性**：覆盖大纲所有章节，无未完成段落
3. **语言一致性**：统一中文，术语前后一致，首次标注英文
4. **格式规范性**：标题层级正确，链接格式统一
5. **关联性**：包含 3-5 个相关词条链接，延伸阅读指向相关专题

### 步骤 5：自动修复

自动修复以下问题：

- 链接格式不一致
- 金融术语首次出现未标注英文
- 提示块使用不当

## 写作策略详解

### 知识科普（基础概念、经济学经典理论）

```markdown
## 概述

- 核心定义（100-150 字）
- 一句话通俗解释

## 演进历史

- 起源与发展脉络
- 关键里程碑

## 核心分类/类型

- 分类 1：说明 + 举例
- 分类 2：说明 + 举例

## 为什么重要

- 在金融体系中的地位
- 对普通人的影响

## 与其他概念的关系

- 链接到相关词条（3-5 个）

## 延伸阅读

- 权威书籍、论文、参考资料
```

### 理论深度解析（金融学核心理论）

```markdown
## 概述

- 核心定义与理论定位
- 解决了什么问题

## 理论前提

- 基本假设
- 适用条件

## 核心公式/模型

- 公式说明（配图）
- 各变量含义
- 计算示例

## 理论意义

- 理论创新点
- 对金融实践的影响

## 局限与批评

- 主要争议
- 现实中的偏离

## 与其他概念的关系

- 链接到相关词条（3-5 个）

## 延伸阅读

- 原始论文、经典教材、权威解读
```

```markdown
## 运作机制

- 流程说明
- 关键环节

## 主要产品/工具

- 产品 2：特点 + 示例

## 风险与注意事项

- 主要风险

## 与其他概念的关系

- 链接到相关词条（3-5 个）

## 延伸阅读

- 权威资料、学习资源
```
