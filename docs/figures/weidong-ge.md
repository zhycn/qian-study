# 葛卫东——混沌投资

> 他是中国最神秘的"期货大佬"。从期货市场起家，到股票、股权投资，葛卫东用宏观对冲和趋势跟踪策略，构建了一个超过 200 亿元的投资帝国。他低调神秘，很少接受采访，但他的每一次操作都在市场上引起关注。

## 人物速写

葛卫东，混沌投资创始人，中国最知名的期货投资者之一。他从期货市场起家，后来扩展到股票和股权投资，是宏观对冲和趋势跟踪策略的代表人物。

## 传奇人生

### 早年经历：期货市场的起步

葛卫东的早年经历并不为外界所知。他出生于浙江，1990 年代开始接触期货市场。

那个年代的中国期货市场：

- 市场规则不完善
- 监管不成熟
- 波动极其剧烈
- 风险极高

葛卫东在这个阶段积累了大量期货交易经验。他后来回忆：

> "期货市场教会了我一件事：趋势是你的朋友。顺势而为，这是期货交易的核心。"

**关于期货市场起家的具体案例：** 1990 年代，葛卫东在上海期货交易所参与铜、大豆等品种交易。据《第一财经》报道，葛卫东在 1990 年代末期的铜期货交易中获利丰厚，完成了原始积累。他善于从宏观经济角度判断商品趋势，结合技术分析进行趋势跟踪（来源：《第一财经》报道、中国期货业协会）。

### 投资启蒙：从期货到股票

2000 年代，葛卫东开始将投资范围从期货扩展到股票。

**扩展逻辑：**

- 期货市场学到的宏观分析能力可以应用于股票市场
- 趋势跟踪策略在股票市场同样有效
- 股票市场的容量更大，可以容纳更多资金
- 多元化投资可以降低风险

葛卫东的投资风格逐渐形成：

- **宏观对冲**——从宏观经济角度判断市场方向
- **趋势跟踪**——顺势而为，不逆势操作
- **科技偏好**——偏好科技股，尤其是硬科技

### 关键转折点：创建混沌投资

葛卫东创立了混沌投资，名字来源于"混沌理论"（Chaos Theory）——一种研究复杂系统的科学。

**混沌理论在投资中的应用：**

混沌理论并非简单的"顺势而为"，而是对复杂系统行为的深刻理解：

- **初始条件敏感性**——市场微小的变化可能引发巨大波动（"蝴蝶效应"），因此需要严格风险管理
- **非线性系统**——市场不是线性可预测的，趋势可能突然反转，需要动态调整
- **自组织临界性**——市场会在稳定和不稳定之间切换，投资者需要识别系统状态
- **分形特征**——市场在不同时间尺度上呈现相似的模式，可用于多周期分析

葛卫东将混沌理论应用于投资实践：

1. **承认不可预测性**——不试图精确预测市场，而是识别趋势和概率
2. **动态适应**——根据市场状态调整策略，而非固守单一模型
3. **风险管理优先**——在复杂系统中，生存比盈利更重要
4. **多维度分析**——结合宏观、行业、技术等多维度信息

混沌投资的核心策略：

1. **宏观对冲**——从宏观经济角度判断市场方向
2. **趋势跟踪**——顺势而为，不逆势操作
3. **科技投资**——偏好科技股，尤其是硬科技
4. **多元化**——期货、股票、股权多元化配置

### 巅峰时刻：科技股投资

葛卫东在科技股投资方面表现出色。

**代表性投资：**

- **科大讯飞**——人工智能龙头
- **用友网络**——企业软件龙头
- **兆易创新**——半导体设计龙头
- **北方华创**——半导体设备龙头

葛卫东的科技股投资逻辑：

- 中国正在经历科技转型
- 硬科技是未来趋势
- 科技股有长期增长潜力
- 宏观趋势有利于科技行业

## 经典战役

### 战役一：期货市场起家（1990 年代）

葛卫东在期货市场积累了第一桶金。

**交易风格：**

- 宏观对冲——从宏观经济角度判断方向
- 趋势跟踪——顺势而为
- 严格止损——控制风险
- 仓位管理——根据趋势调整仓位

**具体案例：** 据《第一财经》报道，葛卫东在 1990 年代末期的铜期货交易中，通过宏观分析判断铜价上涨趋势，大举做多，获利丰厚。这笔交易让他完成了从散户到专业交易者的转变（来源：《第一财经》报道）。

**结果：**

- 在期货市场获得了丰厚回报
- 积累了大量交易经验
- 形成了自己的投资框架
- 为后来的股票投资奠定了基础

### 战役二：投资科大讯飞（2013-2020）

葛卫东是科大讯飞的长期投资者。

**背景：**

- 科大讯飞是中国人工智能龙头企业
- 语音识别技术领先
- 但市场对其商业化能力存在争议

**葛卫东的逻辑：**

- 人工智能是未来趋势
- 科大讯飞在语音识别技术上有优势
- 宏观趋势有利于 AI 行业
- 长期来看，价值会持续增长

**结果：**

- 科大讯飞股价从 2013 年的约 15 元涨到 2020 年的超过 60 元
- 回报率超过 **4 倍**（来源：上市公司前十大流通股东数据、科大讯飞年报，2020）
- 葛卫东在科大讯飞上获利丰厚

### 战役三：投资兆易创新（2019-2021）

葛卫东投资了兆易创新。

**背景：**

- 兆易创新是中国半导体设计龙头
- 存储芯片是核心产品

**结果：**

- 兆易创新股价从 2019 年的约 50 元涨到 2021 年的超过 200 元
- 回报率超过 **4 倍**（来源：上市公司前十大流通股东数据、兆易创新年报，2021）
- 葛卫东在兆易创新上获利丰厚

## 投资哲学

### 原则一：宏观对冲

这是葛卫东投资哲学的核心。

> "从宏观经济角度判断市场方向，这是投资的第一步。"

宏观对冲的体现：

- 关注宏观经济数据
- 判断经济周期位置
- 根据周期调整配置
- 在不同资产间对冲风险

### 原则二：趋势跟踪

葛卫东是趋势跟踪的坚定实践者。

> "趋势是你的朋友。顺势而为，不逆势操作。"

趋势跟踪的体现：

- 识别市场趋势
- 顺势而为
- 不逆势操作
- 趋势反转时及时调整

### 原则三：科技偏好

葛卫东偏好科技股投资。

> "科技是未来。中国正在经历科技转型，这是最大的投资机会。"

科技偏好的逻辑：

- 中国正在经历科技转型
- 硬科技是未来趋势
- 科技股有长期增长潜力
- 宏观趋势有利于科技行业

### 原则四：多元化配置

葛卫东强调多元化配置。

> "不要把所有鸡蛋放在一个篮子里。"

多元化配置的体现：

- 期货、股票、股权多元化
- 不同行业配置
- 不同资产类别配置
- 降低单一资产的风险

### 原则五：严格风险管理

葛卫东强调严格的风险管理。

> "风险管理是投资的核心。"

风险管理的体现：

- 严格止损
- 仓位管理
- 分散投资
- 控制回撤

## 策略拆解

### 选股标准

葛卫东的选股标准：

| 标准 | 具体要求 |
| --- | --- |
| 宏观趋势 | 符合宏观经济趋势 |
| 行业龙头 | 在行业中具有领先地位 |
| 科技偏好 | 偏好科技股 |
| 趋势向上 | 股价趋势向上 |

### 买入时机

葛卫东的买入时机判断：

1. **宏观趋势有利**——宏观经济有利于该行业
2. **趋势向上**——股价趋势向上
3. **估值合理**——价格与价值匹配
4. **行业景气**——行业处于景气周期

### 持仓管理

葛卫东的持仓管理：

- **趋势跟踪**——根据趋势调整仓位
- **严格止损**——趋势反转时及时止损
- **多元化**——不同行业、不同资产配置
- **持续跟踪**——关注宏观和行业变化

### 卖出纪律

葛卫东的卖出条件：

1. **趋势反转**——股价趋势反转
2. **宏观变化**——宏观经济不利于该行业
3. **基本面恶化**——公司竞争力丧失
4. **发现更好的机会**——换仓到趋势更好的标的

### 普通人能学到什么

葛卫东的方法对普通投资者的启示：

1. **关注宏观**——从宏观经济角度判断市场方向
2. **顺势而为**——趋势是你的朋友
3. **科技偏好**——关注科技行业的长期趋势
4. **严格止损**——控制风险
5. **多元化**——不要把所有鸡蛋放在一个篮子里

**具体建议：**

- 关注宏观经济数据
- 识别市场趋势
- 顺势而为，不逆势操作
- 严格止损，控制风险
