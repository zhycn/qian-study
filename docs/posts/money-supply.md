# M0/M1/M2

M0/M1/M2就像衡量市场上"钱有多少"的三把尺子——M0是手头现金,M1是随时能花的钱,M2是连定期存款都算上,口径从小到大,流动性从高到低。

## 概述

货币供应量（Money Supply）是指一个国家在某一时刻社会上流通的货币总量。根据流动性的不同，央行将货币供应量划分为不同层次：

**M0（流通中现金，Currency in Circulation）**：

- 银行体系以外流通的现金
- 包括你钱包里的纸币和硬币
- 不包括银行库存现金
- 流动性最强，随时可以用来买东西
- 在中国，M0 占 M2 的比重约为 4%-5%（来源：中国人民银行），且呈下降趋势（电子支付普及）

**M1（狭义货币，Narrow Money）**：

- M0 + 企业活期存款 + 机关团体部队存款 + 农村存款 + 个人持有的信用卡类存款
- 企业可以随时支取用于支付的资金
- 反映经济中的现实购买力
- M1 增速常被用来衡量经济活跃度

**M2（广义货币，Broad Money）**：

- M1 + 城乡居民储蓄存款 + 企业定期存款 + 外币存款 + 信托类存款 + 其他存款
- 包含了暂时不能直接用于支付、但可以转化为支付能力的资金
- 反映现实购买力 + 潜在购买力
- 是中国最受关注的货币指标

在中国，M2 是最受关注的货币指标，常被用来衡量货币政策的松紧程度。中国央行每月 10 日左右公布上月的金融统计数据（来源：中国人民银行），M2 增速是市场关注的焦点。

不同国家对 M1、M2 的定义略有差异。例如，美国的 M1 包括现金、旅行支票和活期存款，M2 还包括储蓄存款、小额定期存款和零售货币市场基金份额。日本的 M2 加上邮政储蓄和农林中金存款被称为 M2+CD。

## 演进历史

货币供应量统计口径的演变反映了金融体系的不断发展。

**早期阶段**：在现金经济时代，货币供应量的衡量相对简单——主要就是流通中的现金。随着银行体系的发展，存款逐渐成为货币的重要组成部分。

**层次划分阶段（20 世纪中期）**：随着金融工具的多样化，单一口径已无法准确反映货币供应状况。各国央行开始按流动性对货币供应量进行层次划分。美联储在 20 世纪 70 年代正式确立了 M1、M2、M3 的统计框架。

**中国实践**：中国人民银行于 1994 年三季度开始按季公布 M0、M1、M2 统计口径的货币供应量数据（来源：中国人民银行，1994）。此后根据金融创新的发展，对统计口径进行了多次调整。2001 年，将证券公司客户保证金计入 M2；2011 年，将住房公积金存款和非存款类金融机构在存款类金融机构的存款计入 M2；2018 年，将货币市场基金纳入 M2 统计（来源：中国人民银行，2001-2018）。

**数字化时代的挑战**：随着电子支付、数字货币、金融科技的发展，传统货币层次划分的边界日益模糊。例如，余额宝等货币市场基金既具有支付功能又具有投资属性，难以简单归类。中国数字人民币（e-CNY）的推出也可能对货币统计口径产生影响。

## 核心分类/类型

货币供应量的层次划分因国家而异，但基本逻辑相同——按流动性从高到低排列。

**中国的货币层次**：

- **M0** = 流通中现金
- **M1** = M0 + 单位活期存款（企业活期存款 + 机关团体存款等）
- **M2** = M1 + 单位定期存款 + 个人存款 + 其他存款

**美国的货币层次**：

- **M0** = 流通中现金 + 银行库存现金 + 银行在美联储的准备金
- **M1** = 流通中现金 + 旅行支票 + 活期存款 + 其他可开支票存款
- **M2** = M1 + 储蓄存款 + 小额定期存款（10 万美元以下）+ 零售货币市场基金
- **M3**（2006 年停止公布，来源：美联储，2006）= M2 + 大额定期存款 + 机构货币市场基金 + 回购协议等

**日本的货币层次**：

- **M1** = 现金 + 活期存款
- **M2** = M1 + 定期存款 + 储蓄存款（称为"M2+CD"）
- **M3** = M2 + 邮政储蓄 + 农林中金存款

值得注意的是，各国对 M1 的定义差异较大。中国的 M1 不包括个人活期存款（个人活期存款计入 M2），而美国和日本的 M1 包括个人活期存款。这意味着中国的 M1 口径相对较窄，在跨国比较时需要注意口径差异。

## 为什么重要

不同层次的货币供应量提供了不同的经济信号。

**M1 增速快**：说明企业和居民交易活跃，经济景气度高。企业活期存款增加，意味着企业经营活动频繁，资金周转加快。

**M2 增速快**：说明整体货币供应充足，但可能包含大量定期存款。如果 M2 增速主要由定期存款驱动，可能说明企业和居民投资意愿不足，更倾向于"存钱"而非"花钱"。

**M1-M2 剪刀差**：M1 增速高于 M2，说明资金活化（资金从定期转向活期），经济活力增强；M2 增速高于 M1，说明资金定期化（资金从活期转向定期），经济活力不足。这个指标被广泛用于判断经济周期。

**通胀预判**：M2 增速长期大幅高于 GDP 增速，可能引发通货膨胀。根据货币数量论（Quantity Theory of Money），MV = PQ（货币供应量 × 货币流通速度 = 价格水平 × 产出），如果货币流通速度相对稳定，M2 增速持续高于 GDP 增速，多余的货币最终会反映在物价上涨上。

**政策制定参考**：央行通过调控货币供应量来实现物价稳定、经济增长等宏观目标。中国曾将 M2 增速作为货币政策的中间目标，近年来逐步转向价格型调控（利率调控），但 M2 增速仍是重要的监测指标。

## 生活化类比

把货币想象成不同"形态"的水：

- **M0** = 杯子里的水：拿起来就能喝，流动性最强
- **M1** = 杯子里的水 + 水壶里的水：稍微麻烦一点，但也能很快喝到
- **M2** = 杯子里的水 + 水壶里的水 + 冰箱里冻成冰的水：冰需要融化才能喝，但最终也能变成能喝的水

M0 最少但最灵活，M2 最多但包含了一些"暂时不能直接用"的钱。

再换一个角度：把货币想象成不同"距离"的朋友。M0 是你最好的朋友，随叫随到；M1 是你的好朋友，打个电话就能来；M2 包括所有认识的人，有些需要约时间才能见面。人数（钱的总量）越来越多，但"随叫随到"的比例越来越低。

## 关键要点

- M0、M1、M2 是包含关系，口径依次扩大
- M2 是中国最受关注的货币指标
- 各层次货币的增速差异反映经济结构变化
- M2/GDP 比值常被用来衡量一国金融深化程度
- 中国 M2 余额已超过 300 万亿元（来源：中国人民银行，2024），居全球首位
- M1-M2 剪刀差是判断经济活力的重要指标
- 电子支付普及导致 M0 占 M2 比重持续下降
- 跨国比较 M2 时需注意统计口径差异

## 常见误区

- **误区一：M2 越多说明经济越好**。M2 增速需要与 GDP 增速匹配，M2 长期大幅高于 GDP 增速可能意味着资金空转或资产泡沫，而非实体经济繁荣。中国 M2/GDP 比值超过 200%（来源：中国人民银行/国家统计局），远高于美国（约 80%，来源：美联储/BEA），这与中国以间接融资为主的金融结构和高储蓄率有关，不能简单理解为"印钞过多"。

- **误区二：M1 比 M2 更重要**。两者各有侧重，M1 反映短期经济活力，M2 反映整体货币环境。结合分析 M1-M2 剪刀差才能全面判断经济状况。单独看任何一个指标都可能得出片面结论。

- **误区三：中国的 M2 全球第一说明印钞最多**。M2 统计口径各国不同，中国以间接融资（银行贷款）为主、居民储蓄率高，这些因素都推高了 M2 规模。美国以直接融资（股市、债市）为主，大量融资活动不体现在 M2 中。因此不能简单横向比较。

- **误区四：M2 增速下降就是"钱变少了"**。M2 增速下降只说明货币供应增长放缓，不代表绝对量减少。此外，随着金融脱媒（Disintermediation）和理财产品的普及，部分资金从银行存款转向其他金融产品，可能不再计入 M2，但这不意味着资金消失了。
