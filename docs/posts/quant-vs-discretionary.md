# 量化 vs 主观投资

> 一个是"让数据和模型说话"，一个是"让经验和直觉判断"，没有绝对的对错，只有适合与不适合。

## 概述

投资世界里一直存在两大阵营：

**量化投资**（Quantitative Investing）相信数据和模型。它认为市场虽然复杂，但总有规律可循——只要你有足够的数据、合适的模型和强大的算力，就能找到这些规律并从中获利。

**主观投资**（Discretionary Investing）相信人的判断力。它认为市场是无数参与者博弈的结果，充满了非理性行为和突发事件，只有依靠深入的公司研究、行业洞察和投资直觉，才能做出正确的决策。

两种方法各有优劣，也各有成功的代表人物：量化阵营有詹姆斯·西蒙斯（James Simons）和他的大奖章基金（30 年年化 66% 费前（来源：基金年报, SEC 文件）），主观阵营有沃伦·巴菲特（Warren Buffett）和伯克希尔·哈撒韦（57 年年化 20%（来源：Berkshire Hathaway 年度股东信））。

## 为什么重要

理解这两种投资哲学的差异，有助于你：

- **认识自己的投资风格**：你是更相信数据，还是更相信自己的判断？
- **理解市场参与者**：市场上既有量化基金在跑算法，也有主观基金经理在做调研，他们的行为模式不同，对市场的影响也不同
- **构建组合**：很多机构投资者会将量化和主观策略搭配使用，实现互补
- **评估基金产品**：知道一个基金是量化驱动还是主观驱动，才能正确评估其风险和收益特征

## 核心原理

### 量化投资的哲学基础

量化投资的认识论可以追溯到两个学术传统：

**有效市场假说（EMH）的"弱式"版本**：市场并非完全有效，价格会短暂偏离基本面，这些偏离可以被系统性地捕捉。

**行为金融学**：投资者存在系统性的认知偏差——过度自信、损失厌恶、锚定效应、羊群效应——这些偏差会导致价格出现可预测的模式。

量化投资的数学框架基于 CAPM 和多因子模型：

$$R_i = \alpha_i + \beta_{i1}F_1 + \beta_{i2}F_2 + \cdots + \beta_{ik}F_k + \varepsilon_i$$

其中 $\alpha_i$ 是超额收益（量化投资追求的目标），$F_j$ 是各类因子（市场、价值、动量等），$\beta_{ij}$ 是因子暴露。

### 主观投资的哲学基础

主观投资的认识论则建立在不同的前提上：

**市场并非完全可量化**：公司的竞争优势、管理层能力、行业格局变化等关键因素难以用数字精确刻画。

**深度研究创造信息优势**：通过实地调研、产业链访谈、财务分析，可以获得市场尚未充分定价的信息。

**长期主义**：短期市场是投票机，长期是称重机。通过长期持有优质公司，享受企业成长带来的复利。

主观投资的决策框架通常遵循"自上而下"或"自下而上"：

- **自上而下**：先判断宏观经济周期，再选择受益行业，最后精选个股
- **自下而上**：先找到好公司，再判断好价格，最后决定买入时机

## 技术/方法详解

### 量化投资的方法论

1. **数据驱动**：从海量数据中提取信号，覆盖广度优先
2. **系统化执行**：策略规则明确，由计算机自动执行，消除情绪干扰
3. **统计验证**：所有策略必须经过历史数据回测和样本外检验
4. **风险管理**：通过数学模型量化风险，设定严格的止损和仓位限制
5. **持续迭代**：市场结构变化时，模型需要重新训练和参数调整

### 主观投资的方法论

1. **深度研究**：对少数公司进行深入研究，覆盖深度优先
2. **定性判断**：管理层质量、企业文化、行业竞争格局等难以量化的因素
3. **安全边际**：以低于内在价值的价格买入，留出犯错空间
4. **集中持仓**：对最有信心的标的重仓持有，而非均匀分散
5. **长期持有**：减少交易频率，降低交易成本，享受复利效应

## 主流方案对比

| 维度     | 量化投资                        | 主观投资               |
| -------- | ------------------------------- | ---------------------- |
| 决策依据 | 数据、模型、算法                | 研究、经验、直觉       |
| 覆盖范围 | 广（可同时分析数千标的）        | 深（深度研究少数标的） |
| 持仓数量 | 数十~数百只                     | 5~30 只                |
| 情绪影响 | 低（模型无情执行）              | 高（人性弱点难避免）   |
| 透明度   | 高（逻辑可回溯）                | 低（"只可意会"）       |
| 适应变化 | 慢（模型需要重新训练）          | 快（人可以灵活调整）   |
| 换手率   | 中高                            | 低                     |
| 容量     | 大（指数增强）~小（高频）       | 中                     |
| 典型代表 | 西蒙斯、德肖、克利福德·阿斯内斯 | 巴菲特、林奇、段永平   |
| 最佳环境 | 震荡市、结构性行情              | 牛市、价值发现期       |
| 最差环境 | 趋势突变、量化拥挤              | 熊市、市场恐慌         |

### 各自的致命弱点

**量化投资的弱点：**

- **模型风险**：当市场结构发生根本性变化时，基于历史数据的模型可能完全失效。2007 年 8 月的"量化危机"（Quant Quake）（来源：Khandani & Lo, "What Happened to the Quants in August 2007?", Journal of Investment Management, 2007）就是典型案例——多家量化基金同时遭遇罕见回撤
- **过度拟合**：在历史数据上表现完美的策略，可能只是"背下了答案"，样本外表现一塌糊涂
- **策略拥挤**：当太多人使用相似的策略时，超额收益会被摊薄，甚至引发踩踏

**主观投资的弱点：**

- **认知偏差**：确认偏误（只看支持自己观点的信息）、损失厌恶（亏损时不愿止损）、过度自信
- **覆盖有限**：一个人能深度研究的公司数量有限，可能错过其他机会
- **难以复制**：优秀的主观投资者往往依赖个人能力和经验，策略难以规模化

## 实战应用

### 融合之道：量化 + 主观

越来越多的机构采用"量化增强主观"或"主观增强量化"的混合模式：

**量化赋能主观（Quantamental）：**

- 主观基金经理用量化模型做初步筛选，缩小研究范围
- 用量化模型监控组合风险，避免过度暴露
- 用另类数据（如卫星图像、社交媒体情绪）辅助基本面判断

**主观赋能量化：**

- 量化团队引入行业专家的主观判断，作为模型的额外因子
- 在模型失效期（如市场极端波动），由人工介入调整策略
- 用主观逻辑验证量化信号的经济学合理性，避免纯数据挖掘

### 个人投资者如何选择

| 你的特征                       | 推荐方向              |
| ------------------------------ | --------------------- |
| 擅长编程和数学，喜欢系统化思考 | 量化投资              |
| 擅长商业分析，喜欢深入研究公司 | 主观投资              |
| 时间充裕，愿意长期跟踪少数公司 | 主观投资              |
| 时间有限，希望系统化执行       | 量化投资              |
| 两者都感兴趣                   | Quantamental 混合模式 |

## 关键要点

- 量化和主观不是对立的，而是互补的。最成功的投资者往往融合两者
- 量化投资的优势在于广度、纪律和速度，劣势在于模型风险和对历史数据的依赖
- 主观投资的优势在于深度、灵活性和对定性因素的把握，劣势在于认知偏差和覆盖有限
- 选择哪种方法，取决于你的能力圈、时间投入和风险偏好
- 无论选择哪种方法，风险管理和持续学习都是不可或缺的

## 常见误区

- **"量化一定比主观好"**：两种方法各有优劣，巴菲特等主观投资者也取得了传奇业绩
- **"主观投资 = 拍脑袋"**：优秀的主观投资者有深入的研究框架和严格的纪律，不是凭感觉
- **"量化 = 完全客观"**：量化策略的开发过程中，人类的选择（数据、模型、参数）依然带有主观性
- **"两者只能选其一"**：很多机构投资者将量化和主观策略搭配使用，实现互补
- **"量化适应变化慢"**：虽然模型需要重新训练，但量化可以快速调整参数和权重，不一定比人慢
