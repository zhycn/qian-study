# 朱少醒——公募一哥

> 在中国公募基金行业，有一个"神话"：一个人，一只基金，18 年，年化收益超过 19%。这个人叫朱少醒，这只基金叫富国天惠。他用 18 年的专注证明了一件事：**把一件事做到极致，就是最大的竞争力。**

## 人物速写

朱少醒，富国天惠精选成长混合基金基金经理，中国公募基金行业独一无二的"一人一基"传奇。他从 2005 年开始管理富国天惠，18 年来只管理这一只基金，年化收益超过 19%，累计回报超过 20 倍，被誉为"公募一哥"。

## 传奇人生

### 早年经历：从吉大到富国

朱少醒出生于 1970 年代，后来考入吉林大学，攻读经济学博士学位。吉林大学是中国东北地区的重要学府，经济学研究有着深厚的传统。

博士毕业后，朱少醒进入富国基金，开始了他的投研生涯。富国基金成立于 1999 年，是中国最早的公募基金公司之一。

在早期的投研工作中，朱少醒展现出了与众不同的特质：

- **专注**——他不追求覆盖最多的行业，而是深入研究少数公司
- **耐心**——他不急于做出投资决策，而是等待最佳时机
- **独立**——他不盲从市场共识，坚持自己的判断

这些特质后来成为他投资风格的核心。

### 投资启蒙：自下而上的力量

朱少醒的投资方法论深受费雪（Philip Fisher）和巴菲特（Warren Buffett）的影响。他逐渐形成了自己的投资框架：

> "投资的核心是找到好公司，然后长期持有。你不需要预测市场，你只需要找到那些能持续创造价值的公司。"

他的自下而上选股方法：

- **不看宏观**——不过度关注宏观经济和市场走势
- **不看行业**——不刻意配置或回避某个行业
- **只看公司**——深入分析每一家公司的基本面
- **长期持有**——买入后不频繁交易

这种方法在公募基金行业是相对少见的。大多数基金经理会根据宏观判断和行业比较进行配置，但朱少醒选择了一条更"纯粹"的路：**只做好公司，剩下的交给时间。**

### 关键转折点：管理富国天惠（2005）

2005 年 11 月，朱少醒开始管理富国天惠精选成长混合基金。这只基金成立于 2005 年 11 月，是富国基金旗下的旗舰产品之一。

从这一天起，朱少醒做出了一个在公募基金行业极其罕见的决定：**18 年来，只管理这一只基金。**

在中国公募基金行业，基金经理频繁更换管理产品是常态。大多数基金经理会同时管理多只产品，或者在不同产品之间轮换。但朱少醒选择了一条不同的路：

> "我只管理一只基金，这样我可以把所有的精力都放在研究上。我不需要分散注意力，我可以做到最深度地理解每一家公司。"

**关于"一人一基"的说明：** 在中国公募基金行业，朱少醒是极少数长期只管理一只基金的基金经理之一。这种做法让他能够保持投资策略的一致性和深度，但也意味着他放弃了管理更多产品带来的管理费和影响力。（来源：富国基金公告、公开采访）

富国天惠的产品定位是"精选成长"——选择具有长期成长性的优质公司。朱少醒的投资风格与这一定位高度契合：

- **选股标准**——ROE 高、成长性好、竞争优势深
- **持仓风格**——集中持仓、低换手率
- **投资期限**——长期持有，不频繁交易

### 巅峰时刻：18 年 20 倍

从 2005 年到 2024 年，朱少醒管理富国天惠的时间超过了 18 年。在这 18 年中，他创造了中国公募基金行业最令人瞩目的长期业绩之一。

**关键里程碑：**

- **2005 年**：富国天惠成立，朱少醒开始管理
- **2006-2007 年**：A 股大牛市，富国天惠业绩大幅领先
- **2008 年**：全球金融危机，回撤控制优秀
- **2009-2010 年**：四万亿刺激，周期股反弹
- **2013-2015 年**：成长股牛市，业绩持续领先
- **2016-2017 年**：价值投资风格占优
- **2019-2021 年**：核心资产牛市，业绩再创新高
- **2022-2024 年**：市场调整，但长期业绩依然优秀

**关于长期业绩的数据：** 根据富国基金年报和晨星中国数据，富国天惠自成立以来（2005 年 11 月至 2024 年），累计回报超过 20 倍，年化收益超过 19%，大幅跑赢同期沪深 300 指数和同类平均水平（来源：富国基金年报、晨星中国，截至 2024 年）。

朱少醒最核心的贡献是：**证明了在中国 A 股市场，自下而上选股和长期持有是可行的，而且可以做得非常好。**

他的投资框架可以总结为：

1. **自下而上选股**——不看宏观，不看行业，只看公司
2. **深度研究**——对每一家公司进行深入研究
3. **长期持有**——买入后不频繁交易
4. **集中持仓**——前十大重仓股占比高
5. **专注**——18 年只管理一只基金，保持策略一致性

## 经典战役

### 战役一：2006-2007 年牛市——精选成长股

2006-2007 年，A 股经历了历史上最大的牛市之一。朱少醒通过精选成长股，获得了丰厚的回报。

**背景：**

- 2006-2007 年，A 股从 1000 点左右涨到 6124 点
- 股权分置改革释放了市场活力
- 经济高速增长，企业盈利大幅改善
- 金融、地产、有色等板块涨幅巨大

**朱少醒的选择：**

- 不追热点，坚持自下而上选股
- 选择具有长期成长性的优质公司
- 不因为市场狂热而改变投资标准

**重仓方向：**

- 消费——具有品牌护城河的公司
- 金融——低估值、高成长的银行和保险
- 制造业——具有竞争优势的制造企业

**结果：**

- 2006-2007 年，富国天惠业绩大幅领先同类（来源：晨星中国，2007 年基金排名）
- 上证指数全年涨幅超过 100%
- 富国天惠的累计净值从 1 元增长到 3 元以上

| 富国天惠业绩    | 大幅领先同类（来源：晨星中国）   |
| 核心策略        | 自下而上选股、精选成长           |

**数据来源：** 富国基金年报、晨星中国基金评级、Wind 数据

### 战役二：2008 年金融危机——坚守好公司

2008 年全球金融危机爆发，A 股暴跌超过 60%。朱少醒通过坚守好公司，有效控制了回撤。

**背景：**

- 2008 年，全球金融危机爆发
- A 股从 6124 点暴跌至 1664 点，跌幅超过 70%
- 绝大多数基金回撤超过 50%
- 市场恐慌情绪蔓延

**朱少醒的应对：**

- 不恐慌性抛售，坚持持有好公司
- 不盲目减仓，相信好公司的长期价值
- 利用市场恐慌，在低位加仓被错杀的好公司

**结果：**

- 2008 年，富国天惠的回撤小于市场平均水平（来源：晨星中国，2008 年基金年报）
- 在熊市中保存了实力，为下一轮行情做好准备

| 指标         | 数值                             |
| ------------ | -------------------------------- |
| 上证指数跌幅 | 从 6124 点到 1664 点（-73%）     |
| 富国天惠回撤 | 小于市场平均（来源：晨星中国）   |
| 应对策略     | 坚守好公司、不恐慌抛售、低位加仓 |

**数据来源：** 富国基金年报、晨星中国基金评级

### 战役三：2013-2015 年——布局消费医药

2013-2015 年，A 股市场风格从周期转向成长。朱少醒通过布局消费、医药等长期成长赛道，再次获得了优秀业绩。

**背景：**

- 2013 年前后，中国经济增速放缓，传统周期股景气度下降
- 消费升级趋势明显，白酒、家电等消费股表现优异
- 医药行业受益于人口老龄化和创新药发展
- 2015 年，杠杆资金推动 A 股再次冲高

**朱少醒的选择：**

- 坚持自下而上选股，不刻意追逐市场热点
- 选择消费、医药等长期成长确定性高的公司
- 在 2015 年市场狂热时，保持冷静，不追高

**重仓方向：**

- 消费——白酒、家电、食品饮料
- 医药——创新药、医疗器械、医疗服务
- 其他具有长期成长性的公司

**结果：**

- 在 2015 年暴跌中，回撤控制优秀

## 投资哲学

### 原则一：自下而上选股

朱少醒投资哲学的核心是自下而上选股。

> "我不需要预测宏观经济，也不需要判断行业轮动。我只需要找到那些能持续创造价值的好公司。"

他的自下而上选股方法：

- **不看宏观**——不过度关注宏观经济和市场走势
- **不看行业**——不刻意配置或回避某个行业
- **只看公司**——深入分析每一家公司的基本面
- **长期持有**——买入后不频繁交易

### 原则二：深度研究

朱少醒强调深度研究的重要性。

> "你对一家公司了解得越深，你的投资就越有信心。深度研究是长期持有的前提。"

他的深度研究方法：

- **读财报**——至少读过去 5-10 年的年报
- **调研**——实地走访企业，与管理层交流
- **产业链研究**——了解上下游，理解行业格局
- **持续跟踪**——买入后持续关注基本面变化

### 原则三：长期持有

朱少醒是长期持有的坚定践行者。

> "好公司不需要频繁买卖。买入后，让时间成为你的朋友。"

他的长期持有逻辑：

- 好公司的价值会随时间增长
- 频繁交易产生摩擦成本
- 找到好公司很难，不应该轻易卖掉
- A 股市场的波动性大，长期持有可以平滑波动

### 原则四：集中持仓

朱少醒的持仓非常集中。

> "如果你找到了真正的好公司，为什么要分散投资？集中持仓才能最大化回报。"

他的集中持仓逻辑：

- 前十大重仓股占比通常在 60% 以上
- 不为了分散而分散
- 每一笔投资都基于深度研究
- 有信心才能集中

### 原则五：专注

朱少醒最独特的投资哲学是专注。

> "我只管理一只基金，这样我可以把所有的精力都放在研究上。专注是最大的竞争力。"

他的专注体现在：

- 18 年只管理一只基金
- 不追求管理更多产品
- 不追逐市场热点
- 坚持自己的投资框架

## 策略拆解

### 选股标准

朱少醒的选股标准可以总结为"四个好"：

| 标准   | 具体要求                             |
| ------ | ------------------------------------ |
| 好生意 | 商业模式简单易懂，有持久竞争优势     |
| 好公司 | ROE 高、成长性好、现金流强           |
| 好管理 | 管理层诚信、有能力、以股东利益为导向 |
| 好价格 | 估值合理或偏低，有安全边际           |

**具体筛选条件：**

- ROE > 15% 且稳定
- 收入增长 > 15%
- 自由现金流/净利润 > 80%
- 竞争优势深（品牌、技术、渠道等）
- 管理层诚信、有能力

### 买入时机

朱少醒的买入时机判断：

1. **好公司遇到暂时困难**——股价被错杀
2. **市场恐慌**——好公司被连带抛售
3. **估值回到合理区间**——价格低于内在价值
4. **发现新的好公司**——通过深度研究找到新机会

他特别强调：**不要因为市场热点而买入。** 只买你真正理解的好公司。

### 持仓管理

朱少醒的持仓管理极其简单：

- **集中持仓**——前十大重仓股占比 60% 以上
- **买入后不操作**——除非基本面发生变化
- **不关注短期波动**——"股价涨跌不重要，重要的是公司好不好"
- **持续跟踪**——定期关注公司基本面变化

### 卖出纪律

朱少醒的卖出条件：

1. **基本面恶化**——公司核心竞争力丧失
2. **估值极度泡沫**——价格远超内在价值
3. **发现更好的机会**——换仓到性价比更高的标的
4. **逻辑证伪**——当初买入的理由不再成立

他特别强调：**不要因为股价上涨就卖出。** 如果公司还在创造价值，就应该继续持有。

### 普通人能学到什么

朱少醒的方法对普通投资者特别有借鉴意义：

1. **专注**——不要频繁更换投资策略，找到适合你的方法并坚持
2. **自下而上**——不要过度关注宏观和市场，关注公司本身
3. **长期持有**——减少交易频率，让时间创造价值
4. **深度研究**——对你持有的每一家公司都进行深入研究
5. **集中持仓**——找到真正的好公司后，敢于集中投资

**具体建议：**

- 选择 5-10 家你真正理解的好公司
- 以 5 年以上的视角持有
- 每年检查一次基本面是否发生变化
- 保持阅读财报的习惯
- 不要因为短期波动而改变长期策略

> "你对一家公司了解得越深，你的投资就越有信心。"
