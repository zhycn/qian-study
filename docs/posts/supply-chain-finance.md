# 供应链金融

你的公司给大企业供货，大企业 90 天后才付款，但你现在就要钱发工资、买原料怎么办？供应链金融就是来解决这个"有钱的不着急付，着急付的没钱"的矛盾。

## 概述

供应链金融（Supply Chain Finance, SCF）是指**基于供应链中的真实交易，利用核心企业的信用和交易数据，为供应链上下游的中小企业提供融资服务的金融模式**。

供应链金融的核心逻辑：

传统融资：中小企业以自己的信用向银行借钱 → 银行觉得风险高 → 不借或利率很高

供应链金融：中小企业以与核心企业的交易为基础借钱 → 银行看核心企业的信用 → 风险降低 → 愿意借且利率更低

供应链金融的主要模式：

| 模式             | 适用场景               | 融资方        | 还款来源     | 典型产品                 |
| ---------------- | ---------------------- | ------------- | ------------ | ------------------------ |
| 应收账款融资     | 供应商已发货，等待付款 | 上游供应商    | 核心企业付款 | 保理、应收账款质押       |
| 存货融资         | 企业持有存货           | 经销商/制造商 | 存货销售回款 | 存货质押、仓单质押       |
| 预付款融资       | 需要预付货款           | 下游经销商    | 最终销售回款 | 保兑仓、订单融资         |
| 核心企业信用传递 | 整条供应链             | 上下游企业    | 交易回款     | 供应链票据、数字债权凭证 |

供应链金融的参与方：

- **核心企业**：供应链中的大型企业，信用良好
- **上游供应商**：向核心企业提供原材料或服务
- **下游经销商**：从核心企业采购产品进行销售
- **金融机构**：银行、保理公司、金融科技平台
- **物流/仓储企业**：提供存货监管服务

## 为什么重要

- **解决中小企业融资难**：中小企业缺乏抵押物和信用记录，供应链金融利用核心企业信用降低融资门槛
- **优化供应链效率**：资金流畅通，供应链运转更高效
- **降低融资成本**：利用核心企业信用，中小企业可以获得更低利率
- **金融科技应用**：区块链、大数据、物联网等技术在供应链金融中广泛应用
- **政策支持**：各国政府鼓励供应链金融发展，支持实体经济

## 核心原理

### 供应链金融的运作机制

**以应收账款融资为例**：

1. 供应商 A 向核心企业 B 发货，形成 100 万元应收账款，账期 90 天
2. 供应商 A 将应收账款转让给银行 C
3. 银行 C 向供应商 A 支付 95 万元（扣除 5 万元贴现利息）
4. 90 天后，核心企业 B 向银行 C 支付 100 万元
5. 银行 C 获得 5 万元利息收益

**关键要素**：

- **真实交易**：融资基于真实的贸易背景
- **核心企业信用**：还款依赖核心企业的支付能力
- **闭环管理**：资金流向可控，降低挪用风险
- **自偿性**：融资通过交易本身的现金流偿还

### 供应链金融的风险管理

供应链金融的风险管理核心是**控制交易风险，而非主体风险**：

**传统融资**：看借款人的资产负债表、盈利能力、抵押物

**供应链金融**：看交易的真实性、核心企业的信用、现金流的闭环

风险管理的关键点：

| 风险类型 | 管理方式                       |
| -------- | ------------------------------ |
| 信用风险 | 核心企业信用评估、交易历史分析 |
| 操作风险 | 交易真实性验证、资金闭环管理   |
| 法律风险 | 合同规范、权利登记             |
| 市场风险 | 存货价值监控、价格波动管理     |
| 道德风险 | 核心企业配合、信息透明         |

### 供应链金融的定价模型

供应链金融的利率通常低于传统中小企业贷款：

$$\text{利率} = \text{无风险利率} + \text{核心企业信用溢价} + \text{交易风险溢价} + \text{运营成本}$$

- **核心企业信用溢价**：通常较低（核心企业信用好）
- **交易风险溢价**：取决于交易历史、行业特点
- **运营成本**：取决于技术水平和规模效应

典型利率对比：

| 融资方式         | 年化利率 |
| ---------------- | -------- |
| 银行中小企业贷款 | 8-15%    |
| 供应链金融       | 4-8%     |
| P2P 网贷         | 12-24%   |
| 民间借贷         | 15-36%   |

## 模型/方法详解

### 应收账款融资模型

**保理**（Factoring）：

供应商将应收账款转让给保理商，获得融资。

**有追索权保理**：

- 如果核心企业不付款，保理商可以向供应商追索
- 风险主要由供应商承担
- 利率较低

**无追索权保理**：

- 如果核心企业不付款，保理商不能向供应商追索
- 风险主要由保理商承担
- 利率较高

**定价公式**：

$$\text{融资成本} = \text{融资金额} \times \text{利率} \times \frac{\text{账期}}{365} + \text{服务费}$$

### 存货融资模型

**存货质押**：

企业将存货质押给金融机构，获得融资。

**关键要素**：

- **质押率**（Loan-to-Value, LTV）：

$$LTV = \frac{\text{融资金额}}{\text{存货价值}}$$

通常 LTV 为 50-70%。

- **警戒线**：当存货价值下降到一定水平，需要追加质押物或还款
- **平仓线**：当存货价值下降到更低水平，金融机构可以处置质押物

**存货价值监控**：

$$V_t = V_0 \times (1 + r_t)$$

其中 $r_t$ 是存货价格变化率。

如果 $V_t < \text{平仓线}$，触发处置。

### 区块链在供应链金融中的应用

区块链技术解决供应链金融的核心问题：**信息不对称和信任**。

**传统模式的问题**：

- 交易信息分散在多个参与方
- 信息真实性难以验证
- 多方对账成本高

**区块链的优势**：

- 交易信息不可篡改
- 多方共享同一账本
- 智能合约自动执行

**应用场景**：

- 应收账款确权
- 仓单数字化
- 交易溯源
- 自动结算

## 主流方案对比

| 模式           | 优势                 | 局限                           | 适用场景         |
| -------------- | -------------------- | ------------------------------ | ---------------- |
| 银行主导       | 资金成本低、风控成熟 | 流程复杂、审批慢               | 核心企业配合度高 |
| 核心企业主导   | 了解供应链、数据丰富 | 资金成本高、可能占用供应商资金 | 核心企业实力强   |
| 第三方平台主导 | 技术驱动、效率高     | 资金成本高、依赖外部资金       | 数字化程度高     |
| 金融科技平台   | 技术先进、用户体验好 | 监管不确定性、资金成本高       | 创新场景         |

## 实战应用

### 供应链金融的实施步骤

**1. 识别核心企业**

- 信用良好，支付能力强
- 供应链稳定，交易频繁
- 愿意配合（确认应收账款、提供数据）

**2. 设计融资方案**

- 确定融资模式（应收账款、存货、预付款）
- 确定融资额度和利率
- 设计资金闭环

**3. 技术对接**

- 与核心企业 ERP 系统对接
- 实现交易数据实时获取
- 建立风险监控模型

**4. 运营监控**

- 跟踪交易进展
- 监控核心企业信用变化
- 及时处理异常情况

### 供应链金融的风险案例

**案例一：虚假交易**

- 供应商与关联方伪造交易
- 骗取融资后跑路
- 防范：交易真实性验证、关联关系排查

**案例二：核心企业违约**

- 核心企业经营恶化，无法付款
- 供应商和金融机构同时受损
- 防范：核心企业信用评估、分散风险

**案例三：存货价值暴跌**

- 质押的存货价格大幅下跌
- 质押物价值不足以覆盖融资
- 防范：设置合理的质押率、动态监控

### 供应链金融的未来趋势

- **数字化**：大数据、AI、区块链等技术深度应用
- **平台化**：第三方供应链金融平台崛起
- **标准化**：行业标准和监管框架逐步完善
- **全球化**：跨境供应链金融需求增长
- **绿色化**：绿色供应链金融成为新方向

## 关键要点

- 供应链金融利用核心企业信用解决中小企业融资难
- 核心是控制交易风险，而非主体风险
- 应收账款融资是最成熟的模式
- 区块链技术正在改变供应链金融的运作方式
- 供应链金融需要核心企业、金融机构、科技平台的多方协作

## 常见误区

- **误区一**："供应链金融就是保理"——保理只是供应链金融的一种模式，还有存货融资、预付款融资等
- **误区二**："供应链金融没有风险"——供应链金融有信用风险、操作风险、法律风险等
- **误区三**："只有银行能做供应链金融"——核心企业、第三方平台、金融科技公司都可以参与
- **误区四**："供应链金融只适合大企业"——供应链金融的核心是服务中小企业
- **误区五**："区块链能解决所有问题"——区块链是工具，不能替代风险管理和业务逻辑
