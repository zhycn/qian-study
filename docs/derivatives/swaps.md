# 互换

> 两个当事人约定，在未来一段时间内，按照某种规则互相交换现金流的金融合约。

## 概述

互换（Swaps）是场外衍生品（OTC Derivatives）市场中规模最大的品种。它是交易双方约定在未来一段时间内，按照事先确定的规则，定期交换现金流的合约。与期货和期权不同，互换通常不在交易所交易，而是由双方私下协商条款，具有高度的灵活性。

最常见的互换是**利率互换**（Interest Rate Swap）和**货币互换**（Currency Swap）。

**利率互换**的典型结构是：一方支付固定利率，另一方支付浮动利率（如 LIBOR、SOFR 或 SHIBOR）。双方基于相同的名义本金（Notional Principal）计算利息，但只交换利息差额，本金本身并不实际交换。

**货币互换**则涉及两种不同货币：双方在期初交换不同货币的本金，期间交换不同货币的利息，到期后再按约定汇率换回本金。

互换解决的核心问题是：**利用比较优势降低融资成本**，以及**管理利率和汇率风险**。

## 为什么重要

- **全球最大衍生品市场**：互换市场名义本金超过 600 万亿美元（来源：BIS，2023 年），远超期货和期权市场
- **降低融资成本**：企业可以利用自身在不同市场的比较优势，通过互换获得比直接融资更低的利率
- **利率风险管理**：浮动利率债务可以换为固定利率，锁定融资成本；反之亦然
- **跨境投资基础设施**：货币互换帮助投资者规避汇率风险，是全球化投资的必备工具
- **央行政策工具**：各国央行之间签署的本币互换协议（如中美、中日互换）是国际金融安全网的重要组成部分

## 核心原理

### 比较优势与互换收益

互换的核心经济学原理是**比较优势**（Comparative Advantage）。即使一方在所有市场都有绝对优势，双方仍然可以通过互换实现共赢。

假设 A 公司在固定利率市场有比较优势，B 公司在浮动利率市场有比较优势：

| 公司 | 固定利率 | 浮动利率     |
| ---- | -------- | ------------ |
| A    | 4.0%     | LIBOR + 0.3% |
| B    | 5.2%     | LIBOR + 1.0% |
| 利差 | 1.2%     | 0.7%         |

A 在固定利率市场的优势（1.2%）大于在浮动利率市场的优势（0.7%），所以 A 在固定利率市场有比较优势，B 在浮动利率市场有比较优势。

总收益 = 1.2% - 0.7% = 0.5%。双方通过互换可以分享这 0.5% 的收益。

### 利率互换定价

利率互换的定价基于**无套利原理**。固定利率（互换利率 / Swap Rate）的确定使得互换合约在起始时刻的价值为零。

互换利率等于使固定利率现金流的现值等于浮动利率现金流现值的利率：

$$\sum_{i=1}^{n} R \times \Delta t_i \times DF_i = \sum_{i=1}^{n} E(L_i) \times \Delta t_i \times DF_i$$

其中：

- $R$：互换利率（固定利率）
- $\Delta t_i$：第 $i$ 期的时间长度
- $DF_i$：第 $i$ 期的贴现因子
- $L_i$：第 $i$ 期的浮动利率预期值
- $n$：互换的总期数

实际上，互换利率可以通过**零息利率曲线**（Zero-Coupon Yield Curve）通过"Bootstrapping"方法推导出来。

### 货币互换定价

货币互换涉及两个不同利率体系和汇率。定价时需要考虑：

- 两种货币各自的利率曲线
- 即期汇率和远期汇率
- 交叉货币基差（Cross-Currency Basis）

货币互换的定价公式本质上是将两种货币的现金流分别折现，然后通过汇率换算后使其相等。

## 类型/品种详解

### 利率互换（Interest Rate Swap, IRS）

最常见的互换类型，占互换市场总量的 70% 以上（来源：ISDA）。

- **固定换浮动**（Plain Vanilla）：一方支付固定利率，另一方支付浮动利率。用途：将浮动利率债务转为固定利率，或反之
- **基差互换**（Basis Swap）：双方都支付浮动利率，但基于不同的参考利率（如 3 个月 LIBOR 换 6 个月 LIBOR）。用途：管理不同浮动利率基准之间的利差风险
- **远期互换**（Forward Swap）：互换在未来某个时间才开始。用途：锁定未来的利率水平

### 货币互换（Currency Swap）

涉及两种不同货币的本金和利息交换。

- **固定换固定**：双方各自支付固定利率的不同货币利息
- **固定换浮动**：一方支付固定利率，另一方支付浮动利率，货币不同
- **交叉货币基差互换**（Cross-Currency Basis Swap）：双方都支付浮动利率，但货币不同，通常包含一个基差调整

### 信用违约互换（Credit Default Swap, CDS）

虽然名字里有"互换"，但 CDS 更像是一种保险合约。

买方定期向卖方支付保费（Spread），如果参考实体（如某公司或国家）发生信用事件（违约、破产等），卖方赔偿买方的损失。

CDS 的定价基于参考实体的**违约概率**和**违约回收率**：

$$CDS\ Spread \approx (1 - Recovery\ Rate) \times Default\ Probability$$

### 其他互换

- **总收益互换**（Total Return Swap, TRS）：一方支付资产的总收益（价格变动 + 利息/分红），另一方支付浮动利率
- **商品互换**（Commodity Swap）：以商品价格为标的，如石油互换
- **波动率互换**（Volatility Swap）：以实际波动率为标的

## 主流方案对比

| 类型     | 标的        | 交易场所    | 标准化程度  | 主要风险               |
| -------- | ----------- | ----------- | ----------- | ---------------------- |
| 利率互换 | 利率        | 场外（OTC） | 可定制      | 利率风险、交易对手风险 |
| 货币互换 | 汇率 + 利率 | 场外（OTC） | 可定制      | 汇率风险、交易对手风险 |
| CDS      | 信用事件    | 场外（OTC） | ISDA 标准化 | 信用风险、流动性风险   |
| 利率期货 | 利率        | 交易所      | 高度标准化  | 利率风险、基差风险     |

**使用建议**：

- 企业管理利率风险：优先选择利率互换，条款可根据债务结构定制
- 跨境投资对冲汇率：选择货币互换，同时管理汇率和利率风险
- 管理信用风险：使用 CDS，但需注意 CDS 卖方自身的信用质量
- 需要标准化和流动性：考虑利率期货，但灵活性不如互换

## 实战应用

### 企业利率风险管理

某企业有一笔 1 亿元的浮动利率贷款，利率为 SHIBOR + 1%，期限 5 年。企业担心未来利率上升增加财务成本。

操作：与银行签订 5 年期利率互换合约，企业支付固定利率 3.5%，收取 SHIBOR。

效果：

- 贷款利息：SHIBOR + 1%
- 互换收取：SHIBOR
- 互换支付：3.5%
- **综合成本**：3.5% + 1% = 4.5%（固定）

无论 SHIBOR 如何变化，企业的综合融资成本锁定在 4.5%。

### 央行货币互换

中国人民银行与欧洲央行签署了规模为 3500 亿元人民币 / 450 亿欧元的双边本币互换协议（来源：中国人民银行）。这意味着在需要时，双方可以按约定汇率互换本币，为双边贸易和投资提供流动性支持，同时维护金融稳定。

### 注意事项

- **交易对手风险**（Counterparty Risk）：互换是场外合约，一方违约另一方将受损。2008 年 AIG 危机就是 CDS 交易对手风险的典型案例（来源：维基百科，2008 年）
- **流动性风险**：非标准互换可能难以提前终止或转让
- **ISDA 协议**：互换交易通常基于国际互换与衍生品协会（ISDA）的标准协议框架
- **监管变化**：2008 年后，部分互换被要求通过中央对手方（CCP）清算以降低系统性风险（来源：BIS / Dodd-Frank Act，2008 年后）

## 关键要点

1. 互换是**场外合约**，条款可灵活定制，是全球最大的衍生品市场
2. 利率互换只交换**利息差额**，不交换本金，基于相同的名义本金计算
3. 互换的核心经济学原理是**比较优势**，双方通过互换分享融资成本差异
4. 互换定价基于**无套利原理**，互换利率使合约初始价值为零
5. **交易对手风险**是互换的主要风险，2008 年后监管要求部分互换集中清算
6. CDS 本质上是**信用保险**，定价取决于违约概率和回收率
7. 央行间的本币互换协议是国际金融安全网的重要组成部分

## 常见误区

- **误区一**：互换就是交换本金——大多数互换（尤其是利率互换）只交换现金流差额，本金仅作为计算基准（名义本金），不实际交换
- **误区二**：互换没有风险——互换存在显著的交易对手风险，一方违约另一方将面临损失，货币互换还涉及汇率风险
- **误区三**：互换只有银行才能做——大型企业、保险公司、养老基金等机构都可以参与互换市场管理风险
- **误区四**：互换和期货差不多——期货是标准化合约在交易所交易、每日结算；互换是定制化合约在场外交易、通常定期结算，两者在交易机制、风险特征和监管要求上差异很大

## 与其他概念的关系

- [期货](/derivatives/futures)：期货在交易所交易且标准化，互换在场外交易且可定制，两者可互相替代但各有优劣
- [对冲](/derivatives/hedging)：互换是企业对冲利率和汇率风险的主要工具，特别是中长期风险管理
- [保证金](/derivatives/margin)：互换交易在 2008 年后也被要求缴纳初始保证金和变动保证金
- [杠杆](/derivatives/leverage)：互换具有杠杆效应，名义本金远大于实际现金流，小变动可能产生大影响
- [期权](/derivatives/options)：互换期权（Swaption）赋予进入互换的权利，是管理互换风险的重要工具
