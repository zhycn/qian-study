# 学习路径

> 不知道怎么开始？按你的需求选择一条学习路线吧。

## 📖 新手入门

适合零基础读者，从最核心的概念开始建立金融认知框架。

- [什么是金融](/posts/what-is-finance) — 金融是什么？为什么重要？
- [货币](/posts/money) → [利息](/posts/interest) → [通货膨胀](/posts/inflation) — 理解钱的底层逻辑
- [信用](/posts/credit) — 现代金融的基石
- [风险与收益](/posts/risk-return) — 金融决策的核心权衡

**学完你能**：理解日常听到的金融术语，看懂经济新闻的基本逻辑。

## 🎯 实用场景

结合生活实际，学完就能用上的金融知识。

### 个人理财

- [储蓄](/posts/savings) → [信用卡](/posts/credit-card) → [房贷](/posts/mortgage) → [保险](/posts/insurance)
- [资产配置](/posts/asset-allocation) — 不把鸡蛋放在一个篮子里
- [紧急备用金](/posts/emergency-fund) — 应对突发状况的"救命钱"

### 投资入门

- [股票](/posts/stock) → [基金](/posts/fund) → [ETF](/posts/etf) → [指数](/posts/index)
- [市盈率](/posts/pe-ratio) — 判断股票贵不贵
- [分红](/posts/dividend) — 公司给股东的"红包"

### 公司金融基础

- [资产负债表](/posts/balance-sheet) → [利润表](/posts/income-statement) → [现金流量表](/posts/cash-flow-statement)
- [估值](/posts/valuation) — 一个公司到底值多少钱

**学完你能**：管理个人财务，看懂上市公司财报，开始自己的投资实践。
