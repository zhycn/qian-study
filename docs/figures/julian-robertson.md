# 朱利安·罗伯逊——老虎基金之父

> 罗伯逊不是最出名的对冲基金经理，但他可能是**最有影响力**的。他关闭基金后，他的学生们管理着超过 2000 亿美元的资产。

## 人物速写

朱利安·罗伯逊（Julian Robertson），老虎基金（Tiger Management）创始人，1980-2000 年创造了年化 25% 的传奇业绩。他关闭基金后培养了 50 多位知名基金经理，被称为"对冲基金界的教父"。

| 维度     | 数据                                                    |
| -------- | ------------------------------------------------------- |
| 生卒年   | 1932 年 6 月 25 日 - 2022 年 8 月 23 日（享年 90 岁）   |
| 国籍     | 美国                                                    |
| 核心贡献 | 多空策略（Long/Short Equity）、基本面研究、培养"小老虎" |
| 代表基金 | 老虎基金（Tiger Management）                            |
| 管理年限 | 20 年（1980-2000）                                      |
| 年化收益 | 约 25%（扣除费用后，同期标普 500 约 14%）               |
| 管理规模 | 峰值约 220 亿美元（1998 年初）                          |
| 个人财富 | 约 45 亿美元（2022 年福布斯）                           |
| 教育背景 | 北卡罗来纳大学教堂山分校（UNC）                         |

**数据来源：** 《对冲基金怪杰》（Hedge Fund Market Wizards, Jack Schwager, 1992/2012 修订版）、《老虎基金的兴衰》、公开报道、福布斯富豪榜

## 传奇人生

### 早年经历：南方小伙的华尔街梦

1932 年，朱利安·罗伯逊出生于北卡罗来纳州索尔兹伯里市。他的父亲是一名律师，家境中等。

罗伯逊从小就展现出了对数字的敏感。他在北卡罗来纳大学教堂山分校读本科时主修商业，毕业后加入了美国海军。退役后，他前往纽约，开始了他的华尔街生涯。

1950 年代，罗伯逊在基德尔·皮博迪（Kidder, Peabody）工作，主要从事股票经纪业务。这段经历让他学到了两件事：

1. **如何分析公司**——他阅读了大量财报，学会了基本面分析
2. **如何理解市场**——他与客户交流，了解了市场情绪的变化

### 投资启蒙：从经纪人到基金经理

1970 年代，罗伯逊在基德尔·皮博迪逐渐晋升，最终成为股票研究部门的负责人。他的研究报告以深入、准确著称，为公司赢得了大量客户。

1970 年代末，罗伯逊开始考虑自己创业。他看到了一个机会：**用基本面研究驱动的多空策略，可以在任何市场环境下都取得正收益。**

1980 年，48 岁的罗伯逊离开了基德尔·皮博迪，用自己和朋友的 800 万美元，创立了老虎基金管理公司（Tiger Management）。

### 关键转折点：多空策略的成功

罗伯逊的投资策略是典型的"多空策略"（Long/Short Equity）：

- **做多**——买入被低估的好公司
- **做空**——卖出被高估的烂公司
- **净多头**——通常保持 60-70% 的净多头敞口

这种策略的优势在于：

- 市场上涨时，多头获利
- 市场下跌时，空头获利（或至少对冲损失）
- 无论市场涨跌，只要选股能力足够，就能取得正收益

罗伯逊的选股方法非常传统：

- **深入的基本面研究**——阅读财报、分析行业、评估管理层
- **自下而上**——先选个股，再考虑宏观
- **集中持仓**——前十大持仓通常占 50%+的仓位

### 巅峰时刻：20 年的复利奇迹

从 1980 年到 1998 年，老虎基金创造了令人难以置信的业绩：

- **年化收益率约 25%**（扣除费用后），同期标普 500 约 14%
- 基金规模从 **800 万美元** 增长到 **220 亿美元**
- 如果有人在 1980 年投入 1 万美元，到 1998 年将变成约 **50 万美元**

罗伯逊的成功秘诀：

- **严格的选股标准**——只买他真正理解的好公司
- **果断的做空**——敢于做空被高估的股票
- **纪律性**——错了就止损，不犹豫
- **独立判断**——不跟随市场共识

1998 年，罗伯逊做了一件让所有人震惊的事——**他关闭了老虎基金。**

**关闭的原因：**

- 基金规模太大（220 亿美元），难以灵活操作
- 市场变得越来越有效，超额收益越来越难获取
- 互联网泡沫让市场变得非理性，罗伯逊无法理解这种疯狂
- 他想把更多时间用于慈善事业

罗伯逊在给投资者的信中说：

> "我们的策略依赖于发现被错误定价的股票。但随着市场变得越来越有效，这种机会越来越少了。我不想为了收取管理费而继续运营一只无法取得超额收益的基金。"

### "小老虎"传奇：对冲基金界的教父

罗伯逊关闭基金后，做了一件影响深远的事——**支持他的前员工创业。**

罗伯逊的前员工们创立了数十家对冲基金，被称为"小老虎"（Tiger Cubs）。这些基金包括：

| 基金              | 创始人            | 管理规模（约） |
| ----------------- | ----------------- | -------------- |
| Tiger Global      | Chase Coleman     | 800 亿美元     |
| Viking Global     | Andreas Halvorsen | 300 亿美元     |
| Coatue Management | Philippe Laffont  | 200 亿美元     |
| Lone Pine Capital | Stephen Mandel    | 150 亿美元     |
| Maverick Capital  | Lee Ainslie       | 100 亿美元     |

据估计，"小老虎"们管理的总资产超过 **2000 亿美元**。罗伯逊被称为"对冲基金界的教父"，他的投资理念和方法影响了整整一代基金经理。

**数据来源：** Institutional Investor、各基金公开披露文件、《对冲基金怪杰》

## 经典战役

### 战役一：做空日本股市（1980 年代末-1990）

1980 年代末，日本股市和房地产市场出现了巨大的泡沫。日经指数从 1985 年的约 13000 点暴涨到 1989 年底的 38957 点。东京皇居的土地价值据说超过了整个加利福尼亚州。

罗伯逊看到了这个机会。他做了两件事：

- **做空日本股票**——借入日本股票并卖出
- **做多美国股票**——买入被低估的美国公司

**投资逻辑：**

- 日本股市估值极端高企，市盈率超过 60 倍
- 日本房地产市场泡沫严重，不可持续
- 美国股市相对便宜，有上涨空间

**结果：**

- 1990 年，日本股市崩盘，日经指数暴跌约 40%
- 罗伯逊的空头头寸获利丰厚
- 这笔交易为老虎基金贡献了约 **5 亿美元** 的利润（来源：《对冲基金怪杰》，1992）
- 成为老虎基金最赚钱的交易之一

| 指标         | 数值                       |
| ------------ | -------------------------- |
| 交易方向     | 做空日本股票，做多美国股票 |
| 日经指数跌幅 | 约 40%（1990 年）          |
| 持有时间     | 约 2-3 年                  |
| 获利         | 约 5 亿美元                |

**数据来源：** 《对冲基金怪杰》（Jack Schwager, 1992）、日经指数历史记录

### 战役二：做空英镑（1992）

1992 年，罗伯逊也参与了做空英镑的交易——与索罗斯同方向。

**背景：** 与索罗斯的逻辑相同——英国加入 ERM 后，维持英镑汇率需要极高利率，这对英国经济是致命的。

**罗伯逊的操作：**

- 借入英镑，在外汇市场抛售
- 同时做多德国马克
- 使用杠杆放大收益

**结果：**

- 1992 年 9 月 16 日，英国退出 ERM，英镑暴跌
- 老虎基金获利约 **3 亿美元**（来源：《对冲基金怪杰》2012 修订版）

这笔交易与索罗斯的做空英镑几乎同时发生，但罗伯逊的规模较小。两人都看到了同一个宏观机会，只是执行方式不同。

**数据来源：** 《对冲基金怪杰》2012 修订版、英国国家档案馆记录

### 战役三：做多美国航空（1990 年代）

罗伯逊不仅做空，他也做多。美国航空（USAir）是他最成功的多头投资之一。

**背景：**

- 1990 年代初，美国航空遭遇严重困难
- 亏损严重，股价大幅下跌
- 但罗伯逊认为公司正在改善

**投资逻辑：**

- 美国航空正在削减成本、改善运营
- 航空业周期正在触底
- 股价已经反映了最坏情况
- 一旦行业复苏，股价将大幅反弹

**结果：**

- 美国航空确实恢复了盈利
- 股价从约 10 美元涨到超过 50 美元
- 老虎基金获利约 **4 倍**，利润约 **2 亿美元**（来源：公开报道、《对冲基金怪杰》）

| 指标     | 数值          |
| -------- | ------------- |
| 买入时间 | 1990 年代初   |
| 买入价格 | 约 10 美元/股 |
| 卖出价格 | 约 50 美元/股 |
| 回报率   | 约 4 倍       |
| 获利     | 约 2 亿美元   |

**数据来源：** 《对冲基金怪杰》（1992）、美国航空历史记录、公开报道

### 战役四：互联网泡沫时期的坚持与挫折

1990 年代末，互联网泡沫愈演愈烈。罗伯逊坚持自己的价值投资理念，**拒绝买入没有盈利的互联网公司。**

**他的逻辑：**

- 互联网公司没有盈利，估值无法用传统方法评估
- 市场情绪过于狂热，价格严重偏离价值
- 泡沫终将破裂，做空互联网公司会获利

**操作：**

- 做空多家互联网公司
- 做多传统价值股

**结果：**

- 1999 年，互联网泡沫继续膨胀
- 罗伯逊的空头头寸亏损严重
- 老虎基金 1999 年亏损约 **18%**（来源：《金融时报》2000 年报道、老虎基金投资者信函）
- 投资者大量赎回

**反思：**

罗伯逊后来承认：

> "我可能过早了。市场保持非理性的时间，比我保持偿付能力的时间更长。"

这句话后来成为了投资界的经典名言——虽然原话来自凯恩斯，但罗伯逊的经历是最好的注脚。

**数据来源：** 《金融时报》（2000 年 3 月）、老虎基金投资者信函

### 战役五：关闭基金（2000）

2000 年 3 月，互联网泡沫破裂。纳斯达克指数在三个月内暴跌约 50%。

讽刺的是，**罗伯逊在泡沫破裂前关闭了基金。**

**时间线：**

- 1998 年 8 月：罗伯逊宣布将逐步关闭老虎基金
- 1998 年 10 月：开始返还投资者资金
- 2000 年 3 月：基金正式关闭
- 2000 年 3-6 月：互联网泡沫破裂

如果罗伯逊没有关闭基金，他将在 2000 年的科技股崩盘中获利巨大——因为他做空了大量互联网公司。

但罗伯逊并不后悔：

> "我关闭基金不是因为我看空市场，而是因为我觉得自己无法继续取得超额收益。规模太大，市场太有效，这不是一个基金经理应该继续运营的环境。"

## 投资哲学

### 原则一：深入的基本面研究

罗伯逊是基本面研究的坚定信徒。

> "投资的核心是理解一家公司。你必须知道它怎么赚钱、赚多少钱、未来能赚多少钱。"

他的研究方法：

- **阅读财报**——详细分析收入、利润、现金流、负债
- **调研行业**——了解竞争格局、行业趋势
- **评估管理层**——能力、诚信、激励机制
- **估值**——用多种方法评估内在价值

### 原则二：多空策略

罗伯逊的多空策略核心：

| 维度   | 做法               |
| ------ | ------------------ |
| 做多   | 买入被低估的好公司 |
| 做空   | 卖出被高估的烂公司 |
| 净敞口 | 通常 60-70% 净多头 |
| 对冲   | 用空头对冲市场风险 |
| 选股   | 自下而上，独立判断 |

这种策略的优势：

- 市场上涨时，多头获利
- 市场下跌时，空头获利（或减少损失）
- 无论市场涨跌，只要选股能力足够，就能取得正收益

### 原则三：集中持仓

罗伯逊是集中持仓的信徒。

> "如果你最好的想法只占组合的 2%，那你的想法就不够好。"

他的持仓特点：

- 前十大持仓通常占 50%+的仓位
- 单只股票可以占到 5-10% 的仓位
- 组合中通常只有 50-80 只股票

### 原则四：纪律性

罗伯逊非常强调纪律。

> "错了就认错，不要死扛。市场永远是对的。"

他的纪律：

- **止损**——如果逻辑被证伪，立即平仓
- **不平均**——不会在亏损的头寸上继续加仓
- **定期复盘**——每周检查所有头寸

### 原则五：独立判断

罗伯逊从不跟随市场共识。

> "如果你的观点和所有人一样，那你就不可能取得超额收益。"

他的独立判断体现在：

- 不依赖华尔街研究报告
- 不跟随市场热点
- 敢于做空被市场追捧的股票
- 敢于买入被市场抛弃的股票

## 策略拆解

### 选股标准

罗伯逊的选股标准：

| 标准     | 具体要求                   |
| -------- | -------------------------- |
| 好业务   | 商业模式清晰，有竞争优势   |
| 好管理   | 管理层诚信、有能力         |
| 好财务   | 高 ROE、低负债、稳定现金流 |
| 好估值   | 价格低于内在价值           |
| 好催化剂 | 有推动价值回归的催化剂     |

### 买入时机

罗伯逊的买入时机：

1. **公司被市场错误定价**——价格远低于内在价值
2. **有催化剂**——新管理层、新产品、行业复苏
3. **市场恐慌**——好公司被连带抛售

### 持仓管理

罗伯逊的持仓管理：

- **集中持仓**——前十大占 50%+
- **定期复查**——每周检查所有头寸
- **动态调整**——根据基本面变化调整仓位
- **对冲**——用空头对冲市场风险

### 卖出纪律

罗伯逊的卖出条件：

1. **逻辑被证伪**——立即平仓
2. **目标达成**——价格达到内在价值
3. **发现更好的机会**——换仓

### 普通人能学到什么

罗伯逊的方法对普通投资者的启示：

1. **深入研究了再投资**——不要买你看不懂的股票
2. **集中持仓**——如果你最好的想法只占 2%，那想法就不够好
3. **错了就认错**——不要死扛亏损的头寸
4. **独立判断**——不要跟随市场共识
5. **长期视角**——价值回归需要时间
