# 大卫·史文森——耶鲁模式的缔造者

> 史文森去世时，《纽约时报》用了一句话总结他的一生："他可能是过去 50 年里对机构投资影响最大的人。"他没有管理过一只对冲基金，没有上过 CNBC 的头条，但他创造的"耶鲁模式"，改变了全世界捐赠基金、养老金和主权财富基金的投资方式。

## 人物速写

大卫·史文森（David F. Swensen），耶鲁大学首席投资官（Chief Investment Officer, CIO），机构投资领域的教父级人物。他用 38 年时间将耶鲁捐赠基金（Yale Endowment）从 13 亿美元管理到超过 420 亿美元，年化收益率达到 **13.2%**（1985-2021），同期标普 500 指数年化约 11.2%（来源：耶鲁大学捐赠基金年度报告，2021）。

他开创了**耶鲁模式**（Yale Model），核心是将大量资产配置到私募股权、风险投资、实物资产等另类投资领域，以长期视角获取流动性溢价和非市场收益。

## 传奇人生

### 早年经历：威斯康星州的学术少年

1954 年，大卫·史文森出生于威斯康星州一个普通家庭。他的父亲是一名教师，母亲是一名护士。

史文森从小就展现出了对学术的浓厚兴趣。他在当地公立学校就读，成绩优异，尤其擅长数学和科学。但与许多金融天才不同，史文森的早年并没有表现出对金钱或投资的特殊热情。

**生活化类比：** 史文森更像是一个"学者型投资者"——他不追求刺激的交易，不迷恋市场的短期波动，而是用研究学术问题的态度来研究投资。

### 投资启蒙：学术训练与华尔街的碰撞

史文森在威斯康星大学麦迪逊分校（University of Wisconsin-Madison）获得了经济学学士学位，随后进入耶鲁大学攻读经济学博士。

在耶鲁读博期间，史文森的研究方向是**资产定价**（Asset Pricing）和**投资组合理论**（Portfolio Theory）。这段学术训练为他后来的投资实践打下了坚实的理论基础。

1980 年，史文森获得了耶鲁大学经济学博士学位。他的博士论文研究了**个人投资者行为与市场效率**之间的关系——这个主题后来贯穿了他整个职业生涯。

**从学术到实践的关键转折：**

博士毕业后，史文森没有选择学术道路，而是进入了华尔街。他在所罗门兄弟公司（Salomon Brothers）担任固定收益研究员，随后在莱曼兄弟（Lehman Brothers）工作。

这段华尔街经历让史文森深刻认识到：

1. **华尔街的激励机制有问题**——基金经理追求短期业绩以获取更高的管理费，而不是为投资者创造长期价值
2. **个人投资者处于劣势**——高昂的费用、信息不对称和行为偏差让个人投资者很难跑赢市场
3. **机构投资有巨大潜力**——如果能够摆脱短期考核的束缚，长期投资可以创造惊人的回报

> "华尔街的商业模式是向客户收费，而不是为客户赚钱。这两者之间存在着根本的利益冲突。"（来源：《不平凡的成功》，2021）

### 关键转折点：1985 年，回到耶鲁

1985 年，31 岁的史文森做了一个让很多人意外的决定：**离开华尔街，回到耶鲁大学担任捐赠基金的投资主管。**

当时的耶鲁捐赠基金规模约 **13 亿美元**，配置方式与大多数高校一样：

- 约 60% 美国股票
- 约 30% 美国债券
- 约 10% 现金和其他资产

这种传统的 **60/40 组合**（60% 股票 + 40% 债券）在当时的机构投资中是标准配置。但史文森认为，这种方式存在几个致命问题：

| 问题         | 说明                                             |
| ------------ | ------------------------------------------------ |
| 同质化       | 所有高校都用同样的配置，无法获得差异化收益       |
| 费用高       | 主动管理型股票基金的费用侵蚀了收益               |
| 效率悖论     | 在高度有效的公开市场里，主动管理很难持续跑赢指数 |
| 缺乏另类收益 | 完全错过了私募股权、风投等高收益资产类别         |

史文森决定彻底改变这一切。

### 巅峰时刻：耶鲁模式的诞生与验证

从 1985 年开始，史文森逐步将耶鲁捐赠基金的配置转向了**另类资产**（Alternative Assets）：

**耶鲁模式的资产配置演变：**

| 资产类别 | 1985 年 | 2000 年 | 2021 年 |
| -------- | ------- | ------- | ------- |
| 美国股票 | 约 60%  | 约 5%   | 约 0%   |
| 外国股票 | 约 0%   | 约 15%  | 约 13%  |
| 债券     | 约 30%  | 约 5%   | 约 4%   |
| 私募股权 | 约 0%   | 约 25%  | 约 31%  |
| 风险投资 | 约 0%   | 约 5%   | 约 20%  |
| 实物资产 | 约 0%   | 约 15%  | 约 12%  |
| 对冲基金 | 约 0%   | 约 15%  | 约 10%  |
| 现金     | 约 10%  | 约 5%   | 约 10%  |

**数据来源：** 耶鲁大学捐赠基金年度报告（1985、2000、2021）

**这个配置在当时是革命性的。** 大多数机构投资者的另类资产配置比例不到 10%，而耶鲁在 2000 年就已经达到了 **60%+**。

**耶鲁模式的核心逻辑：**

1. **公开市场高度有效**——在美国大盘股等高效市场中，主动管理很难持续创造超额收益（Alpha），因此应该使用低成本指数基金
2. **非公开市场存在机会**——私募股权、风投等信息不对称的市场中，专业投资者可以获得超额收益
3. **流动性溢价**——投资者因承担非流动性风险而获得额外回报
4. **长期视角**——捐赠基金不需要短期变现，可以充分利用时间优势

**结果验证了一切：**

从 1985 年到 2021 年，耶鲁捐赠基金的年化收益率达到 **13.2%**，同期标普 500 指数年化约 **11.2%**，超额收益约 **2 个百分点/年**。

**2 个百分点意味着什么？**

- 13 亿美元以 13.2% 复利 36 年 = **约 1,150 亿美元**
- 13 亿美元以 11.2% 复利 36 年 = **约 590 亿美元**
- 差距：**约 560 亿美元**

当然，实际规模是 420 亿美元（因为每年要提取约 5% 用于学校运营），但即便如此，史文森创造的超额价值也是天文数字。

**数据来源：** 耶鲁大学捐赠基金年度报告（2021）、计算基于复利公式

## 经典战役

### 战役一：重仓私募股权——非公开市场的 Alpha

史文森最早的大胆决策之一，就是**大规模配置私募股权**（Private Equity）。

**背景：**

1980 年代末，私募股权还是一个相对小众的资产类别。大多数机构投资者对其持谨慎态度，原因包括：

- 缺乏透明度
- 流动性极差（锁定期 7-10 年）
- 费用结构复杂（2% 管理费 + 20% 业绩报酬）
- 历史数据有限

但史文森看到了别人没看到的东西：

**私募股权的核心优势：**

1. **信息不对称**——非公开市场中，专业投资者可以通过尽职调查（Due Diligence）获得信息优势
2. **主动创造价值**——PE 基金不仅买公司，还会帮助公司改善经营、优化管理、拓展市场
3. **估值不依赖市场情绪**——非公开市场的估值基于基本面，而非每日市场报价
4. **流动性溢价**——投资者因锁定资金 7-10 年而获得额外回报

**耶鲁的实践：**

- 从 1980 年代末开始逐步增加 PE 配置
- 投资顶级 PE 基金管理人，如 KKR、黑石、凯雷
- 同时也直接投资一些项目，尤其是耶鲁校友创办的企业
- 到 2021 年，私募股权占捐赠基金的 **31%**

**结果：**

- 私募股权板块在 1985-2021 年间贡献了约 **16-18% 的年化收益**（来源：耶鲁年报摘要）
- 成为捐赠基金最重要的收益来源之一
- 验证了"在非有效市场中，主动管理可以创造超额收益"的核心假设

| 指标       | 私募股权            | 公开市场股票         |
| ---------- | ------------------- | -------------------- |
| 年化收益   | 约 16-18%           | 约 10-12%            |
| 波动率     | 较低（估值平滑）    | 较高                 |
| 流动性     | 极差（7-10 年锁定） | 极好（T+2 结算）     |
| 费用       | 高（2/20 结构）     | 低（指数基金 <0.1%） |
| Alpha 机会 | 高                  | 低                   |

**数据来源：** 耶鲁大学捐赠基金年度报告（2021）、Preqin 私募股权行业报告

### 战役二：拥抱风险投资——押注未来

如果说私募股权是史文森的"核心仓位"，那么**风险投资**（Venture Capital）就是他的"卫星仓位"——比例不大，但爆发力极强。

**史文森对 VC 的理解：**

> "风险投资的回报分布极度不均衡——少数项目贡献了绝大部分收益。关键在于，你必须投中那些少数项目。"（来源：《机构投资的创新之路》，2009）

**耶鲁的 VC 策略：**

- 投资顶级 VC 基金，如红杉资本（Sequoia Capital）、Benchmark、Accel
- 分散投资多个基金和多个 vintage year（基金成立年份）
- 重点关注科技、医疗、清洁能源等高增长领域
- 到 2021 年，风险投资占捐赠基金的 **20%**

**经典案例：**

耶鲁捐赠基金通过 VC 基金间接投资了众多科技巨头：

- **谷歌**（Google）——通过红杉资本等基金间接投资
- **Facebook**——通过 Accel 等基金间接投资
- **Uber**——通过 Benchmark 等基金间接投资
- **阿里巴巴**——早期通过软银等基金间接投资

**结果：**

- 风险投资板块在 1985-2021 年间贡献了约 **20%+ 的年化收益**（来源：耶鲁年报摘要）
- 成为捐赠基金收益率最高的资产类别之一
- 但也伴随着极高的波动性和不确定性——并非所有 vintage year 都表现优异

**关键启示：** VC 投资的核心不是"选对项目"，而是**选对基金管理人**。顶级 VC 基金的回报远超行业平均水平，但识别顶级管理人需要专业能力和长期跟踪。

### 战役三：2000 年互联网泡沫——耶鲁模式的压力测试

2000 年 3 月，互联网泡沫破裂，纳斯达克指数暴跌 78%。这是耶鲁模式的第一次重大压力测试。

**耶鲁的表现：**

- 2000 财年，耶鲁捐赠基金收益约 **37%**（来源：耶鲁大学 2000 年年度报告）
- 同期标普 500 指数下跌 **9.1%**
- 纳斯达克指数暴跌 **39.3%**

**为什么耶鲁能在泡沫破裂时大赚？**

1. **低配公开市场股票**——耶鲁当时只有约 20% 的配置在美国和外国股票上
2. **大量配置另类资产**——PE、VC、实物资产的估值不依赖公开市场每日报价
3. **对冲基金策略**——市场中性策略在熊市中依然可以获利
4. **逆向思维**——在市场恐慌时，耶鲁的长期视角使其不受短期波动干扰

**但需要指出的是：** 耶鲁的 37% 收益中，很大一部分来自另类资产的"估值重估"（Valuation Revaluation），而非实际变现收益。这些资产的真实价值可能并没有账面显示的那么高——只是没有被每日标记而已。

| 指标          | 耶鲁捐赠基金 | 标普 500 | 纳斯达克 |
| ------------- | ------------ | -------- | -------- |
| 2000 财年收益 | 约 37%       | -9.1%    | -39.3%   |
| 股票配置      | 约 20%       | 100%     | 100%     |
| 另类资产      | 约 50%+      | 0%       | 0%       |
| 波动率        | 较低         | 高       | 极高     |

**数据来源：** 耶鲁大学年度报告（2000）、Yahoo Finance 历史数据

### 战役四：2008 年金融危机——耶鲁模式的至暗时刻

如果说 2000 年是耶鲁模式的高光时刻，那么 **2008 年就是它的至暗时刻。**

**危机中的耶鲁：**

- 2009 财年，耶鲁捐赠基金亏损约 **25%**（来源：耶鲁大学 2009 年年度报告）
- 捐赠基金规模从约 230 亿美元峰值缩水至约 160 亿美元
- 耶鲁被迫削减预算、推迟建设计划、暂停部分教职招聘

**为什么会亏这么多？**

1. **对冲基金策略失灵**——许多对冲基金在危机中出现了远超预期的亏损，部分基金甚至无法赎回
2. **私募股权估值下调**——虽然 PE 资产不每日报价，但在危机中基金管理人也不得不大幅下调估值
3. **流动性危机**——耶鲁需要从捐赠基金中提取资金维持运营，但大量资产无法快速变现
4. **杠杆放大损失**——部分投资使用了杠杆，在市场暴跌时损失被放大

**史文森的反思：**

> "2008 年是一次痛苦的提醒：**多元化不能完全消除风险。** 当所有资产类别同时下跌时，没有任何地方可以躲藏。"（来源：《不平凡的成功》，2021）

**危机后的恢复：**

- 2010 财年，耶鲁捐赠基金反弹约 **21%**
- 2011 财年，继续增长约 **20%**
- 到 2015 年，捐赠基金规模已超过危机前峰值

**关键教训：**

- 流动性管理至关重要——不能把所有钱都锁在无法变现的资产里
- 对冲基金并非"绝对收益"——它们也会在大危机中亏损
- 长期视角依然有效——危机后的反弹证明了耶鲁模式的韧性

| 指标          | 2008 财年      | 2009 财年      | 2010 财年      |
| ------------- | -------------- | -------------- | -------------- |
| 捐赠基金规模  | 约 230 亿美元  | 约 160 亿美元  | 约 190 亿美元  |
| 年度收益      | 约 12%         | 约 -25%        | 约 21%         |
| 标普 500 收益 | 约 -38.5%      | 约 -22%        | 约 26%         |
| 超额收益      | 约 50 个百分点 | 约 -3 个百分点 | 约 -5 个百分点 |

**数据来源：** 耶鲁大学年度报告（2008-2010）、Yahoo Finance 历史数据

### 战役五：个人投资者指南——《不平凡的成功》

史文森晚年做了一件让很多人意外的事：**他为个人投资者写了一本书。**

2021 年，史文森出版了《不平凡的成功》（Unconventional Success: A Fundamental Approach to Personal Investment），为普通投资者提供了一套基于学术研究和实践经验的投资框架。

**书的核心观点：**

1. **个人投资者应该指数化**——在有效市场中，低成本指数基金是最好的选择
2. **资产配置决定一切**——90% 以上的收益波动来自资产配置，而非选股或择时
3. **再平衡是免费的午餐**——定期再平衡可以自动实现"低买高卖"
4. **费用是确定的敌人**——每年多付 1% 的费用，30 年后差距巨大
5. **行为偏差是最大的敌人**——追涨杀跌、过度自信、损失厌恶等行为偏差让个人投资者跑输市场

**史文森推荐的个人投资者配置：**

| 资产类别                | 配置比例 | 推荐工具         |
| ----------------------- | -------- | ---------------- |
| 美国股票                | 30%      | 全市场指数基金   |
| 外国发达市场股票        | 25%      | 国际指数基金     |
| 新兴市场股票            | 5%       | 新兴市场指数基金 |
| 美国国债                | 20%      | 国债指数基金     |
| 通胀保值债券（TIPS）    | 15%      | TIPS 指数基金    |
| 房地产投资信托（REITs） | 5%       | REIT 指数基金    |

**这个配置的特点：**

- **极度分散**——覆盖全球股票和债券市场
- **低成本**——全部使用指数基金，总费用率 <0.2%
- **简单可执行**——不需要选股、不需要择时、不需要预测市场
- **自动再平衡**——每年调整一次，回到目标配置

> "对大多数个人投资者来说，最好的投资策略就是：买指数基金、定期再平衡、然后忘掉它。"（来源：《不平凡的成功》，2021）

**这本书的意义：**

史文森管理着 420 亿美元的机构资金，但他告诉普通投资者：**你们不需要像我一样投资。你们需要的是简单、低成本、分散化的指数投资。** 这种"机构与个人区别对待"的态度，体现了他对市场有效性的深刻理解和对投资者利益的真诚关怀。

## 投资哲学

### 原则一：市场有效，但不完全有效

史文森对市场有效性（Market Efficiency）的理解极为辩证：

> "美国大盘股市场可能是世界上最有效的市场。在这个市场里，主动管理几乎不可能持续跑赢指数。但在私募股权、风投、新兴市场等领域，信息不对称依然存在，专业投资者可以获得超额收益。"

**他的资产配置逻辑：**

| 市场类型         | 有效性   | 策略                    |
| ---------------- | -------- | ----------------------- |
| 美国大盘股       | 极高     | 低成本指数基金          |
| 外国发达市场股票 | 高       | 低成本指数基金          |
| 新兴市场股票     | 中等     | 指数基金 + 部分主动管理 |
| 私募股权         | 低       | 顶级 PE 基金管理人      |
| 风险投资         | 极低     | 顶级 VC 基金管理人      |
| 实物资产         | 中等偏低 | 主动管理                |

**关键洞察：** 不要在有效的市场里试图战胜市场，而在无效的市场里充分发挥专业能力。

### 原则二：流动性溢价是真实存在的

史文森深信：**投资者因承担非流动性风险而获得额外回报。**

> "流动性是一种保险。你为它付费（放弃流动性溢价），但希望永远不需要用它。"

**流动性溢价的来源：**

1. **供需关系**——愿意锁定资金的投资者更少，资产价格更低，预期回报更高
2. **估值平滑**——非公开市场的估值不每日波动，减少了"恐慌性抛售"
3. **管理改善**——PE/VC 基金可以深度参与被投企业的经营改善
4. **信息优势**——非公开市场中，尽职调查可以创造信息优势

**但史文森也警告：**

- 流动性溢价不是免费的——你需要真正承受非流动性风险
- 不是所有另类资产都能提供流动性溢价——劣质管理人的费用会侵蚀一切
- 流动性管理至关重要——不能把所有钱都锁在无法变现的资产里

### 原则三：多元化是唯一的免费午餐

史文森对多元化（Diversification）的理解超越了传统的"不要把鸡蛋放在一个篮子里"。

**他的多元化框架：**

1. **资产类别多元化**——股票、债券、PE、VC、实物资产、对冲基金
2. **地理多元化**——美国、欧洲、新兴市场
3. **策略多元化**——价值、成长、事件驱动、市场中性
4. **时间多元化**——不同 vintage year 的 PE/VC 基金

**关键洞察：** 真正的多元化不是买更多东西，而是买**相关性低**的东西。当所有资产同时下跌时（如 2008 年），多元化也会失效。

### 原则四：费用是投资世界唯一的确定性

史文森对投资费用（Investment Costs）有着清醒的认识：

> "在投资中，收益是不确定的，风险是不确定的，但费用——一定会有。"

**费用对长期收益的侵蚀：**

| 年费用率             | 30 年后剩余收益（假设年化 8%） |
| -------------------- | ------------------------------ |
| 0.1%（指数基金）     | 约 90%                         |
| 1.0%（普通主动基金） | 约 75%                         |
| 2.0%（PE 管理费）    | 约 55%                         |
| 3.0%（高费用基金）   | 约 41%                         |

**史文森的费用管理策略：**

- 在有效市场使用低成本指数基金（费用率 <0.1%）
- 在非有效市场愿意支付更高费用，但严格筛选管理人
- 定期审查费用结构，避免"费用蠕变"
- 关注"费后收益"而非"费前收益"

### 原则五：行为偏差是个人投资者最大的敌人

史文森晚年特别关注**行为金融学**（Behavioral Finance）对个人投资者的影响。

**他总结的个人投资者常见偏差：**

| 偏差       | 说明                             | 后果                       |
| ---------- | -------------------------------- | -------------------------- |
| 追涨杀跌   | 在市场高点买入，低点卖出         | 大幅跑输市场               |
| 过度自信   | 认为自己能选到牛股               | 集中持仓，风险过高         |
| 损失厌恶   | 对亏损的痛苦感受远大于盈利的快乐 | 过早卖出盈利股，死扛亏损股 |
| 代表性偏差 | 根据近期表现推断未来             | 买入过去表现好的基金       |
| 现状偏差   | 不愿意改变现有配置               | 资产配置失衡               |

**解决方案：**

- 建立规则化的投资流程
- 定期再平衡，自动实现"低买高卖"
- 使用指数基金，减少选股和择时的诱惑
- 长期持有，减少交易频率

## 策略拆解

### 机构投资者的选股（选基金）标准

史文森的框架是**选基金管理人**，而非选个股：

| 维度         | 具体要求                       |
| ------------ | ------------------------------ |
| 长期业绩记录 | 至少 10 年可验证的超额收益     |
| 策略一致性   | 不风格漂移，坚持既定策略       |
| 团队稳定性   | 核心团队成员任职时间长         |
| 利益一致     | 管理人自身大量投资于自己的基金 |
| 费用合理     | 管理费 + 业绩报酬结构合理      |
| 透明度       | 定期披露持仓和业绩             |
| 规模适中     | 不超过策略容量上限             |

### 个人投资者的配置框架

史文森为个人投资者推荐的配置：

| 资产类别     | 比例 | 工具             |
| ------------ | ---- | ---------------- |
| 美国股票     | 30%  | 全市场指数基金   |
| 外国发达市场 | 25%  | 国际指数基金     |
| 新兴市场     | 5%   | 新兴市场指数基金 |
| 美国国债     | 20%  | 国债指数基金     |
| TIPS         | 15%  | TIPS 指数基金    |
| REITs        | 5%   | REIT 指数基金    |

### 买入时机

- **机构投资者**——按照基金募资窗口和战略资产配置执行
- **个人投资者**——定期定额投资（Dollar-Cost Averaging），不择时

### 持仓管理

- **长期持有**——不因短期波动调整战略配置
- **年度再平衡**——每年调整一次，回到目标配置
- **定期审查**——每年审查基金管理人表现

### 卖出纪律

- **基金到期**——PE/VC 基金有固定存续期
- **管理人变更**——核心团队离开或策略变化
- **策略失效**——连续多年无法创造超额收益
- **配置调整**——战略资产配置比例变化

### 普通人能学到什么

史文森对普通投资者的建议极为明确：

1. **指数化**——在有效市场中，低成本指数基金是最好的选择
2. **分散化**——覆盖全球股票和债券市场
3. **低成本**——总费用率控制在 0.2% 以内
4. **再平衡**——每年调整一次，自动实现"低买高卖"
5. **长期持有**——不要试图择时，不要频繁交易
6. **控制行为偏差**——建立规则，减少情绪干扰

**具体建议：**

- 选 3-5 只低成本指数基金，覆盖全球股债市场
- 设定目标配置比例，每年调整一次
- 坚持 20 年以上，让复利发挥作用
- 不要看每日行情，不要听财经新闻
- 把投资变成一件"无聊"的事
