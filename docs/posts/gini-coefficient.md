# 基尼系数

基尼系数就是衡量一个社会"贫富差距"的尺子——0 代表绝对平均（大家收入一样），1 代表绝对不平均（一个人拿走所有钱）。中国基尼系数约 0.47，超过了 0.4 的国际警戒线。

## 概述

基尼系数（Gini Coefficient）是**衡量收入或财富分配不平等程度的统计指标**，由意大利统计学家科拉多·基尼（Corrado Gini）在 1912 年提出（来源：Gini, 1912）。

核心定义：

- **0** = 绝对平均（每个人收入完全相同）
- **1** = 绝对不平均（一个人拥有全部收入，其他人为零）
- **0.2 以下** = 高度平均
- **0.2-0.3** = 比较平均
- **0.3-0.4** = 相对合理
- **0.4-0.5** = 差距较大
- **0.5 以上** = 差距悬殊

## 为什么重要

- **社会稳定的预警指标**：基尼系数超过 0.4 通常被认为存在社会不稳定风险
- **经济政策参考**：政府通过税收、社保、转移支付等政策调节收入分配
- **经济增长影响**：过大的收入差距可能抑制消费、拖累经济增长
- **国际比较**：基尼系数是衡量国家发展水平的重要指标
- **ESG 评估**：收入平等是 ESG 中"S"（社会）维度的核心指标

## 核心原理

### 洛伦兹曲线

基尼系数的几何定义基于洛伦兹曲线（Lorenz Curve）：

- 横轴：累计人口比例（从最穷到最富排序）
- 纵轴：累计收入比例

**绝对平均线**：45 度对角线，表示每 x% 的人口拥有 x% 的收入。

**洛伦兹曲线**：实际收入分配曲线，通常在绝对平均线下方。

基尼系数的几何定义：

$$G = \frac{A}{A + B}$$

其中：
- $A$ = 洛伦兹曲线与绝对平均线之间的面积
- $B$ = 洛伦兹曲线下方的面积
- $A + B = 0.5$（三角形面积）

所以：

$$G = 2A = 1 - 2B$$

### 计算公式

给定 $n$ 个人（或家庭）的收入 $y_1, y_2, \ldots, y_n$（按从小到大排序）：

$$G = \frac{2\sum_{i=1}^{n} i \cdot y_i}{n \sum_{i=1}^{n} y_i} - \frac{n+1}{n}$$

或者使用平均绝对差公式：

$$G = \frac{\sum_{i=1}^{n}\sum_{j=1}^{n} |y_i - y_j|}{2n^2\bar{y}}$$

其中 $\bar{y}$ 是平均收入。

### 基尼系数的分解

基尼系数可以按收入来源分解：

$$G = \sum_{k} S_k \cdot G_k \cdot R_k$$

其中：
- $S_k$ = 收入来源 $k$ 占总收入的比例
- $G_k$ = 收入来源 $k$ 的基尼系数
- $R_k$ = 收入来源 $k$ 与总收入的"伪基尼相关系数"

这有助于识别哪些收入来源对不平等贡献最大。

## 模型/方法详解

### 基尼系数与经济增长

库兹涅茨曲线（Kuznets Curve）假说：

$$\text{不平等} = f(\text{人均 GDP})$$

呈倒 U 型：

- 经济发展初期：不平等上升（工业化、城市化）
- 经济发展中期：不平等达到峰值
- 经济发展后期：不平等下降（福利制度、教育普及）

但皮凯蒂（Thomas Piketty）在《21 世纪资本论》中质疑了这一假说，认为 $r > g$（资本回报率 > 经济增长率）会导致不平等持续扩大。

### 基尼系数的政策含义

| 基尼系数范围 | 政策建议 |
|--------------|----------|
| < 0.3 | 维持现状，关注效率 |
| 0.3-0.4 | 适度调节，关注公平 |
| 0.4-0.5 | 加强调节，完善社保 |
| > 0.5 | 紧急干预，防止社会不稳定 |

## 主流方案对比

| 指标 | 衡量内容 | 优点 | 缺点 |
|------|----------|------|------|
| 基尼系数 | 整体不平等程度 | 单一数值、易于比较 | 不反映分布形状 |
| 收入份额比 | 最高/最低收入比 | 直观 | 忽略中间分布 |
| 泰尔指数 | 信息论不平等度量 | 可分解 | 计算复杂 |
| 阿特金森指数 | 社会福利损失 | 考虑社会偏好 | 参数选择主观 |
| 贫困率 | 贫困人口比例 | 政策导向 | 不反映不平等 |

## 实战应用

### 基尼系数的应用场景

**政策制定**：

- 设定收入分配调节目标
- 评估税收政策效果
- 设计社会保障体系

**投资分析**：

- 高基尼系数国家：奢侈品、高端消费可能有市场
- 低基尼系数国家：大众消费品更受欢迎
- 基尼系数变化趋势影响消费结构

**企业战略**：

- 根据目标市场的收入分布定价
- 高不平等市场需要差异化产品线
- 低不平等市场适合标准化产品

### 调节基尼系数的政策工具

**1. 税收政策**

- 累进所得税：收入越高税率越高
- 财产税、遗产税：调节财富分配
- 消费税：对奢侈品征收高税

**2. 转移支付**

- 社会保障金
- 失业救济
- 最低生活保障

**3. 公共服务**

- 教育公平：缩小教育差距
- 医疗保障：降低因病致贫
- 基础设施：缩小地区差距

## 关键要点

- 基尼系数衡量收入或财富分配的不平等程度
- 0 代表绝对平均，1 代表绝对不平均
- 0.4 是国际警戒线，超过意味着差距较大
- 中国基尼系数约 0.47，处于较高水平
- 基尼系数可以通过税收、社保、公共服务等政策调节

## 常见误区

- **误区一**："基尼系数越低越好"——过低的基尼系数可能抑制效率
- **误区二**："基尼系数反映绝对贫困"——基尼系数衡量相对不平等，不反映绝对贫困
- **误区三**："基尼系数是静态的"——基尼系数随时间变化，需要追踪趋势
- **误区四**："基尼系数能说明一切"——基尼系数不反映分布形状，需要结合其他指标
- **误区五**："基尼系数只适用于国家"——基尼系数可以用于地区、行业、企业等各个层面
