---
description: 词条总览与状态管理
agent: build
---

查看和管理 `docs/glossary/` 目录下所有词条的状态。

执行步骤：

1. 读取 `docs/glossary/index.md` 获取词条分类和列表
2. 扫描 `docs/glossary/` 目录下所有 `.md` 文件
3. 对比 index.md 与实际文件，列出：
   - 已创建且有内容的词条
   - 已创建但仅有标题的词条（空壳）
   - index.md 中列出但文件不存在的词条
4. 按分类统计词条数量和完成状态

输出格式：

```
## 词条状态总览

### 基础概念 (5/5 完成)
- [x] AI
- [x] 机器学习
- [ ] 深度学习 (仅标题)

### 模型与架构 (3/5 完成)
...
```

如果用户要求修复不一致，同步 index.md 与实际文件。

示例：`/ct:list` 或 `/ct:list --fix`
