查看和管理金融词条目录的状态。

执行步骤：

1. 读取 `docs/glossary/index.md` 获取词条分类和列表
2. 扫描 `docs/` 下各分类目录中所有 `.md` 文件
3. 对比 index.md 与实际文件，列出：
   - 已创建且有内容的词条
   - 已创建但仅有标题的词条（空壳）
   - index.md 中列出但文件不存在的词条
4. 按分类统计词条数量和完成状态

输出格式：

```
## 词条状态总览

### 基础概念 (10/10 完成)
- [x] 什么是金融
- [x] 货币
- [x] 利息
- [ ] 信用 (仅标题)

### 货币与银行 (8/10 完成)
...
```

如果用户要求修复不一致，同步 index.md 与实际文件。

示例：`/ct:list` 或 `/ct:list --fix`
