# 约翰·内夫——低市盈率投资之王

> 内夫最让人佩服的地方不是他的业绩——虽然业绩确实惊人——而是他**把一件简单的事情坚持了 31 年。** 在华尔街这个充满复杂策略和花哨术语的地方，内夫用低市盈率（P/E Ratio）这一个指标，打败了绝大多数"聪明人"。

## 人物速写

一句话总结：**从俄亥俄州小镇走出来的年轻人，用低市盈率策略管理温莎基金 31 年，累计跑赢标普 500 指数 55 倍，成为了华尔街最被低估的价值投资大师。**

## 传奇人生

### 早年经历：俄亥俄州的小镇青年

1931 年，约翰·内夫（John Neff）出生于俄亥俄州的一个小镇。他的家庭并不富裕——父亲是一名铁路工人，母亲是家庭主妇。在那个年代，俄亥俄州的小镇生活简单而朴实，但也让内夫养成了一种后来成为他投资哲学核心的品质：**务实。**

内夫后来回忆说：

> "我从小就知道，钱不是天上掉下来的。你必须努力工作，精打细算，才能有所收获。"（来源：《约翰·内夫谈投资》，1999）

这种"精打细算"的思维，后来成为了他低市盈率投资法的心理基础——他本能地不喜欢为任何东西支付过高的价格。

### 求学与启蒙：从会计到投资

内夫在大学期间主修**会计学（Accounting）**。这个选择看似平凡，却为他后来的投资生涯打下了坚实的基础。

会计学教会了他两件事：

1. **如何读懂财务报表**——这是价值投资者的基本功
2. **如何识别数字背后的真相**——公司赚不赚钱，资产负债表上写得清清楚楚

1950 年代末，内夫进入了克利夫兰信托公司（Cleveland Trust Company），开始了他的金融职业生涯。在那里，他从最基础的分析师做起，每天的工作就是阅读公司财报、分析行业数据、撰写投资报告。

**这段经历让他练就了一个硬功夫：在 15 分钟内判断一家公司的财务健康状况。**

### 投资启蒙：低市盈率的发现

1960 年代初，内夫在研究市场数据时发现了一个有趣的现象：

**低市盈率的股票，长期来看，表现优于高市盈率的股票。**

这个发现在今天看来似乎理所当然，但在当时，华尔街的主流观点是"成长型股票（Growth Stocks）才是未来"。投资者愿意为高增长的公司支付极高的估值——市盈率 30 倍、40 倍甚至更高都被认为是合理的。

内夫的逻辑很简单：

- 高市盈率意味着市场已经对未来增长做了乐观预期
- 一旦增长不及预期，股价就会大幅下跌
- 低市盈率意味着市场预期很低，即使公司表现一般，股价也不太会下跌
- 如果低市盈率公司表现好于预期，股价就有上涨空间

**这是一个典型的"下行风险有限、上行空间可观"的策略。**

> 生活化类比：就像你去菜市场买菜。那些标着"有机进口"的蔬菜卖 30 块一斤，而普通蔬菜只卖 5 块。内夫的选择永远是 5 块的——因为即使普通蔬菜没那么好吃，你也不亏；但如果它意外地好吃，你就赚大了。

### 关键转折点：接手温莎基金

1964 年，内夫被任命为温莎基金（Windsor Fund）的基金经理。温莎基金是威灵顿管理公司（Wellington Management Company）旗下的一只共同基金（Mutual Fund），当时规模不大，业绩也平平。

内夫接手后，做了一件在当时看来很"另类"的事：**他系统性地买入低市盈率股票。**

他的选股标准非常明确：

1. **市盈率低于市场平均水平 40-60%**
2. **公司有稳定的盈利增长（至少 7-10%/年）**
3. **股息收益率（Dividend Yield）有吸引力**
4. **基本面健康，没有重大财务风险**

这套方法后来被他称为**"总收益法"（Total Return Approach）**——他不仅关注资本增值（Capital Appreciation），也关注股息收入（Dividend Income）。他认为，对于长期投资者来说，股息是总收益的重要组成部分。

### 巅峰时刻：31 年的复利奇迹

从 1964 年到 1995 年，内夫管理温莎基金整整 31 年。

**结果如何？**

| 指标     | 温莎基金 | 标普 500 | 超额  |
| -------- | -------- | -------- | ----- |
| 年化收益 | 13.7%    | 10.4%    | 3.3%  |
| 累计收益 | 约 55 倍 | 约 22 倍 | 33 倍 |
| 跑赢年份 | 25/31 年 | —        | 80.6% |

（来源：温莎基金年报，1995；《约翰·内夫谈投资》，1999）

**3.3% 的年化超额收益，听起来不多？让我们算一笔账：**

- 如果你在 1964 年投资 1 万美元到温莎基金，到 1995 年它会变成约 **55 万美元**
- 如果投资标普 500 指数，同样 1 万美元会变成约 **22 万美元**
- **温莎基金多赚了 33 万美元——是指数收益的 2.5 倍**

这就是复利的力量。每年多赚 3%，31 年后就是天壤之别。

1995 年，64 岁的内夫决定退休。他给出的理由很简单：

> "我已经做了 31 年同样的事情。是时候让年轻人来试试了。"（来源：《约翰·内夫谈投资》，1999）

退休后，内夫并没有完全离开投资界。他继续写作、演讲，并在 1999 年出版了《约翰·内夫谈投资》一书，系统总结了他 31 年的投资经验。

## 经典战役

### 战役一：福特汽车——逆向投资的经典（1980 年代初）

1980 年代初，美国汽车行业陷入严重危机。日本汽车的崛起让底特律三大汽车制造商（通用、福特、克莱斯勒）面临前所未有的竞争压力。

福特的股价跌到了个位数，市盈率只有 **3-4 倍**——市场认为这家公司可能要完蛋了。

内夫做了什么？

**他大量买入福特股票。**

他的逻辑：

1. 福特不会破产——它是美国第二大汽车制造商，有政府支持
2. 日本汽车虽然质量好，但福特正在改进
3. 3-4 倍的市盈率意味着市场已经定价了最坏的情况
4. 即使福特只能恢复到正常盈利水平，股价也有巨大的上涨空间

**结果：**

随着福特推出新车型、改进质量、降低成本，公司盈利逐渐恢复。到 1980 年代中期，福特股价从个位数涨到了 30 美元以上。

**关键数据：**

| 指标         | 数值           |
| ------------ | -------------- |
| 买入时间     | 1980-1982 年   |
| 买入价格     | 约 5-8 美元/股 |
| 买入时市盈率 | 3-4 倍         |
| 卖出价格     | 约 30 美元+/股 |
| 持有时间     | 3-5 年         |
| 回报率       | 约 300-500%    |

（来源：《约翰·内夫谈投资》，1999；福特汽车年报，1980-1985）

### 战役二：花旗集团——金融股的低谷买入（1990 年代初）

1990-1991 年，美国经历了一次经济衰退。银行业受到房地产贷款损失的严重冲击，花旗集团（Citigroup）的股价暴跌。

花旗的问题：

- 大量的房地产不良贷款（Non-Performing Loans）
- 市场担心花旗可能面临破产风险
- 股价跌到了 10 美元以下，市盈率只有 **4-5 倍**

内夫的分析：

1. 花旗是"大而不能倒"（Too Big to Fail）的银行
2. 房地产损失是一次性的，不是结构性问题
3. 花旗的核心业务（信用卡、零售银行）依然健康
4. 4-5 倍的市盈率意味着市场已经定价了破产风险

**操作：**

内夫在 1990-1991 年大量买入花旗股票。

**结果：**

随着经济复苏和房地产损失的消化，花旗盈利恢复。到 1990 年代中期，花旗股价涨到了 50 美元以上。

**关键数据：**

| 指标         | 数值              |
| ------------ | ----------------- |
| 买入时间     | 1990-1991 年      |
| 买入价格     | 约 10 美元以下/股 |
| 买入时市盈率 | 4-5 倍            |
| 卖出价格     | 约 50 美元+/股    |
| 持有时间     | 3-5 年            |
| 回报率       | 约 400%+          |

（来源：《约翰·内夫谈投资》，1999；花旗集团年报，1990-1995）

### 战役三：菲利普莫里斯——烟草股的长期持有（1980-1990 年代）

菲利普莫里斯（Philip Morris，现为奥驰亚集团 Altria Group）是内夫持仓时间最长的股票之一。

**背景：**

1980 年代，反烟草运动兴起，烟草公司面临巨大的法律和舆论压力。市场极度悲观，菲利普莫里斯的市盈率只有 **5-7 倍**，股息收益率高达 **8-10%**。

内夫的逻辑：

1. 烟草需求不会消失——成瘾性消费品的需求是刚性的
2. 法律风险被市场过度定价
3. 5-7 倍的市盈率 + 8-10% 的股息收益率 = 极高的安全边际
4. 即使股价不涨，仅股息收入就足以跑赢市场

**操作：**

内夫在 1980 年代开始买入菲利普莫里斯，并长期持有。

**结果：**

- 烟草需求确实没有消失
- 法律风险虽然存在，但公司通过和解协议（Settlement Agreements）逐步消化
- 股价从 1980 年代的低位到 1990 年代中期上涨了数倍
- 加上股息收入，总收益极为可观

**关键数据：**

| 指标             | 数值      |
| ---------------- | --------- |
| 买入时间         | 1980 年代 |
| 买入时市盈率     | 5-7 倍    |
| 股息收益率       | 8-10%     |
| 持有时间         | 10+ 年    |
| 总收益（含股息） | 约 500%+  |

（来源：《约翰·内夫谈投资》，1999；菲利普莫里斯年报，1980-1995）

### 战役四：1987 年股灾——低市盈率的防御优势

1987 年 10 月 19 日，"黑色星期一"（Black Monday），道琼斯指数单日暴跌 **22.6%**，创造了历史上最大的单日跌幅记录。

内夫的温莎基金表现如何？

**温莎基金也下跌了，但跌幅远小于市场。**

原因很简单：

- 温莎基金持有的股票市盈率低，估值本来就便宜
- 低估值股票在暴跌中的下行空间有限
- 高股息提供了额外的缓冲

内夫后来回忆：

> "低市盈率策略最大的好处不是涨得多，而是跌得少。"（来源：《约翰·内夫谈投资》，1999）

**1987 年业绩对比：**

| 指标     | 温莎基金 | 标普 500 |
| -------- | -------- | -------- |
| 全年收益 | 约 2%    | 约 -5%   |
| 最大回撤 | 约 -15%  | 约 -23%  |

（来源：温莎基金年报，1987）

### 战役五：31 年的纪律——不追热点的定力

内夫 31 年管理温莎基金期间，经历了无数次市场热点：

- 1960 年代的"漂亮 50"（Nifty Fifty）泡沫
- 1970 年代的石油危机
- 1980 年代的科技股崛起
- 1990 年代的互联网泡沫前夜

**内夫一次都没有追过热点。**

他始终坚持低市盈率策略，即使在某些年份因此跑输市场。

**最困难的时期：1980 年代末到 1990 年代初**

这段时间，成长型股票（尤其是科技股）表现优异，而内夫的低市盈率策略连续几年跑输市场。投资者开始质疑他的方法。

内夫的反应：

> "我不关心市场今年喜欢什么。我关心的是，我买的股票是否足够便宜。"（来源：《约翰·内夫谈投资》，1999）

**结果：**

1990-1991 年经济衰退期间，成长型股票大幅下跌，而内夫的低市盈率股票表现出了强大的防御性。温莎基金不仅追回了之前的差距，还大幅跑赢了市场。

**这证明了内夫的一个核心信念：低市盈率策略可能在某些年份跑输，但长期来看，它一定会赢。**

## 投资哲学

### 原则一：低市盈率是王道

内夫投资哲学的核心可以用一句话概括：**买便宜的股票。**

他的低市盈率标准：

- **市盈率低于市场平均水平 40-60%**
- 如果市场平均市盈率是 15 倍，他只买市盈率在 6-9 倍的股票
- 他绝不买市盈率超过 15 倍的股票

**为什么低市盈率有效？**

1. **市场预期低**——低市盈率意味着市场对公司前景不乐观
2. **下行风险有限**——即使公司表现不好，股价也不太会再跌
3. **上行空间大**——如果公司表现好于预期，股价就会上涨
4. **均值回归（Mean Reversion）**——长期来看，市盈率会向平均水平回归

> 生活化类比：低市盈率就像打折商品。一件原价 100 块的衣服现在卖 40 块，即使它没那么好看，你也不亏；但如果它意外地合身，你就赚大了。

### 原则二：总收益法

内夫不仅关注股价上涨，也关注股息收入。他把两者加在一起，称为**"总收益"（Total Return）。**

他的总收益法逻辑：

- 低市盈率股票通常有高股息
- 股息是确定性的收入，不依赖市场情绪
- 长期来看，股息再投资（Dividend Reinvestment）是复利的重要来源
- 一只市盈率 8 倍、股息率 5% 的股票，即使股价不涨，5 年后仅股息收入就有 25%

**内夫的总收益公式：**

```
总收益 = 资本增值 + 股息收入
```

他认为，对于长期投资者来说，股息收入往往被低估。在温莎基金 31 年的总收益中，**股息贡献了约 30-40%**。

### 原则三：逆向投资

内夫是逆向投资的坚定践行者。

> "最好的投资机会出现在所有人都在卖出的时候。"（来源：《约翰·内夫谈投资》，1999）

他的逆向思维：

- 当市场对某个行业极度悲观时，他反而会关注
- 他不怕买"不受欢迎"的股票——因为不受欢迎，所以便宜
- 他相信均值回归——被过度打压的股票最终会恢复

**但他不是盲目的逆向投资者。** 他的逆向投资有一个前提：**基本面没有恶化。** 如果一家公司的问题是不可逆的（比如技术被淘汰），他不会买。

### 原则四：分散投资

内夫的投资组合通常持有 **50-60 只股票**，覆盖多个行业。

他的分散投资逻辑：

- 低市盈率策略不是每只股票都会赢
- 但作为一个组合，低市盈率股票的整体表现会优于市场
- 分散投资降低了单只股票的风险
- 他不需要每只股票都赚钱——只需要组合整体跑赢市场

**行业配置原则：**

- 单个行业不超过组合的 20%
- 覆盖至少 8-10 个行业
- 不押注单一行业或单一公司

### 原则五：耐心与纪律

内夫最强调的品质是**耐心。**

> "低市盈率策略需要时间。你不能今天买，明天就期望它涨。通常需要 2-3 年，甚至更久，市场才会发现这些股票的价值。"（来源：《约翰·内夫谈投资》，1999）

他的耐心体现在：

- 买入后不频繁交易
- 给市场时间发现价值
- 不因短期波动而改变策略
- 坚持纪律，不追热点

## 策略拆解

### 选股标准

内夫的选股标准非常量化，他提出了著名的"内夫筛选法"：

| 标准                         | 具体要求                         |
| ---------------------------- | -------------------------------- |
| 市盈率（P/E Ratio）          | 低于市场平均水平 40-60%          |
| 盈利增长率                   | 至少 7%/年（内夫认为不需要很高） |
| 股息收益率（Dividend Yield） | 有吸引力（通常高于市场平均）     |
| 市盈率/增长率（PEG Ratio）   | 低于 1（市盈率低于增长率）       |
| 基本面健康                   | 无重大财务风险，负债合理         |
| 行业地位                     | 行业龙头或前几名                 |

**内夫特别强调：盈利增长率不需要很高，7-10% 就足够了。关键是用便宜的价格买入。**

### 买入时机

内夫的买入时机判断：

1. **个股层面**：当市盈率低于市场平均水平 40-60% 时买入
2. **市场层面**：当市场整体估值偏高时，他会更加谨慎；当市场下跌、估值回落时，他会加大投入
3. **行业层面**：当某个行业遭遇短期利空、被市场抛弃时，他会重点关注

**他特别强调：不要试图预测市场短期走势。你只需要知道当前价格是否足够便宜。**

### 持仓管理

内夫的持仓管理原则：

- **买入后不频繁交易**——给市场时间发现价值
- **定期复查持仓**——确认基本面没有恶化
- **不追加投资**——除非价格进一步下跌且基本面不变
- **保持耐心**——价值回归通常需要 2-3 年

### 卖出纪律

内夫的卖出条件非常明确：

1. **市盈率回归到市场平均水平**——获利了结，不贪心
2. **基本面恶化**——公司盈利增长放缓或出现结构性问题
3. **发现更好的机会**——换仓到性价比更高的标的
4. **持有时间过长（3-5 年）仍未回归**——重新评估逻辑是否正确

**他特别强调：卖出不是因为股价涨了，而是因为估值不再便宜。**

### 普通人能学到什么

内夫的方法对普通投资者特别友好，因为：

1. **不需要内幕消息**——所有数据都是公开的财务信息
2. **不需要预测未来**——只需要评估当前估值是否便宜
3. **不需要频繁交易**——买入后持有，等待价值回归
4. **可以系统化执行**——内夫筛选法可以用股票筛选器实现

**具体建议：**

- 用股票筛选器按内夫标准筛选股票（市盈率低于市场 40-60%，盈利增长 7%+，有股息）
- 构建 20-30 只股票的组合，覆盖多个行业
- 每年复查一次，调整不符合标准的持仓
- 保持耐心，不要因短期波动而改变策略
- **股息再投资**——把收到的股息重新买入，加速复利
