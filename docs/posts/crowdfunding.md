# 众筹

你想做一个新产品但缺钱？不用去找VC，让1000个人每人出100块就行了——这就是众筹，"众人拾柴火焰高"的金融版。

## 概述

众筹（Crowdfunding）是指**通过互联网平台，向大量个人（"群众"）募集小额资金来支持项目、创业或创意的一种融资方式**。

众筹的四种主要模式：

| 模式                     | 投资者获得什么           | 典型平台               | 适用场景                 |
| ------------------------ | ------------------------ | ---------------------- | ------------------------ |
| 奖励型（Reward-based）   | 产品或服务（非财务回报） | Kickstarter, Indiegogo | 创意产品、艺术品、游戏   |
| 股权型（Equity-based）   | 公司股权                 | AngelList, Crowdcube   | 创业公司融资             |
| 债权型（Lending-based）  | 本金+利息                | LendingClub, Prosper   | 个人贷款、中小企业贷款   |
| 捐赠型（Donation-based） | 无回报（慈善）           | GoFundMe, 轻松筹       | 慈善、医疗救助、公益项目 |

众筹的发展历程：

- **2003 年**：ArtistShare 成立，被认为是第一个众筹平台（音乐项目）
- **2009 年**：Kickstarter 成立，推动奖励型众筹普及
- **2012 年**：美国 JOBS 法案（Jumpstart Our Business Startups Act）通过，为股权众筹提供法律框架
- **2015 年**：中国众筹市场爆发，京东众筹、淘宝众筹上线
- **2016 年**：SEC Regulation Crowdfunding（Reg CF）生效，允许非合格投资者参与股权众筹
- **2021 年**：SEC 将 Reg CF 年度融资上限从 500 万美元提高到 500 万美元

## 为什么重要

- **融资民主化**：众筹让普通人也能参与早期投资，打破了传统风投的垄断
- **创业门槛降低**：创业者可以直接面向消费者验证产品和获取资金
- **市场验证**：众筹成功本身就是对产品市场需求的验证
- **长尾市场**：小众产品通过众筹找到目标用户，这在传统融资中很难实现
- **监管创新**：股权众筹推动了证券监管的改革（如 JOBS 法案、Reg CF）

## 核心原理

### 众筹的经济学逻辑

**1. 信息聚合**

众筹平台聚合了大量投资者的信息和资金：

$$\text{总融资额} = \sum_{i=1}^N \text{贡献}_i$$

其中 $N$ 是支持者数量，通常 $N$ 很大（数百到数万），每个 $\text{贡献}_i$ 很小（几十到几百元）。

**2. 信号传递**

众筹成功向市场传递积极信号：

- 产品有市场需求
- 创业者有执行力
- 后续投资者更愿意跟进

**3. 网络效应**

众筹平台具有网络效应：

- 更多项目 → 吸引更多支持者
- 更多支持者 → 吸引更多项目
- 形成正向循环

### 奖励型众筹的机制设计

Kickstarter 使用 **All-or-Nothing** 模式：

- 设定融资目标和截止日期
- 如果达到目标，支持者被扣款，项目获得资金
- 如果未达到目标，支持者不被扣款，项目获得零资金

这种设计的好处：

- 保护支持者：如果资金不足，项目可能无法完成
- 激励项目方：设定合理的目标
- 创造紧迫感：倒计时促使支持者行动

Indiegogo 提供 **Flexible Funding** 选项：

- 即使未达到目标，项目方也能获得已筹集的资金
- 适合不需要全部资金也能推进的项目

### 股权众筹的监管框架

**美国 Reg CF**（来源：SEC Regulation Crowdfunding, 2016）：

- 融资上限：12 个月内最多 500 万美元
- 投资者限制：
  - 年收入/净资产 < 12.4 万美元：投资上限为 2400 美元或年收入/净资产的 5%（取较大值）
  - 年收入/净资产 ≥ 12.4 万美元：投资上限为年收入/净资产的 10%，最高 12.4 万美元
- 信息披露：项目方需要提供财务报表、业务计划等

**中国众筹监管**：

- 股权众筹被纳入《证券法》监管范围
- 需要通过持牌平台进行
- 对投资者有合格投资者要求

## 模型/方法详解

### 众筹成功的影响因素

Mollick（2014）（来源：Mollick, 2014, Research Policy）对 Kickstarter 项目的研究发现：

| 因素         | 影响方向 | 说明                                   |
| ------------ | -------- | -------------------------------------- |
| 目标金额     | 负       | 目标越高，成功率越低                   |
| 项目持续时间 | 倒 U 型  | 太短或太长都不好，30 天左右最佳        |
| 社交网络     | 正       | Facebook 好友数量与成功率正相关        |
| 视频质量     | 正       | 有视频的项目成功率更高                 |
| 更新频率     | 正       | 频繁更新的项目更受信任                 |
| 地理位置     | 正       | 科技中心（硅谷、纽约）的项目成功率更高 |

### 众筹中的行为偏差

**1. 羊群效应**（Herd Behavior）

- 早期支持者影响后续支持者的决策
- "已筹集金额"是重要的社会证明信号
- 项目方经常用"种子资金"（自己的钱）来启动

**2. 目标梯度效应**（Goal Gradient Effect）

- 接近目标时，支持速度加快
- 人们更愿意支持"即将成功"的项目

**3. 社会偏好**

- 支持者不仅关心回报，还关心帮助创业者实现梦想
- 故事性和情感共鸣影响支持决策

### 众筹的定价模型

股权众筹的定价是一个难题：

**传统估值方法不适用**：

- 早期创业公司没有盈利，DCF 不适用
- 没有可比公司，相对估值不适用
- 风险极高，传统风险调整不适用

**常用方法**：

- **SAFE**（Simple Agreement for Future Equity）：Y Combinator 发明，将估值推迟到下一轮
- **可转换票据**（Convertible Note）：类似 SAFE，但有利息和到期日
- **收入分成**（Revenue Sharing）：按未来收入的一定比例回报投资者

## 主流方案对比

| 平台        | 模式   | 费用                  | 特点                         |
| ----------- | ------ | --------------------- | ---------------------------- |
| Kickstarter | 奖励型 | 5% + 支付处理费       | All-or-Nothing，创意产品为主 |
| Indiegogo   | 奖励型 | 5% + 支付处理费       | Flexible Funding 选项        |
| AngelList   | 股权型 | 成功费                | 合格投资者，Syndicate 模式   |
| Crowdcube   | 股权型 | 7% + 增值税           | 英国最大股权众筹平台         |
| LendingClub | 债权型 | 服务费                | P2P 借贷，已转型为持牌银行   |
| GoFundMe    | 捐赠型 | 0% 平台费（可选小费） | 个人筹款，医疗、教育为主     |

## 实战应用

### 如何成功发起众筹项目

**1. 准备阶段**

- **验证需求**：在发起前，通过社交媒体、邮件列表测试兴趣
- **设定合理目标**：目标是成功的关键因素，宁低勿高
- **制作高质量视频**：视频是项目最重要的展示方式
- **准备奖励/回报**：设计有吸引力的回报方案

**2. 启动阶段**

- **种子资金**：用自己的钱或亲友的钱启动，创造早期势头
- **社交媒体推广**：利用个人社交网络传播
- **媒体联系**：联系科技媒体、博客进行报道

**3. 运营阶段**

- **频繁更新**：每周至少更新一次，保持支持者参与
- **回应评论**：积极回应支持者的问题和建议
- **管理预期**：诚实地沟通进展和挑战

### 众筹投资的风险

**对投资者**：

- **项目失败风险**：大多数众筹项目无法完成或交付
- **流动性风险**：众筹投资通常无法转让或退出
- **信息不对称**：项目方可能夸大或隐瞒信息
- **欺诈风险**：少数项目方可能挪用资金

**对创业者**：

- **公开失败**：如果众筹失败，公开记录可能影响未来融资
- **知识产权风险**：公开创意可能被抄袭
- **交付压力**：承诺的回报可能难以按时按质交付
- **监管风险**：股权众筹涉及证券法合规

### 众筹的未来趋势

- **监管沙盒**：各国监管机构探索更灵活的众筹监管框架
- **区块链众筹**（ICO/IEO/IDO）：基于加密货币的众筹模式（但风险极高）
- **房地产众筹**：通过众筹投资房地产项目
- **NFT 众筹**：通过 NFT 为创意项目融资

## 关键要点

- 众筹让融资民主化，但也带来了新的风险和挑战
- 奖励型众筹适合产品验证，股权型众筹适合早期融资
- 众筹成功的关键因素包括目标设定、社交网络、项目展示
- 投资者需要理解众筹的高风险性和低流动性
- 监管框架正在不断完善，但各国差异较大

## 常见误区

- **误区一**："众筹就是免费的钱"——众筹需要交付回报（产品、股权、利息），不是免费的
- **误区二**："众筹成功=商业成功"——众筹成功只是第一步，后续执行和商业化同样重要
- **误区三**："众筹适合所有项目"——众筹适合有明确产品/服务的项目，不适合纯研发或 B2B 项目
- **误区四**："股权众筹是稳赚的"——早期创业投资失败率极高，股权众筹风险更大
- **误区五**："众筹不需要营销"——众筹本身就是营销活动，需要大量推广才能成功
